# Administrative Documents — Norwegian (Bokmal)

## 1. Reader Model

### 1.1 Norwegian Government Structure

Norway is a **constitutional monarchy** with a parliamentary system. The government is organized at three levels:

| Level | Norwegian | Examples |
|-------|-----------|---------|
| National (State) | Stat / Regjeringen | Ministries, directorates |
| County | Fylke | County councils (11 counties) |
| Municipal | Kommune | Municipal councils (356 municipalities) |

### 1.2 Key Government Institutions

| Institution | Norwegian Name | Function |
|-------------|----------------|----------|
| Parliament | Stortinget | Legislative body |
| Government | Regjeringen | Executive branch |
| Prime Minister | Statsminister | Head of government |
| Ministry | Departement | Policy development |
| Directorate | Direktorat | Implementation |
| Agency | Etat | Service delivery |
| County Governor | Statsforvalteren | Regional representation |
| Municipal Council | Kommunestyret | Local governance |

### 1.3 Administrative Document Types

| Document Type | Norwegian Markers | Register |
|---------------|-------------------|----------|
| Decision (Vedtak) | "I medhold av... fattes folgende vedtak:" | Formal, imperative |
| Regulation (Forskrift) | "Med hjemmel i... fastsettes:" | Legislative, precise |
| Circular (Rundskriv) | "Vi viser til..." | Administrative, explanatory |
| Consultation (Hoering) | "Departementet onsker innspill..." | Consultative |
| Report (Rapport) | "Rapporten oppsummerer..." | Analytical, factual |
| White Paper (Stortingsmelding) | "Stortinget far forelagt..." | Political, strategic |
| Public Notice ( kunngjoring) | "Kunngjoring i Norsk Lovtidend" | Official, public |

## 2. Decision Framework

### 2.1 Norwegian Administrative Hierarchy

| Level | Norwegian | Authority |
|-------|-----------|-----------|
| Constitution | Grunnloven | Supreme law |
| Acts (Lover) | Lov / Lovdata | Parliamentary legislation |
| Regulations (Forskrifter) | Forskrift | Delegated legislation |
| Circulars (Rundskriv) | Rundschriv | Administrative guidance |
| Guidelines (Retningslinjer) | Retningslinjer | Internal guidance |
| Practice (Praksis) | Praksis | Administrative practice |

### 2.2 Norwegian Administrative Decision Structure

| Element | Norwegian | Notes |
|---------|-----------|-------|
| Legal Basis | "Hjemmel" | Authority for the decision |
| Parties | "Part" | Affected persons/entities |
| Facts | "Faktum" | Factual basis |
| Assessment | "Vurdering" | Legal/factual assessment |
| Decision | "Vedtak" | The actual decision |
| Reasoning | "Begrunnelse" | Explanation |
| Appeal Rights | "Klagerett" | Right to appeal |
| Effective Date | "Dato for ikrafttredelse" | When decision takes effect |

### 2.3 Norwegian Public Communication Style

| Feature | Norwegian Approach | Notes |
|---------|-------------------|-------|
| Formality | Semi-formal, clear | Avoid legalese |
| Tone | Direct, helpful | Service-oriented |
| Pronoun | "du" (universal) | NOT "De" (archaic) |
| Passive Voice | Sometimes used | For objectivity |
| Active Voice | Preferred | For clarity |
| Jargon | Minimize | Use plain language |
| Structure | Clear headings, numbered lists | Scannable format |

## 3. Error Pattern Library

### 3.1 False Friends in Norwegian Administrative Translation

| English False Friend | Incorrect Norwegian | Correct Norwegian | Notes |
|----------------------|---------------------|-------------------|-------|
| "eventually" | "til slutt" | "etter hvert" / "på sikt" | NOT "til slutt" |
| "actually" | "faktisk" | "egentlig" / "faktisk" | Context-dependent |
| "realize" | "realisere" | "forsta" / "innse" | "realize" = "forsta" |
| "premises" | "forutsetninger" | "lokaler" / "premisser" | "premises" = "lokaler" |
| "instrument" | "instrument" | "redskap" / "dokument" | "legal instrument" = "rettslig grunnlag" |
| "implement" | "implementere" | "innfore" / "gjennomfore" | "implement" = "innfore" |
| "policy" | "politikk" | "politikk" / "retningslinjer" | "policy" = "retningslinjer" (guidelines) |
| "sustainable" | "bærekraftig" | "bærekraftig" / "holdbar" | Same, but check context |
| "inclusive" | "inkluderende" | "inkluderende" / "åpen" | Same |
| "transparent" | "gjennomsiktig" | "åpen" / "transparent" | "transparent" = "åpen" (open) |

### 3.2 Common Structural Errors

| Error Pattern | Wrong Norwegian | Correct Norwegian | Notes |
|---------------|-----------------|-------------------|-------|
| Calque: "in accordance with" | "i overensstemmelse med" | "i henhold til" | Standard phrase |
| Calque: "with regard to" | "med hensyn til" | "nar det gjelder" / "vedrørende" | Norwegian prefers shorter |
| Calque: "for the purpose of" | "for formal av" | "for å" / "i den hensikt å" | Simplify |
| Calque: "in the event that" | "i tilfelle at" | "dersom" / "hvis" | Simple conditional |
| Calque: "at the time of" | "på tidspunktet for" | "ved" / "da" | More natural |
| Calque: "with respect to" | "med hensyn til" | "nar det gjelder" | Norwegian prefers shorter |
| Calque: "in order to" | "for å" | "for å" | Same (acceptable) |

## 4. Domain-Specific Reference

### 4.1 Norwegian Government Communication Templates

**Public Notice (Kunngjoring):**
```
KUNNGJORING

[Etatens navn]
[Dato]

[Vedrørende: Emne]

[Innledning]

[Hoveddel]

[Avslutning]

[Navn]
[Tittel]
[Etat]
```

**Administrative Decision (Vedtak):**
```
VEDTAK

Saksnr: [Nummer]
Dato: [Dato]
Saksbehandler: [Navn]

I medhold av [lov/grunnlag] fattes folgende vedtak:

1. [Vedtak 1]
2. [Vedtak 2]

Begrunnelse:
[ begrunnelse ]

Klagefrist: [Dato]
Klageinstans: [Navn]
```
