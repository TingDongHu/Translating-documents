# Commercial Documents — Norwegian (Bokmal)

## 1. Reader Model

### 1.1 Norwegian Business Environment

Norway has a highly developed, export-oriented economy with strong institutions and transparent business practices.

| Aspect | Norwegian Reality | Implication for Translation |
|--------|-------------------|----------------------------|
| Business Culture | Egalitarian, direct, informal | Use "du" address, avoid overly formal language |
| Language | Bokmal dominant (85-90%) | Default to Bokmal; avoid Nynorsk unless targeted |
| Company Types | AS, ASA, ENK, ANS, KS | Translate legal forms precisely |
| Regulation | Finanstilsynet, Konkurransetilsynet | Use official Norwegian regulatory terms |
| Tax System | High transparency, MVA (VAT) | Use standard Norwegian tax terminology |
| Payment Terms | Bank transfer common, Faktura standard | Use Norwegian payment terminology |
| Business Hours | 08:00-16:00 (standard) | 24-hour format in formal contexts |

### 1.2 Norwegian Company Types

| Type | Norwegian | Abbreviation | Characteristics |
|------|-----------|--------------|----------------|
| Private Limited Company | Aksjeselskap | AS | Min. NOK 30,000 share capital |
| Public Limited Company | Allmennaksjeselskap | ASA | Min. NOK 1,000,000 share capital |
| Sole Proprietorship | Enkeltpersonforetak | ENK | Unlimited liability |
| General Partnership | Ansvarlig selskap | ANS | Unlimited liability |
| Limited Partnership | Kommandittselskap | KS | Mixed liability |
| Foundation | Stiftelse | — | Non-profit |
| Cooperative | Samvirke | SA | Democratic governance |

## 2. Decision Framework

### 2.1 Norwegian Business Letter Structure

| Element | Norwegian | Notes |
|---------|-----------|-------|
| Sender Address | "Avsender:" | Top left or letterhead |
| Recipient Address | "Mottaker:" | Below sender |
| Date | "Dato:" | DD.MM.YYYY |
| Reference | "Vår ref:" / "Deres ref:" | Reference numbers |
| Subject Line | "Vedrørende:" / "Emne:" | Clear, concise |
| Salutation | "Hei [navn]" / "Kjære [navn]" | Use "du" (first name) |
| Body | Main content | Clear, direct, professional |
| Closing | "Med vennlig hilsen" | Standard closing |
| Signature | Signature + printed name | Optional title |

### 2.3 Norwegian Marketing Copy Adaptation

| English Approach | Norwegian Adaptation | Notes |
|------------------|---------------------|-------|
| Superlatives ("best", "ultimate") | Understated ("god", "pålitelig") | Norwegians prefer modesty |
| "World's #1" | "En av de ledende" | Avoid extreme claims |
| Emotional appeal | Rational, factual approach | Trust-based culture |
| "Buy now!" | "Les mer" / "Ta kontakt" | Less aggressive CTAs |
| "Amazing deal!" | "Godt tilbud" / "Konkurransedyktig pris" | Moderate enthusiasm |
| Celebrity endorsements | User testimonials, case studies | Credibility over glamour |
| "革命性" (revolutionary) | "Ny" / "Forbedret" | Avoid hype |

## 3. Error Pattern Library

### 3.1 False Friends in Norwegian Commercial Translation

| English False Friend | Incorrect Norwegian | Correct Norwegian | Notes |
|----------------------|---------------------|-------------------|-------|
| "actually" | "faktisk" | "egentlig" / "faktisk" | Context-dependent |
| "eventually" | "til slutt" | "etter hvert" / "på sikt" | NOT "til slutt" |
| "realize" | "realisere" | "innse" / "forsta" | "realize profit" = "realisere gevinst" |
| "presently" | "for tiden" | "om kort tid" / "for tiden" | Context-dependent |
| "resume" | "gjenoppta" | "fortsette" / "gjenoppta" | "resume" = "fortsette" (continue) |
| "excellent" | "utmerket" | "utmerket" / "god" | Same word, but "utmerket" may sound too formal |
| "promote" | "promovere" | "fremme" / "anbefale" | "promote" = "fremme" (to further) |
| "eventually" | "til slutt" | "på sikt" / "etter hvert" | "eventually" = "etter hvert" |
| "sensible" | "sensibel" | "forstandig" / "sensibel" | "sensible" = "forstandig" (wise) |
| "fabricate" | "fabrisere" | "dikte opp" / "produsere" | "fabricate" (lie) = "dikte opp" |

### 3.2 Common Structural Errors

| Error Pattern | Wrong Norwegian | Correct Norwegian | Notes |
|---------------|-----------------|-------------------|-------|
| Calque: "looking forward to" | "ser frem til" | "gleder meg til" | More natural |
| Calque: "in case of" | "i tilfelle av" | "ved" / "hvis" | Simpler |
| Calque: "please find attached" | "vedlagt finner du" | "Vedlagt" / "Som vedlagt" | Shorter |
| Calque: "do not hesitate to" | "du nøl ikke med" | "ta gjerne kontakt" | More natural |
| Calque: "we are pleased to" | "vi er glade for a" | "vi har gleden av å" | Standard phrase |
| Calque: "at your earliest convenience" | "ved din tidligste bekvemmelighet" | "nar det passer" / "snart" | Norwegian prefers directness |
| Calque: "enclosed please find" | "vedlagt finner du" | "Vedlagt" / "Som vedlagt" | Shorter |

### 3.3 Cultural Adaptation Errors

| English Expression | Wrong Norwegian | Correct Norwegian | Notes |
|---------------------|-----------------|-------------------|-------|
| "Best regards" | "Mvh" (too informal) | "Med vennlig hilsen" | Standard formal closing |
| "Sincerely" | "Oppriktig" | "Med vennlig hilsen" | "Sincerely" = "Med vennlig hilsen" |
| "To whom it may concern" | "Til hvem det ma anga" | "Til den det ma anga" | Correct form |
| "Dear Sir/Madam" | "Kjære Herr/Fru" | "Vaer sa venlig" | More natural |
| "Yours faithfully" | "Deres trofast" | "Med vennlig hilsen" | Norwegian standard |
| "Please advise" | "Vennligst rad" | "Gi oss beskjed" / "Ta kontakt" | More natural |

## 4. Domain-Specific Reference

### 4.1 Norwegian Business Communication Templates

**Formal Business Letter Opening:**
```
[Bedriftsnavn]
[Adresse]
[Dato]

Vedrørende: [Emne]

Hei [Navn],

[Innledning]

[Hoveddel]

[Avslutning]

Med vennlig hilsen,
[Navn]
[Tittel]
```

**Invoice Header:**
```
FAKTURA
Fakturanummer: [Nummer]
Fakturadato: [Dato]
Forfallsdato: [Dato]
Kundenummer: [Nummer]
```

**Quotation Header:**
```
TILBUD
Tilbudsnummer: [Nummer]
Dato: [Dato]
Gyldig til: [Dato]
```
