# Legal Documents — Norwegian (Bokmal)

## 1. Reader Model

### 1.1 Norwegian Legal System Overview

Norway operates under a **civil law** system based on codified statutes. The court system is organized into three tiers:

| Court Level | Norwegian Name | Function |
|-------------|----------------|----------|
| District Court | Tingrett | First instance for most civil and criminal cases |
| Court of Appeal | Lagmannsrett | Appeals from tingrett |
| Supreme Court | Høyesterett | Final appeals, constitutional matters |

### 1.2 Key Legal Codes

| Code | Norwegian Name | Domain |
|------|----------------|--------|
| Contract Act | Avtaleloven | Contract formation, interpretation |
| Tort Act | Skadeserstatningsloven | Tort liability, damages |
| Sale of Goods Act | Kjøpsloven | Commercial sales |
| Consumer Purchase Act | Forbrukerkjøpsloven | Consumer protection |
| Companies Act | Aksjeloven | Company governance |
| Partnership Act | Sameieloven | Partnerships |
| Property Act | Tinglysningsloven | Real property registration |
| Criminal Act | Straffeloven | Criminal law |
| Civil Procedure Act | Trafikkloven | Court procedures |
| Arbitration Act | Voldgiftsloven | Arbitration |

### 1.3 Document Types and Their Register

| Document Type | Norwegian Markers | Register |
|---------------|-------------------|----------|
| Contract (Kontrakt) | "Partene avtaler herved..." | Obligatory / Volitional |
| Convention (Konvensjon) | "De undertegnede stater..." | Solemn |
| Act/Law (Lov) | "§ 1. Lovens formål er..." | Legislative imperative |
| Court Decision (Dom) | "Retten finner at..." / "Etter stemmegivningen..." | Judicial reasoned |
| Regulation (Forskrift) | "Med hjemmel i... fastsettes..." | Regulatory |
| Notarial Act (Notarial dokument) | "For Notarius Publicus..." | Authentic |
| Memorandum of Understanding (Mellomregning) | "Partene er enige om..." | Agreed |

## 2. Decision Framework

### 2.1 Norwegian Contract Structure vs. Anglo-American Contract Structure

| Anglo-American Element | Norwegian Equivalent | Decision |
|------------------------|----------------------|----------|
| "Parties" | "Partene" | Preserve party identification |
| "Recitals / Whereas clauses" | "Preambulere" / "Forutsetninger" | Translate as "Forutsetninger:" |
| "Definitions" | "Definisjoner" | Preserve structure |
| "Representations and Warranties" | "Erklæringer og garantier" | Adapt to Norwegian contract law |
| "Covenants" | "Forpliktelser" | Translate as "forpliktelser" |
| "Conditions Precedent" | "Vilkår for ikrafttredelse" | Concept identical |
| "Conditions Subsequent" | "Opphørsvilkår" | Concept identical |
| "Indemnification" | "Skadeslosholdelse" / "Erstatning" | Requires adaptation |
| "Limitation of Liability" | "Ansvarsbegrensning" | Valid in Norwegian law |
| "Termination" | "Opphør" / "Heving" | "Heving" (for breach), "Opphør" (general) |
| "Governing Law" | "Anvendelig rett" / "Lovvalg" | Standard clause |
| "Dispute Resolution" | "Tvisteløsning" | Arbitration or courts |
| "Force Majeure" | "Force majeure" / "Overmakt" | Preserve Latin or use Norwegian |
| "Entire Agreement" | "Avtalens samlede innhold" / "Avtale完整性" | Concept known |
| "Amendments" | "Endringer" / "Tillegg" | "Tillegg" for written modifications |
| "Notices" | "Varsler" / "Meldinger" | Preserve |
| "Signatures" | "Underskrifter" | Format: "Utskrevet i [by], den [dato]" |
| "Schedules / Exhibits" | "Vedlegg" | "Vedlegg" (not "tidsplan") |

### 2.2 Modal Verbs — Legal Value in Norwegian

| English Modal | Norwegian Translation | Legal Value |
|---------------|----------------------|-------------|
| "shall" | "skal" | Strict obligation (universal in contracts) |
| "shall not" | "skal ikke" | Prohibition |
| "must" | "ma" | Obligation (less frequent in contracts than "skal") |
| "must not" | "ma ikke" | Prohibition |
| "may" | "kan" | Faculty / permission |
| "may not" | "kan ikke" | Absence of right / prohibition |
| "need not" | "er ikke forpliktet til" | Absence of obligation |
| "will" | "vil" / "skal" | Future commitment |
| "should" | "bor" | Recommendation (rare in contracts) |
| "is entitled to" | "har rett til" | Subjective right |
| "is required to" | "er forpliktet til" | Obligation |
| "is prohibited from" | "har ikke lov til" | Prohibition |

## 3. Error Pattern Library

### 3.1 False Friends in Norwegian Legal Translation

| English False Friend | Incorrect Norwegian | Correct Norwegian | Notes |
|----------------------|---------------------|-------------------|-------|
| "eventually" | "til slutt" | "eventuelt" / "om nodvendig" | "eventually" = "om nødvendig", NOT "til slutt" |
| "actually" | "faktisk" | "faktisk" / "egentlig" | "actually" = "egentlig" in legal context |
| "realize" | "realisere" | "innse" / "forsta" | "realize" = "innse" (to understand), NOT "realisere" (to achieve) |
| "consequently" | "konsekvent" | "følgelig" / "derfor" | "consequently" = "følgelig", NOT "konsekvent" |
| "premises" | "forutsetninger" | "lokaler" / "premisser" | "premises" (building) = "lokaler", (legal basis) = "premisser" |
| "execute" | "utføre" | "underteckne" | "execute a contract" = "underteckne en kontrakt" |
| "consideration" | "betraktning" | "motytelse" / "vederlag" | "consideration" (contract) = "motytelse" |
| "discharge" | "utladning" | "fritakelse" / "oppfyllelse" | "discharge of duty" = "oppfyllelse av plikt" |
| "instrument" | "instrument" | "dokument" / "redskap" | "legal instrument" = "juridisk dokument" |
| "specific" | "spesifikk" | "nærmere bestemt" / "bestemt" | "specific performance" = "tvangsfullbyrdelse" |

### 3.2 Common Structural Errors

| Error Pattern | Wrong Norwegian | Correct Norwegian | Notes |
|---------------|-----------------|-------------------|-------|
| Calque: "I agree to" | "Jeg er enig til" | "Jeg samtykker til" | Use legal register |
| Calque: "in case of" | "i tilfelle av" | "ved" / "dersom" | Shorter, more natural |
| Calque: "in accordance with" | "i overensstemmelse med" | "i henhold til" | Standard legal phrase |
| Calque: "for the purpose of" | "for formal av" | "for å" / "i den hensikt å" | Simplify |
| Calque: "with regard to" | "med hensyn til" | "når det gjelder" / "vedrørende" | Norwegian prefers shorter |
| Calque: "in the event that" | "i tilfelle at" | "dersom" / "hvis" | Simple conditional |
| Calque: "at the time of" | "på tidspunktet for" | "ved" / "på tidspunktet da" | More natural |

### 4.1 Norwegian Court System — Detailed

| Court | Norwegian | Jurisdiction |
|-------|-----------|-------------|
| Supreme Court | Høyesterett | Final appeals, constitutional review |
| Court of Appeal | Lagmannsrett | Appeals from tingrett (7 courts nationwide) |
| District Court | Tingrett | First instance (70 courts nationwide) |
| Specialized Courts | Særlige domstoler | Labour, maritime, IP, etc. |
| conciliation Board | Forliksråd | Mandatory first step in civil disputes |

### 4.4 Legal References Format

Norwegian legal references follow this structure:
- "Lov om avtaler med forbruker om fast eiendom (avhendingsloven)"
- "Avtaleloven § 36"
- "Forbrukerkjøpsloven med kommentarer"
- "Rettspraksis: Rt. 2020 s. 1234" (Supreme Court decision)
- "LG-2020-123456" (Lagmannsrett case number)

### 4.5 Notarial and Formal Requirements

| Requirement | Norwegian | Notes |
|-------------|-----------|-------|
| Witness requirement | "Vitne" | Two witnesses for wills |
| Notarial certification | "Notarius publicus" | For international documents |
| Apostille | "Apostille" | For Hague Convention countries |
| Power of attorney | "Fullmakt" | Must be specific and dated |
| Signature requirement | "Underskrift" | Must be in own hand |
| Date requirement | "Dato" | Must be specified |
| Seal/stamp | "Stempel" / "Segl" | For official documents |
