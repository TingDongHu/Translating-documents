# Financial Documents — Norwegian (Bokmal)

## 1. Reader Model

### 1.1 Norwegian Financial System Overview

Norway's financial system is regulated by the **Financial Supervisory Authority of Norway (Finanstilsynet)** and operates under the **Accounting Act (Regnskapsloven)** and related legislation.

| Institution | Norwegian Name | Function |
|-------------|----------------|----------|
| Oslo Stock Exchange | Oslo Børs | Primary stock exchange |
| Norwegian Central Bank | Norges Sentralbank (Norges Bank) | Monetary policy |
| Financial Supervisory Authority | Finanstilsynet | Regulation and supervision |
| Norwegian Accounting Standards Board | Regnskapsstyret | Accounting standards |
| Oslo Clearing | Oslo Clearing | Securities settlement |

### 1.2 Key Financial Legislation

| Act | Norwegian Name | Domain |
|-----|----------------|--------|
| Accounting Act | Regnskapsloven | Financial reporting |
| Securities Trading Act | Verdipapirhandelloven | Securities regulation |
| Public Limited Companies Act | Allmennaksjeloven | ASA governance |
| Private Limited Companies Act | Aksjeloven | AS governance |
| Financial Institutions Act | Finansforetaksloven | Banking, insurance |
| Anti-Money Laundering Act | Hvitvaskingsloven | AML compliance |
| Annual Accounts Act | Arsregnskapsloven | Annual reporting |

### 1.3 Document Types and Register

| Document Type | Norwegian Markers | Register |
|---------------|-------------------|----------|
| Annual Report (Arsrapport) | "Styret legger frem..." | Formal, statutory |
| Financial Statement (Regnskap) | "Resultatregnskapet viser..." | Technical, precise |
| Audit Report (Revisjonsberetning) | "Vi har revidert..." | Independent, assurance |
| Prospectus | "Emisjonsprospekt" | Legal, disclosure |
| Board Report (Styreberetning) | "Styret anser..." | Governance, narrative |
| Budget (Budsjett) | "Budsjettet for 2026..." | Planning, forecast |
| Quarterly Report | "Kvartalsrapport Q2..." | Timely, interim |

## 2. Decision Framework

### 2.1 Norwegian Financial Statement Structure

| Element | Norwegian | Notes |
|---------|-----------|-------|
| Balance Sheet | Balanse / Status | Asset = Eiendeler, Equity+Liability = Egenkapital og gjeld |
| Income Statement | Resultatregnskap | Revenue = Inntekter, Expenses = Kostnader |
| Cash Flow Statement | Kontantstromoppgjor | Operating, investing, financing activities |
| Notes to Accounts | Noter | Detailed explanations |
| Auditor's Report | Revisjonsberetning | Independent opinion |
| Director's Report | Direktoberetning | Management narrative |
| Statement of Changes in Equity | Endringer i egenkapital | Equity movements |

### 2.2 Norwegian Chart of Accounts Standard

Norwegian companies typically use a standard chart of accounts (kontoplan) with the following structure:

| Account Range | Category | Norwegian |
|---------------|----------|-----------|
| 1000-1999 | Assets (Eiendeler) | Anleggsmidler, omloepsmidler |
| 2000-2999 | Debt (Gjeld) | Egenkapital, gjeld |
| 3000-3999 | Revenue (Inntekter) | Salgsinntekter, finansinntekter |
| 4000-4999 | Operating Costs (Driftskostnader) | Varekostnad, lonn |
| 5000-5999 | Other Costs (Andre kostnader) | Avskrivninger, renter |
| 6000-6999 | Financial Items (Finansposter) | Rentekostnader, valuta |
| 7000-7999 | Taxes (Skatter) | Skattekostnad |
| 8000-8999 | Extraordinary Items | Uvanlige poster |
| 9000-9999 | Memorandum Accounts | Noteopplysninger |

### 2.3 Norwegian GAAP (NRS) vs. IFRS

| Feature | Norwegian GAAP (NRS) | IFRS |
|---------|---------------------|------|
| Required for | Most Norwegian companies | Large publicly traded companies |
| Revenue recognition | Realisasjonsprinsippet | IFRS 15 |
| Leases | Leieforhold etter regnskapsloven | IFRS 16 |
| Financial instruments | Finansielle instrumenter | IFRS 9 |
| Consolidation | Konsernregnskap | IFRS 10 |
| Presentation | Balanse, resultatregnskap | Statement of financial position |

## 3. Error Pattern Library

### 3.1 False Friends in Norwegian Financial Translation

| English False Friend | Incorrect Norwegian | Correct Norwegian | Notes |
|----------------------|---------------------|-------------------|-------|
| "capital" | "kapital" | "egenkapital" / "kapital" | Context-dependent: "share capital" = "aksjekapital" |
| "securities" | "sikkerhet" | "verdipapirer" | "Securities" = "verdipapirer", NOT "sikkerhet" |
| "stock" | "lagerbeholdning" | "aksjer" | "Stock" (shares) = "aksjer", (inventory) = "lager" |
| "bonds" | "bann" | "obligasjoner" | "Bonds" = "obligasjoner" |
| "gross" | "grovt" | "brutto" | "Gross" = "brutto" |
| "net" | "netto" | "netto" | Same, but "net income" = "nettoinntekt" |
| "yield" | "utbytte" | "avkastning" | "Yield" = "avkastning", "dividend" = "utbytte" |
| "equity" | "rettferdighet" | "egenkapital" | "Equity" = "egenkapital", NOT "rettferdighet" |
| "balance" | "balanse" | "balanse" | Same, but check context |
| "fiscal" | "fiskal" | "finanspolitisk" / "skattemessig" | "Fiscal year" = "regnskapsar" |

### 3.2 Common Formatting Errors

| Error Pattern | Wrong Norwegian | Correct Norwegian | Notes |
|---------------|-----------------|-------------------|-------|
| Number format | "1,000.50" | "1 000,50" | Space thousands, comma decimal |
| Currency symbol position | "kr 100" | "100 kr" | Symbol AFTER amount |
| Percentage format | "5%" | "5 %" | Space before % |
| Date format | "05/25/2026" | "25.05.2026" | DD.MM.YYYY |
| Negative numbers | "(1,000)" | "(1 000)" or "-1 000" | Use parentheses or minus sign |
| Fiscal year reference | "FY 2026" | "Regnskapsar 2026" | Norwegian abbreviation |

### 3.3 Gender and Number Agreement in Financial Terms

| Noun | Gender | Wrong | Correct | Notes |
|------|--------|-------|---------|-------|
| overskudd (profit) | neuter | "et overskudd" | "et overskudd" | Correct (neuter) |
| underskudd (loss) | neuter | "et underskudd" | "et underskudd" | Correct (neuter) |
| fortjeneste (profit) | feminine | "et fortjeneste" | "en fortjeneste" | Feminine (Bokmal "en") |
| utbytte (dividend) | neuter | "et utbytte" | "et utbytte" | Correct (neuter) |
| gjeld (debt) | masculine | "et gjeld" | "en gjeld" | Masculine |
| eiendeler (assets) | masculine | "et eiendeler" | "en eiendeler" | Masculine (plural: "eiendeler") |
