# Technical Documents — Norwegian (Bokmal)

## 1. Reader Model

### 1.1 Norwegian Technical Standards System

Norway uses the **Norwegian Standards (SN)** system, aligned with European (EN) and international (ISO) standards.

| Organization | Norwegian Name | Function |
|--------------|----------------|----------|
| Standards Norway | Standard Norge | National standards body |
| Norwegian Electrician Association | Nelfo | Electrical standards |
| Norwegian Maritime Authority | Sjøfartsdirektoratet | Maritime standards |
| Petroleum Safety Authority | Petroleumstilsynet | Oil & gas safety |
| Norwegian Environment Agency | Miljødirektoratet | Environmental standards |

### 1.2 Key Technical Standards

| Standard | Norwegian Name | Domain |
|----------|----------------|--------|
| NEK 400 | NEK 400 | Electrical installations |
| NS 3420 | NS 3420 | Technical specifications for building |

## 2. Decision Framework

### 2.1 Norwegian Technical Writing Style

| Feature | Norwegian Approach | Notes |
|---------|-------------------|-------|
| Formality | Technical, precise | Use standard terminology |
| Tone | Direct, instructional | Clear, unambiguous |
| Pronoun | Impersonal or "du" | Avoid "man" (one) |
| Passive Voice | Sometimes used | For objectivity |
| Active Voice | Preferred | For instructions |
| Sentence Length | Short to medium | Avoid complex sentences |
| Lists | Numbered for steps | Bulleted for features |

### 2.2 Norwegian Technical Translation Principles

| Principle | Norwegian Approach | Notes |
|-----------|-------------------|-------|
| Terminology | Use SN/NEK/ISO standard terms | Consult Standard Norge |
| Units | SI units (metric) | Use Norwegian unit names |
| Numbers | Space thousands, comma decimal | `1 000,50` not `1,000.50` |
| Abbreviations | Define on first use | Use Norwegian expansion |
| Cross-references | Use § (paragraph) | Follow Norwegian convention |
| Warnings | Use standard symbols | Follow ISO 3864 |
| Instructions | Use imperative mood | "Koble til" (Connect) |

## 3. Error Pattern Library

### 3.1 False Friends in Norwegian Technical Translation

| English False Friend | Incorrect Norwegian | Correct Norwegian | Notes |
|----------------------|---------------------|-------------------|-------|
| "eventually" | "til slutt" | "etter hvert" / "på sikt" | NOT "til slutt" |
| "actually" | "faktisk" | "egentlig" / "faktisk" | Context-dependent |
| "realize" | "realisere" | "forsta" / "innse" | "realize" = "forsta" |
| "execute" | "utføre" | "kjøre" / "utføre" | "execute a program" = "kjøre et program" |
| "suspend" | "utsette" | "avbryte" / "midlertidig stanse" | "suspend operation" = "avbryte drift" |
| "critical" | "kritisk" | "kritisk" / "viktig" | "critical error" = "kritisk feil" |
| "current" | "nåværende" | "strøm" / "nåværende" | "electric current" = "strøm" |
| "drive" | "drive" | "stasjon" / "disk" | "hard drive" = "harddisk" |
| "memory" | "minne" | "minne" / "hukommelse" | "RAM" = "minne" |
| "terminal" | "terminal" | "terminal" / "konsoll" | Same |
