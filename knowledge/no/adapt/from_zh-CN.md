# Adaptation to Norwegian from Simplified Chinese (zh-CN)

## 1. Formality and Register

| Chinese (zh-CN) | Norwegian (Bokmal) | Notes |
|-----------------|-------------------|-------|
| 您 | du / deg / din | Always "du" — Norway completed du-reformen in 1970s. NEVER use "De". |
| 你 | du / deg / din | Same as above — universal address |
| 尊敬的客户 | Kjare kunde / Vaer sa venlig | Avoid calques; "尊重的" is not translated literally |
| 贵公司 | Deres selskap / Deres firma | "贵" marks deference; in Norwegian, simple polite possessive |
| 本公司 / 我公司 | Vaart selskap / Vaer | Chinese modesty ("our humble") disappears in Norwegian |
| 敞公司 | Vaar bedrift / Vaar virksomhet | Same: "敞" (humble) is not translated |
| 请 | Vaer sa snill / Vennligst / Gi | "请" followed by imperative can become "vaer sa snill og + inf." |
| 谢谢 | Takk / Tusen takk | "Tusen takk" more emphatic |
| 不好意思 | Unnskyld / Beklager | Less formal than "Jeg er lei meg" |
| 抱歉 | Beklager / Vi beklager | More formal than "不好意思" |
| 你好 | Hei / God dag | "Hei" for informal, "God dag" for formal |
| 大家好 | Hei alle sammen / God dag alle sammen | Inclusive greeting |
| 先生 | Hr. (herre) | Abbreviated in correspondence |
| 女士 | Fru | Use "Fru" for all women |
| 小姐 | Fru | "Froenn" is outdated; use "Fru" |
| 致 | Til / I anledning av | For letter headers |
| 此致 | Med vennlig hilsen | Standard closing |
| 顺颂商祺 | Med vennlig hilsen | Business closing; do not translate literally |
| 敬礼 | Med vennlig hilsen | "敬礼" (salute) is obsolete in Norwegian |
| 拜上 | Med vennlig hilsen | Archaic Chinese; use standard Norwegian closing |

### 1.1 Chinese to Norwegian Formality Mapping

| Chinese Level | Norwegian Equivalent | Context |
|---------------|---------------------|---------|
| 您 (formal you) | du | ALL contexts — Norway has no formal "you" |
| 你 (informal you) | du | ALL contexts |
| 请 (please) | vaer sa snill / vennligst | Polite requests |
| 谢谢 (thank you) | takk | Standard |
| 对不起 (sorry) | beklager / unnskyld | Apologies |
| 你好 (hello) | hei | Standard greeting |
| 再见 (goodbye) | ha det / farvel | "Ha det" informal, "farvel" formal |
| 您好 (formal hello) | god dag / hei | "God dag" more formal |
| 请问 (excuse me) | unnskyld / kan jeg spørre | Getting attention |
| 久仰大名 (I've heard so much about you) | hyggelig a mote deg | Norwegian equivalent |

## 2. Dates, Numbers, and Currency

### 2.1 Date Conversion

| Chinese Format | Norwegian Format | Notes |
|----------------|------------------|-------|
| 2024年5月25日 | 25.05.2024 | Year-Month-Day → Day.Month.Year |
| 2024年5月25日 | 25. mai 2024 | With month name |
| 2024年5月 | mai 2024 | Month in lowercase |
| 上午8点 | 08:00 | 24-hour format |
| 下午3点 | 15:00 | Convert to 24-hour |
| 下午3点30分 | 15:30 | Convert to 24-hour |
| 公元前206年 | 206 f.Kr. | "for Kristus" |
| 公元2024年 | 2024 e.Kr. | "etter Kristus" (usually omitted) |
| 第1条 | § 1 / Artikkel 1 | Paragraph/article reference |
| 第一季度 | forste kvartal / Q1 | Quarter reference |

### 2.2 Number System Conversion

| Chinese Number | Norwegian Number | Notes |
|----------------|------------------|-------|
| 一百 | 100 | "ett hundre" |
| 一千 | 1 000 | "ett tusen" |
| 一万 | 10 000 | "ti tusen" — Chinese "wan" = 10,000 |
| 十万 | 100 000 | "hundre tusen" |
| 一百万 | 1 000 000 | "en million" |
| 一亿 | 100 000 000 | "hundre millioner" — Chinese "yi" = 100 million |
| 十亿 | 1 000 000 000 | "en milliard" |
| 一百亿 | 10 000 000 000 | "ti milliarder" |

### 2.3 Critical Number Differences

| Feature | Chinese | Norwegian |
|---------|---------|-----------|
| Thousands separator | 万 (10,000 block) | Space (1,000 block) |
| Decimal separator | 点 (.) | Comma (,) |
| Percent | 百分之X | X % |
| Currency symbol | ¥ (before amount) | kr (after amount) |
| Large numbers | 亿 (10^8) | Milliard (10^9) |
| Negative | 负 | minus sign (-) |

### 2.4 Currency Conversion

| Chinese | Norwegian | Notes |
|---------|-----------|-------|
| 人民币 (RMB/CNY) | kinesisk yuan (CNY) | ISO code preferred |
| ¥100 | 100 CNY | Symbol after amount |
| 100元 | 100 CNY | "元" → "CNY" |
| 美元 (USD) | amerikanske dollar (USD) | Full name |
| 欧元 (EUR) | euro (EUR) | Same |
| 日元 (JPY) | japanske yen (JPY) | Same |
| 英镑 (GBP) | britiske pund (GBP) | Same |

## 3. Structural Differences

### 3.1 Word Order

| Feature | Chinese | Norwegian | Notes |
|---------|---------|-----------|-------|
| Basic word order | SVO | SVO | Same |
| Adjective position | Before noun | Before noun | Same |
| Possessive | 我的书 (my book) | boka mi (book-the my) | Possessive follows noun in Norwegian |
| Question word | At beginning | In situ (same position) | Different placement |
| Negation | 不 + verb | verb + ikke | Different position |
| Measure words | Required (一本书) | Not required (en bok) | Measure words removed |
| Topic-prominent | Yes (话题) | No | Restructure topic-comment |

### 3.2 Definiteness

| Chinese | Norwegian | Notes |
|---------|-----------|-------|
| 一本书 (a book) | en bok | Indefinite: article + noun |
| 这本书 (this book) | denne boka | Demonstrative + definite suffix |
| 那本书 (that book) | den boka | Demonstrative + definite suffix |
| 书 (book, general) | bok / boka | Context-dependent |
| 书的 (of the book) | boka til | Possessive construction |

### 3.3 Measure Words (Classifiers)

| Chinese Measure Word | Norwegian Equivalent | Notes |
|----------------------|---------------------|-------|
| 个 (general) | (omit) | Norwegian has no measure words |
| 本 (books) | (omit) | "en bok" not "en bok bok" |
| 条 (long objects) | (omit) | "en vei" not "en lang vei" |
| 张 (flat objects) | (omit) | "et bord" not "et flatt bord" |
| 台 (machines) | (omit) | "en maskin" not "en maskin maskin" |
| 位 (people, polite) | (omit) | "en person" not "en vennlig person" |

## 6. Punctuation and Formatting

### 6.1 Punctuation Differences

| Feature | Chinese | Norwegian |
|---------|---------|-----------|
| Period | 。 (hollow circle) | . (dot) |
| Comma | ， (full-width) | , (half-width) |
| Semicolon | ；(full-width) | ; (half-width) |
| Colon | ：(full-width) | : (half-width) |
| Question mark | ？(full-width) | ? (half-width) |
| Exclamation mark | ！(full-width) | ! (half-width) |
| Quotation marks | ""（CJK）| ""（straight）|
| Brackets | 【】| [] / () |
| Ellipsis | ……（two）| ... (three) |
| Em dash | ——（two）| — (one) |

### 6.2 Text Direction

| Feature | Chinese | Norwegian |
|---------|---------|-----------|
| Direction | Horizontal, left-to-right | Horizontal, left-to-right |
| Paragraph indent | 2 characters | Not standard (use spacing) |
| Line spacing | Varies | Single or 1.5 |
| Justification | Common | Common |

### 6.3 Number Formatting Summary
