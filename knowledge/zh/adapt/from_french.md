## 称呼和敬语

- 法语 "vous"（正式）→ 中文无正式/非正式之分
- 法语正式结尾 → 中文正式结尾（"此致敬礼"）
- 法语称谓（M.、Mme）→ 中文称谓（先生、女士）

## 商务文化

- 法语商务信函比中文更正式 → 适当调整语气
- 法语 "Nous" 在企业语境 → "我司" 或 "本公司"
- 法语被动语态 → 中文主动语态

## 法律和行政参考

- 法国法律体系（民法典、商法典）→ 保留法语名称，首次出现加注中文
- 法国行政机关 → 保留法语名称
- "SIRET"、"SIREN"、"RCS" → 保留缩写，法国特有

## 文化参考

- 法国节日、文化活动 → 保留原文，必要时加注
- 法国地理名称 → 使用官方中文译名（巴黎、里昂、马赛）
- 法国历史参考 → 保留原文

## 法语修辞手法

- 法语商务写作比中文更文学化 → 中文版本适当简化
- "Je me permets de vous écrire" → "兹致函"（不要字面翻译法语礼貌公式）
- "J'ai l'honneur de vous demander" → "谨此请求"
- 法语低调表达（"Ce n'est pas mal"）→ 中文直接表达（"相当不错"）

## 行政语言处理

- 法语行政公文有固定格式和套话 → 中文使用对应的行政公文格式
- "Il est porté à votre connaissance" → "特此通知"
- "En application de" → "根据"
- "Le préfet arrête que" → "省长命令"

## 教育和职业头衔

- "Ingénieur" → "工程师"（在法国是受保护的头衔，来自大学校）
- "Docteur" → "博士" 或 "医生"
- "Professeur" → "教授"
- "Maître"（律师）→ 保留"Maître"或使用"Me"缩写

## 数字和日期格式

- 法语数字格式（1.234,56 €）→ 中文格式（1,234.56 欧元）
- 法语日期格式（DD/MM/YYYY）→ 中文格式（YYYY年MM月DD日）
- 法语小数逗号 → 中文小数点
- 法语空格千位分隔符 → 中文逗号
