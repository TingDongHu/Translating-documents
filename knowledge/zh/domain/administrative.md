## 读者模型

**行政文书的目标读者是谁，他们期望什么？**

行政文书的读者包括政府官员、监管审查人员、申请人、公证人及相关实体。他们阅读文件是为了验证合规性、处理申请或确立法律身份。核心期望如下：

- **流程合规** — 表格、许可证和申报材料必须遵循规定的行政程序。任何偏差都可能导致退件或延误。
- **机构可识别性** — 政府机构名称、部门头衔和官方称谓必须能让受理机关立即辨识，不能使用非官方译名。
- **可追溯性** — 每一个编号、案号、文号和日期戳都必须完整保留，以确保审计和追踪链路完整。
- **正式但清晰的语言** — 行政文书的中文应直接、专业，避免法律用语的过度复杂，同时保持公文应有的正式感。

**什么会破坏读者的信任？**

- 翻译后的机构名称无法与官方机构对应，导致文件无法归档。
- 编号缺失或被修改，打断了文件的追溯链条。
- 表格字段标签误译，导致所填信息与原始要求不符。
- 官方印章或签章的描述遗漏了关键细节。
- 日期格式不统一（如法式日期与中式日期混用），造成审核混乱。
- 行政层级翻译不当（如将省级机构误译为市级），导致管辖权混淆。

## 决策框架

### 1. 政府机构名称

| 场景 | 处理规则 |
|---|---|
| 机构已有官方中文译名 | 使用官方译名（如"经济部"对应"Ministère de l'Économie"），需核实权威来源 |
| 机构无官方中文译名 | 首次出现使用全称 [中文描述性翻译]，后续直接使用法文原名 |
| 机构为某部下属司局 | 使用"部—司/局—处"层级格式，如"经济部—企业局—中小企业处" |
| 机构名称含缩写 | 保留法文缩写，首次出现时附加全称，如"OFII（法国移民融入局）" |

### 2. 表格字段标签

| 字段类型 | 中文标准标签 |
|---|---|
| 个人身份信息 | 姓名、出生日期、出生地点、国籍、护照号码、身份证号码 |
| 联系方式 | 通讯地址、联系电话、电子邮箱、邮政编码 |
| 声明与确认 | 保留勾选框/声明结构，翻译为"本人特此声明……" |
| 多选项 | 逐项翻译每个选项，保留选择机制（复选框、单选按钮、下拉菜单） |
| 引用其他文件或编号 | 保留引用原编号，不翻译编号本身，如"参见编号 AB-123/45" |

### 3. 公章和签章

| 印章类型 | 中文处理规则 |
|---|---|
| 纯文字印章 | 翻译印章文字，必要时在方括号内描述格式，如"[圆形公章：市政府]" |
| 图文混合印章 | 翻译文字部分，在方括号内描述徽标类型，不复制或修改徽标 |
| 含日期的印章 | 精确保留日期，转换为明确格式："2025年4月30日" |
| 手写批注 | 转录手写内容，标注"[手写批注]"并翻译内容 |

### 4. 文号和编号

| 编号类型 | 处理规则 |
|---|---|
| 案号、文号、许可号、登记号 | 精确保留全部字符，包括连字符、斜杠、点号和空格 |
| 条形码或二维码描述 | 如可读取则转录编码值，否则标注"[此处有条形码]" |
| 日期戳 | 精确保留日期，附加中文翻译周围文字 |
| 引用其他文件的编号 | 保留原始编号格式，不进行任何转换或重新排列 |

## 错误模式库

### 严重错误（零容忍）

| 错误类型 | 后果说明 |
|---|---|
| **文号连字符错误** | 将"AB-123/45"译为"AB12345"或"AB 123-45"，导致文件无法在档案系统中定位 |
| **日期戳与手写日期混淆** | 印刷日期和手写日期可能不同，若只保留其中一个会丢失关键信息 |
| **公章格式错误** | 漏译印章文字或错误描述印章类型（如将圆形章误述为方形章），损害文件真实性 |
| **边注遗漏** | 手写批注、缩写签章和边栏标注是文件的组成部分，遗漏会导致信息不完整 |

### 常见错误（高频出现）

| 错误类型 | 纠正指引 |
|---|---|
| **机构名称过度翻译** | 已有官方译名的机构应直接使用官方译名，不应自行创造新译法 |
| **表格字段错位** | 翻译填写式表格时，译文标签必须与对应的输入框对齐，字段顺序影响表单流程 |
| **语域不一致** | 行政文书要求正式但非法律术语的语域，避免口语化表达和过度法律化用语 |
| **文件类型术语混用** | 同一文件类型应统一使用"证明"或"许可证"等术语，不可交替使用近义词 |

### 隐蔽错误（容易遗漏）

| 错误类型 | 纠正指引 |
|---|---|
| **行政层级对应错误** | 法国的省（département）对应中国的"省"或"自治区"级别，不可与"市"混淆 |
| **法国特有制度直译** | 如"conseil municipal"不可直译为"市议会"，应译为"市政委员会"以符合中国行政体制语境 |
| **日期格式不统一** | 法式日期（30/04/2025）应统一转换为中文格式（2025年4月30日），不可混用 |
| **缩写首次展开遗漏** | 法文缩写首次出现时必须附加全称中文翻译，后续方可直接使用缩写 |

## 参考表

### 机构名称格式

| 场景 | 格式规范 |
|---|---|
| 文件中首次出现 | 法文全称 [中文翻译]，如"Préfecture de Paris [巴黎省长办公室]" |
| 后续出现 | 直接使用法文原名或已确定的中文译名 |
| 缩写首次出现 | 全称后附加括号内缩写，如"法国内政部（MINISTÈRE DE L'INTÉRIEUR）" |
| 国际知名机构 | 使用约定俗成的中文名称，如"世界卫生组织""联合国教科文组织" |
