## 通用原则

- 使用正式书面语
- 法律术语精确，避免口语化表达
- 保持法律论证的逻辑结构（鉴于、因此、前提是）
- 精确保留条款编号（第1条、第2.3款）

## 条款表达

- 义务性条款使用"应当"（对应 shall）
- 授权性条款使用"可以"（对应 may）
- 禁止性条款使用"不得"（对应 shall not）

## 签署栏

- "Fait à Paris, le ..." → "签署于巴黎，……"
- "Le Gérant" → "管理人"
- "Pour la société" → "代表公司"
- 签名行：保持格式，不翻译人名

## 日期条款

- 精确保留所有日期
- "à compter du" → "自……起"
- "pour une durée de" → "期限为"

## 法律模糊性处理

### 一词多义
法律术语常有多个含义，需根据上下文判断：
- "acte" → 文件 / 行为 / 证书（取决于上下文）
- "engagement" → 承诺 / 义务 / 债务
- "titre" → 所有权 / 证券 / 标题
- "droit" → 权利 / 法律 / 费用

处理策略：
1. 分析条款的法律意图
2. 参考同文档中其他用法保持一致
3. 如无法确定，保留原文 + 注释可能的含义

### 管辖权差异
不同法律体系的同一概念可能不完全对等：
- 法国 "SARL" ≠ 美国 "LLC"（近似但治理结构不同）
- 中国 "有限责任公司" ≠ 法国 "SARL"（管理层构成有差异）

处理策略：
1. 使用目标语言中最接近的对等术语
2. 首次出现时加注说明差异（如 "SARL（类似于有限责任公司）"）
3. 不要自行创造法律概念

### 双语合同格式
- 保留两种语言的条款编号对应
- "以中文为准" / "la version française fait foi" → "以中文版本为准"
- 定义条款中的术语必须在两种语言中严格对应
- 交叉引用必须匹配目标语言的条款编号
