## 分数区间

| 分数 | 含义 | 处理方式 |
|------|------|----------|
| 100 | 完美 | 无需处理 |
| 90-99 | 小问题，不影响理解 | 可选修复 |
| 70-89 | 若干问题，建议修复 | 推荐修复 |
| 50-69 | 系统性问题，需要修订 | 必须修订 |
| <50 | 该维度不合格，必须重译 | 全文重译 |

包含任何 **严重** 问题的译文总分不得超过 60。

---

## 1. 数值准确性（权重 30%）

| 分数 | 标准 |
|------|------|
| 100 | 所有数值、单位、日期、百分比、数量与原文完全匹配 |
| 90 | 1-2 处格式细微差异（如单位前空格、小数点样式），不影响数值本身 |
| 80 | 1-2 处数值格式不一致（如千位分隔符风格、单位缩写不统一） |
| 70 | 3-5 处格式问题或 1 处不影响整体理解的数值错误 |
| 60 | 超过 1 处实际数值错误；或 5 处以上格式问题；或任何日期理解错误 |
| <50 | 任何严重数值错误：数额错误、日期颠倒、擅自换算单位、货币不匹配 |

**零容忍项（专业级）：** 金额、日期、百分比——任何不匹配自动标记为严重问题。

---

## 2. 术语一致性（权重 20%）

| 分数 | 标准 |
|------|------|
| 100 | 所有重复出现的术语使用同一译文；完全遵守术语表 |
| 90 | 1-2 处有上下文合理依据的术语变化（如为可读性使用同义词） |
| 80 | 1-2 处无正当理由的术语不一致；或 1 处术语表违反 |
| 70 | 3-5 处术语不一致或 2-3 处术语表违反 |
| 60 | 5 处以上术语不一致；关键术语系统性混乱 |
| <50 | 系统性无视术语表；关键术语每次翻译不同，导致理解困难 |

**语境变化 vs 无理由变化：** 同一术语可能因语境不同而合理变化（如 "run" → "执行"（命令）vs "运行"（机器））。仅在无语境理由时才标记为问题。

---

## 3. 语义忠实度（权重 25%）

| 分数 | 标准 |
|------|------|
| 100 | 完全语义对等——无遗漏、无添加、无扭曲 |
| 90 | 1-2 处微调性意译，保留了原意但略微改变了强调重点 |
| 80 | 1-2 处遗漏或添加，不影响整体理解 |
| 70 | 3-5 处小遗漏或解释性添加；或 1 个句子含义偏差 |
| 60 | 遗漏句子；添加原文没有的内容；2 个以上句子含义错误 |
| <50 | 段落含义根本性错误；关键信息丢失；系统性遗漏 |

**检查方法：** 读原文段落，读译文段落。两者回答的是同一个问题吗？译文中是否增加了原文没有的信息？是否遗漏了关键细节？

---

## 4. 格式规范性（权重 10%）

| 分数 | 标准 |
|------|------|
| 100 | 所有数字格式、地址格式、标点符号符合目标语言规范 |
| 90 | 1-2 处格式偏差（如列表中漏了一个逗号、一个多余空格） |
| 80 | 2-3 处格式不一致或混用不同语言规范的标点 |
| 70 | 4-5 处格式问题；系统性使用源语言格式（如中文标点出现在英文译文中） |
| 60 | 大量格式问题；标点风格通篇不一致 |
| <50 | 格式严重影响文档专业度；严重违反目标语言规范 |

**常见检查项：**
- 小数点和千位分隔符格式
- 引号和破折号格式
- 日期和时间格式
- 单位缩写格式
- 地址格式

---

## 5. 完整性（权重 10%）

| 分数 | 标准 |
|------|------|
| 100 | 每个段落都已翻译；所有 [Pn] 标记完整；无缺失章节 |
| 90 | 1-2 个段落含有未翻译片段；或 1 个 [Pn] 标记缺失但无实际段落缺失 |
| 80 | 1-2 个段落完全未翻译；或一个小节标题未翻译 |
| 70 | 3-5 个段落未翻译或缺失；或一个章节被完全省略 |
| 60 | 5 个以上段落未翻译；或内容在文档末尾截断 |
| <50 | 大段内容缺失；文档不完整；存在系统性空白 |

---

## 6. 推断/OCR 质量（权重 5%）

| 分数 | 标准 |
|------|------|
| 100 | 所有推断标记合理、有充分依据，且正确使用 `[inferred:]` 标签 |
| 90 | 1 处推断标记理由稍弱但尚可接受 |
| 80 | 1-2 处推断可能不准确但已明确标注；或 1 处应标注推断而未标注 |
| 70 | 2-3 处可疑推断；或 2 处未标注的推断 |
| 60 | 多处可疑推断；系统性过度猜测而不标注 |
| <50 | 将凭空编造的内容当作翻译输出；编造数值或事实 |

**原则：** 将不确定内容标注为推断是良好实践。正确标注不确定性不是缺陷。
