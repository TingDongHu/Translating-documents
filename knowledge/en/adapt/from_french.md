# French to English Adaptation Guide

> Target language: English
> Source language: French
> Purpose: Standardized adaptation rules for French-to-English document translation
> Last updated: 2026-05-27

---

## 1. Formality and Address

### Overview

French has a grammatical distinction between formal "vous" and informal "tu" that permeates every level of address, politeness, and institutional communication. English uses a single "you" for all contexts. French also employs elaborate formal closing formulas, gendered titles, and passive/modal verb constructions that require systematic adaptation to natural English style.

### Rule 1.1: French "vous" / "tu" → English "you"

English has no formal/informal grammatical distinction for second-person address. The translator must choose an English register that matches the source intent without over-formalizing or under-formalizing.

| French | English | Strategy |
|--------|---------|----------|
| "vous" (formal singular) | "you" | Standard English "you" |
| "tu" (informal singular) | "you" | Same form; convey informality through tone |
| "Monsieur" / "Madame" | "Mr." / "Mrs." / "Ms." | Gendered title equivalent |
| "Cher collègue" | "Dear colleague" | Direct equivalent |
| "Cher Monsieur Dupont" | "Dear Mr. Dupont" | Formal salutation |
| "Tu peux" (informal) | "You can" | Neutral English |
| "Vous pouvez" (formal) | "You can" | Same English form |
| "Mesdames et Messieurs" | "Ladies and Gentlemen" / "Dear colleagues" | Context-dependent |

**Rule:** Do not over-formalize English to compensate for the loss of the vous/tu distinction. French formal tone should map to standard professional English, not archaic English. Conversely, informal "tu" in French does not warrant casual English ("hey", "hi") unless the source is genuinely colloquial.

### Rule 1.2: French formal closings → English equivalents

| French closing | English equivalent | Notes |
|---------------|-------------------|-------|
| "Veuillez agréer, Monsieur, l'expression de mes salutations distinguées" | "Yours faithfully" / "Yours sincerely" | Most formal; use "faithfully" if recipient is unknown |
| "Je vous prie d'agréer l'expression de mes sentiments les meilleurs" | "Kind regards" / "Best regards" | Standard business |
| "Cordialement" | "Kind regards" / "Best regards" | Semi-formal |
| "Bien à vous" | "Kind regards" / "Best wishes" | Warm professional |
| "Salutations distinguées" | "Yours faithfully" | Very formal |
| "Avec mes remerciements" | "With thanks" / "Thank you for your attention" | Gratitude closing |
| "En vous remerciant par avance" | "Thank you in advance" | Pre-emptive thanks |

**Rule:** French formal closings are significantly longer and more formulaic than English equivalents. Never translate them literally. Use the standard English business closing that matches the register.

### Rule 1.3: French titles → English equivalents

| French title | English equivalent | Notes |
|-------------|-------------------|-------|
| M. (Monsieur) | Mr. | Standard |
| Mme (Madame) | Mrs. or Ms. | Use "Ms." unless marital status is known and context requires "Mrs." |
| Mlle (Mademoiselle) | Ms. | Standard modern usage |
| Dr. / Docteur | Dr. | Medical or PhD |
| Professeur | Prof. / Professor | Academic |
| Ingénieur | Engineer | Note: French "Ingénieur" is a protected title from Grandes Écoles |
| Maître (Me) | Counsel / Attorney (for lawyers) | Keep "Maître" only in French-law contexts |
| Maire | Mayor | Municipal |
| Ministre | Minister | Government |
| Général / Colonel | General / Colonel | Military (same in English) |

**Rule:** French gendered titles (M./Mme/Mlle) should be adapted to English equivalents. "Mlle" is obsolete in official French documents since 2012; use "Ms." The title "Maître" has no English equivalent; use the person's role ("counsel", "attorney") unless the document specifically concerns French legal proceedings.

### Rule 1.4: French passive voice → English active voice

French uses passive constructions far more frequently than English, especially in administrative, legal, and academic writing. English prefers active voice for clarity.

| French (passive) | English (active) | Notes |
|-----------------|------------------|-------|
| "Il a été décidé que..." | "It was decided that..." / "The board decided that..." | Prefer active if agent is identifiable |
| "Le rapport a été rédigé par..." | "The team drafted the report..." | Agent available |
| "Il est interdit de fumer" | "Smoking is prohibited" | Keep passive if no clear agent |
| "La décision a été prise" | "The decision was taken" / "They took the decision" | Active preferred |
| "Il est demandé de" | "Please" / "You are requested to" | Rewrite as direct request |

**Rule:** Convert French passive to English active voice wherever the agent is identifiable. Where the agent is absent and the sentence is impersonal, English passive is acceptable.

### Rule 1.5: French modal verbs → English modals

| French | English | Strength |
|--------|---------|----------|
| "doit" (devoir) | "shall" (legal) / "must" / "should" | Obligation |
| "peut" (pouvoir) | "may" / "can" | Permission / ability |
| "pourrait" (conditional) | "might" / "could" | Polite suggestion / possibility |
| "devra" (future devoir) | "shall" / "will need to" | Future obligation |
| "est tenu de" | "is required to" / "shall" | Formal obligation |
| "a le droit de" | "is entitled to" / "may" | Right |
| "devrait" (conditional) | "should" / "ought to" | Recommendation |
| "pourra" (future pouvoir) | "may" / "will be able to" | Future permission |

**Rule:** "Doit" in legal context maps to "shall." In non-legal context, "must" or "should" is more natural. "Peut" maps to "may" (permission) or "can" (ability). Avoid overusing "shall" in non-legal English, as it sounds archaic.

## 2. Dates, Times, and Numbers

### Overview

French and English use different conventions for date ordering, number grouping, decimal separators, time notation, and measurement units. The metric system is standard in both French and British English, but American English uses imperial units. Currency formatting also differs. Incorrect conversion of these elements is a common source of translation errors.

### Rule 2.1: French date format → English date format

| French | English (UK/International) | English (US) | Notes |
|--------|---------------------------|-------------|-------|
| 27 février 2026 | 27 February 2026 | February 27, 2026 | Full date |
| 27/02/2026 | 27/02/2026 | 02/27/2026 | Numeric — watch for ambiguity |
| 27 fév. 2026 | 27 Feb 2026 | Feb 27, 2026 | Abbreviation |
| Le 1er janvier | 1 January | January 1st | "1er" = premier = 1st |
| fin février | late February | end of February | Approximate |
| début mars | early March | beginning of March | Approximate |
| mi-juin | mid-June | mid-June | Same concept |

**Rule:** French uses DD/MM/YYYY. US English uses MM/DD/YYYY. Always clarify or use the unambiguous written form (Month DD, YYYY). "Le 1er" (premier) must be rendered as "1st" or written out. When dates appear in legal or formal contexts, write the month in full.

### Rule 2.2: French number format → English number format

| French | English | Notes |
|--------|---------|-------|
| 1 234,56 | 1,234.56 | Space = thousands separator; comma = decimal |
| 1,5 | 1.5 | French decimal comma |
| 1 000 000 | 1,000,000 | French space vs English comma for thousands |
| 10 000 | 10,000 | Large numbers |
| 99,9 % | 99.9% | Percentage with decimal |
| 0,25 | 0.25 | Small decimals |
| 12 345,67 € | EUR 12,345.67 | Combined currency + number |

**Rule:** French uses a space as the thousands separator and a comma as the decimal separator. English uses a comma for thousands and a period for decimals. This is the most critical conversion in numerical translation — getting it wrong inverts the meaning of the number.

### Rule 2.3: French time format → English time format

| French | English (12-hour) | English (24-hour) | Notes |
|--------|-------------------|-------------------|-------|
| 14h30 | 2:30 PM | 14:30 | Standard conversion |
| 9h00 | 9:00 AM | 09:00 | Morning |
| 17h15 | 5:15 PM | 17:15 | Afternoon |
| 8h30 – 12h00 | 8:30 AM – 12:00 PM | 08:30–12:00 | Time range |
| 13h – 14h30 | 1:00 PM – 2:30 PM | 13:00–14:30 | Lunch break |
| 10h | 10:00 AM | 10:00 | Without minutes |
| à 15h précises | at 3:00 PM sharp | at 15:00 sharp | Exact time |

**Rule:** French uses the 24-hour clock with "h" as separator (14h30). English uses either 12-hour (AM/PM) or 24-hour. Match the target audience: British English accepts both; American English strongly prefers 12-hour. Use AM/PM for formal American English.

### Rule 2.4: French currency → English currency

| French | English | Notes |
|--------|---------|-------|
| 100 € | EUR 100.00 / €100 | Euro — both notations acceptable |
| 1 500,00 € | EUR 1,500.00 | Space removed, comma → period |
| 50 $ US | USD 50.00 / $50 US | US dollar |
| 250 £ | GBP 250.00 / £250 | British pound |
| CHF 300 | CHF 300.00 | Swiss franc — keep ISO code |
| 10 000 € HT | EUR 10,000.00 ex. VAT | "HT" = hors taxes = excluding tax |
| 12 000 € TTC | EUR 12,000.00 incl. VAT | "TTC" = toutes taxes comprises = including tax |

**Rule:** "HT" (hors taxes) → "ex. VAT" or "excluding tax." "TTC" (toutes taxes comprises) → "incl. VAT" or "including tax." These are critical in commercial and financial documents.

### Rule 2.5: French measurement units → English

| French (metric) | English (US imperial) | Notes |
|-----------------|----------------------|-------|
| kilomètre (km) | mile | Convert if US audience |
| mètre (m) | yard / foot | Convert if US audience |
| centimètre (cm) | inch | Convert if US audience |
| litre (L) | gallon | Convert if US audience |
| kilogramme (kg) | pound (lb) | Convert if US audience |
| gramme (g) | ounce (oz) | Convert if US audience |
| degré Celsius (°C) | degree Fahrenheit (°F) | Convert if US audience |

**Rule:** Both French and British English use the metric system — no conversion needed. Only convert to imperial for American audiences. When converting, always provide the original metric value in parentheses: "The road is approximately 50 miles (80 km) long." Temperature: °C → °F (multiply by 1.8, add 32).

## 3. Cultural References

### Overview

French text contains idioms, cultural references, institutional concepts, holidays, and food/culture terms that may be unfamiliar or misleading to English-speaking readers. The strategy ranges from direct translation (for well-known references) to explanatory adaptation (for France-specific concepts) to substitution (for idioms with clear English equivalents).

### Rule 3.1: French idioms → English equivalents

| French idiom | Literal translation | English equivalent | Strategy |
|-------------|--------------------|--------------------|----------|
| "Avoir le cafard" | "To have the cockroach" | "To feel down" / "To be blue" | Substitute English idiom |
| "Casser les pieds" | "To break someone's feet" | "To bore someone to death" | Substitute |
| "Donner sa langue au chat" | "Give one's tongue to the cat" | "To give up guessing" | Explain or substitute |
| "Il pleut des cordes" | "It's raining ropes" | "It's raining cats and dogs" | Substitute |
| "Mettre les pieds dans le plat" | "To put one's feet in the dish" | "To put one's foot in it" / "To blurt out" | Substitute |
| "Coûter les yeux de la tête" | "To cost the eyes of the head" | "To cost an arm and a leg" | Substitute |
| "Poser un lapin" | "To put down a rabbit" | "To stand someone up" | Substitute |
| "Tomber dans les pommes" | "To fall in the apples" | "To faint" / "To pass out" | Explain |
| "Avoir le beurre et l'argent du beurre" | "To have the butter and the money for the butter" | "To want to have one's cake and eat it too" | Substitute |
| "Tourner autour du pot" | "To turn around the pot" | "To beat around the bush" | Substitute |
| "Courir sur le haricot" | "To run on the bean" | "To get on someone's nerves" | Substitute |
| "Mettre de l'eau dans son vin" | "To put water in one's wine" | "To temper one's enthusiasm" / "To back down" | Explain |
| "Ne pas y aller par quatre chemins" | "Not to take four paths" | "To not mince words" / "To be direct" | Substitute |
| "Se creuser la tête" | "To dig one's head" | "To rack one's brains" | Substitute |
| "L'habit ne fait pas le moine" | "The habit doesn't make the monk" | "Don't judge a book by its cover" | Substitute |
| "Après nous, le déluge" | "After us, the flood" | "After us, the deluge" — well known | Keep literal (understood) |

**Rule:** French idioms should be replaced with natural English equivalents when available. Never translate literally without an English equivalent. If no equivalent exists, use a plain-English paraphrase that conveys the meaning without the idiom.

### Rule 3.2: French cultural/gallicisms → English

| French expression | English equivalent | Notes |
|------------------|--------------------|-------|
| "c'est la vie" | "that's life" | Widely understood; can keep French |
| "l'esprit de l'escalier" | "staircase wit" / "the wit of the staircase" | Explain: thinking of the perfect reply too late |
| "sang-froid" | "composure" / "cool-headedness" | Widely adopted in English |
| "savoir-faire" | "know-how" / "tact" | Widely adopted |
| "joie de vivre" | "exuberance" / "love of life" | Widely adopted |
| "bon vivant" | "epicure" / "person who enjoys the good life" | Known but less common |
| "ennui" | "boredom" / "tedium" | Widely adopted |
| "déjà vu" | "déjà vu" | Universal; keep as-is |
| "raison d'être" | "reason for being" / "purpose" | Widely adopted |
| "nouveau riche" | "new money" / "nouveau riche" | Both used |
| "faux pas" | "faux pas" / "social blunder" | Widely adopted |
| "art de vivre" | "art of living" / "art of life" | Known |
| "bonhomie" | "geniality" / "good-natured warmth" | English borrowing |
| "je ne sais quoi" | "a certain something" | Widely understood |

**Rule:** Many French gallicisms are already absorbed into English and need no translation. When the term is unfamiliar, provide a plain-English explanation. Never leave a French term unexplained if it is not a recognized English loanword.

### Rule 3.3: French holidays and institutional references

| French holiday/event | English equivalent | Notes |
|---------------------|--------------------|-------|
| Fête nationale (14 juillet) | Bastille Day (14 July) / French National Day | "Bastille Day" is the standard English name |
| Jour de l'Armistice (11 novembre) | Armistice Day / Remembrance Day | 11 November — same date as UK |
| La Toussaint (1 novembre) | All Saints' Day | Same concept |
| Fête du Travail (1 mai) | Labour Day / May Day | Note spelling: "Labour" (UK), "Labor" (US) |
| Boxing Day / Noël | Christmas Day / Boxing Day | France observes 25 Dec only |
| Grande École | "Grande École" (keep French) | Elite French institution; explain on first use |
| Code civil | Civil Code / Napoleonic Code | Both are accepted in English |
| Code de commerce | Commercial Code | Keep French name, add English gloss |
| Conseil d'État | Council of State / Conseil d'État | France-specific administrative court |
| Préfet | Prefect | France-specific administrative authority |
| Mairie | Town Hall / City Hall | Municipal building |
| Conseil régional | Regional Council | Regional government body |
| Département | Department (keep) | Administrative division |

**Rule:** French institutional and administrative references are France-specific and should keep their French names with an English explanation on first mention. Do not invent English equivalents for bodies that have no English-language counterpart.

### Rule 3.4: French food and culture

| French term | English equivalent | Notes |
|------------|--------------------|-------|
| fromage | cheese | Generic |
| charcuterie | charcuterie / deli meats | Widely adopted in English |
| boulangerie | bakery | Specific type |
| pâtisserie | pastry shop | Specific type |
| crèche | nursery / daycare (UK) | Not "Nativity scene" in this context |
| terroir | terroir / regional character | Widely used in wine/food contexts |
| mise en place | mise en place | Kitchen term; widely adopted |
| carte blanche | carte blanche / free hand | Widely adopted |
| prototype | prototype | Direct borrowing already in English |
| dossier | file / dossier | Both used in English |
| bureau | office / bureau | "Bureau" is known in English |
| résumé | résumé / curriculum vitae (CV) | In US, "résumé"; in UK, "CV" |
| entrée | appetizer (US) / starter (UK) | Note: in France, "entrée" = first course |
| plat principal | main course / entrée (US) | Avoid confusion with US "entrée" |
| dessert | dessert | Same |

**Rule:** Many French food terms are already in English. For terms like "entrée" that have different meanings in French and English, clarify the French meaning explicitly to avoid confusion.

## 4. Terminology

### Overview

French and English share many Latinate roots, but legal, business, administrative, technical, and financial terminology diverges significantly. False cognates (faux amis) are a particular danger. Each domain below provides the standard English equivalents for common French terms.

### Rule 4.1: Legal terminology

| French term | English term | Notes |
|------------|-------------|-------|
| contrat | contract | |
| clause | clause | Same root |
| obligation | obligation | Same root |
| responsabilité civile | civil liability | Not "civil responsibility" |
| dommages-intérêts | damages | Not "prejudice" |
| prescription | limitation (of actions) | False cognate: not "prescription" |
| mandat | mandate / power of attorney | Context-dependent |
| caution | deposit / security / guarantee | Not "caution" as in warning |
| hypothèque | mortgage | False cognate |
| avocat | lawyer / attorney | Not "advocate" in most contexts |
| notaire | notary public (UK: notary) | French notaire is a public official — different role |
| greffier | clerk of court | Court official |
| jugement | judgment | Not "jugement" cognate |
| arrêt | ruling / judgment | Used by courts of appeal and Conseil d'État |
| ordonnance | order / ordinance | Legal or administrative order |
| garde à vue | police custody | Not "guard in sight" |
| récidive | recidivism / repeat offending | |

**Rule:** Watch for false cognates carefully. "Prescription" in French law means "limitation period," not "prescription" (medication). "Domages-intérêts" means "damages" (compensation), not "prejudice." "Caution" means "security deposit," not "caution" (warning).

## 5. Administrative/Legal Style

### Overview

French legal and administrative writing is characterized by elaborate formulas, extensive use of passive voice, long subordinate clauses, nominalization, and impersonal constructions. English legal and administrative writing favors active voice, shorter sentences, direct address, and simpler clause structures. Adapting French administrative style to English requires systematic restructuring, not word-for-word translation.

### Rule 5.1: French legal formulas → English equivalents

| French formula | English equivalent | Context |
|---------------|-------------------|---------|
| "Il est convenu que" | "It is agreed that" / "The parties agree that" | Contract preamble |
| "Il est stipulé que" | "It is provided that" / "This Agreement provides that" | Contract clause |
| "Les soussignés" | "The undersigned" | Signature block |
| "Fait à [ville], le [date]" | "Done at [city], on [date]" | Execution clause |
| "En foi de quoi" | "In witness whereof" | Closing of contract |
| "Les présentes" | "This Agreement" / "This document" | Reference to current document |
| "Sous réserve de" | "Subject to" / "Without prejudice to" | Condition/limitation |
| "À défaut de" | "In the absence of" / "If...fails to" | Default clause |
| "Nonobstant toute disposition contraire" | "Notwithstanding any provision to the contrary" | Override clause |
| "Pour tout ce qui n'est pas prévu" | "For anything not covered herein" / "In the event of a gap" | Gap-filling clause |

**Rule:** French legal formulas are highly standardized. Never translate them literally if a recognized English legal formula exists. Use the English legal convention that conveys the same legal effect.

### Rule 5.2: French passive constructions → English active voice

| French (passive/impersonal) | English (active) | Notes |
|----------------------------|------------------|-------|
| "Il a été décidé que..." | "The board decided that..." | Active if agent known |
| "Il est porté à votre connaissance que..." | "We wish to inform you that..." | Rewrite as direct address |
| "Il vous sera demandé de..." | "You will need to..." / "You are required to..." | Direct address |
| "Il est interdit de..." | "Do not..." / "Smoking is prohibited" | Command or passive |
| "Il est rappelé que..." | "Please note that..." / "This is a reminder that..." | Rewrite |
| "Il a été constaté que..." | "We found that..." / "The audit found that..." | Active if possible |
| "Il est conseillé de..." | "It is advisable to..." / "We recommend..." | Active preferred |

**Rule:** French administrative writing relies heavily on impersonal constructions ("il est", "il a été"). In English, rewrite these as direct statements or active sentences. The passive/impersonal form should only be retained when no agent is known or when the passive voice serves a deliberate stylistic purpose.

### Rule 5.3: French subordinate clauses → English restructuring

French legal writing frequently embeds multiple subordinate clauses within a single sentence. English prefers shorter, clearer sentences.

| French style | English adaptation |
|-------------|-------------------|
| "Le vendeur, qui a fourni les informations mentionnées ci-dessus, et qui a garanti leur exactitude, sera tenu responsable de tout préjudice en résultant." | "The seller provided the above information and guaranteed its accuracy. The seller shall be liable for any resulting damages." |
| "Sous réserve des dispositions légales et réglementaires en vigueur, et dans le respect des obligations contractuelles, le fournisseur s'engage à..." | "Subject to applicable laws and regulations, and in compliance with its contractual obligations, the supplier undertakes to..." |
| "Le souscripteur, en signant le présent formulaire, déclare avoir pris connaissance des conditions générales et accepte sans réserve l'ensemble de leurs dispositions." | "By signing this form, the subscriber acknowledges that they have read the general terms and conditions and accepts all provisions without reservation." |

**Rule:** Break long French sentences into two or three shorter English sentences. Move conditional clauses to the beginning of the sentence for clarity. Replace embedded relative clauses with coordinate clauses (using "and" or separate sentences).

### Rule 5.4: French administrative language → English direct style

| French administrative | English direct | Notes |
|----------------------|---------------|-------|
| "Vu" (legal decree opening) | "Having regard to" / "Considering" | Formal decree opening |
| "Arrête" (administrative order) | "Hereby orders" / "Decrees" | Dispositive formula |
| "Le préfet arrête que" | "The Prefect orders that" / "The Prefect decrees that" | Authority exercise |
| "Le ministre arrête que" | "The Minister hereby orders" | Ministerial order |
| "Article 1er : Est autorisé..." | "Article 1: [Party] is authorized..." | Active reformulation |
| "Est approuvé" | "Is approved" / "Approves" | Approval formula |
| "Il est créé..." | "There is hereby established..." / "This order establishes..." | Creation formula |
| "Le présent arrêté entre en vigueur..." | "This order takes effect..." | Entry into force |

**Rule:** French administrative decrees follow a rigid structure: "Vu" (legal basis) + "Le [authority] arrête que" (dispositive formula). Translate the "Vu" as "Having regard to" and the "Arrête" as "hereby orders" or "decrees." Keep the formal register but avoid archaic English.

### Rule 5.5: Sample paragraph comparison

**French original:**
"Il est convenu et stipulé entre les parties que le prestataire s'engage à fournir les services décrits en annexe 1 dans un délai de trente (30) jours à compter de la date de signature du présent contrat, sous réserve du paiement anticipé d'un acompte représentant vingt pour cent (20 %) du montant total. À défaut de paiement dans le délai imparti, le prestateur se réserve le droit de suspendre l'exécution de ses obligations, sans préjudice de tout recours ultérieur."

**English translation:**
"The parties agree that the service provider shall deliver the services described in Schedule 1 within thirty (30) days of the date of this Agreement, subject to prior payment of a deposit representing twenty percent (20%) of the total amount. If payment is not received within the specified period, the service provider reserves the right to suspend performance of its obligations, without prejudice to any further remedy."

**Key adaptation points:**
- "Il est convenu et stipulé" → "The parties agree" (active voice, simplified formula)
- "s'engage à fournir" → "shall deliver" (modal verb "shall" for legal obligation)
- "dans un délai de" → "within" (more concise English)
- "à compter de" → "from" / "of" (simplified)
- "sous réserve du paiement anticipé" → "subject to prior payment" (standard English condition)
- "À défaut de" → "If...is not" (conditional rewrite)
- "sans préjudice de" → "without prejudice to" (direct cognate that works)

## 6. Punctuation

### Overview

French and English share most punctuation marks but differ significantly in quotation conventions, spacing rules, and dash usage. French uses guillemets « » instead of double quotes " ", inserts a non-breaking space before colons, semicolons, exclamation marks, and question marks, and uses different dash conventions. These differences are small individually but cumulatively important for producing natural English text.

### Rule 6.1: French guillemets « » → English quotation marks " "

| French | English | Notes |
|--------|---------|-------|
| « bonjour » | "hello" | Standard conversion |
| « Bonjour, comment allez-vous ? » | "Hello, how are you?" | With interior punctuation |
| « dit-il » | "he said" | Reported speech |
| « le « vrai » problème » | "the 'real' problem" | Nested quotes: single inside double |

**Rule:** French guillemets « » are always replaced by English double quotation marks " ". French guillemets use « (with non-breaking space before) and » (with non-breaking space after). English uses " and " with no extra spacing. For nested quotations, English uses single quotes inside double quotes (opposite of French convention in some styles).

### Rule 6.2: French spacing before punctuation → English no spacing

| French | English | Notes |
|--------|---------|-------|
| "coût : 100 €" | "cost: EUR 100" | No space before colon in English |
| "trois éléments : un, deux, trois" | "three elements: one, two, three" | Same rule |
| "vraiment ?" | "really?" | No space before ? in English |
| "merci !" | "thank you!" | No space before ! in English |
| "il dit ; elle répond" | "he says; she replies" | No space before ; in English |
| "Paris ; Lyon ; Marseille" | "Paris; Lyon; Marseille" | Semicolon spacing |

**Rule:** French typography requires a non-breaking space before :, ;, !, and ?. English typography does NOT use spaces before these punctuation marks. Always remove the space in translation. This is one of the most visible differences between French and English formatting.

### Rule 6.3: French dash conventions → English

| French | English | Notes |
|--------|---------|-------|
| " — " (em dash with spaces) | " — " (em dash with spaces) | Same usage for parenthetical |
| " - " (hyphen for range) | " – " (en dash for range) | French often uses hyphen; English prefers en dash |
| "de 2020 - 2025" | "from 2020 to 2025" / "2020–2025" | Range notation |
| "téléphone - 01 23 45 67 89" | "Telephone: 01 23 45 67 89" | Label separator |
| "nom - prénom" | "last name / first name" | Field separator |

**Rule:** French often uses a hyphen (-) where English uses an en dash (–) for ranges. French em dashes (—) are used similarly to English em dashes, with spaces on both sides. Convert French hyphens to en dashes for ranges: "2020–2025" not "2020-2025."

### Rule 6.4: French quotation conventions in running text

| French | English | Notes |
|--------|---------|-------|
| Le mot « liberté » est central. | The word "freedom" is central. | Terminology emphasis |
| Il a dit « non ». | He said "no." | Direct quote |
| « Comme dit l'autre... » | "As the saying goes..." | Reported speech |
| Les « experts » ont réagi. | The "experts" reacted. | Scare quotes |

**Rule:** French uses guillemets for emphasis, terminology, scare quotes, and direct speech — exactly as English uses double quotation marks. The function is the same; only the mark changes.

### Rule 6.5: French italics and underlining → English

| French formatting | English equivalent | Notes |
|------------------|-------------------|-------|
| Texte en italique | Italic text | Same convention |
| Texte souligné (underlined) | *Italic text* | English prefers italics over underlining for emphasis |
| Titre en majuscules | Title Case or SMALL CAPS | Adapt to English title conventions |
| **Texte en gras** | **Bold text** | Same convention |

**Rule:** French documents frequently use underlining for emphasis where English would use italics. Convert underlined text to italicized text in English. Capitalization for emphasis should be converted to italics or bold.

### Rule 6.6: French number and list formatting

| French | English | Notes |
|--------|---------|-------|
| 1) Premier point | (a) First point / 1. First point | Adapt numbering style |
| a) Premier point | (a) First point | Lowercase letter lists |
| I. Premier point | (i) First point / I. First point | Roman numerals |
| • Point | • Bullet point / - Bullet point | Bullet style varies |
| 1, 2, 3... | 1, 2, 3... | Same for inline lists |
| 1er, 2e, 3e | 1st, 2nd, 3rd | Ordinals |

**Rule:** French list formatting is similar to English. The main difference is ordinal notation: "1er" (premier) → "1st", "2e" (deuxième) → "2nd", "3e" (troisième) → "3rd". Use English ordinal suffixes.

### Rule 6.7: Summary table — French vs English punctuation

| Element | French | English | Action |
|---------|--------|---------|--------|
| Quotation marks | « » (guillemets) | " " (double quotes) | Replace |
| Nested quotes | « " " » | " ' ' " | Replace (reverse nesting) |
| Space before : | Yes (non-breaking) | No | Remove space |
| Space before ; | Yes (non-breaking) | No | Remove space |
| Space before ! | Yes (non-breaking) | No | Remove space |
| Space before ? | Yes (non-breaking) | No | Remove space |
| Em dash | « — » (with spaces) | " — " (with spaces) | Same |
| Range dash | - (hyphen) | – (en dash) | Replace |
| Apostrophe (elision) | l'homme | the man | No apostrophe needed in English |
| Decimal separator | comma (1,5) | period (1.5) | Replace |
| Thousands separator | space (1 000) | comma (1,000) | Replace |
| Date separator | / (27/02/2026) | / or - (27/02/2026 or 2026-02-27) | Same or convert |
| Time separator | h (14h30) | : (2:30 PM) | Replace and adjust format |
| Percentage | espace before % (99,9 %) | No space (99.9%) | Remove space, fix decimal |
| Euro symbol | after number (100 €) | before or after (EUR 100 or €100) | Adapt to convention |

**Rule:** The most impactful punctuation changes are: (1) replacing guillemets with quotation marks, (2) removing spaces before colons/semicolons/exclamation/question marks, and (3) converting decimal commas to decimal periods. These three changes alone account for the majority of visible French-to-English punctuation differences.
