## General Principles

- Use clear, direct English appropriate for government and regulatory contexts
- Preserve all reference numbers, filing numbers, and case numbers exactly
- Maintain formal register but avoid overly legalistic language
- Translate administrative titles to their closest English equivalents
- Keep institution names in original with English translation in brackets on first mention

## Government Bodies

- Translate ministry names: "XX部" → "Ministry of XX"
- Translate bureau names: "XX局" → "XX Bureau"
- Keep regional government body names with standard English translations
- If no established English name exists, transliterate + add description

## Permits and Authorizations

- "许可证" → "license" / "permit" (context-dependent)
- "批准" → "approval"
- "登记证" → "registration certificate"
- Use the specific term: "building permit", "business license", "operating license"

## Forms and Procedures

- "申请表" → "application form"
- "提交申请" → "submit an application"
- "处理时间" → "processing time"
- "申诉渠道" → "appeal channel" / "appeal options"
- "身份证明" → "identity document" / "ID"
- "地址证明" → "proof of address"

## Certificates and Seals

- "证书" → "certificate"
- "公证" → "notarization" / "notarized"
- "复印件" → "copy" / "photocopy"
- "原件" → "original"
- "公章" → "official seal" / "company stamp"
- "签字" → "signature"
- "见证" → "witness" (verb or noun as context requires)
