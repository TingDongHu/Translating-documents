# 中→德 (zh-CN → de-DE) 适应规则

- **id**: de/adapt/from_zh-CN
- **type**: adapt
- **target_lang**: de

## 1. Formality and Address — 您/Sie, 你/du, Chinese Humility → German Neutral

中文通过"您/你"区分正式程度，但谦辞体系（"拙见""敝公司""贵方"等）在德语中无直接对应，需转为德语中性正式表达。

| 中文原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| 您今天有空吗？ | Haben Sie heute Zeit? | 您 → Sie（大写），商业/正式场景 |
| 你帮我看看这个 | Kannst du dir das ansehen? | 你 → du（小写），熟人/内部场景 |
| 请坐 | Bitte nehmen Sie Platz / Setz dich bitte | 依语境必须二选一 |
| 尊敬的各位领导 | Sehr geehrte Damen und Herren | "各位领导"在德语无对应，改用通用敬语 |
| 张局长 | Herr Direktor Zhang / Herr Amtsleiter Zhang | 中文"姓+职务简称"需展开为全称 |
| 李老师 | Herr Li / Frau Li / Herr Professor Li | "老师"不可直译为 Lehrer，根据身份转换 |
| 王总 | Herr Wang (Geschäftsführer) / Herr Wang | "总"需视具体职务展开 |
| 赵工 | Herr Zhao (Ingenieur) | "工"（工程师简称）需展开 |
| 拙见 | meiner bescheidenen Meinung nach / ich denke | 谦辞"拙"→ 中性表达 |
| 敝公司 | unser Unternehmen / unsere Firma | 谦辞"敝"→ 中性表达 |
| 贵方 | Ihr Unternehmen / Ihre Seite | 敬辞"贵"→ 德语 Ihr |
| 贵国 | Ihr Land | "贵国"→ Ihr Land，非 Euer Land |
| 请多关照 | Ich freue mich auf die Zusammenarbeit | 中文客套语 → 德语合作期待 |
| 不敢当 | Danke für das Kompliment | 谦让回应 → 德语谢辞 |
| 哪里哪里 | Danke sehr / Vielen Dank | 中文谦让 → 德语接受感谢 |
| 承蒙招待 | Vielen Dank für Ihre Gastfreundschaft | 中文谦辞 → 德语正式感谢 |
| 区区小事 | Gern geschehen / Keine Ursache | 自谦小事 → 德语淡化回应 |
| 恕我直言 | mit Verlaub / wenn ich offen sein darf | 谦辞缓冲语 → 德语正式缓冲 |
| 在下 | ich (keine Höflichkeitsform nötig) | 古代谦称 → 直接用 ich |
| 同志们 | liebe Kolleginnen und Kollegen | 政治称呼 → 性别双称，避免"Genossen" |

**核心规则**：
- **您/你强制转换**：中文商业/正式文件中的"您"全译为 `Sie`（大写），物主代词同步为 `Ihr/Ihre/Ihnen`；内部/非正式场景的"你"译为 `du`，物主代词为 `dein/deine/dir`
- **谦辞消除**：中文"拙/敝/贱/寒"等自我贬抑表达 → 德语中性表达（mein, unser），不保留自谦色彩
- **敬辞转换**：中文"贵/尊/令"等敬辞 → 德语 `Ihr`（正式）或直接使用相应名词（如"令尊"→ Ihr Vater）
- **职务称呼**：中文习惯"姓+职务简称"（张局、李总、王处）→ 德语需展开为完整职称，格式为 `Herr/Frau + [Vorname] + [Titel] + [Nachname]`
- **"老师"处理**：学术界 → Professor/Lehrer，企业/职场 → Herr/Frau，不可直译 Lehrer
- **自谦结构**：中文"让我抛砖引玉"→ 德语直述意图（Ich möchte eine kurze Einführung geben）

## 2. Numbers and Dates — 万/亿-Umrechnung, TT.MM.JJJJ

中文使用万（10⁴）、亿（10⁸）进制，与德语千（10³）进制完全不同。日期格式从"年月日"改为"日月年"。

| 中文原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| 一万 | 10.000 | 万（10⁴）→ 1.000的十倍 |
| 十万 | 100.000 | 十万 = 100千 |
| 一百万 | 1.000.000 | 百万一致 |
| 一千万 | 10.000.000 | 千万 = 10百万 |
| 一亿 | 100.000.000 | 亿（10⁸）→ 100 Millionen |
| 十亿 | 1.000.000.000 | 十亿 = 1 Milliarde |
| 一百亿 | 10.000.000.000 | 百亿 = 10 Milliarden |
| 一千亿 | 100.000.000.000 | 千亿 = 100 Milliarden |
| 一兆（万亿） | 1.000.000.000.000 | 兆（10¹²）= 1 Billion |
| 2024年3月15日 | 15. März 2024 / 15.03.2024 | 年月日 → 日月年 |
| 2024年3月 | März 2024 | 仅年月时无需转换日 |
| 3月15日 | 15. März | 仅月日格式 |
| 2024年第1季度 | Q1 2024 / 1. Quartal 2024 | 季度表达可直接使用 |
| 20世纪90年代 | die 1990er-Jahre | 年代表达转换 |
| 下午2点 | 14:00 Uhr | 12小时制 → 24小时制 |
| 上午9点半 | 09:30 Uhr | 上午/下午→24小时制 |
| 3.5公里 | 3,5 km | 小数点：中文`.`→德语`,` |
| 1,200人 | 1.200 Personen | 千位分隔：中文`,`→德语`.` |
| 3.14% | 3,14 % | 小数+百分号格式 |
| 第5页 | Seite 5 | 中文"第X"→ 德语无前缀，数字置后 |
| 第1条 | § 1 / Artikel 1 | 条款标记转换 |
| 5号楼 | Gebäude 5 | 编号后置 |
| 30℃ | 30 °C | 度符号前加空格 |
| 5% | 5 % | 百分号前加空格 |
| 约50人 | etwa 50 Personen / ca. 50 Personen | 约数表达 |
| 5000余人 | über 5.000 Personen / mehr als 5.000 | "余"→ über/mehr als |
| 约3.5万个 | ca. 35.000 | 万进制转换+约数表达 |
| 8亿人次 | 800 Millionen Personenzugänge | 亿人次 → Millionen Personenzugänge |

**核心规则**：
- **万进制 → 千进制**：中文 `万` = `10.000`，`亿` = `100.000.000` = `100 Millionen`
  - 1万=10千，10万=100千，1000万=10百万，1亿=100百万
  - 使用德语千位分组写法（`.`作为千位分隔）
- **日期系统性转换**：`YYYY年M月D日` → `DD.MM.YYYY`（数字格式）或 `D. Month YYYY`（全写格式）
- **时间强制24小时制**：上午/下午消失，全部以 `00:00`–`23:59` 表示；正式文本加 `Uhr`（14:00 Uhr）
- **小数与千位分隔对调**：`3.14` 变 `3,14`；`1,200` 变 `1.200`
- **单位与编号**："第X"习惯改为数字后置（Seite 5, Gebäude 7）；百分号/度符号前加空格

## 3. Cultural Concepts — Chinese Idioms → German Equivalents, China-Specific Terms

中文成语、俗语及中国特色文化概念不能直译，需找到德语功能对等表达或加注处理。

| 中文原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| 画蛇添足 | etwas Überflüssiges tun | 直译"die Schlange mit Füßen malen"无意义，用意译 |
| 对牛弹琴 | Perlen vor die Säue werfen | 德语固有对应谚语 |
| 一石二鸟 | zwei Fliegen mit einer Klappe schlagen | 德语完全对应成语 |
| 亡羊补牢 | Vorsicht ist besser als Nachsicht | 中文"亡羊补牢"→ 德语"预防胜于补救" |
| 塞翁失马 | Pech im Spiel, Glück in der Liebe / jedes Unglück hat sein Glück | 成语故事需注明文化背景 |
| 班门弄斧 | Eulen nach Athen tragen | 德语对应谚语 |
| 马马虎虎 | na ja / geht so / mittelmäßig | 口语化处理 |
| 功夫 | Kampfkunst / Können/Fähigkeit (nicht: harte Arbeit) | "功夫"多义，依语境选译 |
| 关系 | Beziehungen / Guanxi (mit Fußnote) | 复杂文化概念，首次出现需加注 |
| 面子 | Gesicht (wahren/verlieren) | 已成德语借词，可直接使用 |
| 人情 | persönliche Gefälligkeiten / Beziehungsnetz | 无直接对应，意译+加注 |
| 缘分 | Schicksal / Fügung | 文化概念需功能性对应 |
| 风水 | Feng Shui | 已成德语借词 |
| 阴阳 | Yin und Yang | 已成德语借词 |
| 中庸之道 | der Weg der Mitte / Mittelweg | 儒家概念需标准化译法 |
| 道 | Dao / Weg / Prinzip | 哲学概念保留拼音+加注 |
| 和谐 | Harmonie | 政治/哲学通用 |
| 已所不欲勿施于人 | Was du nicht willst, das man dir tut, das füg auch keinem anderen zu | 德语已有对应黄金法则 |
| 千里之行始于足下 | Eine Reise von tausend Meilen beginnt mit dem ersten Schritt | 标准德译 |
| 实事求是 | Tatsachen entsprechen / auf Tatsachen basiert | 政治+哲学双重含义 |
| 改革开放 | Reform- und Öffnungspolitik | 固定政治术语 |
| 中国特色社会主义 | Sozialismus chinesischer Prägung | 标准政治术语 |
| 中华民族伟大复兴 | die große Wiederbelebung der chinesischen Nation | 标准政治术语 |
| 撸起袖子加油干 | mit angepackten Ärmeln / mit vereinten Kräften | 政治口号 → 描述性翻译 |
| 小康社会 | Gesellschaft des bescheidenen Wohlstands | 标准政治经济术语 |
| 人类命运共同体 | Schicksalsgemeinschaft der Menschheit | 外交术语定译 |
| 黑马 | Überraschungssieger / Dark Horse | 借词可用 |
| 山寨 | Fälschung / Nachahmung / Billigkopie | 需根据语境选词 |
| 菜鸟 | Anfänger / Neuling | 口语化对应 |
| 土豪 | Neureicher / protziger Reicher | 概念对应而非字译 |

**核心规则**：
- **成语/谚语**：优先找德语固有对应谚语（Perlen vor die Säue = 对牛弹琴）；无直接对应时用意译 + 括号直译
- **中国特有概念**（关系、面子、功夫）：在德语语境中已有借词地位的保留借词，否则用描述性翻译 + 首次加注拼音
- **政治口号**：严格使用新华社/外交部标准德译，不自创译法
- **哲学概念**（道、阴阳、中庸）：首次使用保留拼音 + 短解释，后续全文统一
- **网络新词**（菜鸟、土豪、山寨）：尚无固定译法时采用描述性翻译，加括号说明中文原文

## 4. Terminology — Chinese Technical Terms → German, False Friends

中德术语体系存在大量不对应和"假朋友"（形同义异），需特别注意。

| 中文原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| 计算机 | Computer / Rechner | 两者皆可，"Rechner"更正式 |
| 软件 | Software (f.) | 德语阴性名词 |
| 硬件 | Hardware (f.) | 德语阴性名词 |
| 数据库 | Datenbank | 标准对应 |
| 网络 | Netzwerk | 标准对应 |
| 服务器 | Server | 标准借词 |
| 防火墙 | Firewall (f.) | 德语阴性 |
| 程序 / 应用程序 | Programm / Anwendung / App | 依语境选词 |
| 算法 | Algorithmus | 标准对应 |
| 大数据 | Big Data / Massendaten | 两种皆可 |
| 人工智能 | Künstliche Intelligenz (KI) | 标准术语+缩写 |
| 机器学习 | Maschinelles Lernen | 标准术语 |
| 深度学习 | Deep Learning / Tiefes Lernen | 两种皆可，需统一 |
| 云计算 | Cloud-Computing | 德语正写法：连字符+大写 |
| 接口 | Schnittstelle (优先) / Interface | 技术文档优先德语化 |
| 芯片 | Chip / Mikrochip | 统一用 Chip |
| 传感器 | Sensor | 标准对应 |
| 自动化 | Automatisierung | 标准对应 |
| 机器人 | Roboter | 标准对应 |
| 自动驾驶 | autonomes Fahren / automatisiertes Fahren | 技术层面区分 |
| 零配件 | Ersatzteile / Komponenten | 标准对应 |
| 合同 | Vertrag | 法律文本标准用词 |
| 甲方 / 乙方 | Partei A / Partei B | 合同当事人标记 |
| 法定代表人 | gesetzlicher Vertreter | 法律术语 |
| 营业执照 | Gewerbeanmeldung / Gewerbeschein | 注意德国无完全对应的"执照"概念 |
| 注册资本 | Stammkapital / Grundkapital | 公司类型不同用词不同（GmbH→Stammkapital, AG→Grundkapital） |
| 增值税 | Mehrwertsteuer (MwSt.) / Umsatzsteuer (USt.) | 需区分MwSt和USt |
| 企业所得税 | Körperschaftsteuer (KSt.) | 德国公司税 |
| 个人所得税 | Einkommensteuer (ESt.) | 德国个税 |
| 海关 | Zoll | 标准对应 |
| 检疫 | Quarantäne / Gesundheitskontrolle | 标准对应 |
| 备案 | Registrierung / Hinterlegung / Anmeldung | "备案"概念需根据语境选择 |
| 审批 | Genehmigung / Prüfung und Genehmigung | "审"+"批"在德语中可能只需Genehmigung |
| 验收 | Abnahme / Abnahmeprüfung | 工程技术术语 |
| 实施 | Umsetzung / Implementierung | 依语境 |
| 质量监督 | Qualitätsüberwachung | 标准对应 |
| 基础设施 | Infrastruktur | 标准对应 |
| 轨道交通 | Schienenverkehr | 标准对应 |
| 预算 | Budget / Haushalt | 企业/政府不同用词 |
| 累计 | kumuliert / insgesamt | 标准对应 |

**🔴 False Friends（假朋友）特别注意**：

| 中文词 | 相似德语词 | 实际含义 | 正确译法 |
|--------|-----------|----------|---------|
| 批评 | prüfen（prüfen = 检查/测试） | 批评 = kritisieren | 不可混淆 |
| 学校 | Schule（正确对应） | 大多数情况下一致 | 但"师范大学"≠Schule，用 Pädagogische Universität |
| 银行 | Bank（正确对应） | 一般一致 | 但"银行"某些语境指"河岸"，德语也用Bank |
| 信息 | Information（正确对应） | 基本一致 | 注意复数形式 Informationen |
| 保险 | Versicherung（正确对应） | 一致 | 但"社会保险"≠Sozialversicherung |
| 敏感 | sensibel | sensibel = 敏感的（情感上）；sensitiv = 敏感的（政治上） | 政治敏感 = politisch sensibel/sensitiv |
| 实际 | praktisch | praktisch = 实用的，并非"实际"= tatsächlich/wirklich | 混淆会导致误译 |
| 控制 | kontrollieren | 德语kontrollieren = 检查/监督；控制 = steuern/beherrschen | ❌ Kontrolle über etwas haben（可以） |
| 经理 | Manager（正确） | 大多数一致 | 但"部门经理"= Abteilungsleiter |
| 可能 | möglich | möglich = 可能的；können = 能够 | 语态差异 |
| 平均 | durchschnittlich | 一致 | 但注意词性：副词/形容词 |
| 生产 | produzieren | 一致 | 但"生产线"= Fertigungslinie |
| 分析 | Analyse / analysieren | 一致 | 注意德语重音在第二个音节 |
| 系统 | System | 一致 | 注意德语大写 |
| 组织 | Organisation / organisieren | 一致 | 注意"组织"在生物语境和机构语境用词不同 |
| 自然 | Natur（正确对应） | 一致 | "自然"做副词时=selbstverständlich/natürlich |

**核心规则**：
- 技术术语优先使用 **DIN 标准** 和 **ISO 标准德语版** 规定的术语
- 英语源术语在德语中已标准化的（Computer, Server, Chip）可保留，但需遵守德语正写法（首字母大写、确冠词性）
- "假朋友"词（与德语拼写相似但含义不同）必须逐一核查，不可凭中文直觉使用
- 法律会计术语（注册资本、增值税等）必须对应德国法律体系的具体概念，而非字面翻译
- "备案/审批/验收"等中国特色管理术语需转为德国实务中的对应流程概念
- 同一文件内术语须建表统一，用 Terminologie-Management 工具管理

## 5. Administrative and Legal Style — 行政公文转换, 根据→gemäß, 特此通知→hiermit

中文行政公文习惯使用四字套语、流水短句和主动语态；德语 Amtsdeutsch 偏好名词化、被动语态和长复合句。

| 中文原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| 根据《中华人民共和国对外贸易法》 | gemäß dem Außenhandelsgesetz der Volksrepublik China | 根据 → gemäß（法律引用） |
| 按照相关规定 | laut einschlägiger Bestimmungen / nach den einschlägigen Vorschriften | 按照 → laut/nach |
| 依照本法执行 | wird nach Maßgabe dieses Gesetzes durchgeführt | 依照 → nach Maßgabe von |
| 特此通知 | Hiermit wird bekannt gegeben / Hiermit teilen wir mit | 公文标准开头套语 |
| 特此公告 | Hiermit wird Folgendes bekannt gegeben | 公告开头 |
| 现就有关事项通知如下 | Hierzu wird Folgendes mitgeteilt: | 通知过渡语 |
| 经研究，决定如下 | Nach Prüfung wurde Folgendes beschlossen: | 经研究 → 被动结构 |
| 兹 | hiermit / in diesem Zusammenhang | "兹"为公文起首词 → hiermit |
| 鉴于 | in Anbetracht / angesichts / in Erwägung | 法律鉴于条款 |
| 为此 | zu diesem Zweck / hierzu | 公文衔接 |
| 请予以配合 | Wir bitten um Ihre Unterstützung / Mitwirkung | 请求语 |
| 请遵照执行 | Wir bitten um Beachtung und Umsetzung | 执行要求 |
| 自发布之日起施行 | Diese Regelung tritt am Tag ihrer Verkündung in Kraft | 生效条款 |
| 有效期至 | gültig bis zum | 有效期表达 |
| 构成犯罪的，依法追究刑事责任 | Strafbare Handlungen werden nach Maßgabe des Gesetzes verfolgt | 刑事条款 |
| 由XX部负责解释 | Die Auslegung dieser Verordnung obliegt dem Ministerium XX | 解释权条款 |
| 本办法 | diese Verordnung / diese Verwaltungsvorschrift | "办法"→ Verordnung/Vorschrift |
| 未尽事宜 | nicht geregelte Angelegenheiten | 兜底条款 |
| 与其他规定不一致时 | bei Widerspruch zu anderen Bestimmungen | 冲突条款 |
| 必须 | ist verpflichtet / muss / hat zu | 强制性规范 |
| 不得 | ist nicht zulässig / darf nicht / ist verboten | 禁止性规范 |
| 可以 | kann / ist berechtigt | 授权性规范 |
| 应当 | soll / hat ... zu | 义务性规范（弱于"必须"） |
| 由...负责 | liegt in der Zuständigkeit von / obliegt | 职责分配 |
| 参照执行 | sinngemäß anwenden / entsprechend anwenden | 参照适用 |
| 试行 | vorläufig / probeweise | 试验性法规 |
| 暂行 | vorläufig / einstweilig | 临时性法规 |
| 特此证明 | Hiermit wird bescheinigt | 证明类文书 |
| 特此批复 | Hiermit wird wie folgt verfügt / beschieden | 批复/决定 |
| 此致 | Mit freundlichen Grüßen | 信函结尾 |

**核心规则**：
- **根据/按照/依照**体系：
  - `根据/gemäß`：法律依据引用（gemäß dem Gesetz）
  - `按照/entsprechend, laut`：程序性引用（laut Verfahrensordnung）
  - `依照/nach Maßgabe von`：执行性引用（nach Maßgabe des Beschlusses）
- **主动→被动**：中文大量主动表达（"经研究决定"）→ 德语转为被动/无人称（"Nach Prüfung wurde beschlossen"）
- **套语转换**：中文四字公文套语（"特此通知""现就有关事项"）→ 德语固定公文套语（"Hiermit wird...""Hierzu wird Folgendes..."）
- **短句→长句**：中文流水短句 → 德语复合长句（Schachtelsatz），但需控制在可读范围内
- **情态动词体系**：中文"必须/应当/可以/不得" → 德语规范情态动词体系（muss/soll/kann/darf nicht），法律文本还可用"hat zu / ist verpflichtet"
- **法规层级**：中文"法/条例/办法/规定" → 德语需对应德国法规层级（Gesetz/Verordnung/Satzung/Richtlinie）
- **书名号处理**：中文《》法律名称 → 德语取消书名号，使用斜体或引号或直接文字表述
- **"有关"表达**："有关部门"→ die zuständigen Behörden，"有关事项"→ die betreffenden Angelegenheiten
- **"单位"处理**："事业单位/用人单位"→ 无法直接为 Einheit，需按组织类型具体化

## 6. Punctuation Conversion — 。→ ., 、→ ,, 「」→ „"

中文与德语的标点符号体系存在系统性差异，需逐项转换。

| 中文原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| 你好。 | Hallo. | 中文句号 → 德语句号（一致，但中文为全角，德语为半角） |
| 一、二、三 | eins, zwei, drei / erstens, zweitens, drittens | 中文顿号 → 德语逗号或 und/sowie/erstens... |
| 第一，第二，第三 | erstens, zweitens, drittens | 列举时改用 erstens...zweitens... |
| 他说："你好。" | Er sagte: „Guten Tag.“ | 双引号：中文 `""` → 德语 `„ “`（先低后高） |
| 他回答："是的。" | Er antwortete: „Ja.“ | 双引号转换 |
| '单引号示例' | ‚Beispiel mit einfachen Anführungszeichen‘ | 单引号：中文 `''` → 德语 `‚ ‘` |
| 「重要提示」 | „Wichtiger Hinweis“ | 日语式引号 → 德语标准引号 |
| 『引用中的引用』 | „Zitat ‚inneres Zitat‘ “ | 嵌套引号：外层„ "，内层‚ ' |
| 《中华人民共和国宪法》 | die Verfassung der Volksrepublik China | 书名号 → 文字描述或斜体 |
| 《红楼梦》 | „Der Traum der Roten Kammer“ | 书名号 → 德语引号或斜体表示作品 |
| 《人民日报》 | die Volkszeitung (Renmin Ribao) | 书名号 → 文字说明 |
| ——重要通知—— | – Wichtiger Hinweis – | 中文双字线（——）→ 德语半角短破折号（–），两侧空格 |
| 3～5天 | 3 bis 5 Tage / 3–5 Tage | 波浪连接号 → bis 或半角短线 |
| 10～15人 | 10 bis 15 Personen / 10–15 Personen | 范围表达转换 |
| …… | ... (drei Punkte) | 中文六点省略号 → 德语三点 |
| 列奥纳多·达·芬奇 | Leonardo da Vinci | 间隔号→空格或直接德语写法 |
| 托马斯·爱迪生 | Thomas Edison | 间隔号删除或改为空格 |
| 第一章 | Kapitel 1 | 章节标记：中文"第X"→ 德语词 + 数字 |
| 第1款 | Absatz 1 | 法律条款标记 |
| 第(一)项 | lit. a / Buchstabe a | 法律次级编号 |
| 五、六人 | fünf oder sechs Personen | 约数表达：中文顿号→oder/bis |
| （一） | (1) / 1. | 中文括号数字编号 → 德语数字编号 |
| 1. 引言 | 1 Einleitung / 1. Einleitung | 编号列表：德语句点后可加空格或不加 |
| ——（破折号作为强调） | – (Gedankenstrich als Hervorhebung) | 强调性破折号 → 德语用加空格短破折号 |
| ？！ | !? (ohne Kombination) | 中文可混用"？！"，德语避免，选其一 |
| ～ | ca. / etwa / ungefähr | 中文波浪号表示"大约" → ca. |
| 〃（同上号） | ders. / dies. / wiederholt | 中文重复号 → 德语用缩写或重复 |
| ※（米号/强调标记） | (ohne Entsprechung) → weglassen | 中文强调标记 → 德语删除或用其他强调手段 |

**核心规则**：
- **句号**：中文全角句号 `。` → 德语半角句号 `.`。德语句号也用在小数点（反之中文全角）和序数词后（1. = erstes）
- **顿号**：中文 `、` **无直接德语对应**，必须转换为：
  - 列举词 → 逗号 `,`（Äpfel, Birnen, Bananen）
  - 列举句 → `und / sowie / oder`
  - 编号列举 → `erstens, zweitens, drittens`
- **引号彻底转换**：
  - 中文双引号 `""`（位于右上角）→ 德语下上引号 `„ “`（„ 在基线， " 在上角）
  - 中文单引号 `''` → 德语单引号 `‚ ‘`
  - 中文「」（日语式引号）→ 德语 „ "
  - 嵌套引用：外层 „ "，内层 ‚ '
- **书名号**《》→ 德语中 **无书名号**，改为：斜体（*Titel*）、引号（„Titel"）、或用文字描述（der Roman/der Film + Titel）
- **波浪连接号** `~` → 德语 `bis`（3 bis 5 Tage）或短破折号 `3–5 Tage`
- **省略号**：中文六点 `……` → 德语三点 `...`（德语省略号始终为三个点，不加多）
- **间隔号** `·` → 外文人名中直接删除或转为空格；中文书名中可保留但罕见
- **双字线破折号** `——` → 德语 **Gedankenstrich** `–`（半角短破折号），两侧各加空格
- **括号**：中文全角括号 `（）` → 德语半角 `()`，或保留全角但统一风格
- **感叹号+问号**：中文可连用「？！」，德语通常只选其一（! 或 ?，不用组合形式）

## 附：中文→德语文体转换速查表

| 中文特征 | 德语特征 | 变化方向 |
|---------|---------|---------|
| 主动语态为主 | 被动语态常见 | 中文主动 → 德语被动（wird...gemacht） |
| 短句、流水句 | 复合句、从句嵌套 | 中文短句合并为德语从句结构 |
| 四字成语/套语 | 功能对等表达 | 成语 → 德语谚语或意译 |
| 主语省略频繁 | 主语必现 | 补充主语（es/man/wir） |
| 修饰语前置 | 修饰语可后置（从句） | 中文"的"结构 → 德语关系从句或复合词 |
| 名词无性数格 | 名词有性数格 | 确定冠词（der/die/das）和变格 |
| 动量词/名量词体系 | 无量词 | 删除量词（"一本书"→ ein Buch） |
| 谦辞/敬辞体系 | Ih-/中性表达 | 谦辞消除，敬辞转为 Ihr- |
