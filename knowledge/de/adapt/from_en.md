# 英→德 (en → de-DE) 适应规则

- **id**: de/adapt/from_en
- **type**: adapt
- **target_lang**: de

---

## 1. Formality and Address — You → Sie/du, Passive Voice, Titles

英语仅存一个 `you` 统摄所有场景，德语必须在 `Sie（正式）` 和 `du（非正式）` 之间强制二选一。此外，英语被动语态的用法与德语不同，头衔体系也有显著差异。

| 英语原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| You are invited to attend | Sie sind herzlich eingeladen | 正式邀请 → Sie（大写） |
| You should check this out | Du solltest dir das ansehen | 内部非正式 → du |
| You will receive a response | Sie erhalten eine Rückmeldung | B2B通信 → Sie |
| Your application has been approved | Ihr Antrag wurde genehmigt | 物主代词 Ihr |
| Thank you for your email | Vielen Dank für Ihre E-Mail | 正式邮件 |
| Please let us know | Bitte teilen Sie uns mit | 正式请求 |
| You can pay online | Sie können online bezahlen | 电商客服 → Sie |
| Dear Mr. Anderson | Sehr geehrter Herr Anderson | 正式信件抬头 |
| Dear Ms. Carter | Sehr geehrte Frau Carter | 德语统一用 Frau，不区分 Ms./Mrs. |
| Dear Dr. Miller | Sehr geehrter Herr Dr. Miller | 德语保留博士头衔 |
| Dear Sir or Madam | Sehr geehrte Damen und Herren | 无性别不确定时标准用法 |
| Dear Professor Schmidt | Sehr geehrter Herr Professor Schmidt | 教授头衔在德语中必须保留 |
| To whom it may concern | An die zuständige Stelle / Sehr geehrte Damen und Herren | 英语不确切称呼 → 德语两选一 |
| Hi John | Hallo Johann | 仅限于熟人/非正式 |
| Dear all | Liebe Kolleginnen und Kollegen | 集体称呼 → 性别双称 |
| Ladies and gentlemen | Sehr geehrte Damen und Herren | 正式演讲开场 |
| Mr. President | Herr Präsident | 头衔在人名前，不带冠词 |
| Your Honor | Euer Ehren / Hohes Gericht | 法庭称呼 |
| Your Excellency | Eure Exzellenz | 外交称呼 |

**被动语态对比**：

| 英语原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| It is believed that | Es wird angenommen, dass | 无人称被动 → 一致 |
| It has been decided that | Es wurde beschlossen, dass | 无人称被动 → 一致 |
| The report was submitted | Der Bericht wurde eingereicht | wurden + Partizip II |
| The contract is being signed | Der Vertrag wird gerade unterschrieben | werden + Partizip II |
| You are required to | Sie sind verpflichtet, ... / Sie müssen | 英语被动 → 德语可转为 müssen |
| It must be noted that | Es ist anzumerken, dass | sein + zu + Infinitiv |
| No smoking is permitted | Rauchen ist nicht gestattet | 禁止类被动 |
| It was agreed that | Es wurde vereinbart, dass | 会议纪要典型表达 |
| It is understood that | Es besteht Einigkeit darüber, dass | 法律/合同套语 |
| Mistakes were made | Es sind Fehler unterlaufen | 回避直接归责 |

**核心规则**：
- **You → 强制区分**：B2B、法律、政府文件 → 一律 `Sie`（且代词的物主、反身形式都大写：Ihr, Ihre, Ihnen, Sich）。B2C、社交媒体、面向年轻群体 → `du`（小写：dein, dich, dir）
- **Headings/称呼**：英语 `Dear` → 德语正式场合用 `Sehr geehrte/r`，不能用 `Liebe/r`（后者仅用于熟人）
- **Ms./Mrs./Miss** → 统一德语 `Frau`，德语不区分婚姻状况
- **职称处理**：英语 `President/CEO/Director` → 德语 `Präsident/Geschäftsführer/Direktor`，头衔在姓名后而非前
- **被动差异**：英德被动结构大致对应，但德语使用频率更高且多用无人称 `es` 引导。英语 `be + past participle` ↔ 德语 `werden + Partizip II`，某些场景用 `sein + zu` 表达被动含义
- **Sie/du 一致性**：全文/全段落保持统一，不可在 Sie 场景中混入 du 物主代词

## 2. Dates and Times — MM/DD/YYYY → DD.MM.YYYY, AM/PM → 24-Stunden-Uhr

英语（尤其美式）与德语的日期、时间、日历表达存在根本差异，必须系统性转换。

| 英语原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| March 15, 2024 | 15. März 2024 | 美式全写 → 德语全写（日.月 年） |
| 03/15/2024 | 15.03.2024 | 美式数字日期 → DD.MM.YYYY |
| 2024-03-15 | 15.03.2024 | ISO格式 → 德语日常格式 |
| March 2024 | März 2024 | 年月无日时格式一致 |
| Jan. 5, 2024 | 5. Januar 2024 | 月份缩写展开 |
| 9:30 AM | 09:30 Uhr | 12小时制上午 → 24小时制 |
| 9:30 PM | 21:30 Uhr | 12小时制下午 → 24小时制 |
| 12:00 PM (noon) | 12:00 Uhr (Mittag) | 注意12PM为中午 |
| 12:00 AM (midnight) | 00:00 Uhr (Mitternacht) | 注意12AM为午夜 |
| 3:00 PM EST | 15:00 Uhr (MEZ) | 时区转换 → MEZ/MESZ |
| 10:30 AM PDT | 19:30 Uhr MESZ | 美国时区转为欧洲时区 |
| 9 am – 5 pm | 09:00 – 17:00 Uhr | 工作时间范围表达 |
| Monday, March 15 | Montag, 15. März | 星期格式 |
| 5/3/2024 (ambiguous) | 3. Mai 2024 (or 5. März) | 英语歧义日期需上下文判断 |
| Last updated: 01/15/2024 | Letzte Aktualisierung: 15.01.2024 | 软件界面日期 |
| the 1990s | die 1990er-Jahre / die Neunziger | 年代表达 |
| the 21st century | das 21. Jahrhundert | 世纪表达 |
| Q1 2024 | Q1 2024 / 1. Quartal 2024 | 季度表达 |
| FY2024 (fiscal year) | Geschäftsjahr 2024 (GJ 2024) | 财年表达 |
| 2 weeks | 2 Wochen / 14 Tage | "14 Tage"在德语商业语境常见 |
| bi-weekly | alle zwei Wochen / zweimal wöchentlich | "bi-"歧义需消除 |
| semi-annual | halbjährlich | 半年度 |
| 5/5 through 5/10 | 5. bis 10. Mai | 日期范围 → bis |
| Monday through Friday | Montag bis Freitag | 范围表达 |
| 10:00-11:30 | 10:00 – 11:30 Uhr | 时间段表达 |
| Early 2024 | Anfang 2024 | 年初表达 |
| Mid-2024 | Mitte 2024 | 年中表达 |
| Late 2024 | Ende 2024 | 年末表达 |
| 2023/2024 season | Saison 2023/2024 | 跨年表达 |

**核心规则**：
- **美式日期 MM/DD/YYYY → 德语 DD.MM.YYYY**：这是最常出错的转换，必须严格执行
- **月份名称**：英语缩写（Jan./Feb./Mar.）→ 德语展开全写（Januar/Februar/März）或用数字格式
- **12小时制→24小时制**：AM/PM 完全消除，全部转为 `00:00`–`23:59`。正式文本加 `Uhr`（09:30 Uhr）
- **时区注意**：EST/EDT/PST → MEZ/MESZ，根据德国受众调整时区参考
- **歧义日期**：`03/04/2024` 在美式=3月4日，在英式=4月3日 → 在德语输出中必须明确为 `4. März 2024` 或 `3. April 2024`
- **年代表达**：`the 1990s` → `die 1990er-Jahre`（年份+er+Jahre）
- **序数词**：英语 `1st, 2nd, 3rd, 4th` → 德语 `1., 2., 3., 4.`（数字+句点）
- **财年/季度**：保留 FY/Q 前缀或转为德语 Geschäftsjahr/Quartal

## 3. Cultural Concepts — English Idioms → German Equivalents

英语成语、惯用语和文化典故需要使用德语的功能对等表达，而非字译。

| 英语原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| It's raining cats and dogs | Es regnet in Strömen / Es regnet Bindfäden | 德语有多个对应雨天谚语 |
| To kill two birds with one stone | Zwei Fliegen mit einer Klappe schlagen | 直接对应 |
| The ball is in your court | Jetzt liegt der Ball bei Ihnen | 体坛隐喻保留 |
| To hit the nail on the head | Den Nagel auf den Kopf treffen | 直接对应 |
| To burn the midnight oil | Bis spät in die Nacht arbeiten | 直译不可理解，用意译 |
| To break the ice | Das Eis brechen | 直接对应 |
| A piece of cake | Ein Kinderspiel / Ein Klacks | 德语对应表达 |
| To cost an arm and a leg | Einen Arm und ein Bein kosten | 直接对应 |
| To be on the same page | Auf derselben Wellenlänge sein | "在同一页"→"在同一波长" |
| To think outside the box | Über den Tellerrand hinausschauen | "盒子外思考"→"越过餐盘边缘看" |
| The elephant in the room | Der Elefant im Raum | 已进入德语，或：Das Offensichtliche aussprechen |
| To let the cat out of the bag | Die Katze aus dem Sack lassen | 直接对应 |
| To bite the bullet | In den sauren Apfel beißen | "咬子弹"→"咬酸苹果" |
| To add fuel to the fire | Öl ins Feuer gießen | 直接对应 |
| To be in the same boat | Im selben Boot sitzen | 直接对应 |
| To pull someone's leg | Jemanden auf den Arm nehmen | "拉腿"→"架在胳膊上" |
| Once in a blue moon | Alle Jubeljahre einmal / sehr selten | 德语用"每 jubilee 年一次" |
| To save something for a rainy day | Etwas für schlechte Zeiten zurücklegen | "雨天"→"困难时期" |
| To go the extra mile | Einen Schritt weitergehen / sich besonders anstrengen | 功能性意译 |
| The best of both worlds | Das Beste aus beiden Welten | 直接对应 |
| To be caught between a rock and a hard place | Zwischen zwei Stühlen sitzen / in der Zwickmühle sein | "岩石和硬地"→"两把椅子" |
| To make a long story short | Kurz gesagt / Um es kurz zu machen | 功能性对应 |
| To spill the beans | Die Katze aus dem Sack lassen | 与 let the cat out of the bag 相同德语对应 |
| To feel under the weather | Sich angeschlagen fühlen / nicht gut drauf sein | "天气下"→"感到不适" |
| To take it with a grain of salt | Etwas mit Vorsicht genießen | "一粒盐"→"谨慎对待" |
| Thanksgiving Thanksgiving | Erntedankfest / Thanksgiving (mit Fußnote) | 文化节日需解释 |
| Halloween | Halloween | 已在德语文化中普及 |
| Super Bowl | Super Bowl | 专有名词保留 |
| High school | weiterführende Schule / Sekundarstufe II | 教育制度不同，功能对应 |
| College | Hochschule / Studium | 注意德语 Hochschule 与 Universität 区别 |
| Undergraduate | Grundstudium / Bachelorstudium | 教育体系差异 |
| Graduate | Masterstudium / Postgraduiertenstudium | 教育体系差异 |
| Homecoming | Absolvententreffen / Alumni-Treffen | 美式校园文化 → 德语描述 |
| Baby shower | Babyparty | 已在德语文化中普及 |
| Potluck | Mitbringparty / Buffet beitragen | 美式社交 → 描述性翻译 |
| Pottery Barn / Home Depot | keine Entsprechung → beschreibend | 品牌名需加注说明 |

**核心规则**：
- **英语习语**：优先查找德语是否已有对应成语（Zwei Fliegen mit einer Klappe）；无对应时用意译，避免字面翻译造成困惑
- **文化特定概念**：美式/英式特有的教育、节日、制度概念 — 找德国功能等效物或用描述性翻译 + 括号注
- **借词处理**：英语借词在德语中已普及的（Halloween, Brainstorming, Meeting）可直接使用，但需遵守德语正写法
- **度量衡**：英制（mile, inch, pound, ounce, gallon）→ 全部转换为公制（km, cm, kg, g, l）
- **Soccer vs. Football**：soccer → Fußball，football → American Football（需加说明）
- **楼层计数**：US first floor = Erdgeschoss（EG），US second floor = 1. Stock（德国1楼实际是2层）
- **教育对照**：需要一张英美-德国教育阶段对照表，不可直译 grade/college/freshman 等

## 4. Terminology — English Technical → German (DIN Standards), Compound Noun Formation

英语技术术语进入德语需符合 DIN 标准、德国工业规范及德语构词法，特别是复合名词的构成规则。

**通用技术术语**：

| 英语原文 | 适应后（德语） | 来源/标准 |
|----------|----------------|-----------|
| Artificial Intelligence | Künstliche Intelligenz (KI) | DIN 标准术语 |
| Machine Learning | Maschinelles Lernen | 技术标准 |
| Deep Learning | Deep Learning / Tiefes Lernen | 两种皆可，需统一 |
| Natural Language Processing | Natürliche Sprachverarbeitung (NSV) | 计算机语言学 |
| Cloud Computing | Cloud-Computing | 德语正写法：连字+大写 |

| Firmware | Firmware (f.) | 德语阴性 |
| User Interface (UI) | Benutzeroberfläche (BO) | DIN 术语优先 |
| User Experience (UX) | Benutzererfahrung (UX) | 需统一 |
| Human-Machine Interface | Mensch-Maschine-Schnittstelle | DIN 标准 |
| Application Programming Interface | Anwendungsprogrammierschnittstelle (API) | 全称→缩写保留英文 |
| Database | Datenbank | DIN 标准 |
| Network | Netzwerk | DIN 标准 |

**DIN / ISO 工程技术术语**：

|----------|----------------|------|

| Technical Documentation | Technische Dokumentation | DIN EN 82079 |
| Standard Operating Procedure | Standardarbeitsanweisung (SAA) | DIN ISO |
| Bill of Materials | Stückliste | DIN 标准 |
| Computer-Aided Design | Rechnerunterstützte Konstruktion / CAD | DIN |
| Computer-Aided Manufacturing | Rechnerunterstützte Fertigung / CAM | DIN |
| Computer Numerical Control | Computerized Numerical Control / CNC | 保留英文缩写 |
| Overall Equipment Effectiveness | Gesamtanlageneffektivität (GAE / OEE) | DIN |

| Logistics | Logistik | DIN 标准 |
| Lean Production | Schlanke Produktion / Lean Production | 两种皆可 |
| Six Sigma | Six Sigma | 保留英文 |
| Just-in-Time | Just-in-time (JIT) | 保留英文写法 |

**复合名词构成规则（Komposita）**：

| 英语短语 | 德语复合词 | 构词规则 |
|---------|-----------|---------|
| quality management system | Qualitätsmanagementsystem | 多重复合：QM + System |
| risk assessment process | Risikobewertungsprozess | - |
| data protection regulation | Datenschutzverordnung | 去除连字符直接合并 |
| customer relationship management | Kundenbeziehungsmanagement / CRM | 连写或用缩写 |
| supply chain management | Lieferkettenmanagement | - |
| product lifecycle management | Produktlebenszyklusmanagement | 注意长度，过长时缩写 |
| human resources department | Personalabteilung | 非直译，德语习惯用法 |
| software development kit | Softwareentwicklungskit (SDK) | - |
| machine tool manufacturer | Werkzeugmaschinenhersteller | - |
| energy efficiency class | Energieeffizienzklasse | - |
| building automation system | Gebäudeautomationssystem | - |
| heating ventilation air conditioning | Heizungs-, Lüftungs- und Klimatechnik (HLK) | 列举式复合 |
| parts per million | Millionstel (ppm) | 单位表达 |
| research and development | Forschung und Entwicklung (F&E) | 保持 und |
| terms and conditions | Allgemeine Geschäftsbedingungen (AGB) | 固定术语 |

**核心规则**：
- **DIN/ISO 优先**：技术术语首先查 DIN 标准和 ISO 德语版，使用标准化术语，不自创
- **复合名词**：英语介词短语（quality of service）→ 德语复合名词（Dienstqualität / Quality of Service QoS）
  - 德语复合名词连写（Werkzeugmaschinenhersteller），不加空格或连字符（Cloud-Computing 例外）
  - 三词以上复合需考虑可读性，过长时使用缩写（QMS statt Qualitätsmanagementsystem）
  - 复合词性别由最后一个词决定（das Management → das Qualitätsmanagement）
- **英语借词**：在技术语境中广泛存在，保留时需确定：
  - 性别（der Server, die Software, das Interface）
  - 首字母大写（德语名词均大写）
  - 复合时加连字符（Cloud-Computing, Plug-in）
- **缩写管理**：英语缩写首次出现时须给出全称：`SOP (Standard Operating Procedure)` → 同时给出德语对应 `Standardarbeitsanweisung (SAA)`
- **领域敏感**：同一个英语词在不同领域的德语对应不同：
  - Interface → IT: Schnittstelle / 法律: Schnittstelle / UI设计: Benutzeroberfläche
  - Compliance → 金融: Compliance / 法律: Rechtskonformität / 技术: Normenkonformität
- **避免伪英语（Scheinanglizismen）**：
  - 德语自造的"英语"词（Handy = 手机, Bodybag = 邮差包, Public Viewing = 集体观赛）不可用于翻译
  - 真实语境中 Handy → Mobiltelefon，Public Viewing → Public Viewing（但需确认受众）

## 5. Administrative and Legal Style — Legal English → Amtssprache

英语法律/行政文本（Legal English）与德语公务语言（Amtssprache/Amtsdeutsch）在句式结构、术语和篇章风格上差异显著。

| 英语原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| This Agreement is entered into on this [date] | Diese Vereinbarung wird am [Datum] geschlossen | 合同开头标准句式 |
| This Agreement is made and entered into | Dieser Vertrag wird geschlossen und tritt in Kraft | 英语赘词（made and entered）简化 |
| Whereas the Parties intend to | In Erwägung nachstehender Gründe beabsichtigen die Parteien | "鉴于"条款缩写 |
| In witness whereof the Parties have executed | Zu Urkund dessen haben die Parteien diese Vereinbarung unterzeichnet | 合同结尾套语 |
| Now, therefore, the Parties agree as follows | Daher vereinbaren die Parteien Folgendes | 合同主文过渡 |
| The Company shall indemnify and hold harmless | Die Gesellschaft stellt die andere Partei von allen Ansprüchen frei | "indemnify + hold harmless"为赘词，合并 |
| The Lessee shall not assign or sublet | Der Mieter ist nicht berechtigt, den Mietvertrag abzutreten oder unterzuvermieten | shall not → ist nicht berechtigt |
| This Agreement shall be governed by | Diese Vereinbarung unterliegt dem Recht von / Es gilt das Recht der | 准据法条款 |
| Any dispute arising out of or in connection with | Alle Streitigkeiten aus oder im Zusammenhang mit | 仲裁条款 |
| Force Majeure | Höhere Gewalt | 固定法律术语 |
| Without prejudice to | Unbeschadet / Vorbehaltlich | 法律保留条款 |
| Subject to | Vorbehaltlich / Unter dem Vorbehalt | 条件前提 |
| Notwithstanding the foregoing | Ungeachtet des Vorstehenden / Hiervon abweichend | 即使有前述规定 |
| The Parties hereby agree as follows | Die Parteien vereinbaren hiermit Folgendes | hiermit可保留直译 |
| This Agreement constitutes the entire agreement | Diese Vereinbarung stellt die vollständige Vereinbarung dar | 完整协议条款 |
| In the event of termination | Im Falle der Kündigung / Bei Beendigung | 条件从句简化 |
| Default | Verzug / Vertragsverletzung / Leistungsstörung | 依上下文差异大 |
| Termination for convenience | Ordentliche Kündigung (ohne wichtigen Grund) | 德国法分类不同 |
| Termination for cause | Außerordentliche Kündigung (aus wichtigem Grund) | 德国法分类不同 |
| Liquidated damages | Vertragsstrafe / Pauschalierter Schadensersatz | 需区分两种概念 |
| Consequential damages | Mangelfolgeschaden / mittelbarer Schaden | 德国法独特分类 |
| Limitation of liability | Haftungsbeschränkung / Haftungsbegrenzung | 标准条款 |
| Intellectual Property (IP) | Geistiges Eigentum (GE) | 法律术语 |
| Know-how | Know-how / Fachwissen | 法律语境保留Know-how |
| Non-disclosure Agreement (NDA) | Vertraulichkeitsvereinbarung / Geheimhaltungsvereinbarung | 德语多用Vertraulichkeitsvereinbarung |
| Letter of Intent (LOI) | Absichtserklärung | 法律固定对应 |
| Memorandum of Understanding (MOU) | Absichtserklärung / Vereinbarung | 与LOI在德语常合并 |
| Due Diligence | Due Diligence / Sorgfaltsprüfung | 商业并购语境保留英文 |
| Best Endeavors / Reasonable Endeavors | Bestmögliche Bemühungen / Angemessene Bemühungen | 英国法特有，需解释 |
| Services | Dienstleistungen | 标准对应 |
| Deliverables | Leistungsgegenstände / Liefergegenstände | 合同交付物 |
| Warranty | Garantie / Gewährleistung | 德语法律严格区分两种 |

**行政用语转换**：

| 英语原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| Please find attached | Anbei erhalten Sie / Anbei senden wir Ihnen | 正式商务用语 |
| For your reference | Zu Ihrer Information / Für Ihre Unterlagen | 公文附注 |
| We hereby inform you that | Hiermit teilen wir Ihnen mit, dass | 官方通知开头 |
| Please do not hesitate to contact us | Bei Rückfragen stehen wir Ihnen gern zur Verfügung | 德语标准客套结尾 |
| We are writing to you regarding | Wir wenden uns an Sie bezüglich | 正式信函开头 |
| In response to your inquiry | In Beantwortung Ihrer Anfrage | 正式回复 |
| As per your request | Wie von Ihnen gewünscht / Gemäß Ihrer Bitte | 公文引述请求 |
| We acknowledge receipt of | Wir bestätigen den Erhalt | 回执确认 |
| Please be advised that | Wir möchten Sie darauf hinweisen, dass | 正式通知 |
| Effective immediately | Mit sofortiger Wirkung | 生效时间 |
| As of [date] | Mit Wirkung vom [Datum] / Ab dem [Datum] | 截止/起始日期 |
| On a without prejudice basis | Vorbehaltlich / Unbeschadet | 法律保留 |
| Hereinafter referred to as | Nachfolgend genannt / Im Folgenden als ... bezeichnet | 定义引用 |

**核心规则**：
- **Shall 转换**：英语 `shall` 是法律文本中最复杂的情态动词，在德语中无直接对应：
  - 义务/要求 → `muss / hat ... zu / ist verpflichtet`
  - 禁止 → `darf nicht / ist nicht zulässig / ist verboten`
  - 授权 → `kann / ist berechtigt / darf`
  - 宣示性 → 现在时（tritt in Kraft / gilt）
- **名词化倾向**：英语动词表达 → 德语名词化（comply with → unter Einhaltung von; terminate → Kündigung; notify → Mitteilung）
- **句法结构**：英语右分支从句（right-branching）→ 德语左分支结构（left-branching），动词置于句末。这要求在翻译时重组大段英文从句
- **赘词消除**：英语法律文本的固定赘词（made and entered into, indemnify and hold harmless, null and void）→ 德语用一个词表达即可（geschlossen, freistellen, nichtig）
- **法律术语精确性**：英语法律术语（Termination for cause, Default, Liquidated damages）需映射到德国法律体系的特定概念，不可简单字译。必要时使用德国 BGB（民法典）和 HGB（商法典）对应概念
- **条款引用格式**：`Section 3.1(a)` → `§ 3 Abs. 1 lit. a`（德国引用标准）
- **被动语态**：英语法律文本被动 → 德语同样常用被动，但德语还可用 `sein + zu`、`sich lassen` 等替代形式

## 6. Punctuation Conversion — "" → „", ' → ‚', Comma Rules

英语与德语的标点符号在引号、撇号、逗号规则和空格习惯上存在重要差异。

| 英语原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| He said, "Hello." | Er sagte: „Hallo.“ | 双引号：英语 `""` → 德语 `„ “`（先下后上） |
| She asked, "How are you?" | Sie fragte: „Wie geht es Ihnen?“ | 问号在引号内规则一致 |
| 'Single quote example' | ‚Beispiel mit einfachen Anführungszeichen‘ | 单引号：英语 `''` → 德语 `‚ ‘` |
| He said, "She said, 'Hi.'" | Er sagte: „Sie sagte: ‚Hallo.‘“ | 嵌套：外层„ "，内层‚ ' |
| "Quote within 'single quote'" | „Anführungszeichen ‚innen‘ wiedergeben“ | 嵌套引号规则 |
| It's a beautiful day. | Es ist ein schöner Tag. | 英语撇号缩写 → 德语展开全写 |
| Don't / Can't / Won't | nicht / kann nicht / wird nicht | 英语缩写展开 |
| I'm / You're / We're | ich bin / du bist / wir sind | 人称缩写展开 |
| It'll / They'll | es wird / sie werden | 未来缩写展开 |
| rock 'n' roll | Rock ’n‘ Roll | 撇号方向调整 |
| O'Connor | O’Connor | 爱尔兰姓名撇号可保留 |
| 'Twas the night | Es war die Nacht | 诗歌缩写展开 |
| 9 o'clock | 9 Uhr | 缩写o'clock → Uhr |
| The 1990s | die 1990er-Jahre | 年代缩写 → 德语完整表达 |
| e.g. | z. B. (zum Beispiel) | 拉丁缩写替换 |
| i.e. | d. h. (das heißt) | 拉丁缩写替换 |
| etc. | usw. (und so weiter) | 拉丁缩写替换 |
| vs. | vs. / gegen / gegenüber | 或保留 vs. |
| NB / N.B. | Anm. / Hinweis | 注意标记替换 |
| P.S. | PS (ohne Punkte) | 附言德语写法无点 |
| PhD | Dr. (Promotion) | 缩写在德语不同 |
| & (ampersand) | und / sowie | 仅公司名中保留（AT&T） |
| @ (at sign) | @ (at) / bei | 邮件地址保留，正文转换 |
| # (hash/number) | Nr. / Nummer | 标签符号在德语非正式 |
| 1st, 2nd, 3rd | 1., 2., 3. | 序数词转换：数字+句点 |
| 4th | 4. | 同上 |
| Page 5 | Seite 5 | 一致 |
| The comma, however, is different | Das Komma jedoch ist anders | 逗号规则差异 |

**逗号规则差异详解**：

| 英语规则 | 德语规则 | 示例 |
|---------|---------|------|
| 限制性定语从句前无逗号 | 关系从句前必须有逗号 | The man who came → Der Mann, der kam, |
| 非限制性定语从句前有逗号 | 关系从句前必须有逗号 | Mr. Smith, who... → Herr Smith, der... |
| and/or 列举时最后的逗号可选（Oxford comma） | 德语无 Oxford comma，und/sowie/oder 前无逗号 | A, B and C → A, B und C |
| 不定式短语前可加逗号或不加 | 扩展不定式前必须加逗号 | um...zu, ohne...zu 前必加逗号 |
| 主从句间逗号较随意 | 从句前置时主句前必加逗号 | Wenn ..., dann ... |
| 插入语前后逗号 | 插入语前后逗号一致 | however → jedoch 两侧逗号 |
| 日期中逗号（March 15, 2024） | 日期中无逗号（15. März 2024） | 删除逗号 |
| 千位分隔使用逗号 | 千位分隔使用句点，无逗号 | 1,000 → 1.000 |
| 小数使用句点 | 小数使用逗号 | 3.14 → 3,14 |

**破折号与连字符**：

| 英语原文 | 适应后（德语） | 说明 |
|----------|----------------|------|
| — (em dash, no spaces) | – (Gedankenstrich, mit Leerzeichen) | 英语长破折号 → 德语短破折号，两侧空格 |
| – (en dash for ranges) | – (behalten) / bis | 范围破折号可保留或用bis |
| hyphen (compound modifier) | - (behalten bei Komposita) | 复合词修饰语连字符一致 |
| `-` vs `–` vs `—` confusion | `–` Gedankenstrich, `-` Bindestrich | 德语明确区分三种横线 |
| word–word (en dash) | Wort–Wort (behalten) | 对等连接保留 |

**其他标点差异**：

| 英语习惯 | 德语习惯 | 示例 |
|---------|---------|------|
| 引号内保留句号 | 引号外保留句号（Duden规则允许两种） | Er sagte: „Guten Tag“. |
| all' (意大利语冠词省略) | all' (可保留) | 外语借词中撇号可保留 |
| 印刷品引用用斜体 | 印刷品引用用„ "或斜体 | „Faust“ oder *Faust* |
| 百分比 5% | 5 %（空格） | 百分号前空格 |
| 温度 30°C | 30 °C（空格） | 度符号前空格 |
| 货币 US$1,234.56 | 1.234,56 USD | 货币符号位置 + 数字格式 |
| 段落缩进 | 段落缩进或无缩进 | 一致 |
| 冠词在文章标题大写 | 仅首词和名词大写 | The Road Not Taken → Der nicht genommene Weg |
| 标题：每个实词大写 | 标题：仅首词和名词大写 | 标题大小写规则不同 |

**核心规则**：
- **引号互换**：英语起始 `"` → 德语起始 `„`（低位9形）；英语结束 `"` → 德语结束 `"`（高位6形）。嵌套英语 `''` → 德语 `‚ '`
- **缩写展开**：英语有大量缩写（don't, I'm, it'll）→ 德语 **必须展开** 为完整形式（nicht, ich bin, es wird）。德语自身极少使用撇号表示省略，仅在方言或诗歌中使用
- **拉丁缩写替换**：`e.g.` → `z. B.`；`i.e.` → `d. h.`；`etc.` → `usw.`；`NB` → `Anm.`；`vs.` → 保留 or `gegenüber`
- **逗号规则**：德语关系从句的前后 **必须加逗号**；列举时 `und/sowie/oder` 前 **不加逗号**（无 Oxford comma）；扩展不定式前须加逗号
- **序数词**：英语 `1st/2nd/3rd` → 德语 `1./2./3.`（数字后直接加句点）
- **所有格**：英语 `'s` → 德语属格（Genitiv: `des Kunden`）或 `von + Dativ`，视正式程度选择
- **破折号**：英语用 `—`（长破折号，无空格）→ 德语用 `–`（短破折号，两侧各加一个空格）
- **百分号/度符号**：德语须在 % 和 ° 前加空格（5 % 而非 5%，30 °C 而非 30°C）
- **空格转换**：英语在标点后（如冒号、分号后）的空格习惯与德语一致，但德语在括号/引号前的空格需注意 Duden 规范

## 附：英→德总体文体转换参考

| 英语特征 | 德语特征 | 转换策略 |
|---------|---------|---------|
| 主动语态占主导 | 被动语态更常用 | 适当转为 werden + Partizip II |
| 短句、直接 | 长复合句、从句结构 | 合并短句为 Nebensatz 结构 |
| 情态动词（can/may/must） | 情态动词 + 替代结构 | können/dürfen/müssen + 名词化 |
| -ing 分词短语 | 关系从句或不定式 | Converting → der/die/das umwandelt |
| 后置定语（the man involved） | 前置定语（der beteiligte Mann） | 后置定语前移 |
| 介词短语链 | 复合名词 | the quality of service → Dienstqualität |
| 节奏紧凑、中性 | 正式化、名词化倾向 | 增加名词化表达 |
| 文化负载习语 | 功能性对等习语或意译 | 查德语对应谚语 |
| 广告/营销中的英语术语 | 德语化或保持借词+首字母大写 | 按受众决定 |
| 名词大小写：专有名词大写 | 所有名词大写 | 系统性大写转换 |
