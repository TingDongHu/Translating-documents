# Domain: Financial (AR) — Arabic Financial Documentation Translation

## Reader Model

**Primary Reader:** Arabic-speaking accountant, auditor, CFO, financial analyst, or banking professional in Saudi Arabia, UAE, Egypt, Jordan, Kuwait, or Lebanon.
**Secondary Readers:** Tax authority auditors (ZATCA, FTA, Egyptian Tax Authority); central bank regulators; IFRS compliance reviewers.
**Reader Expectations:**
- Demands **perfect numerical accuracy** — a single mistranslated digit is catastrophic (zero-tolerance).
- Expects **standard accounting terminology** aligned with IFRS Arabic translations and local GAAP.
- Requires **dual presentation** (Arabic numbers for text, Western/Arabic digits for figures depending on country).
- Sensitive to **currency code precision** — ر.س (SAR) vs د.إ (AED) vs ج.م (EGP) — never misused.
- Expects financial statements in **vertical (عمودي)** format, not horizontal.

### Framework 3: Amount in Words (Arabic) Rules

| Rule | Example |
|---|---|
| Use فقط لا غير at end | ألف ريال سعودي فقط لا غير |
| 1–10 feminine for currency | 3 ريالات / 5 جنيهات / 7 دراهم |
| Compound 11–19: first part feminine | 11 ريالاً / 12 جنيهاً / 13 درهماً |
| 20–99: use plural after | 25 ريالاً / 100 جنيه |
| 100 and 1000: masculine form | مائة جنيه / ألف ريال |
| ألف followed by singular | ألف ريال (not ألف ريالات) |
| Million: مليون followed by singular | مليون ريال |
| Billion: مليار | مليار ريال (not بليون in Arabic) |
| Piaster/subunit: mention after main | ١٠٠.٥٠ ج.م = مائة جنيه وخمسون قرشاً فقط لا غير |
| Fils (Gulf): هللة / فلس | ريال واحد وخمس هللات |
| Decimals in words | فاصلة / وأجزاء من المائة |
| Zero amounts | صفر ريال / لا شيء |
| Correction of amount | Amount in words overrides figures in case of discrepancy |

## Standard Conventions

### 1. Arabic Financial Document Format

1. **Layout:** Financial statements presented vertically (قوائم عمودية). Column layout: description right, current year left, prior year further left.
2. **Currency symbol:** After amount with space. ١٠٠,٠٠٠ ر.س (KSA), 100,000 ج.م (Egypt), 100,000 د.إ (UAE).
3. **Negative numbers:** Parentheses (١٠٠,٠٠٠) or minus sign (−). Parentheses preferred in Arabic accounting.
4. **Rounding:** Round to nearest whole unit for presentation. Note: تم تقريب الأرقام إلى أقرب ألف / ريال.
5. **Comparative figures:** Current year column first, prior year column second. Column headers: 2025 / 2024.
6. **Notes cross-reference:** (1), (2), (3) — notes appear at end. Referenced in Arabic: (انظر الإيضاح رقم 1).
7. **Date of authorization:** تاريخ الإذن بإصدار القوائم المالية: ___/___/____
8. **Board of Directors sign-off:** توقيع مجلس الإدارة — typically Chairman + CFO + Audit Committee Chair.
9. **Auditor's report page:** Separate page, placed before financial statements or after, per local custom.
10. **Arabic (IFRS) official translations:** Use SOCPA-published IFRS Arabic translations for KSA; Egyptian Accounting Standards for Egypt.

### 2. Amount in Words — Complete Conjugation Rules

1. **1-2:** واحد (masc) / واحدة (fem) → ريال واحد / ريالان
2. **3-10:** Opposite gender of counted noun → ثلاثة ريالات / ثلاث نساء
3. **11-19:** First part follows gender of noun → أحد عشر ريالاً / اثنتا عشرة جنيهاً
4. **20-90:** Same for both genders → عشرون ريالاً / ثلاثون جنيهاً
5. **100:** مائة → مائة ريال
6. **200:** مائتا → مائتا ريال (or مائتين in some dialects, but مائتا in fus-ha is standard in formal docs)
7. **1000:** ألف → ألف ريال (not ألف ريالات)
8. **2000:** ألفا → ألفا ريال
9. **Compound:** 1,234 → ألف ومائتان وأربعة وثلاثون ريالاً
10. **Large numbers:** 1,000,000 → مليون; 1,000,000,000 → مليار (not بليون in Arabic financial usage)
11. **Subunits:** Connect with و (and): مائة ريال وخمسون هللة
12. **Closing:** فقط لا غير (only, no more) — mandatory at end of amount in words in formal Arabic financial documents

**Examples:**
- SAR 1,250.75: ألف ومائتان وخمسون ريالاً وخمس وسبعون هللة فقط لا غير
- EGP 500,000.00: خمسمائة ألف جنيه فقط لا غير
- AED 1,000,000.00: مليون درهم فقط لا غير
- KWD 10,500.500: عشرة آلاف وخمسمائة دينار وخمسمائة فلس فقط لا غير
