# Domain: Technical (AR) — Arabic Technical Documentation

## Reader Model

**Primary Reader:** Arabic-speaking engineer, technician, or technical translator in Saudi Arabia, UAE, Egypt, Jordan, or Iraq.
**Secondary Readers:** Non-Arabic-speaking engineers reading Arabic-translated specs; SASO/ESO standardization review boards.
**Reader Expectations:**
- Prefers **short, active-voice sentences** in Arabic technical prose.
- Expects **consistent loanword transliteration** (e.g., فولط for volt, أمبير for ampere — never mixed with French/English spellings).
- Tolerates **anglicized technical terms** only when no Arabic standard exists; otherwise expects موحّد المصطلحات (unified terminology) per SASO or Arabization bureau standards.
- Requires **unit symbols in Latin script** (m, kg, s, A, V, W, Hz) — never translated into Arabic letters — but the unit name spelled out in Arabic in running text.
- Expects: **SI/metric exclusively**; imperial conversions shown in parentheses only when the source document is American.

## Decision Framework

### Framework 1: Unit Conversion & Measurement Style

| Condition | Decision | Example |
|---|---|---|
| Source uses imperial (°F, psi, ft, lb) | Convert to SI; add imperial in parentheses once | °C (°F) at first occurrence only |
| Source uses US Customary volumes (gal, bbl) | Convert to m³ or L | 100 gal → 378.5 L |
| Source uses kip, ksi | Convert to kN, MPa | 50 ksi → 345 MPa |
| Source uses HP (horsepower) | Convert to kW | 100 HP → 74.6 kW |
| Temperature in running text | Format: "درجة حرارة 40° مئوية" with space before °C | 40 °C |
| Pressure | Pa / kPa / MPa; بار (bar) acceptable in Arab oil & gas | 200 kPa = 2 بار |
| Pipe diameter | DN (Diameter Nominal) preferred: قطر 100 مم (DN 100) | DN 100 |
| Torque | N·m (نيوتن.متر) | 50 N·m |
| Rotational speed | rpm (دورة/دقيقة) write as "لفة/دقيقة" | 3000 لفة/دقيقة |
| Electrical power | kW / MW / W | 5 ميجاواط (5 MW) |
| Energy | kWh (كيلوواط ساعي) | 500 كيلوواط ساعي |
| Frequency | Hz (هرتز) | 50 هرتز / 60 هرتز |
| Sound level | dB(A) (ديسيبل (أ)) | 85 ديسيبل (أ) |

## Standard Conventions

### 1. Arabic Technical Writing Style Guide

1. **Sentence structure:** Prefer SVO (subject-verb-object). يحدد المعيار متطلبات الأداء before يتم تحديد متطلبات الأداء في المعيار.
2. **Verb tense:** Present tense for specifications (يجب / ينص / يحدد). Past tense for completed tests (تم اختبار / تم قياس).
3. **Plural of non-human nouns:** Feminine singular adjective agreement. الأجهزة الحديثة not الأجهزة الحديثون.
4. **Definiteness:** Technical terms are usually definite: الجهد الكهربائي, المقاومة النوعية.
5. **Idafa construction:** Limit to 2–3 terms in a chain. تجاوز درجة حرارة التشغيل القصوى (3 terms acceptable). For longer chains, use prepositions: متطلبات الأداء للنظام الهيدروليكي.
6. **Preposition choice:** بـ for instrument (يقاس بـ), لـ for purpose (صمام لتنظيم الضغط), من for material (مصنوع من الفولاذ), في for location (الضغط في النظام).
7. **Relative clauses:** Use الذي / التي / اللذان / اللاتي. Keep relative pronouns adjacent to antecedent in technical writing — ambiguity is unacceptable.
8. **Connectors:** بينما (whereas), حيث (where/in which), وذلك (namely), كما (as/likewise), بحيث (such that), بالتالي (therefore), بالإضافة إلى (in addition to).
9. **Negation:** لا (does not) for present tense, لم (did not) for past. لا يعمل الجهاز if malfunctioning, لم يعمل الجهاز if it failed to operate.
10. **Conditions:** إذا (if, likely) for real conditions, لو (if, unlikely) for hypothetical. إذا تجاوز الضغط الحد المسموح به for real condition. لو تعطل النظام for hypothetical.
11. **Exception:** باستثناء / ما عدا / عدا / إلا (except). ما علا ذلك (otherwise).
12. **Causality:** بسبب / نظراً لـ / نتيجة لـ (due to / because of). نظراً لارتفاع درجة الحرارة.
13. **Purpose:** بهدف / بغرض / وذلك لـ (in order to). بهدف ضمان السلامة.
14. **Sequence:** أولاً / ثانياً / ثالثاً (first / second / third — written أ / ب / ت when enumerating items).
15. **Summary:** خلاصة / باختصار / إجمالاً (in summary). إجمالاً, تستوفي المواد جميع المتطلبات.

### 2. Metrology & Measurement Conventions

1. SI base units are always in Latin script: m, kg, s, A, K, mol, cd.
2. Prefixes: k (كيلو), M (ميجا), G (جيجا), m (ملي), μ (ميكرو), n (نانو), p (بيكو). Arabic expansion optional on first use.
3. Compound units: N·m, m/s², kg/m³ — use middle dot (·) or slash as in source.
4. Uncertainty format: القيمة ± عدم اليقين (25.0 ± 0.1) mm.
5. Calibration interval: فترة المعايرة (e.g., 12 شهراً / سنوياً).
6. Traceability: إمكانية الإرجاع (Traceability to SI / NIST / PTB).
7. Accuracy vs Precision: الدقة (accuracy — closeness to true value) vs الضبط (precision — repeatability). Never conflate them.
8. Resolution: أقل تدريج / قابلية القراءة (resolution of instrument).
9. Range: مجال القياس (من ... إلى ...). e.g., مجال القياس: من 0 إلى 100 كيلوباسكال.
10. Sensitivity: الحساسية (output per unit input).
11. Drift: الانجراف (drift over time/temperature).
12. Hysteresis: التخلفية (hysteresis).
13. Linearity: الخطية (linearity error).
14. Repeatability: التكرارية (repeatability under same conditions).
15. Reproducibility: الاستنساخية (reproducibility under different conditions).

### 3. SASO/ESO/ADI Documentation Protocols

1. **SASO standard citation format:** SASO ISO XXX:YYYY or SASO XXX:YYYY. Always include year.
2. **Conformity mark:** علامة الجودة السعودية / علامة المطابقة الخليجية.
3. **Halal certification:** شهادة حلال (SASO/GSO requirement for food/chemical exports).
4. **Energy efficiency:** بطاقة كفاءة الطاقة (SASO 2663 for AC units, SASO 2870 for appliances).
5. **Low voltage:** SASO 2404 / IEC 60364 series for electrical installations.
6. **Plumbing:** SASO 1060 / SASO 765 for water fixtures.
7. **Building code:** الكود السعودي للبناء (SBC) — mandatory reference in KSA construction docs.
8. **Fire safety:** متطلبات الدفاع المدني السعودي / UAE Civil Defense.
9. **Environmental:** SASO 2995 / SASO 3000 for environmental management.
10. **ESO (Egyptian):** المواصفات القياسية المصرية — EOS publishes ANSI-aligned standards with Arabic introductions.
11. **ADI/ADIM (Arab Industrial Development):** المواصفات العربية — pan-Arab standards, often referenced for inter-Arab trade.
12. **GCC mark:** علامة الجودة الخليجية — mandatory for certain products traded in Gulf states.
13. **Emirates Conformity Assessment Scheme (ECAS):** mandatory for UAE market.
14. **Iraqi Central Organization for Standardization:** COSQC standards — often reference ISO/ASTM with Arabic translation.
15. **Metrology law:** قانون القياس والمعايرة (Saudi Metrology Law / Egyptian Law No. 48 for Metrology).

### 4. Typographic and Formatting Standards

1. **Headings:** Hierarchical numbering: 1 / 1.1 / 1.1.1 (Arabic numerals for levels), or Arabic alphabet for appendices.
2. **Numbered clauses:** Use parentheses Arabic style: (1) (2) (3) or 1- 2- 3-
3. **Block diagrams / flowcharts:** Labels in Arabic; text left-to-right only for equations/English terms.
4. **Chemical formulas:** Left-to-right within RTL paragraph — use Unicode LTR mark (U+200E) if needed.
5. **Equations:** Centered, equation number right-aligned in parentheses. Symbols remain Latin.
6. **Table direction:** RTL. Column headers on the rightmost column. If English terms are in the table, use dual-direction support.
7. **Cross-referencing:** انظر القسم (see section), انظر الجدول (see table), انظر الشكل (see figure).
8. **Figure captions:** Above figure in Arabic (not below as in English). الشكل 1: رسم توضيحي للنظام.
9. **Table captions:** Above table in Arabic. الجدول 1: الخصائص التقنية.
10. **Page numbering:** Arabic numerals (1, 2, 3) throughout. Preface may use ت-ث-ج (Arabic letter numbering).
11. **Margins:** Right margin wider (3–4 cm) for binding in Arabic scripts (right-bound book).
12. **Headers and footers:** Company logo right-aligned, document title left-aligned in header. Footer: page number center.
