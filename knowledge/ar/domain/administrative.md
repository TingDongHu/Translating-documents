# Domain: Administrative (AR) — Arabic Official & Administrative Documentation

## Reader Model

**Primary Reader:** Arabic-speaking civil servant, government clerk, administrative officer, minister's office staff, or public sector translator in Saudi Arabia, Egypt, UAE, Jordan, or Iraq.
**Secondary Readers:** Consular officials legalizing documents; international organizations (UN, World Bank, Arab League) reviewing Arabic administrative correspondence.
**Reader Expectations:**
- Demands **protocol-perfect addressing** (المخاطبات الرسمية) — honorifics, titles, and chains of command strictly observed.
- Expects **formulaic bureaucratic phrasing** that follows established patterns — creativity is NOT desired.
- Requires **correct seal/stamp protocol**:
  - Official round seal: ختم رسمي (دائري) — must include是国家 emblem + ministry/authority name.
  - Square/rectangle incoming stamp: ختم الوارد containing date, number, classification.
- Highly sensitive to **formatting hierarchy**: order of senders, carbon copy (CC) protocol, enclosure notation.
- Expects **document numbering** conventions specific to each issuing body.

### Framework 2: Official Letter Structure (الكتاب الرسمي)

| Element | Position in Document | Arabic Content |
|---|---|---|
| 1 | National Emblem (top center) | شعار الدولة (eagle / palm / falcon) |
| 2 | Authority name (below emblem) | اسم الجهة / الوزارة / الهيئة |
| 3 | Document type header | خطاب رسمي / قرار / تعميم |
| 4 | Reference number (top right) | الرقم: ______ / رقم الخطاب: ______ |
| 5 | Date (top right, below ref) | التاريخ: ___/___/____ (هـ / م) |

| 9 | Opening | السلام عليكم ورحمة الله وبركاته، وبعد / تحية طيبة وبعد |
| 10 | Body (right-aligned paragraphs) | إشارة إلى / بناءً على / نود إفادتكم ... |

**Handling instructions (تعليمات التداول):**
- For official use only: لاستعمال رسمي فقط
- Not to be reproduced: لا يصور / لا ينسخ
- To be returned after reading: يعاد بعد الاطلاع
- To be destroyed after reading: يتلف بعد الاطلاع
- Keep in safe custody: يحفظ في مكان آمن
- Access by named individuals only: لا يطلع عليه إلا المذكورون
- Deliver in sealed envelope: يسلم في ظرف مغلق

**Stamp types:**
1. **Round seal (الختم الدائري):** Official seal of ministry/authority. Contains national emblem + name. Used on all official correspondence.
2. **Incoming stamp (ختم الوارد):** Rectangle (6×4 cm). Contains: رقم الوارد, التاريخ, عدد المرفقات, التصنيف.
3. **Outgoing stamp (ختم الصادر):** Rectangle. Contains: رقم الصادر, التاريخ.
4. **Classification stamp (ختم التصنيف):** Red ink. سري / سري للغاية / رسمي.
5. **Registration stamp (ختم التسجيل):** For registering documents in the log.
6. **Urgency stamp (ختم العاجل):** Red ink. مستعجل / عاجل / عاجل جداً.

**Signature protocol:**
- Signature must cross over the edge of the seal (يمتد التوقيع على جزء من الختم والحافة).
- Initials (الأحرف الأولى) on each page of multi-page documents.
- Correction initials at margin for any amendments.
- Signature must be by authorized person (مخوّل بالتوقيع).

## Standard Conventions

### 1. Official Letter Format — Complete Standard

**Paper specifications:**
- Size: A4 (210 × 297 mm)
- Weight: 80–100 gsm (official letterhead 100 gsm minimum)
- Colour: White (cream for heritage/ceremonial)
- Security features: Watermark, microprinting (on high-security documents)

**Margins:**
- Right: 3.5 cm (binding edge for RTL)
- Left: 2.5 cm
- Top: 3.0 cm
- Bottom: 2.5 cm

**Font:**
- Body: Simplified Arabic 14 pt (KSA/UAE) or Traditional Arabic 14 pt (Egypt)
- Headings: Same font, 16 pt bold for subject line
- Footer/CC: 12 pt
- Alignment: Right-aligned (RTL)

**Document numbering placement:**
- KSA: الرقم: ___ وتاريخ: ___ هـ الموافق: ___ م (top right)
- Egypt: الرقم: ___ في ___ / ___ / ____ (top right)
- UAE: الرقم المرجعي: ___ التاريخ: ___ (top right)

**Structure sequence:**
1. National emblem (top center)
2. Authority name (center below emblem)
3. Document type or blank line
4. Reference no. + date + attachments (right-aligned, same line or stacked)
5. Recipient line (from far right margin)
6. Subject line (bold, below recipient)
7. Opening greeting (new paragraph)
8. Body text (right-aligned paragraphs)
9. Closing phrase
10. Signature block: title left, name below, signature above
11. Seal overlapping signature edge
12. Date of signing
13. CC list (bottom, left-aligned)
14. Enclosures notation (below CC)
15. File code / archive code (footer)
16. Page numbering (bottom center)

**Multi-page protocol:**
- Page numbering: ___ / ___ (page X of Y) centered at bottom
- Continuation header on page 2+: اسم الجهة — الرقم — التاريخ (right-aligned)
- No letterhead on continuation pages
- Initials at bottom of each page by signatory or clerk

### 2. Diplomatic & International Correspondence Conventions

1. **Note Verbale (مذكرة شفوية):** Third-person, no salutation, no signature. Initialed by Director. Used between embassies and foreign ministries.
2. **Diplomatic Note (مذكرة دبلوماسية):** First-person from ambassador to foreign minister. Formal salutation and closing. Signed.
3. **Aide-Memoire (مذكرة مساعدة):** No salutation, no signature. Delivered personally. Records oral communication.
4. **Letter of Credence (أوراق اعتماد):** Ambassador credentials. Presented to head of state.
5. **Letter of Recall (خطاب الاستدعاء):** Recalling an ambassador.
6. **Pro Memoria (مذكرة تذكيرية):** Informal summary of discussion.
7. **Demarche (خطوة دبلوماسية):** Formal representation by ambassador to host government.
8. **Demarche paper (ورقة دبلوماسية):** Written document left after oral demarche.
9. **Embassy note format:** سفارة ___ / رقم: ___ / التاريخ: ___ / الموضوع: ___

**Diplomatic openings:**
- تتشرف سفارة ___ بأن تتقدم باحترامها إلى وزارة الخارجية ___ وتشرفها بعلمها بأن ...
- تحية طيبة وبعد، في إطار العلاقات الثنائية المتميزة بين بلدينا ...

**Diplomatic closings:**
- وتغتنم سفارة ___ هذه المناسبة لتجدد لوزارة الخارجية ___ فائق احترامها
- وتفضلوا بقبول فائق الاحترام والتقدير

### 3. Government Meeting and Committee Minutes

**Minute components (محضر الاجتماع):**
1. **Header:** محضر اجتماع / مضبطة جلسة رقم ___ بتاريخ ___
2. **Authority:** الجهة / اللجنة / المجلس
3. **Presided by:** برئاسة ___ (Chairperson)
4. **Attendees:** الحضور (list with titles)
5. **Absentees:** المتغيبون / المعتذرون
6. **Agenda:** جدول الأعمال
7. **Minutes of previous meeting:** مصادقة على محضر الجلسة السابقة
8. **Discussion:** الموضوع الأول / ثانياً / ثالثاً (numbered agenda items)
9. **Decisions:** القرارات (numbered sequentially)
10. **Next meeting:** موعد الجلسة القادمة
11. **Closing:** واختتمت الجلسة في الساعة ___
12. **Signatures:** رئيس الجلسة / مقرر الجلسة / أمين السر

**Decision numbering format:**
- قرار رقم (1) لسنة 2025 بتاريخ ___/___/____ بشأن ___
- Each decision has: رقم / موضوع / نص القرار / الجهة المنفذة / المهلة

**Minutes verbs:**
- استمع (heard) — استمعت اللجنة إلى عرض حول
- ناقش (discussed) — ناقش المجتمعون
- قرر (decided) — قررت اللجنة بالإجماع / بالأغلبية
- أوصى (recommended) — أوصت اللجنة بـ
- أحال (referred) — أحيل الموضوع إلى
- وافق (approved) — وافق المجتمعون على ما يلي
- رفع (adjourned) — رفعت الجلسة
- أرجأ (postponed) — أرجئ الموضوع إلى الجلسة القادمة

### 4. Administrative Numbers & Dates

**Date formats in official documents:**
- Hijri: ___/___/____ هـ
- Gregorian: ___/___/____ م
- Dual (KSA): ___/___/____ هـ — الموافق ___/___/____ م
- Written (official): يوم ___ شهر ___ سنة ___ هجرية / ميلادية
- Fiscal year reference: السنة المالية ___ / ___

**Number style in administrative text:**
- Arabic (Eastern) numerals (٠-٩) in Saudi government documents
- Arabic (Western) numerals (0-9) in Egypt, UAE, Levant and North Africa
- Consistency within a single document is mandatory
- Amounts in text spelled out for clarity, especially for sums of money

**Time format:**
- 24-hour format for official documents: الساعة 14:30
- 12-hour format optional in less formal contexts: الساعة 2:30 مساءً
- Full written: الساعة العاشرة والدقيقة الخامسة والثلاثين صباحاً
