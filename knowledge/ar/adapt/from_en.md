## 1. مستويات الرسمية وطرق المخاطبة (Formality Levels and Register)

### 1.1 ضمير المخاطب (Second-Person Pronouns)

اللغة الإنجليزية لا تميز بين صيغتي المخاطبة الرسمية وغير الرسمية بضمير "you"، بينما العربية تميز تمييزاً واضحاً:

| الإنجليزية | العربية (رسمي) | العربية (غير رسمي) | ملاحظات |
|------------|----------------|-------------------|---------|
| You (formal address) | حضرتك / سيادتك / حضرتكم | أنتَ / أنتِ | حضرتك = Egyptian/Levantine formal |
| You (very formal) | سعادتكم / فخامتكم / سماحتكم | — | للمسؤولين رفيعي المستوى |
| You (plural formal) | حضرتكم / أنتم | أنتم | |
| Your request | طلبكم / طلب حضرتك | طلبك | حسب درجة الرسمية |
| Please send | نرجو التفضل بالإرسال / من فضلكم | لو سمحت / من فضلك | |

- **قاعدة عامة**: عند ترجمة "You" من الإنجليزية إلى العربية، يجب استنتاج درجة الرسمية من السياق:
  - خطاب رسمي → "حضرتك" / "سيادتك"
  - خطاب رسمي جداً (وزراء، سفراء) → "سعادتكم" / "فخامتكم"
  - بريد إلكتروني تجاري → "حضرتك" / "سيادتكم"
  - بين الأصدقاء/الزملاء المقربين → "أنتَ" / "أنتِ"
  - جمهور عام غير محدد → "أنت" (في الإعلانات العامة)

- **مثال**: "Thank you for your cooperation" → "نشكركم على تعاونكم / نشكر حضرتك على تعاونك"
- **مثال**: "You are invited to attend" → "يتشرف حضرتكم بالحضور / نرجو التكرم بالحضور"
- **مثال**: "You can start now" → "يمكنك البدء الآن" (غير رسمي) / "يمكن لحضرتك البدء الآن" (رسمي)

### 1.2 المبني للمعلوم والمبني للمجهول (Active vs. Passive Voice)

الإنجليزية تستخدم المبني للمجهول بكثرة في الكتابة الأكاديمية والرسمية، بينما العربية تفضل المبني للمعلوم:

| الإنجليزية (Passive) | العربية (Active preferred) | ملاحظات |
|---------------------|---------------------------|---------|
| It was decided that... | تقرر أن... / تم اتخاذ قرار بـ... | مقبول في العربية إذا كان الفاعل غير محدد |
| The meeting was held | عُقد الاجتماع / تم عقد الاجتماع | "تم + مصدر" شائع في العربية الحديثة |
| It is believed that... | يُعتقد أن... / من المعتقد أن... | مقبول لعدم تحديد الفاعل |
| It has been noted that... | لوحظ أن... / تجدر الإشارة إلى أن... | |
| The report was submitted | تم تقديم التقرير / قُدِّم التقرير | |
| It was agreed that... | تم الاتفاق على أن... / اتُّفق على أن... | |
| The results were analyzed | تم تحليل النتائج / حُلِّلت النتائج | |
| The letter was received | تم استلام الخطاب / ورد الخطاب | |
| It was found that... | تبيَّن أن... / وُجِد أن... | |
| The system was designed | صُمِّم النظام / تم تصميم النظام | |

- **قاعدة**: عند الترجمة من الإنجليزية إلى العربية:
  - المبني للمجهول الإنجليزي → "تم + مصدر" (الأكثر شيوعاً في العربية الحديثة)
  - أو → الفعل المبني للمجهول الصرفي (فُعِلَ، يُفعَل) للغة أكثر فصاحة
  - أو → إعادة صياغة الجملة بصيغة المبني للمعلوم إذا أمكن تحديد الفاعل

- **مثال**: "The project was completed on time" → "تم إنجاز المشروع في الوقت المحدد" / "أُنجز المشروع في الوقت المحدد"
- **مثال**: "It was suggested that the policy be revised" → "اقتُرِح إعادة النظر في السياسة / تم اقتراح مراجعة السياسة"
- **مثال**: "Participants were asked to complete a survey" → "طُلب من المشاركين تعبئة استبيان / تم توجيه طلب للمشاركين بتعبئة استبيان"

### 1.3 صيغ التخاطب الرسمي في المراسلات (Formal Correspondence Formulas)

| الإنجليزية | العربية | السياق |
|------------|---------|--------|
| Dear Sir / Madam | السيد / السيدة المحترم / المحترمة | افتتاحية رسمية |
| To Whom It May Concern | إلى من يهمه الأمر | خطاب مرجعي/شهادة |
| Dear Mr. Smith | عزيزي السيد سميث / السيد سميث المحترم | شبه رسمي |
| Dear Dr. Ahmed | عزيزي الدكتور أحمد / الدكتور أحمد المحترم | |
| Dear Colleagues | الزملاء الأعزاء / السادة الزملاء | داخل المؤسسة |
| Dear Team | فريق العمل الأعزاء | داخلي |
| Hi John | عزيزي جون / مرحباً جون | غير رسمي |
| Hello | مرحباً / أهلاً | غير رسمي |
| Sincerely | مع خالص التحية / مع خالص الشكر والتقدير | رسمي |
| Best regards | مع أطيب التحيات / وأطيب التمنيات | شبه رسمي |
| Yours faithfully | وتفضلوا بقبول فائق الاحترام (لخطاب يبدأ بـ Dear Sir) | رسمي جداً |
| Yours sincerely | مع خالص الاحترام والتقدير (لخطاب يبدأ باسم) | رسمي |
| Regards | مع التحية / تحياتي | شبه رسمي/غير رسمي |
| Best | مع أطيب التمنيات / دمتم بخير | غير رسمي |
| Cheers | في أمان الله / مع السلامة | غير رسمي جداً |
| Warm regards | مع أطيب التحيات وأصدق التمنيات | ودود/رسمي |
| Respectfully | مع فائق الاحترام والتقدير | رسمي جداً |

- **مثال**: "Dear Mr. Abdullah, Thank you for your email." → "السيد عبد الله المحترم، نشكركم على بريدكم الإلكتروني"
- **مثال**: "To Whom It May Concern, This is to certify that..." → "إلى من يهمه الأمر، نشهد بموجب هذا بأن..."
- **مثال**: "Best regards, John" → "مع أطيب التحيات، جون"
- **قاعدة**: الافتتاحيات والختاميات العربية أكثر طولاً وبلاغة من الإنجليزية. لا تترجم حرفياً.

### 1.4 التمييز بين الجنسين في العربية (Gender Differentiation)

الإنجليزية لا تميز بين المذكر والمؤنث في الضمائر (you, they, we) بينما العربية تفعل:

| الإنجليزية | العربية (مذكر) | العربية (مؤنث) |
|------------|----------------|----------------|
| You are welcome | على الرحب والسعة / أهلاً بك | على الرحب والسعة / أهلاً بكِ |
| You said that | قُلتَ ذلك | قُلتِ ذلك |
| Are you ready? | هل أنت مستعد؟ | هل أنتِ مستعدة؟ |
| You did a great job | قُمتَ بعمل رائع | قُمتِ بعمل رائع |

- **استراتيجية**: إذا كان الجنس غير محدد في النص الإنجليزي، استخدم صيغة المذكر كافتراض افتراضي (في العربية المذكر يشمل المؤنث في صيغ العموم).
- **مثال**: "Each participant must submit his/her report" → "يجب على كل مشارك تقديم تقريره" (صيغة مذكر شاملة للجنسين)

## 2. التواريخ والأوقات (Dates and Time)

### 2.1 تحويل صيغ التواريخ (Date Format Conversion)

| الإنجليزية | العربية | ملاحظات |
|------------|---------|---------|
| MM/DD/YYYY (05/15/2026) | YYYY/MM/DD (2026/05/15) أو DD/MM/YYYY (15/05/2026) | النظام الإداري العربي يفضل YYYY/MM/DD أو DD/MM/YYYY |
| May 15, 2026 | 15 مايو 2026 / 15 أيار 2026 | أسماء الأشهر تختلف بين المشرق (أيار) والمغرب العربي (ماي) |
| 15th May 2026 | 15 مايو 2026 | لا حاجة لـ "th" في العربية |
| 2026-05-15 | 2026-05-15 | صيغة ISO مقبولة دولياً |
| May 2026 | مايو 2026 | شهر فقط |
| 15 May | 15 مايو | يوم وشهر |
| May 15th | 15 مايو | |
| the 15th of May, 2026 | 15 مايو 2026 | |
| Monday, May 15, 2026 | الاثنين، 15 مايو 2026 | |
| 5/15/2026 | 15/5/2026 أو 2026/5/15 | تجنب MM/DD/YYYY تماماً! |

- **قاعدة مهمة**: صيغة MM/DD/YYYY الأمريكية غير مفهومة في العالم العربي ويجب تحويلها إلى DD/MM/YYYY أو YYYY/MM/DD.
- **خطأ شائع**: ترجمة "05/15/2026" حرفياً إلى "05/15/2026" ← خطأ! الصواب: "15/05/2026" أو "2026/05/15"
- **مثال**: "The deadline is 12/31/2026" (American) → "الموعد النهائي هو 31/12/2026"
- **مثال**: "Your appointment is on 07/04/2026" (July 4) → "موعدك هو 4 يوليو 2026"

### 2.2 أسماء الأشهر (Month Names)

| الإنجليزية | العربية (مشرق / بلاد الشام) | العربية (مغرب / مصر) | ملاحظات |
|------------|---------------------------|---------------------|---------|
| January | كانون الثاني | يناير | |
| February | شباط | فبراير | |
| March | آذار | مارس | |
| April | نيسان | أبريل / إبريل | |
| May | أيار | مايو | نفس الاسم تقريباً |
| June | حزيران | يونيو | |
| July | تموز | يوليو | |
| August | آب | أغسطس | |
| September | أيلول | سبتمبر | |
| October | تشرين الأول | أكتوبر | |
| November | تشرين الثاني | نوفمبر | |
| December | كانون الأول | ديسمبر | |

- **قاعدة**: اختر نظام أسماء أشهر واحد يتناسب مع الجمهور المستهدف:
  - السعودية، الخليج، مصر، شمال أفريقيا → الأسماء الغربية (يناير، فبراير...)
  - بلاد الشام (سوريا، لبنان، الأردن، فلسطين) → الأسماء الشرقية (كانون الثاني، شباط...)
  - العراق → الأسماء الشرقية
- **مثال**: "Meeting on March 15, 2026" → "اجتماع في 15 مارس 2026" أو "اجتماع في 15 آذار 2026"

### 2.3 أسماء أيام الأسبوع (Day Names)

| الإنجليزية | العربية | ملاحظات |
|------------|---------|---------|
| Monday | الاثنين | |
| Tuesday | الثلاثاء | |
| Wednesday | الأربعاء | |
| Thursday | الخميس | |
| Friday | الجمعة | يوم العطلة الرسمية في الدول العربية |
| Saturday | السبت | أول أيام العطلة الأسبوعية في بعض الدول |
| Sunday | الأحد | أول يوم عمل في معظم الدول العربية |

- **ملاحظة ثقافية مهمة**: في العالم العربي، الأسبوع يبدأ يوم الأحد (أول أيام العمل) وينتهي يوم الخميس. الجمعة والسبت هما عطلة نهاية الأسبوع في معظم الدول العربية. هذا يختلف عن النمط الغربي (Saturday-Sunday).

### 2.4 صيغ الوقت (Time Formats)

| الإنجليزية | العربية | ملاحظات |
|------------|---------|---------|
| 9:00 AM | 9:00 صباحاً / الساعة التاسعة صباحاً | |
| 3:30 PM | 3:30 مساءً / الساعة الثالثة والنصف مساءً | |
| 12:00 PM | 12:00 ظهراً | |
| 12:00 AM | 12:00 منتصف الليل | |
| 9:45 | 9:45 / الساعة التاسعة وخمس وأربعون دقيقة | |
| 14:00 (24-hour) | 14:00 / الساعة الثانية ظهراً | صيغة 24 ساعة مقبولة في العربية الفصحى |
| Half past three | الثالثة والنصف | |
| Quarter past four | الرابعة والربع | |
| Quarter to five | الخامسة إلا ربع | |

- **قاعدة**: صيغة 12 ساعة مع تحديد (صباحاً/مساءً) هي الأكثر شيوعاً في العالم العربي للمراسلات اليومية.
- صيغة 24 ساعة تستخدم في المراسلات الرسمية والعسكرية والطبية.
- **مثال**: "The meeting starts at 2:00 PM" → "يبدأ الاجتماع في الساعة الثانية مساءً / 14:00"

### 2.5 التقويم الهجري (Hijri Calendar)

عند التعامل مع السياقات العربية، قد نحتاج إلى التقويم الهجري بالإضافة إلى الميلادي:

| المفهوم | العربية |
|---------|---------|
| AD / CE | ميلادي / م |
| AH (After Hijra) | هجري / هـ |
| BC / BCE | قبل الميلاد / ق.م |
| 1447 AH | 1447 هـ |
| 2026 CE | 2026 م |

- **قاعدة**: في المستندات الرسمية في الدول العربية، قد يذكر التاريخ الهجري والميلادي معاً.
- **مثال**: "Date: 15 May 2026" → "التاريخ: 15 مايو 2026م / 15 مايو 2026 ميلادي"
- **مثال**: في المملكة العربية السعودية: "15/05/2026 م الموافق 28/10/1447 هـ"

## 3. التكيف الثقافي (Cultural Adaptation)

### 3.1 التعابير الاصطلاحية الغربية إلى العربية (Western Idioms to Arabic Equivalents)

التعابير الإنجليزية تحتاج إلى تكيف ثقافي، لا ترجمة حرفية:

| الإنجليزية | المعنى الحرفي | العربية المناسبة | استراتيجية |
|------------|---------------|-------------------|-----------|
| Break a leg | حظ سعيد (للممثلين) | "بالتوفيق" / "حظاً موفقاً" | استخدام التحية العربية |
| Piece of cake | سهل جداً | "سهل جداً" / "هيّن" / "لعبة" | استخدام المكافئ العربي |
| Spill the beans | إفشاء سر | "أفشى السر" / "فتح فمه" | ترجمة المعنى |
| Hit the nail on the head | أصاب تماماً | "أصاب عين الحقيقة" / "أصاب كبد الحقيقة" | استعارة عربية مشابهة |
| Once in a blue moon | نادر جداً | "مرة في العام / نادراً ما" / "في الأسبوع دلو" | |
| The ball is in your court | دورك الآن | "الكرة في ملعبك / الكرة الآن في ملعبك الآن" | استعارة مشتركة في العربية الحديثة |
| Bite the bullet | تحمل موقف صعب | "تحمل المسؤولية" / "قَبِل الأمر الواقع" | |
| Beat around the bush | اللف والدوران | "يدور حول الموضوع" / "يلف ويدور" | مقبول في العربية |
| Let the cat out of the bag | أفشى السر | "أفشى السر" / "أخرج ما في جعبته" | |
| Under the weather | مريض / متعب | "مُتعب" / "على غير ما يرام" / "مرهق" | |
| Call it a day | أنهى العمل لليوم | "ننهي عملنا لهذا اليوم" / "هذا ما لدينا لليوم" | |
| The best of both worlds | أفضل ما في العالمين | "أفضل ما في الأمرين" / "الجمع بين الميزتين" | |
| Cost an arm and a leg | مكلف جداً | "مكلف جداً" / "غالي الثمن" / "يكلف الكثير" | |
| Actions speak louder than words | الأفعال أقوى من الأقوال | "الأفعال أبلغ من الأقوال" | مكافئ عربي مباشر |
| When in Rome, do as the Romans do | إذا زرت روما فافعل كأهلها | "لكل بلد عاداته" / "إذا حللت بينهم فاحكم بأعرافهم" | بديل ثقافي عربي |
| Curiosity killed the cat | الفضول قد يسبب المتاعب | "الفضول مقتلة" / "كثرة السؤال تُتعب" | مثل عربي مشابه |

- **قاعدة أساسية**: لا تترجم التعابير الاصطلاحية الإنجليزية حرفياً إلى العربية:
  - إذا وجد تعبير عربي مكافئ: استخدمه.
  - إذا لم يوجد: اشرح المعنى بطريقة مباشرة.
  - إذا كان التعبير معروفاً في الثقافة العربية المعاصرة (مثل "الكرة في ملعبك"): يمكن استخدامه.

- **مثال**: "Let's call it a day and review this tomorrow" → "لننهي عملنا اليوم ونراجع هذا غداً"
- **مثال**: "He was under the weather, so he couldn't attend" → "كان على غير ما يرام، لذا لم يستطع الحضور"
- **مثال**: "Don't beat around the bush, just tell me" → "لا تلف وتدور، قل لي مباشرة"

### 3.2 الفكاهة والتهكم والسخرية (Humor, Irony, and Sarcasm)

| النمط | التكيف العربي | ملاحظات |
|-------|---------------|---------|
| الفكاهة الإنجليزية الجافة (Dry humor) | غالباً لا تترجم بنفس التأثير | استخدم تعابير عربية بديلة أو تجاهل |
| السخرية (Sarcasm) | يجب إعادة الصياغة للحفاظ على النبرة | السخرية الإنجليزية أكثر حدة من العربية الرسمية |
| التهكم (Irony) | يترجم المعنى وليس الصياغة | تخفيف حدة التهكم في السياقات الرسمية |
| النكات الثقافية (Cultural jokes) | إذا كانت تعتمد على ثقافة غربية: اشرحها أو استبدلها | لا تترجم حرفياً أبداً |
| التلاعب بالألفاظ (Puns/Wordplay) | إذا تعذر التكافؤ: ترجمة حرفية + شرح بين قوسين | الأفضل إعادة صياغة المزاح بطريقة عربية |

- **مثال**: "Oh great, another Monday!" (sarcastic) → "يا للروعة، يوم الاثنين مرة أخرى!" (لا تنقل السخرية كاملة، الأفضل: "يوم الاثنين من جديد!" بنبرة محايدة)
- **قاعدة**: في الكتابة الرسمية العربية، السخرية نادرة وغير مرغوب فيها. في الكتابات غير الرسمية أو الإبداعية، يمكن محاولة نقل النبرة.

### 3.3 المراجع الثقافية الغربية (Western Cultural References)

| المرجع | المشكلة | التكيف العربي |
|--------|---------|---------------|
| Greek mythology (Achilles' heel, Pandora's box) | معروفة في الأوساط المثقفة العربية | احتفظ بها مع شرح إن لزم |
| Shakespeare quotes | معروفة لدى المثقفين العرب | استخدم الترجمات العربية الثابتة |
| Biblical references | مفهومة في العالم العربي (للنصارى والمسلمين بقصص مشتركة) | احتفظ بالمرجع الديني المشترك |
| Western holidays (Thanksgiving, Halloween) | غير معروفة في العالم العربي | اشرح أو استبدل بمناسبة مناسبة |
| Historical Western figures | غير معروفة للقارئ العربي العام | اشرح من هم أو استخدم شخصيات معروفة عربياً |
| Sports references (baseball, American football) | غير مفهومة في العالم العربي | استبدل بكرة القدم أو الرياضات المعروفة |
| TV shows / pop culture | غير معروفة بشكل عام | احذف أو اشرح أو استبدل |
| Western legal/political systems | مختلفة عن الأنظمة العربية | اشرح النظام مع الاحتفاظ بالمصطلح |

- **مثال**: "He has an Achilles' heel when it comes to deadlines" → "لديه نقطة ضعف (كعب أخيل) فيما يتعلق بالمواعيد النهائية"
- **مثال**: "This is a Cinderella story" → "هذه قصة سندريلا / هذه قصة نجاح مذهلة من الصفر إلى القمة"
- **مثال**: "Opening Pandora's box" → "فتح صندوق باندورا / فتح باب لا يُحمد عقباه"

### 3.4 التعابير الدينية (Religious Expressions)

اللغة العربية مشبعة بالتعابير الدينية الإسلامية التي لا نظير لها في الإنجليزية:

| السياق | الإنجليزية | العربية | ملاحظات |
|--------|------------|---------|---------|
| افتتاحية | — | بسم الله الرحمن الرحيم | في أول المستندات الرسمية الإسلامية |
| شكر | Thank God / Thankfully | الحمد لله | شائعة في كل السياقات |
| مستقبل | — | إن شاء الله | للتحدث عن المستقبل |
| مدح | — | ما شاء الله / تبارك الله | عند الإعجاب بشيء |
| بداية | In the name of God | بسم الله | للبدء بعمل |
| اعتذار | Sorry / Excuse me | المعذرة / سمح لي / الله يسامحك | |
| تعزية | Condolences | البقاء لله / عزائي / إنا لله وإنا إليه راجعون | |
| تهنئة | Congratulations | مبارك / ألف مبارك | |
| فراق | Goodbye | في أمان الله / مع السلامة | |

- **قاعدة**: عند الترجمة من الإنجليزية إلى العربية:
  - لا تفرض تعابير دينية إسلامية إذا لم تكن موجودة في النص الإنجليزي الأصلي
  - إذا كان النص الإنجليزي يحتوي على إشارات دينية مسيحية ("Thank God", "God bless you")، فالأفضل استخدام الصيغ العربية المحايدة أو الإسلامية المقابلة
  - إذا كان النص الأصلي علمانياً بحتاً، تجنب إضافة التعابير الدينية

- **مثال**: "Thank God everything went well" → "الحمد لله، كل شيء سار على ما يرام"
- **مثال**: "See you next week, God willing" → "أراك الأسبوع القادم إن شاء الله"
- **مثال**: "Congratulations on your promotion!" → "مبارك الترقية! / ألف مبارك على الترقية"

### 3.5 التكيف مع الثقافة التجارية (Business Culture Adaptation)

| المفهوم الإنجليزي | المشكلة | العربية المناسبة |
|-------------------|---------|-------------------|
| Directness in requests | الإنجليزية أكثر مباشرة | أضف مجاملات افتتاحية قبل الطلب |
| "I want" / "I need" | قد تبدو وقحة في العربية | "أود" / "أرجو" / "نأمل التفضل بـ" |
| "Unfortunately" | مقبولة في الإنجليزية | "مع الأسف" / "للأسف" — في العربية الفصحى "لَسوف" |
| "No problem" | مقبولة في الإنجليزية | "عفواً" / "تفضل" / "لا داعي للشكر" |
| Addressing by first name | مألوف في الإنجليزية | في العربية الرسمية: اللقب + الاسم الأول |
| "ASAP" / "urgent" | قد تبدو وقحة إذا لم تخفف | "في أقرب وقت ممكن" / "عند التيسر" |
| Self-praise | مقبولة في الإنجليزية التجارية | في العربية: التواضع مطلوب، استخدم صيغاً أقل مباشرة |
| "I think" / "In my opinion" | مقبولة | "من وجهة نظري" / "أعتقد أن" — ممكنة ولكن أفضل: "يرى الكاتب أن" |
| Negative feedback | مباشرة في الإنجليزية | خفف النقد بالمجاملات في العربية |

- **مثال**: "I need the report by Friday" → "أرجو التكرم بتقديم التقرير بحلول يوم الجمعة إن أمكن" (أكثر تليقاً)
- **مثال**: "Unfortunately, your application has been rejected" → "نأسف لإعلامكم بأن طلبكم لم يتم قبوله / مع الأسف، لم يتم قبول طلبكم"
- **مثال**: "No problem at all" → "عفواً، لا داعي للشكر / هذا واجبنا"
- **قاعدة**: أضف طبقة من المجاملة العربية عند ترجمة الرسائل التجارية الإنجليزية. الرسائل التجارية العربية أكثر رسمية واحتراماً.

### 3.6 الحساسيات الثقافية (Cultural Sensitivities)

| الموضوع | التكيف | ملاحظات |
|---------|--------|---------|
| الكحول | تجنب الإشارات الإيجابية أو استبدلها | حساسية دينية في العالم العربي |
| العلاقات خارج الزواج | تجنب الإشارات الصريحة | غير مقبولة في大多数 الدول العربية |
| المثلية الجنسية (LGBTQ+) | غير مقبولة في الخطاب الرسمي العربي | قد تحتاج إلى حذف أو ترجمة محايدة جداً |
| الإشارات إلى إسرائيل | حساسية سياسية عالية | استخدم "إسرائيل" أو "الكيان الإسرائيلي" حسب التوجه |
| الإشارات إلى الدين | احترام جميع الأديان | لا تسيء لأي دين |
| الصور النمطية | تجنب أي صورة نمطية عن العرب أو المسلمين | حساسية عالية |
| الملابس (خاصة المرأة) | استخدم أوصافاً محتشمة | |

- **قاعدة**: إذا كان النص الإنجليزي يحتوي على إشارات غير مقبولة في السياق العربي، يجب التكيف بحذر:
  - إعادة الصياغة بطريقة مقبولة ثقافياً
  - الحذف إذا كان المحتوى لا يمكن تكييفه
  - إضافة توضيح أو تحذير إذا لزم الأمر

## 4. المصطلحات التقنية والعلمية (Technical and Scientific Terminology)

### 4.1 الترجمة مقابل التعريب (Translation vs. Transliteration)

هناك جدل في الأوساط العلمية العربية بين الترجمة والتعريب:

| النهج | الوصف | مثال |
|-------|-------|------|
| الترجمة الكاملة | استخدام مقابل عربي خالص | Computer → حاسوب / كمبيوتر |
| التعريب الصوتي | كتابة المصطلح بالأحرف العربية | Computer → كمبيوتر |
| الترجمة + الأصل | استخدام المكافئ العربي مع الأصل الإنجليزي بين قوسين | الحاسوب (Computer) |
| الاحتفاظ بالإنجليزية + شرط | استخدام المصطلح الإنجليزي مع شرط عربي | The system → النظام / النظام (System) |

- **الأفضل**: الترجمة + الأصل الإنجليزي بين قوسين عند أول ظهور للمصطلح في النص، ثم استخدام المكافئ العربي فقط بعد ذلك.

## 6. علامات الترقيم وضبط الاتجاه (Punctuation and RTL Adjustment)

### 6.1 تحويل علامات الاقتباس (Quotation Mark Conversion)

| الإنجليزية | العربية | ملاحظات |
|------------|---------|---------|
| " " (double quotes) | « » (guillemets / علامتا اقتباس عربيتان) | العلامتان: « » |
| ' ' (single quotes) | " " (نصف غيوميه / used within guillemets) | للاقتباس ضمن الاقتباس |
| "quote" within "quote" | «اقتباس "داخل" اقتباس» | |

- **مثال**: He said, "I will come tomorrow." → قال: «سأحضر غداً».
- **مثال**: The manager replied, "Please read the 'policy' carefully." → ردَّ المدير: «يرجى قراءة "السياسة" بعناية.»
- **ملاحظة**: في بعض الدول العربية (خاصة مصر)، قد تستخدم " " بدلاً من « » في الكتابات غير الرسمية. يُفضل استخدام « » في المستندات الرسمية.

### 6.2 علامات الترقيم الأساسية (Basic Punctuation Marks)

| الإنجليزية | العربية | ملاحظات |
|------------|---------|---------|
| . (period / full stop) | . (نقطة) | نفس الشكل، لا تغيير |
| , (comma) | ، (فاصلة) | الفاصلة العربية معكوسة (U+060C) |
| ; (semicolon) | ؛ (فاصلة منقوطة) | شكل عربي مختلف (U+061B) |
| ? (question mark) | ؟ (علامة استفهام) | شكل عربي مختلف (U+061F) |
| ! (exclamation mark) | ! (علامة تعجب) | نفس الشكل |
| : (colon) | : (نقطتان رأسيتان) | نفس الشكل (قاعدة المسافة مختلفة) |
| ( ) (parentheses) | ( ) (قوسان) | نفس الشكل |
| [ ] (brackets) | [ ] (قوسان مربعان) | نفس الشكل |
| { } (braces) | { } (قوسان معقوفان) | نفس الشكل |
| - (hyphen) | - (شرطة / واصلة) | نفس الشكل |
| — (em dash) | — (شرطة طويلة) | نفس الشكل |
| ... (ellipsis) | ... (نقاط حذف) | نفس الشكل (3 نقاط) |
| / (slash) | / (شرطة مائلة) | نفس الشكل |
| * (asterisk) | * (نجمة) | نفس الشكل |
| @ (at sign) | @ (عند/آت) | يقرأ "عند" في النطق العربي |
| & (ampersand) | و (واو العطف) | لا تستخدم & في العربية، استخدم "و" |

- **مثال**: "Hello, how are you? I am fine, thanks." → "مرحباً، كيف حالك؟ أنا بخير، شكراً."
- **مثال**: "Wait! Where are you going?" → "انتظر! أين تذهب؟"

### 6.3 قواعد المسافات (Spacing Rules)

تختلف قواعد المسافات العربية عن الإنجليزية:

| الإنجليزية | العربية | ملاحظات |
|------------|---------|---------|
| مسافة بعد الفاصلة: "word, word" | مسافة بعد الفاصلة: "كلمة، كلمة" | نفس القاعدة |
| مسافة بعد النقطة: "end. Next" | مسافة بعد النقطة: "انتهى. التالي" | نفس القاعدة |
| مسافة قبل علامة الاستفهام: "what?" | مسافة قبلها: "ماذا؟" — لا مسافة قبلها | اختلاف! في العربية، لا مسافة قبل علامة الاستفهام |
| مسافة قبل النقطتين: "note: text" | مسافة قبلها: "ملاحظة: النص" أو لا مسافة | في العربية، قد توضع مسافة قبل النقطتين |
| مسافة بعد علامات الاقتباس | مسافة بعد « » | وضع مسافة قبل وبعد علامات الاقتباس |
| مسافة قبل القوس: "text (info)" | مسافة قبل القوس أو لا مسافة | حسب السياق |
| مسافة حول الشرطة: "word - word" | مسافة حول الشرطة: "كلمة - كلمة" | عادة بمسافات |

- **قاعدة**: في العربية، لا توضع مسافة قبل علامات الترقيم (؟، !، ،، .) بل بعدها. هذه قاعدة تختلف عن الفرنسية (التي تضع مسافة قبل ? و !).

### 6.4 ضبط اتجاه الكتابة من اليمين لليسار (RTL / Right-to-Left Adjustment)

| القضية | المشكلة | الحل |
|--------|---------|------|
| خلط RTL + LTR (أرقام إنجليزية، أسماء إنكليزية) | اتجاه النص المختلط | استخدام علامات الاتجاه (LRM, RLM) |
| الأرقام في النص العربي | الأرقام تكتب LTR في نص RTL | استخدام الأرقام العربية الشرقية (١٢٣) لتجنب المشكلة |
| عنوان بريد إلكتروني في نص عربي | يظهر معكوساً | استخدم علامة LRM (U+200E) قبل عنوان البريد |
| علامات الترقيم في نهاية الجملة | موضع علامة الاستفهام | في نص RTL، علامة الاستفهام موضوعة بشكل صحيح تلقائياً |
| القوائم الرقمية | الأرقام تظهر بشكل صحيح | استخدم CSS: direction: rtl; text-align: right; |
| HTML/CSS للصفحات العربية | direction: rtl | جميع الاتجاهات معكوسة (right↔left) |

- **مثال**: "Please email us at info@company.com" في نص عربي:
  - بدون معالجة: «info@company.com» قد يظهر مقلوباً
  - مع المعالجة: أضف علامة LRM أو استخدم HTML Unicode controls
- **قاعدة**: عند خلط النص العربي بالإنجليزية/الأرقام، انتبه لاتجاه الكتابة ثنائي الاتجاه (BiDi)، خاصة في:
  - عناوين البريد الإلكتروني
  - أسماء النطاقات
  - الأكواد البرمجية
  - الأرقام المختلطة مع النص

### 6.5 كتابة الأرقام في النص العربي (Numbers in Arabic Text)

| القاعدة | الإنجليزية | العربية | ملاحظات |
|---------|------------|---------|---------|
| ترقيم الصفحات | Page 1, Page 2 | صفحة ١، صفحة ٢ / صفحة 1، صفحة 2 | حسب تفضيل الأرقام |
| أرقام الهواتف | +1 (555) 123-4567 | +20 100 123 4567 (تنسيق عربي) | تنسيق الهاتف العربي |
| العناوين | 123 Main Street | شارع الملك فهد، رقم ١٢٣ / 123 | تنسيق العنوان العربي |
| المبالغ المالية | $1,000.00 | ١٬٠٠٠٫٠٠ دولار / 1,000.00 دولار | |
| الرموز البريدية | ZIP Code 12345 | الرمز البريدي ١٢٣٤٥ / 12345 | |
| النسب المئوية | 15% | ١٥٪ / 15% | |

- **مثال**: "Our office is at 123 Main Street, Suite 200" → "يقع مكتبنا في شارع مين الرئيسي، رقم ١٢٣ / 123، جناح ٢٠٠ / 200"
- **قاعدة**: في النص العربي RTL، الأرقام تُكتب من اليسار إلى اليمين. هذا طبيعي ولا يحتاج إلى تعديل.

### 6.6 تنسيق القوائم والعناوين (List and Heading Formatting)

| الميزة | الإنجليزية | العربية |
|--------|------------|---------|
| محاذاة العناوين | يسار (Left) | يمين (Right) |
| محاذاة الفقرات | يسار (Left) / ضبط (Justify) | يمين (Right) / ضبط (Justify) |
| تنسيق القوائم النقطية | • نقطة يسار | • نقطة يمين |
| القوائم المرقمة | 1. | ١. / 1. |
| التباعد بين السطور | قياسي | قياسي (نفس الإنجليزية) |
| المسافة البادئة (Indent) | يسار | يمين |
| اتجاه الجدول | LTR | RTL (خانات اليمين أولاً) |

- **قاعدة**: أي مستند عربي يجب أن يكون اتجاهه العام RTL:
  - المحاذاة الرئيسية: يمين (Right / Justify)
  - القوائم: تبدأ من اليمين
  - الأعمدة في الجداول: من اليمين إلى اليسار
  - الترقيم: ١، ٢، ٣ أو 1، 2، 3 مع محاذاة RTL

### 6.7 قواعد إضافية (Additional Rules)

- **فصل الكلمات**: لا يجوز قطع الكلمة العربية بين سطرين (لا واصلة في نهاية السطر للكلمات العربية).
- **المسافات**: الكلمة العربية لا تنفصل عن علامات الترقيم التي تليها (لا مسافة قبل الفاصلة أو النقطة أو علامة الاستفهام).
- **الاقتباس**: علامات الاقتباس العربية « » تفتح من اليمين وتغلق من اليسار.
- **الأرقام المركبة**: تكتب الأرقام في العربية من اليسار إلى اليمين (تنسيق LTR ضمن RTL)، مثل: 15, 2026, 1,000.
- **التاريخ**: لا تستخدم الأرقام الترتيبية مع أسماء الأشهر العربية (لا تقل "15th مايو" بل "15 مايو").
- **الترقيم في المستندات الرسمية**: يفضل استخدام "أولاً، ثانياً، ثالثاً" بدلاً من "1، 2، 3" في العناوين الرئيسية للمستندات الرسمية العربية.
- **رمز العملة**: رمز العملة يكتب بعد المبلغ في العربية: "دولار 100" أو "100 دولار". التنسيق الأكثر شيوعاً: "100 دولار أمريكي".
