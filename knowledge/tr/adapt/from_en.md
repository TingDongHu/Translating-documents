# Turkish Adaptation Guide: English → Turkish

## 1. Formality and Voice

English uses _you_ for both formal and informal address. Turkish has a single second-person pronoun _sen_ (informal) as the default, with _siz_ reserved for highly formal contexts. Turkish strongly prefers active voice over passive voice.

| English | Turkish | Notes |
|---|---|---|
| You (general/indefinite) | — (omit) / insan / -ebilirsiniz | Impersonal "you" → use impersonal constructions |
| You (singular informal) | sen | Use in user interfaces, personal communication |
| You (singular formal) | siz | Use in official letters, formal documents |
| You (plural) | siz (plural context) / hepiniz | Turkish _siz_ covers both formal singular and plural; clarify when needed |
| Please | lütfen / rica ederim | _Lütfen_ is standard; _rica ederim_ for polite requests |
| Thank you | teşekkür ederim / teşekkürler | _Teşekkür ederim_ is formal; _teşekkürler_ is slightly less formal |
| Dear [Name] | Sayın [Name] / Sevgili [Name] | Sayın (formal), Sevgili (informal) |
| Mr. / Mrs. / Ms. | Sayın / Bey / Hanım | English titles have no direct equivalent; use name-level honorifics |
| Sir / Madam | Beyefendi / Hanımefendi | Very formal address |
| Passive: "It is believed that..." | ...olduğu düşünülmektedir / Kanıya göre... | Turkish uses passive less; prefer active or impersonal forms |
| Passive: "The system was designed..." | Sistem ... için tasarlandı | Convert to active or agentless passive; Turkish -DIr suffix |
| Passive: "It was decided that..." | ...kararlaştırıldı / ...kararı alındı | Use active voice: "Karar alındı" or "Kurul ... karar verdi" |
| "One must consider..." | ...göz önünde bulundurulmalıdır | Impersonal construction; acceptable in formal Turkish |
| "It can be argued that..." | ...tartışılabilir / söylenebilir ki | Active impersonal works well in Turkish |
| "The document is attached." | Belge ektedir. | Turkish prefers direct statement over passive |
| "It should be noted that..." | Şunu belirtmek gerekir ki / Unutulmamalıdır ki | Both forms acceptable; active is more natural |
| "I was told that..." | ...olduğunu duydum / bana söylendi | Prefer active "duydum" over passive "söylendi" |
| "It is recommended that..." | ...önerilir / tavsiye edilir | Impersonal active form is natural in Turkish |
| "He is considered to be..." | ...olarak kabul edilir / sayılır | Acceptable impersonal form |

## 2. Dates, Times, and Numbers

English (US) uses MM/DD/YYYY and AM/PM. Turkish uses DD.MM.YYYY and 24-hour format exclusively.

| English (US) | Turkish | Notes |
|---|---|---|
| MM/DD/YYYY (e.g., 01/15/2025) | DD.MM.YYYY (e.g., 15.01.2025) | Month and day swapped; dots as separators |
| January 15, 2025 | 15 Ocak 2025 | Day before month name |
| 1/15/2025 | 15.01.2025 | Never keep US short date format |
| 2025-01-15 (ISO) | 15.01.2025 | ISO → Turkish numeric date |
| Week of 1/15 | 15 Ocak haftası | First convert date |
| 9:00 AM | 09:00 | Convert to 24-hour format |
| 12:00 PM (noon) | 12:00 | 24-hour: midday is 12:00 |
| 12:00 AM (midnight) | 00:00 / 24:00 | 00:00 is standard |
| 3:30 PM | 15:30 | Add 12 hours to PM times |
| 1:00 PM | 13:00 | — |
| 11:45 PM | 23:45 | — |
| 6:00 AM | 06:00 | — |
| 10,000 (comma as separator) | 10.000 (dot as separator) | Turkish uses . for thousands separator |
| 1,000,000 (comma) | 1.000.000 (dot) | Same rule |
| 3.14 (dot as decimal) | 3,14 (comma as decimal) | Turkish uses , for decimal separator |
| $100 (dollar) | 100 USD / 100 ABD doları | Indicate currency explicitly |
| 5'10" (height) | 1,78 m / 178 cm | Convert imperial to metric |
| 32°F (temperature) | 0°C | Convert Fahrenheit to Celsius |
| Phone: (555) 123-4567 | Telefon: +90 555 123 4567 | Convert to Turkish format; add +90 country code |
| Zip code: 90210 | Posta kodu: 34000 (example) | US 5-digit code; flag for Turkish equivalent |
| Q1 2025 / FY2025 | 2025 ilk çeyrek / 2025 mali yılı | Quarter/fiscal year |
| 50% (no space before %) | %50 (no space) | Turkish places % before the number without space |
| Page 1-5 | Sayfa 1–5 | Use en dash in Turkish |

## 3. Cultural References and Idioms

English idioms and culturally specific expressions need Turkish equivalents, not literal translations.

| English Idiom | Turkish Equivalent | Notes |
|---|---|---|
| It's raining cats and dogs | Bardaktan boşanırcasına yağıyor | Turkish uses "pouring from a bucket" |
| Piece of cake | Çocuk oyuncağı / Çok kolay | Turkish uses "child's play" |
| Break a leg | Kolay gelsin / Bol şans | "Break a leg" → Turkish well-wishing phrase |
| The ball is in your court | Top sende | Same metaphor, well understood in Turkish |
| Hit the nail on the head | Tam isabet / Çiviyi tam başına vurmak | Same metaphor exists |
| Better late than never | Geç olsun güç olmasın / Zararın neresinden dönülse kardır | Equivalent proverbs |
| Bite the bullet | Dişini sıkıp katlanmak / Göze almak | No direct equivalent; functional translation |
| Spill the beans | Ağzından baklayı çıkarmak / Sırrı ifşa etmek | Turkish uses "remove the chickpea from mouth" |
| Once in a blue moon | Haftada bir pazar / Çok nadir | Turkish uses "once a week on Sunday" = very rare |
| Cost an arm and a leg | Çok pahalı / Servete mal olmak | No equivalent idiom; translate meaning |
| Let the cat out of the bag | Ağzından kaçırmak / Sırrı açıklamak | No equivalent idiom |
| Under the weather | Keyifsiz / Hastalıklı / Kendini iyi hissetmemek | Translate functionally |
| The early bird catches the worm | Erken kalkan yol alır | Close equivalent exists |
| Don't count your chickens | Deveye hendek atlatmak (no) / Dereyi görmeden paçaları sıvama | Turkish: "Don't roll up your pants before seeing the stream" |
| Actions speak louder than words | Lafla peynir gemisi yürümez | Turkish: "Cheese boat doesn't sail with words" |
| Rome wasn't built in a day | Bir günde (hemen) olmaz | Direct functional translation works |
| When in Rome... | Roma'da Romalılar gibi davran | Same loaned idiom exists |
| The straw that broke the camel's back | Bardaktaki son damla / Deveyi sırtındaki son saman çöpü devirir | Equivalent exists |
| To kill two birds with one stone | Bir taşla iki kuş vurmak | Same idiom exists in Turkish |
| To add insult to injury | Tuz biber ekmek / Dert üstüne dert | Turkish: "to add salt and pepper" |
| To beat around the bush | Lak lak etmek / Konuyu gevellemek | Turkish: "to mumble the topic" |

## 5. Administrative and Legal Language

English modal verbs and legal phrases map to Turkish suffixes and structures.

| English | Turkish | Notes |
|---|---|---|
| shall (obligation) | -meli / -malı | Suffix attached to verb stem; e.g., yapılmalıdır |
| shall not (prohibition) | -mez / -maz / yasaktır | Negative obligation suffix |
| must | zorunlu / -meli / -malı | Stronger than shall; -meli/-malı suffices in most texts |
| must not | kesinlikle -mez / -maz | Stronger prohibition |
| may (permission) | -ebilir / -abilir | Permission suffix |
| may (possibility) | olabilir / ihtimali vardır | Possibility; distinguish from permission |
| should (recommendation) | -meli / -malı (weak) / önerilir | Weaker than shall; önerilir is explicit recommendation |
| should not | -memeli / -memalı / önerilmez | Negative recommendation |
| will (future) | -ecek / -acak | Simple future suffix or geniş zaman for habitual |
| can (ability) | -ebilir / -abilir | Ability suffix; same form as permission |
| cannot / can't | -emez / -amaz | Negative ability |
| need to | gerek / gereklidir | Necessity |
| pursuant to | uyarınca | Legal phrase |
| in accordance with | uyarınca / gereğince | Formal correspondence |
| notwithstanding | ...ne rağmen / ...e bakılmaksızın | Typically clause-initial in Turkish |
| without prejudice to | ...e halel getirmeksizin | Legal term; use sparingly |
| subject to | ...e tabi olmak / ...e bağlı olarak | Legal contexts |
| effective as of | ... tarihinden itibaren geçerli | Date bound |
| hereby | işbu / işbu belgeyle | Formal legal term |
| therein | orada / bundan / onda | Translate contextually; avoid legalese if inappropriate |
| hereto | buna / işbu belgeye | Legal formal; consider if plain Turkish is better |
| party (legal: "the Parties") | taraf / Taraflar | Capitalize in legal Turkish |
| agreement / contract | sözleşme / anlaşma | Sözleşme = contract; anlaşma = agreement (slightly broader) |
| clause / article | madde | Standard legal term |
| section | bölüm | Standard term |
| appendix / annex | ek | Standard term |
| enter into force | yürürlüğe girmek | Legal phrase |
| terminate / termination | feshetmek / fesih | Standard term |
| indemnify | tazmin etmek / zararı karşılamak | Legal context |
| liability | sorumluluk / yükümlülük | Distinguish: sorumluluk (responsibility), yükümlülük (obligation) |

## 6. Punctuation

Turkish punctuation mostly follows English conventions, with key differences in quotation marks and apostrophe use.

| English | Turkish | Rule |
|---|---|---|
| " " (double quotation marks) | "" (Turkish guillemets-like double quotes) | Turkish officially uses the same " " as English; no change required for simple quotes |
| ' ' (single quotation marks) | ' ' | Same usage for nested quotes |
| "quote" (within quotation) | "..." | No change needed |
| ... (ellipsis, 3 dots) | ... (3 dots) | Same |
| — (em dash) | — (em dash) | Same usage |
| – (en dash for ranges) | – (en dash) | Same, used for 5–10, Istanbul–Ankara |
| - (hyphen) | - (hyphen) | Same |
| ' (apostrophe) | ' (apostrophe) | **Critical**: Turkish uses the apostrophe for proper noun suffix boundaries: _İstanbul'da_, _Ali'nin_, _Türkiye'ye_. Always preserve apostrophe before case suffixes on proper nouns. |
| , (comma) | , (comma) | Same |
| . (period) | . (period) | Same |
| : (colon) | : (colon) | Same |
| ; (semicolon) | ; (semicolon) | Same |
| ? (question mark) | ? | Same |
| ! (exclamation) | ! | Same |
| % (percent) | % | Turkish places % before the number: %50 (not 50%) |
| No space after % (50%) | No space either (%50) | % moves to front, no space after it |
| $100 | 100 USD | Currency symbol → written after amount with code |
| "Chapter 1" | "Bölüm 1" / 1. Bölüm | Turkish may use numbered expressions differently |
| "e.g." / "i.e." | "ör." / "yani" | Use Turkish abbreviations; or write out "örneğin" / "yani" |
| "etc." | "vb." (ve benzeri) / "vs." (ve saire) | Turkish uses vb. or vs. |
| "i.e." | "yani" | No direct equivalent abbreviation; write out |
| "c.f." | "krş." (karşılaştırınız) | Use Turkish abbreviation |
| "N.B." | "önemli not" / "dikkat" | No standard Latin abbreviation; use Turkish |
| "Inc." | "A.Ş." / "Ltd. Şti." | Company suffix localization depending on legal form |
| "Ltd." | "Ltd. Şti." | Limited company |
| "PLC" | "A.Ş." | Halka açık anonim şirket |
| Bullet list punctuation | No punctuation after bullet items | Turkish convention: lowercase items without final punctuation |
