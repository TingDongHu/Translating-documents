# Turkish Adaptation Guide: Chinese (Simplified) → Turkish

## 1. Formality

Chinese uses two second-person pronouns: 您 (formal) and 你 (informal). Turkish uses a single pronoun _sen_ for most contexts, with formality expressed through verb endings, titles, and indirect phrasing rather than a dedicated formal pronoun. _Siz_ (plural/polite _you_) can be used sparingly in very formal or official texts.

| Chinese | Turkish | Notes |
|---|---|---|
| 您 | siz | Reserved for highly formal/official contexts; use sparingly |
| 你 | sen | Default second person in Turkish |
| 先生 (xiānsheng) | Sayın + [surname] / Bey | Sayın for formal titles, Bey for general address |
| 女士 (nǚshì) | Sayın + [surname] / Hanım | Same distinction as above |
| 小姐 (xiǎojiě) | Bayan / Hanım | Bayan is slightly dated; prefer Hanım |
| 同志 (tóngzhì) | — (omit) | Not used in modern Turkish; omit or use meslektaşım if colleague |
| 师傅 (shīfu) | Usta / Beyefendi | Context-dependent; usta for tradespeople |
| 老师 (lǎoshī) | Hoca / Öğretmenim | Hoca for academic/instructor contexts |
| 尊敬的 (zūnjìng de) | Saygıdeğer | Use in formal correspondence openings |
| 亲爱的 (qīn'ài de) | Sevgili | Use in informal correspondence; avoid in official texts |
| 贵姓 (guìxìng) | Adınız nedir? / İsminizi öğrenebilir miyim? | Turkish lacks the "honorable surname" construct; use polite question forms |
| 请 (qǐng) | Lütfen / rica ederim | Lütfen is standard polite marker |
| 麻烦您 (máfan nín) | Zahmet olacak ama / Kusura bakmayın | Expresses polite inconvenience |
| 您先请 (nín xiān qǐng) | Buyurun, siz önden | Literally "please, you go first" |
| 大家好 (dàjiā hǎo) | Herkese merhaba | Standard greeting to a group |
| 诸位 (zhūwèi) | Hepiniz / Herkes | Literary "everyone"; no direct equivalent in Turkish |

## 2. Numbers and Dates

Chinese uses unique number words (万 10,000, 亿 100,000,000) that do not exist in Turkish. Turkish uses the standard European grouping system. Date formats also differ.

| Chinese | Turkish | Notes |
|---|---|---|
| 万 (10,000) | 10.000 | Never translate as "on bin" only; use digit format |
| 亿 (100,000,000) | 100.000.000 | Chinese 亿 has no single-word Turkish equivalent |
| 百万 (1,000,000) | 1.000.000 | Same concept, different digit grouping |
| 千万 (10,000,000) | 10.000.000 | — |
| 万亿 / 兆 (10¹²) | 1.000.000.000.000 | Verify context; 兆 can also mean 10⁶ in some Chinese contexts |
| 几十万 (hundreds of thousands) | yüz binlerce | Convert to Turkish numeric phrasing |
| 百分之... (%) | yüzde ... | E.g., 百分之五十 → yüzde elli |
| 2025年1月15日 | 15 Ocak 2025 | Turkish: DD Month YYYY (month name, not number by default) |
| 2025-01-15 | 15.01.2025 | ISO → Turkish numeric date |
| 2025/01/15 | 15.01.2025 | Slash-separated → dot-separated |
| 1月 (January) | Ocak | Turkish month names are lowercase unless sentence-initial |
| 第1 (ordinal marker) | 1. (with dot) | Turkish uses a dot after the numeral for ordinals |
| 第一 | 1. / birinci | Can use numeral+dot or written-out form |
| 1点半 (1:30) | 01:30 / 1.30 | Turkish uses 24-hour format or colon |
| 上午9点 (9 AM) | 09:00 | Turkish uses 24-hour format exclusively in formal writing |
| 下午3点 (3 PM) | 15:00 | Convert PM to 24-hour clock |
| 电话号码: 12345678 | Telefon: 12345678 | Same format; area codes may need +90 prefix |
| 邮政编码 (100000) | Posta kodu: 100000 | Chinese postal codes are 6 digits; Turkish are 5 digits; flag for review |
| 平方米 (m²) | m² | Same unit symbol |

## 3. Cultural References and Idioms

Chinese idioms (成语 chéngyǔ) and culturally specific expressions rarely have direct Turkish equivalents. Prefer functional translation or find a matching Turkish proverb/deyim.

| Chinese | Turkish Equivalent | Literal / Notes |
|---|---|---|
| 画蛇添足 (huà shé tiān zú) | Gerekmedikçe işe karışma / Fazladan iş yapma | "Drawing legs on a snake" = unnecessary addition |
| 对牛弹琴 (duì niú tán qín) | Deveye hendek atlatmak / Neye uğradığını şaşırmak | "Playing music to a cow" = explaining to the wrong audience |
| 井底之蛙 (jǐng dǐ zhī wā) | Havuza düşen kurbağa / Dar görüşlü | "Frog at the bottom of a well" = narrow-minded |
| 千里送鹅毛 (qiān lǐ sòng é máo) | Uzaktan davulun sesi hoş gelir | "Sending a goose feather from 1000 li away" = the thought counts |
| 守株待兔 (shǒu zhū dài tù) | Boş bekleyen boş kazanır / Tembellik | "Waiting by the tree stump for rabbits" = waiting idly for luck |
| 掩耳盗铃 (yǎn ěr dào líng) | Deve kuşu gibi başını kuma gömmek | "Covering ears to steal a bell" = self-deception |
| 亡羊补牢 (wáng yáng bǔ láo) | Zararın neresinden dönülse kardır | "Mending the pen after the sheep is lost" = better late than never |
| 塞翁失马 (sài wēng shī mǎ) | Her şeyde bir hayır vardır | "The old man lost his horse" = every cloud has a silver lining |
| 笨鸟先飞 (bèn niǎo xiān fēi) | Erken kalkan yol alır | "Clumsy bird flies first" = early preparation |
| 入乡随俗 (rù xiāng suí sú) | Roma'da Romalılar gibi davran | "When in Rome, do as the Romans do" |
| 一举两得 (yī jǔ liǎng dé) | Bir taşla iki kuş vurmak | "One move, two gains" = kill two birds with one stone |
| 半途而废 (bàn tú ér fèi) | İşten artmaz, dişten artar (contextual) | "Give up halfway" = leave unfinished |
| 不可思议 (bù kě sī yì) | Akıl almaz / İnanılmaz | "Unimaginable" — not culturally specific, direct translation works |
| 马马虎虎 (mǎ mǎ hǔ hǔ) | İdare eder / Şöyle böyle | "Horse horse tiger tiger" = so-so, passable |
| 加油 (jiā yóu) | Kolay gelsin / Haydi / Başarılar | "Add oil" = encouragement; use Turkish equivalents |
| 面子 (miànzi) | İtibar / Onur / Saygınlık | "Face" (social standing); translate concept, not word |
| 关系 (guānxi) | Bağlantı / İlişki / Torpil (negative) | "Connections"; use context-appropriate term |
| 龙 (lóng) | Ejderha | Chinese lóng is benevolent vs. Turkish ejderha may be neutral/negative; flag for review |
| 风水 (fēngshuǐ) | Feng shui (loanword) | Borrowed term accepted; explain if needed |
| 功夫 (gōngfu) | Kung fu / çaba (figurative) | Loanword for martial arts; use çaba for "effort/skill" sense |
| 红包 (hóngbāo) | Kırmızı zarf / hediye parası | Explain as "red envelope with money for special occasions" |

## 6. Punctuation

Chinese and Turkish punctuation differ in several key areas.

| Chinese | Turkish | Rule |
|---|---|---|
| 。 (period) | . | Chinese full-width period → standard Latin period |
| ， (comma) | , | Chinese full-width comma → standard Latin comma |
| 、 (enumeration comma) | , (Latin comma) | Chinese dunhao (、) has no Turkish equivalent; use standard comma |
| "" (double quotation marks) | "" (Turkish quotation marks) | Chinese "" (full-width) → Turkish "" (standard double quotes) |
| '' (single quotation marks) | '' | Same as above |
| 《》(book title marks) | "italics" or "..." | No direct equivalent; use italics or standard quotes |
| 【】(bracket) | [] | Chinese square brackets → Turkish square brackets |
| —— (em dash, 2-wide) | — (em dash, standard) | Chinese double em dash → single em dash |
| …… (ellipsis, 6 dots) | ... (3 dots) | Chinese six-dot ellipsis → standard three-dot ellipsis |
| · (middle dot) | - (hyphen) or omit | Used in Chinese for translated foreign names; adapt to Turkish convention |
| No space after punctuation | Space after punctuation | Turkish requires a space after ., , : ; ? ! |
| ： (full-width colon) | : | Full-width → half-width |
| ； (full-width semicolon) | ; | Full-width → half-width |
| ？ (full-width question mark) | ? | Full-width → half-width |
| ！ (full-width exclamation) | ! | Full-width → half-width |
| （） (full-width parentheses) | () | Full-width → half-width |
| — (number dash in ranges) | – (en dash) | Use en dash for number ranges (e.g., 5–10) |
| No hyphenation at line breaks | Hyphenation allowed | Turkish allows word splitting at line breaks |
| % (percentage, no space before) | % (space before in TR) | Turkish convention places a space before % |
