# Türkçe Hukuki Belge Çeviri Alan Bilgisi

## Okuyucu Modeli (Reader Model)

### Hedef Okuyucu Profili
- **Birincil Okuyucu**: Türk avukatlar, hukuk danışmanları, noterler, hâkimler, arabulucular
- **İkincil Okuyucu**: Şirket hukuk müşavirleri, sözleşme yöneticileri, insan kaynakları uzmanları
- **Üçüncül Okuyucu**: Mahkemeler, tahkim heyetleri, icra daireleri, kamu denetçileri

### Okuyucu Beklentileri
1. **Terminoloji Kesinliği**: Her hukuki terimin kanunlarda tanımlanmış karşılığı kullanılmalı, yakın anlamlı terimler karıştırılmamalıdır
2. **Türk Borçlar Kanunu (TBK) Uyumu**: Sözleşme hukuku terimleri TBK m.1-649 arasındaki tanımlara sadık kalmalıdır
3. **Cümle Yapısı**: Türk hukuk dilinde uzun, bağlaçlı, dönük cümle yapıları yaygındır; kısa cümleler bazen yetersiz kalır
4. **Resmî Yazışma Usulü**: Dilekçe, sözleşme, ihtar gibi belgelerde belirli şablonlar kullanılır
5. **Latin Kökenli Terimler**: Hukuk Latincesi terimler Türkçede yerleşmiş biçimleriyle kullanılır (ex parte, prima facie vb.)

### Okuyucu Beklenti Matrisi

| Beklenti | Önem Derecesi | Karşılanmazsa Sonuç |
|----------|--------------|---------------------|
| Kanuni terminoloji uyumu | Kritik | Hukuki geçersizlik, davanın kaybı |
| TBK/TTK/TMK terimleri | Kritik | Sözleşme ihtilafı |
| Resmî yazışma formatı | Yüksek | Belgenin işleme alınmaması |
| Latin terimlerin doğru kullanımı | Yüksek | Anlam kayması, yanlış yorum |
| Dönük cümle yapısı | Orta | Doğal olmayan metin |
| Islah/ıslah ayrımı | Kritik | Hukuki anlam farkı |

### Tablo L12: Çeviride Kritik Anlam Farkı Olan Terimler

| Terim Çifti | Fark | Doğru Kullanım |
|-------------|------|----------------|
| **Ceza** vs **Tazminat** | Ceza = punitif (kamu), Tazminat = compensatory (özel hukuk) | "Penalty" = "ceza" değil "cezai şart" (TBK) |
| **Feragat** vs **Vazgeçme** | Feragat = usuli (davadan), Vazgeçme = maddi (haktan) | HMK m.307 vs m.310 |
| **Fesih** vs **İptal** | Fesih = ileriye etkili, İptal = geçmişe etkili | Sözleşme sona erme türleri |
| **Butlan** vs **İptal edilebilirlik** | Butlan = kesin hükümsüz, İptal = dava yoluyla geçersiz | TBK m.27 vs m.39 |
| **Temerrüt** vs **Kusur** | Temerrüt = ihtar gerektirir, Kusur = kusursuz sorumluluk da var | TBK m.117 vs m.49 |
| **Islah** vs **Düzeltme** | Islah = HMK'da usuli işlem, Düzeltme = genel | HMK m.176-187 |
| **İsticvap** vs **Sorgu** | İsticvap = medeni hukuk, Sorgu = ceza hukuku | HMK vs CMK |
| **Mübayaa** vs **Satın alma** | Mübayaa = eski kullanım, Satın alma = güncel | TBK güncel tercih |
| **Akdetmek** vs **Yapmak** | Akdetmek = sözleşme için, Yapmak = genel | Sözleşme akdedilir, iş yapılır |
| **İbra** vs **İnfaz** | İbra = borçtan kurtarma, İnfaz = cezanın çekilmesi | Ayrı alanlar |

## Standartlar (Standards)

### Türk Hukuk Sistemi Yapısı

**Anayasal Düzen (1982 Anayasası)**
- Cumhurbaşkanlığı Hükûmet Sistemi (2017 değişikliği ile)
- Kanunların Anayasa'ya uygunluğu denetimi: Anayasa Mahkemesi
- Temel hak ve özgürlükler: Anayasa m.12-74

**Kanunlaşma Süreci**
- Kanun teklifi: Milletvekilleri veya Cumhurbaşkanı
- Kanunun yayımı: Resmî Gazete'de yayım tarihi
- Yürürlük: Kanunda belirtilen tarih, belirtilmemişse yayımı izleyen 45. gün

**Normlar Hiyerarşisi**
1. Anayasa (1982 Anayasası)
2. Kanun (TBK, TMK, TTK, HMK, CMK vb.)
3. Cumhurbaşkanlığı Kararnamesi
4. Yönetmelik
5. Tebliğ / Genelge

### Yargı Sistemi

**Adli Yargı (Hukuk + Ceza)**
- İlk Derece Mahkemeleri (Hukuk Mahkemeleri, Ceza Mahkemeleri)
- Bölge Adliye Mahkemeleri (İstinaf)
- Yargıtay (Temyiz)

**İdari Yargı**
- İdare Mahkemeleri
- Bölge İdare Mahkemeleri
- Danıştay

**Anayasa Yargısı**
- Anayasa Mahkemesi (Bireysel başvuru, norm denetimi)

**Alternatif Uyuşmazlık Çözümü**
- Arabuluculuk (HUAK, 6325 sayılı)
- Tahkim (ICC, ISTAC, TTK m.)
- Uzlaştırma (CMK)

### Hukuki Belge Çeviri Standartları

1. **Yeminli Tercüman Zorunluluğu**: Resmî belgelerde noter onaylı tercüme
2. **Apostil Şerhi**: Lahey Sözleşmesi'ne taraf ülkelerde
3. **Aslına Uygunluk Beyanı**: "İşbu çeviri, aslına uygundur" ibaresi
4. **Boşluk Doldurma Yasağı**: Anlam boşluğu varsa dipnot eklenir
5. **Değişiklik Gösterimi**: Sözleşme değişikliklerinde üstü çizili/kırmızı metin
6. **İmza Çevirisi**: İmzalar çevrilmez, yalnızca imza sahibinin adı yazılır
