# Dutch Adaptation Guide: English -> Dutch

## 1. Formality

| English | Dutch Translation | Notes |
|---|---|---|
| You (formal) | u | Capitalise _U_ in formal correspondence; lowercase _u_ in body text |
| You (informal) | je / jij | _Jij_ stressed, _je_ unstressed; _jij/jouw_ for emphasis |
| Your (formal) | uw | _Uw brief van 15 mei_ |
| Your (informal) | jouw / je | _Jouw_ stressed, _je_ unstressed |
| Dear Sir / Madam | Geachte heer/mevrouw | Salutation; note _Geachte_ not _Beste_ in very formal letters |
| Dear Mr. Smith | Geachte heer Smith | Full salutation with surname |
| Dear Ms. Jones | Geachte mevrouw Jones | _Mevrouw_ not abbreviated in salutation |
| Dear Dr. Brown | Geachte dr. Brown | Academic titles retained |
| Dear Prof. White | Geachte professor White | _Professor_ not abbreviated in salutation |
| Hi / Hello | Hallo / Hoi / Beste | _Beste_ is neutral-friendly; _Hoi_ is informal |
| Sincerely | Hoogachtend | Formal closing (letter) |
| Kind regards | Met vriendelijke groet | Semi-formal closing |
| Best regards | Met vriendelijke groet | Same as Kind regards |
| Cheers | Groetjes / Doei | Informal closing |
| Mr. | de heer (dhr.) | Abbreviated in running text: _dhr. Jansen_ |
| Ms. / Mrs. | mevrouw (mw.) | _Mw. de Wit_ (no distinction between Ms./Mrs. in NL) |
| Ladies and gentlemen | Dames en heren | Note: _dames_ before _heren_ |
| Please | alstublieft (formal) / alsjeblieft (informal) | Distinguish register; attach to request tone |
| Thank you | dank u (formal) / dank je (informal) | |
| You're welcome | Graag gedaan / Geen dank | |

## 2. Dates

| English | Dutch Translation | Notes |
|---|---|---|
| MM/DD/YYYY | DD-MM-YYYY | e.g. 05/25/2025 → 25-05-2025 |
| May 25, 2025 | 25 mei 2025 | NL: day + month name + year; no comma |
| January 1, 2025 | 1 januari 2025 | Month names lowercase in NL |
| 2024-2025 | 2024-2025 | Same range format |
| 9:00 AM | 9:00 uur / 09:00 | _uur_ optional; 24h preferred in formal texts |
| 3:30 PM | 15:30 | 24-hour clock; use colon (not dot) |
| 12:00 PM | 12:00 uur (middag) | Clarify: _12.00 uur 's middags_ if ambiguity |
| 12:00 AM | 00:00 uur (middernacht) | Clarify: _middernacht_ |
| 1:00 PM | 13:00 | Always convert PM hours |
| 11:45 AM | 11:45 | Before noon stays the same |
| 2:30 PM EST | 14:30 uur (EST) | Retain timezone abbreviation |
| March | maart | Note: _maart_ not _Maart_ (months lowercase) |
| April | april | |
| August | augustus | |
| The 1990s | de jaren 1990 | No apostrophe: _jaren 1990_ (not _jaren '90_) |
| 21st century | de 21e eeuw / de eenentwintigste eeuw | Ordinal with _e_ suffix |
| Week 12 | week 12 | Same format |
| Q1 2025 | eerste kwartaal 2025 | Or _Q1 2025_ is understood in business NL |

## 4. Terminology

| English | Dutch Translation | Notes |
|---|---|---|
| computer | computer | Same spelling; NL pronunciation /kɔmˈpjutər/ |
| software | software | _de software_ or _het software_; plural _softwares_ rare |
| hardware | hardware | _de hardware_ |
| website | website | Same word; _de website_ (not _web site_) |
| email / e-mail | e-mail | Hyphenated in NL; _de e-mail_ |
| internet | internet | _het internet_ (lowercase) |
| login / log in | inloggen / aanmelden | Verb: _inloggen_; noun: _login_ or _inloggegevens_ |
| password | wachtwoord | Not _paswoord_ |
| username | gebruikersnaam | _gebruikersnaam_ (not _username_) |
| cloud | cloud / de cloud | _de cloud_ is common; prefer _cloudomgeving_ in formal texts |
| database | database | _de database_; plural _databases_ |
| server | server | _de server_; plural _servers_ |
| network | netwerk | _het netwerk_ (not _network_) |
| printer | printer | _de printer_ |
| scanner | scanner | _de scanner_ |
| attachment | bijlage | _de bijlage_ (not _attachment_) |
| folder | map | _de map_ (not _folder_ in standard NL) |
| file | bestand | _het bestand_ |
| screenshot | screenshot / schermafbeelding | _Screenshot_ common; _schermafbeelding_ formal |
| update | update / bijwerking | _Update_ is common; _bijwerken_ as verb |
| backup | back-up | With hyphen in NL; _de back-up_, plural _back-ups_ |

## 5. Administrative

| English | Dutch Translation | Notes |
|---|---|---|
| shall | moet / dient te | _Dient te_ is formal; _moet_ is neutral |
| shall not | mag niet / dient niet te | Prohibition; formal variant: _dient niet te_ |
| must | moet / is verplicht | Strong obligation |
| may | mag / kan | Permission or possibility |
| should | zou moeten / behoort te | Recommendation |
| will | zal / zullen | Future or intention |
| party (in contract) | partij | _de partij_ (legal entity) |
| agreement | overeenkomst | _de overeenkomst_; _akkoord_ is less formal |
| contract | contract / overeenkomst | Both are used |
| pursuant to | overeenkomstig / krachtens | Formal legal reference |
| in accordance with | in overeenstemming met | Compliance |
| notwithstanding | niettegenstaande / onverminderd | _Niettegenstaande_ formal; _onverminderd_ in legal clauses |
| hereunder | hieronder / onderstaand | Below (in document) |
| hereto | hieraan / aan deze | Formal; often replaced with _hieraan_ |
| thereof | daarvan / hiervan | Formal; _overeenkomstig artikel 5 daarvan_ |
| hereby | hierbij | _Hierbij verklaart partij A..._ |
| whereas | overwegende dat | Recital clause opening |
| execute (a document) | ondertekenen | Sign; not _executeren_ |
| effective date | ingangsdatum | _de ingangsdatum_ |
| termination | beëindiging / opzegging | _Beëindiging_ general; _opzegging_ by notice |

## 6. Punctuation

| English | Dutch | Notes / Examples |
|---|---|---|
| " " (curly double quotes) | " " or „ " | NL convention: „ “ (low-9 opening, high-9 closing) or regular " "; be consistent |
| ' ' (single quotes) | ' ' | Same; often used for quotes within quotes |
| , (comma before coordinating conj.) | , (no serial comma) | Dutch typically omits comma before _en/of_: _appels, peren en bananen_ |
| , (Oxford comma) | ❌ | Do not use Oxford comma in NL |
| . (decimal point) | , (decimal comma) | 3.14 → 3,14 |
| , (thousands separator) | . (dot) or space | 1,000 → 1.000 or 1 000 |
| $1,000 | $ 1.000 / € 1.000 | Currency symbol with space before amount? NL: _€ 1.000_ (space) |
| $ | US$ / Amerikaanse dollar | Clarify currency when needed; € for euro |
| @ (at sign) | @ (apenstaartje) | Called _apenstaartje_ in NL |
| # (hash) | # (hekje / hashtag) | _Hekje_ in NL |
| / (slash) | / (schuine streep) | Same; avoid in formal text where possible |
| - (hyphen) | - (koppelteken) | Same; used in compound words like _e-mail_, _back-up_ |
| — (em-dash) | – (en-dash) or — (em-dash) | NL prefers spaced en-dash: _Amsterdam – Rotterdam_ |
| … (ellipsis) | ... (3 dots) | Same; optional space before in NL |
| ! (exclamation) | ! | Use sparingly; NL same as EN |
| ? (question mark) | ? | Same |
| ( ) (parentheses) | ( ) | Same; no space before opening paren in NL |
| ; (semicolon) | ; | Used similarly but less frequent in NL |
| : (colon) | : | Lowercase after colon in NL (unless proper noun) |
| % (percent) | % | NL convention: space before % (10 %) |
| °C | °C | NL space before °C (25 °C) |
| No unit space (50ml) | Space before unit (50 ml) | NL always inserts space: _50 ml_ |
