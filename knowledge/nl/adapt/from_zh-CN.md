# Dutch Adaptation Guide: Chinese (Simplified) -> Dutch

## 1. Formality

| Chinese (Simplified) | Dutch Translation | Notes |
|---|---|---|
| 您 | u | Formal "you"; always capitalise _u_ in correspondence |
| 你 | je / jij | Informal "you"; _jij_ is stressed, _je_ is unstressed |
| 你好 | Hallo / Hoi | Neutral-to-informal greeting |
| 您好 | Geachte / Beste | Formal salutation in letters/email |
| 先生 | de heer (dhr.) | Used before a surname, e.g. _de heer Wang_ |
| 女士 | mevrouw (mw.) | Used before a surname, e.g. _mevrouw Li_ |
| 同志 | (omit / kameraad) | Archaic in NL; omit or use _collega_ if workplace |
| 师傅 | meneer / mevrouw | Polite address for a worker/driver (no direct equivalent) |
| 老师 | leraar / docent | "Teacher" as a title is not used in NL address; use _docent_ when referring |
| 王总 | directeur Wang | _总_ as title → NL job title (directeur, manager, etc.) |
| 李经理 | manager Li | Keep surname + Dutch job title |
| 张博士 | dr. Zhang | Retain academic titles; note lowercase _dr._ in NL |
| 请 | alstublieft / alsjeblieft | Formal vs informal "please" |
| 谢谢 | dank u / dank je | Formal vs informal "thank you" |
| 不客气 | graag gedaan / geen dank | Corresponding polite replies |
| 贵公司 | uw bedrijf | _贵_ (honorific "your") → simply _uw_ |
| 敝公司 | ons bedrijf | _敝_ (humble "our") → simply _ons/onze_ |

## 2. Numbers

| Chinese (Simplified) | Dutch Translation | Notes |
|---|---|---|
| 万 | 10.000 | _tienduizend_; never use _10 duizend_ |
| 亿 | 100.000.000 | _honderd miljoen_ |
| 万亿 | 1.000.000.000.000 | _biljoen_ (NL short scale; note EN _trillion_) |
| 5万 | 50.000 | _vijftigduizend_ |
| 3亿 | 300.000.000 | _driehonderd miljoen_ |
| 2025年5月25日 | 25-05-2025 | Ordnung: DD-MM-YYYY |
| 2025年12月1日 | 01-12-2025 | Always pad day/month to 2 digits |
| 2020年代 | de jaren 2020 | _jaren_ + year; no apostrophe (not _jaren '20_) |
| 50% | 50% | Same symbol; note NL space before _%_ in running text (50 %) |
| 1.5万 | 15.000 | _vijftien duizend_ |
| 2.5亿 | 250.000.000 | _tweehonderdvijftig miljoen_ |
| 第1 | 1e / 1ste | Ordinal suffix: _1e_ common, _1ste_ formal |
| 第2 | 2e / 2de | Ordinal suffix: _2e_ or _2de_ |
| 电话号码 | telefoonnummer | Format +31 (0)6 12 34 56 78 |
| 10多 | meer dan 10 / 10+ | Approximations: _meer dan_ or _circa_ |
| 几十 | tientallen | _tientallen_ (tens of) |
| 几百 | honderden | _honderden_ (hundreds of) |
| 上千 | duizenden | _duizenden_ (thousands of) |

## 4. Terminology

|---|---|---|
| 计算机 | computer | Same word; NL pronunciation: /kɔmˈpjutər/ |
| 软件 | software | Same word, neuter gender: _het software_ (but _de software_ also used) |
| 硬件 | hardware | Same word |
| 数据 | gegevens / data | _data_ common in IT; _gegevens_ in administrative context |
| 信息 | informatie | Standard; avoid _informatie_ for large data sets |
| 算法 | algoritme | _algoritme_ (het); plural _algoritmen_ or _algoritmes_ |
| 下载 | downloaden | Dutch verb: _ik download, hij downloadt_ |
| 上传 | uploaden | Dutch verb: _uploaden, geüpload_ |
| 邮件 | e-mail | With hyphen in NL (not _email_) |
| 互联网 | internet | _het internet_ (always lowercase) |
| 打印机 | printer | Same word |
| 文件 | bestand / document | _bestand_ for computer files; _document_ for paper-like docs |
| 文件夹 | map | _map_ (not _folder_ in standard NL) |
| 点击 | klikken | _klik op_ (not _click op_) |
| 密码 | wachtwoord | _wachtwoord_ (not _paswoord_) |

| 浏览器 | browser | Same word; NL verb: _rondstruinen / browsen_ |

## 6. Punctuation

| Chinese | Dutch | Examples / Notes |
|---|---|---|
| 。(full stop) | . (punt) | Same glyph, NL uses no extra space |
| ，(comma) | , (komma) | NL same comma symbol |
| 、(enumeration comma) | , (komma) | Always replace with Dutch comma |
| ；(semicolon) | ; (puntkomma) | Same; NL uses sparingly |
| ：(colon) | : (dubbele punt) | Same symbol; lowercase after colon in NL |
| "" (curly quotes, fullwidth) | " " or „ " | NL prefers „ “ (low-9 opening, high-9 closing) or regular " " |
| （ ）(fullwidth parens) | ( ) (halfwidth parens) | Always convert to halfwidth |
| 【 】(brackets) | [ ] (rechte haken) | Replace with square brackets |
| 《 》(book title marks) | " " or cursief | Book titles: use _cursief_ (italics) or regular quotes |
| ——(em-dash, 2汉字宽) | — (halfwidth em-dash) | NL uses spaced en-dash or unspaced em-dash |
| ……(6-dot ellipsis) | ... (3-dot ellipsis) | NL uses 3 dots, optionally with space before |
| ·(middle dot) | - (koppelteken) / · (middenpunt) | Used in translated names; keep as middenpunt if needed |
| ～(wave dash) | tot / ongeveer | Replace with _tot_ (range) or _circa_ (approximate) |
| No space before `%` | Optional thin space | NL convention: space before % (10 %) |
| No space before `℃` | Space before °C | 25 °C |
| No space before `ml` | Space before unit | 50 ml |
| 《About》 | "Over" or _Over_ | Use Dutch quotation or italics for titles |
| 、、 | , (single comma) | Multiple enumeration commas merge into one |
