# Chinese (Simplified) to Korean Adaptation Guide

> Target language: Korean (한국어 / 韓國語)  
> Source language: Chinese Simplified (简体中文 / 簡體中文)  
> Purpose: Standardized adaptation rules for Chinese-to-Korean document translation  
> Last updated: 2026-05-25

---

## 1. Formality: Chinese Honorifics to Korean 존댓말

### Overview

Chinese has a limited honorific system (primarily 您 and a few polite particles), while Korean has an elaborate honorific system (경어법 / 敬語法) with multiple speech levels. The primary formal levels used in formal document translation are **하십시오체** (extremely formal) and **해요체** (polite informal). This section covers how to map Chinese formality cues into the Korean honorific system.

### Rule 1.1: Chinese 您 → Korean honorific verb endings

The Chinese honorific pronoun 您 has no direct Korean equivalent. Korean expresses respect through verb endings (the `-시-` infix) and formal sentence endings rather than through a separate pronoun.

| Chinese | Korean (하십시오체) | Korean (해요체) | Notes |
|---------|-------------------|-----------------|-------|
| 您请坐 | 앉으십시오 | 앉으세요 | Use `-시-` honorific infix |
| 您要什么？ | 무엇을 원하십니까? | 무엇을 원하세요? | Question with honorific |
| 您先走 | 먼저 가십시오 | 먼저 가세요 | Direction with honorific |
| 您贵姓？ | 성함이 어떻게 되십니까? | 성함이 어떻게 되세요? | Name inquiry (uses humble form) |
| 您好 | 안녕하십니까? | 안녕하세요? | Standard greeting |

**Rule:** Never translate 您 as a separate pronoun like `당신` in formal contexts (당신 can be confrontational in Korean). Instead, embed respect through the honorific suffix `-시-` and formal endings `-(스)ㅂ니다` / `-요`.

### Rule 1.2: Chinese 请 → Korean request forms

Chinese 请 is a versatile politeness marker. Korean distinguishes multiple levels of request depending on formality.

| Chinese | Korean | Formality | Context |
|---------|--------|-----------|---------|
| 请稍等 | 잠시만 기다려 주십시오 | Highest (하십시오체) | Formal wait request |
| 请进 | 들어오십시오 | Highest | Formal entry invitation |
| 请坐 | 앉으십시오 | Highest | Very formal seating |
| 请坐 | 앉으세요 | Mid-high (해요체) | General polite seating |
| 请告诉我 | 알려 주십시오 | Highest | Formal request for info |
| 请问 | 여쭙겠습니다 | Highest | "May I ask" (humble form) |
| 请确认 | 확인해 주시기 바랍니다 | Highest | Very formal (business writing) |
| 请参考附件 | 첨부 파일을 참고해 주십시오 | Highest | Attachment reference |
| 请勿吸烟 | 흡연하지 마십시오 | Highest | Prohibition (signage) |
| 请在这里签字 | 이곳에 서명해 주십시오 | Highest | Signature request |

**Rule:** Use `-아/어 주십시오` for formal documents, `-아/어 주시기 바랍니다` for extremely formal/official requests, and `-아/어 주세요` for general polite communication. Never translate 请 as 제발 (which implies desperation).

### Rule 1.3: Chinese honorific address nouns → Korean equivalents

Chinese uses specific honorific nouns for the addressee's affiliations and family members.

| Chinese | Korean | Korean (Hanja) | Notes |
|---------|--------|----------------|-------|
| 贵公司 | 귀사 | 貴社 | Formal company address |
| 贵校 | 귀교 | 貴校 | Formal school address |
| 贵国 | 귀국 | 貴國 | Formal country address |
| 贵方 | 귀측 | 貴側 | Formal counterparty |
| 阁下 | 각하 | 閣下 | Very formal, for dignitaries |
| 先生 | 선생님 | — | General respectful address |
| 女士 | ~님 / 숙녀 | — | Context-dependent; ~님 is safer |
| 令尊 | 댁의 부친 / 아버님 | — | "Your father" |
| 令堂 | 댁의 모친 / 어머님 | — | "Your mother" |
| 令郎 | 댁의 아드님 | — | "Your son" |
| 令爱 | 댁의 따님 | — | "Your daughter" |
| 您老 | 어르신 | — | For elderly persons |
| 贤弟 | ~군 / ~ 씨 | — | Addressing junior (archaic) |

### Rule 1.4: Chinese humble self-reference → Korean humble forms

Chinese uses humble self-references to show respect to the addressee. Korean has a parallel system.

| Chinese | Korean | Korean (Hanja) | Notes |
|---------|--------|----------------|-------|
| 本人 (formal I) | 본인 | 本人 | Official self-reference |
| 我 (I) | 저 / 제가 | — | Humble form of 나 |
| 我们 (we) | 저희 | — | Humble form of 우리 |
| 敝公司 (our company) | 당사 / 저희 회사 | 當社 | Formal humble reference |
| 敝校 (our school) | 본교 | 本校 | Formal humble reference |
| 拙作 (my work) | 졸작 | 拙作 | Humble reference to own work |
| 愚见 (my opinion) | 우견 / 제 의견 | 愚見 | Humble opinion |
| 寒舍 (my home) | 저희 집 | — | Humble home reference |
| 贱内 (my wife) | 제 아내 | — | Archaic humble wife reference |
| 小弟 (I, junior) | 저 / 후배 | — | Junior's self-reference |

**Rule:** Match the Korean humble form to the Chinese humble form level. `저` is the standard humble equivalent of Chinese `我`.

### Rule 1.5: Chinese formal sentence endings → Korean formal endings

Chinese uses specific formal closing formulas. Korean has corresponding formulas with different grammar structure.

| Chinese | Korean | Context |
|---------|--------|---------|
| 特此通知 | 이에 통지합니다 / 이에 알려드립니다 | Official notice opening/closing |
| 特此声明 | 이에 밝힙니다 / 이에 선언합니다 | Formal declaration |
| 特此证明 | 이를 증명합니다 | Certificate closing |
| 为盼 | 바랍니다 / 부탁드립니다 | Expressing expectation |
| 此致 | 이만 줄입니다 / 이상입니다 | Letter closing (brief) |
| 此致敬礼 | 이만 줄이며, 안녕히 계십시오 | Full formal letter closing |
| 敬礼 | 안녕히 계십시오 | Respectful salutation in letters |
| 谨上 | 올림 / 드림 | Sign-off in formal letters |
| 顺颂商祺 | 번창하시길 기원합니다 | Business correspondence closing |
| 专此函达 | 이에 특별히 알려드립니다 | Special notice |
| 特此批复 | 이에 회신합니다 | Official reply closing |
| 请予批准 | 승인하여 주시기 바랍니다 | Request for approval |
| 妥否，请批示 | 적절한지 검토하여 지시해 주시기 바랍니다 | Request for instructions |

### Rule 1.6: Chinese 的 in formal context → Korean renderings

The ubiquitous Chinese particle 的 maps to several Korean grammatical forms depending on function.

| Chinese function | Korean | Example (Chinese → Korean) |
|-----------------|--------|---------------------------|
| Possessive (~的) | ~의 | 公司的决定 → 회사의 결정 |
| Attributive (的 → adjective) | ~ㄴ/은/는 | 重要的项目 → 중요한 프로젝트 |
| Descriptive (的 → ~적) | ~적인 | 科学的分析 → 과학적인 분석 |
| Nominalizer (的 → ~것) | ~것 / ~거 | 我要的 → 제가 원하는 것 |
| Relative clause (V+的+N) | ~는/은 + N | 来的人 → 오는 사람 |
| Modal (会/能+的) | ~ㄹ/을 거예요 | 他会来的 → 그가 올 거예요 |
| Time phrase (的) | omit | 昨天来的 → 어제 온 |

**Rule:** Chinese 的 should NOT be mechanically translated as 의. Analyze the grammatical function and render accordingly.

### Rule 1.7: Chinese address forms → Korean address forms

Chinese address patterns (title + surname) differ from Korean address patterns.

| Chinese | Korean | Notes |
|---------|--------|-------|
| 王经理 | 왕 과장님 / 왕 부장님 | Korean uses position title + 님 |
| 李老师 | 이 선생님 | Teacher/professor + 님 |
| 张总 | 장 사장님 | General manager → 사장님 |
| 赵博士 | 조 박사님 | Doctor + 님 |
| 刘教授 | 유 교수님 | Professor + 님 |
| 小张 (junior) | 장 군 / 장 씨 | Addressing junior familiarly |
| 老李 (senior) | 이 선배님 / 이 씨 | Addressing senior familiarly |
| 王工 (engineer) | 왕 엔지니어님 | Technical position |

**Rule:** In Korean, always attach `님` to titles when addressing someone respectfully. Chinese surname + position → Korean surname + position + 님.

### Rule 1.8: Chinese command vs Korean request

Chinese direct imperatives are softened in Korean to request forms.

| Chinese command | Korean request | Tone level |
|----------------|---------------|------------|
| 马上做！ | 즉시 해 주십시오 | Formal request |
| 进来！ | 들어오십시오 / 들어오세요 | Polite request |
| 看这里 | 이쪽을 봐 주십시오 | Polite request |
| 签字！ | 서명해 주시기 바랍니다 | Very formal request |
| 安静！ (in writing) | 조용히 해 주시기 바랍니다 | Written notice |
| 不要说话！ | 말씀하지 마십시오 | Formal prohibition |
| 排队！ | 줄을 서 주십시오 | Polite direction |
| 停下！ | 멈추어 주십시오 | Polite request to stop |

### Rule 1.9: Chinese address in official letters → Korean letter format

| Chinese | Korean | Notes |
|---------|--------|-------|
| 尊敬的…… | …… 귀하 / ……님께 | Formal salutation |
| 亲爱的…… (formal) | ……께 | Dear (formal, no direct equivalent) |
| 各位领导 | 관계 제위께 / 여러분 | Addressing a group |
| 同志们 | 동지 여러분 | Political group address |
| 全体同事 | 전 직원 여러분 | All colleagues |

**Rule:** Korean formal letters use `귀하` or `님께` rather than a direct equivalent of "Dear". The salutation is often omitted entirely in Korean formal documents.

### Rule 1.10: Chinese 你们 → Korean polite plural address

| Chinese | Korean | Context |
|---------|--------|---------|
| 你们 (general) | 여러분 | Polite group address |
| 你们公司 | 귀사 / 여러분 회사 | Company address |
| 你们国家 | 귀국 | Country address |
| 你们部门 | 귀 부서 | Department address |

**Rule:** 你们 in formal context → 여러분 or 귀-. Avoid using 너희 or 너희들 in formal settings.

## 2. Numbers: Chinese Numeric Expressions to Korean

### Overview

Chinese and Korean share sino-Korean numerals (일, 이, 삼, 사... / 一, 二, 三, 四...), but numeric conventions differ in date format, unit grouping, and usage patterns. Korean also has native numerals (하나, 둘, 셋...) for certain contexts.

### Rule 2.1: Chinese 万 → Korean 만

Chinese uses 万 as a base unit at 10,000. Korean uses 만 (同源字: 萬) with identical value.

| Chinese | Korean | Numeric value |
|---------|--------|---------------|
| 一万 | 1만 (일만) | 10,000 |
| 十万 | 10만 (십만) | 100,000 |
| 一百万 | 100만 (백만) | 1,000,000 |
| 一千万 | 1,000만 (천만) | 10,000,000 |
| 五万 | 5만 (오만) | 50,000 |
| 五十万 | 50만 (오십만) | 500,000 |

**Rule:** 万 always becomes 만. Keep the Korean 4-digit grouping system (만, 억, 조).

### Rule 2.2: Chinese 亿 → Korean 억

| Chinese | Korean | Numeric value |
|---------|--------|---------------|
| 一亿 | 1억 (일억) | 100,000,000 |
| 十亿 | 10억 (십억) | 1,000,000,000 |
| 一百亿 | 100억 (백억) | 10,000,000,000 |
| 一千亿 | 1,000억 (천억) | 100,000,000,000 |
| 两亿 | 2억 (이억) | 200,000,000 |
| 二十亿 | 20억 (이십억) | 2,000,000,000 |

### Rule 2.3: Chinese 万亿 → Korean 조

| Chinese | Korean | Numeric value |
|---------|--------|---------------|
| 一万亿 | 1조 (일조) | 1,000,000,000,000 |
| 十万亿 | 10조 (십조) | 10,000,000,000,000 |
| 一百万亿 | 100조 (백조) | 100,000,000,000,000 |

### Rule 2.4: Chinese date format → Korean date format

Chinese uses `YYYY年M月D日`. Korean uses `YYYY년 M월 D일`.

| Chinese | Korean | Notes |
|---------|--------|-------|
| 2024年1月1日 | 2024년 1월 1일 | Standard date conversion |
| 2024年01月01日 | 2024년 01월 01일 | With leading zeros (optional) |
| 2024年1月 | 2024년 1월 | Month only |
| 1月1日 | 1월 1일 | Without year |
| 2024年第一季度 | 2024년 1분기 | Quarter expression |
| 2024年上半年 | 2024년 상반기 | Half-year expression |
| 20世纪90年代 | 1990년대 | Century expression |
| 2024年1月1日至1月5日 | 2024년 1월 1일부터 1월 5일까지 | Date range |

**Rule:** Change 年→년, 月→월, 日→일. Keep Arabic numerals. The word order (Y/M/D) is the same as Chinese.

### Rule 2.5: Chinese ordinal 第 → Korean 제 or ~째

| Chinese | Korean | Context |
|---------|--------|---------|
| 第一 | 제1 / 첫째 | Articles/chapters (제) vs general (째) |
| 第二条 | 제2조 | Legal articles |
| 第一章 | 제1장 | Book chapters |
| 第一次 | 제1차 / 첫 번째 | First time/instance |
| 第十届 | 제10회 / 제10차 | 10th edition/conference |
| 第三代 | 3세대 / 제3세대 | Third generation |
| 第一点 | 첫째 / 첫 번째 | First point in a list |

**Rule:** Use `제` for numbered sections, articles, and official designations. Use `째` or `번째` for ordinal in prose.

### Rule 2.6: Chinese percentages (百分之) → Korean 퍼센트 / %

| Chinese | Korean | Notes |
|---------|--------|-------|
| 百分之十 | 10% / 10퍼센트 | Standard expression |
| 百分之百 | 100% / 100퍼센트 | 100 percent |
| 百分之三点五 | 3.5% | Decimal percentage |
| 增长了百分之五 | 5% 증가했다 | Growth rate |
| 占百分之二十 | 20%를 차지한다 | Proportion |
| 百分之零点一 | 0.1% | Fractional percentage |

**Rule:** 百分之X → X%. Never keep the Chinese 百分之 format in Korean text.

### Rule 2.7: Chinese 成 (10%) → Korean 할

| Chinese | Korean | Notes |
|---------|--------|-------|
| 七成 | 70% / 7할 (칠할) | Formal usage |
| 九成九 | 99% / 9할 9푼 | Colloquial |
| 十成 | 100% / 10할 | Complete |
| 五五分成 | 5:5로 나누다 / 절반씩 | Equal split |

**Rule:** 成 → 할 in formal context. For modern documents, % is preferred.

### Rule 2.8: Chinese fractions (几分之几) → Korean 분수

| Chinese | Korean | Notes |
|---------|--------|-------|
| 三分之一 | 3분의 1 | Standard |
| 四分之三 | 4분의 3 | Standard |
| 二分之一 | 2분의 1 / 반 | 반 (half) is common |
| 三分之二 | 3분의 2 | Standard |
| 十分之一 | 10분의 1 | Standard |
| 千分之一 | 1,000분의 1 | Standard |

**Rule:** X分之Y → X분의 Y. Word order is identical.

### Rule 2.9: Chinese 两 vs 二 → Korean

Chinese distinguishes 两 (counting/quantity) from 二 (ordinal/digit). Korean has different forms.

| Chinese | Korean | Context |
|---------|--------|---------|
| 两个人 | 두 사람 / 2명 | Counting people |
| 两个 | 두 개 / 2개 | Counting items |
| 两百 | 200 (이백) | Number 200 |
| 第二 | 두 번째 / 제2 | Ordinal |
| 二楼 | 2층 | Second floor |
| 二月 | 2월 (이월) | February |
| 两万 | 2만 (이만) | 20,000 |
| 两点钟 | 2시 (두 시) | Two o'clock |

**Rule:** Chinese 两 in counting → Korean 두 + classifier / Arabic numeral. Chinese 二 → Korean 이 + noun / Arabic numeral.

### Rule 2.10: Chinese approximate numbers → Korean

| Chinese | Korean | Notes |
|---------|--------|-------|
| 五六个 | 5~6개 / 대여섯 개 | Korean native: 대여섯 |
| 二十余人 | 20여 명 / 20명 이상 | Formal |
| 数十 | 수십 | Tens of |
| 数百人 | 수백 명 | Hundreds of people |
| 数千 | 수천 | Thousands of |
| 数万人 | 수만 명 | Tens of thousands |
| 十余年 | 10여 년 | More than ten years |
| 三十多年 | 30여 년 / 30년 이상 | Over thirty years |
| 大约五十个 | 약 50개 / 50개 정도 | Approximately |
| 近百个 | 100개 가까이 | Nearly 100 |

### Rule 2.11: Chinese 半 → Korean 반

| Chinese | Korean | Notes |
|---------|--------|-------|
| 一半 | 절반 / 반 | Half |
| 一个半 | 한 개 반 | One and a half |
| 半年 | 반년 | Half year (formal) |
| 半小时 | 30분 | 30 minutes (more common than 반시간) |
| 半个月 | 보름 / 반달 | 보름 = 15 days |
| 一半以上 | 절반 이상 / 반수 이상 | More than half |

### Rule 2.12: Chinese multiples (倍) → Korean 배

| Chinese | Korean | Example |
|---------|--------|---------|
| 三倍 | 3배 | 3배 증가 |
| 翻一番 | 2배가 되다 / 두 배 | Double |
| 翻两番 | 4배가 되다 | Quadruple |
| 翻三番 | 8배가 되다 | Octuple |
| 增长了五倍 | 5배 증가했다 | Five-fold increase |
| 五倍于 | ~의 5배 | Five times as much as |

**Rule:** 倍 → 배. Same concept and word order.

### Rule 2.13: Chinese time-of-day → Korean

| Chinese | Korean | Notes |
|---------|--------|-------|
| 上午8点 | 오전 8시 | AM |
| 下午3点 | 오후 3시 | PM |
| 早上 | 아침 / 오전 | Morning (6-9) |

| 晚上 | 저녁 / 밤 | Evening/night |

| 差五分三点 | 3시 5분 전 | Five to three |
| 三点过五分 | 3시 5분 | Five past three |

### Rule 2.14: Chinese 多 (more than) → Korean

| Chinese | Korean | Usage |
|---------|--------|-------|

## 3. Cultural: Chinese Cultural References to Korean

### Overview

Chinese and Korean share a common heritage of Classical Chinese (한문/漢文) and Sino-Korean vocabulary, but modern cultural references, idiomatic expressions, and institutional terms require careful adaptation. Some concepts have direct Korean equivalents; others require explanation or substitution.

### Rule 3.1: Chinese idioms (成语) → Korean 사자성어

Many Chinese idioms have word-for-word equivalent Korean 사자성어 (四字成语), often identical in Hanja form.

| Chinese 成语 | Korean 사자성어 | Meaning | Strategy |
|-------------|---------------|---------|----------|
| 一举两得 | 일거양득 (一舉兩得) | Kill two birds with one stone | Direct equivalent |
| 对牛弹琴 | 대우탄금 (對牛彈琴) | Cast pearls before swine | Direct equivalent |
| 画蛇添足 | 화사첨족 (畫蛇添足) | Gild the lily | Direct equivalent |
| 井底之蛙 | 정저지와 (井底之蛙) | Frog in a well | Direct equivalent |
| 亡羊补牢 | 망양보뢰 (亡羊補牢) | Better late than never | Direct equivalent |
| 塞翁失马 | 새옹지마 (塞翁之馬) | Blessing in disguise | Direct equivalent |
| 刻舟求剑 | 각주구검 (刻舟求劍) | Inflexible | Direct equivalent |
| 狐假虎威 | 호가호위 (狐假虎威) | Bully by proxy | Direct equivalent |
| 鹤立鸡群 | 학립계군 (鶴立雞群) | Stand out | Direct equivalent |
| 自相矛盾 | 자가당착 (自家撞着) | Self-contradiction | Slightly different form |
| 掩耳盗铃 | 엄이도령 (掩耳盜鈴) | Fooling oneself | Direct equivalent |
| 守株待兔 | 수주대토 (守株待兔) | Waiting idly | Direct equivalent |
| 叶公好龙 | 섭공호룡 (葉公好龍) | Pretend to like | Direct equivalent |
| 愚公移山 | 우공이산 (愚公移山) | Perseverance | Direct equivalent |
| 百发百中 | 백발백중 (百發百中) | Always right | Direct equivalent |

**Rule:** When a Chinese idiom has a standard Korean 사자성어 equivalent, use it. Korean 사자성어 is read in Korean Hanja pronunciation, not Chinese pinyin.

### Rule 3.2: Chinese idioms without direct 사자성어 → Korean proverb or explanation

| Chinese idiom | Korean rendering | Strategy |
|---------------|-----------------|----------|

### Rule 5.1: 根据/依据 → Korean 에 따라 / 에 의거하여

|---------|--------|-----------|-------|
| 根据规定 | 규정에 따라 | Standard | General reference |
| 根据法律 | 법률에 의거하여 | Very formal | Legal context |
| 根据合同 | 계약에 따라 | Standard | Contract reference |
| 根据上述 | 상기에 따라 | Formal | Referring to above |

**Rule:** 根据 → `-에 따라` (general) or `-에 의거하여` (formal/legal). 依据 follows the same pattern.

## 6. Punctuation: Chinese Punctuation to Korean

### Overview

Chinese uses exclusively fullwidth (全角) punctuation marks. Korean primarily uses halfwidth punctuation similar to English, with some exceptions. Systematic conversion is required.

### Rule 6.1: Chinese period 。 → Korean period .

Chinese uses `。` (full stop, fullwidth). Korean uses `.` (ASCII period/matkhimpyo).

| Chinese | Korean | Example |
|---------|--------|---------|
| 你好。 | 안녕하세요. | Simple sentence end |
| 再见。 | 안녕히 계세요. | Goodbye |
| 谢谢。 | 감사합니다. | Thanks |
| 请稍等。 | 잠시만 기다려 주십시오. | Request |
| 通知如下。 | 다음과 같이 통지합니다. | Document |

**Rule:** Convert all fullwidth `。` to halfwidth `.`. Exception: literary/calligraphic texts may retain `。`.

### Rule 6.2: Chinese enumeration comma 、 → Korean comma ,

Chinese uses `、` (顿号 / dunhao) for enumeration. Korean does not use `、` and uses `,` instead.

| Chinese | Korean | Notes |
|---------|--------|-------|
| 苹果、香蕉、橘子 | 사과, 바나나, 귤 | Fruit list |
| 第一、第二、第三 | 첫째, 둘째, 셋째 | Ordinal list |
| 北京、上海、广州 | 베이징, 상하이, 광저우 | City list |
| 语文、数学、英语 | 국어, 수학, 영어 | Subject list |
| 研究、开发、生产 | 연구, 개발, 생산 | Process list |
| 依照、参照、比照 | 준하여, 참작하여, 비교하여 | Legal reference list |

**Rule:** All Chinese `、` become Korean `,`. Korean never uses the enum comma character.

### Rule 6.3: Chinese quotation marks "" → Korean ""

Both Chinese and Korean use `""` for quotations. However, usage conventions differ slightly.

| Chinese | Korean | Notes |
|---------|--------|-------|
| 他说：“你好。” | 그는 "안녕하세요."라고 말했다. | Chinese uses colon before quote; Korean does not |
| 所谓“专家” | 이른바 "전문가" | Scare quotes — same |
| 所谓“三个代表” | 이른바 "삼개대표" | Political term quotes |
| 他问：“为什么？” | 그는 "왜요?"라고 물었다. | Question in quotes |

**Rule:** Korean does NOT put a colon before quoted speech. Use `라고 말하다/묻다/하다` after the quote instead. The quote content retains its own punctuation.

### Rule 6.4: Chinese book title marks 《》 → Korean

Chinese uses `《》` for book titles. Korean equivalents vary.

| Chinese | Korean option | Preferred |
|---------|---------------|-----------|
| 《红楼梦》 | 《홍루몽》 / 「홍루몽」 / "홍루몽" | 《》 for titles of Chinese works |
| 《人民日报》 | 《인민일보》 | 《》 for Chinese publications |
| 《XXX法》 | 「XXX법」 / 《XXX법》 | 「」 is standard for Korean laws |
| 《关于XXX的通知》 | "XXX에 관한 통지" | "" for document names |
| 《诗经》 | 《시경》 | 《》 for Chinese classics |

**Rule:** Keep `《》` for Chinese book/publication titles. For Korean laws, prefer `「」`. For document names, `""` is acceptable.

### Rule 6.5: Chinese dash/hyphen → Korean

| Chinese | Korean | Usage |
|---------|--------|-------|
| 2018–2020年 | 2018~2020년 | Range: ~ preferred in Korean |
| 5–10人 | 5~10명 | Quantity range |
| 北京—上海 | 베이징~상하이 | Route range |
| —— (em dash for interruption) | — | Same em dash usage |
| ——(引文破折号) | — | Same function |
| - (hyphen in compounds) | - | Same hyphen usage |

**Rule:** Range dash `–` or `-` → `~` in Korean. Em dash `—` is used identically in both languages for breaks/interruptions.

### Rule 6.6: Chinese ellipsis …… → Korean …

Chinese uses 6 dots (…… fullwidth). Korean standard uses 3 dots (…) or 6 dots (……).

| Chinese | Korean standard | Notes |
|---------|----------------|-------|
| 等等…… | 등등… | Korean prefers 3 dots |
| 原来如此…… | 그렇군요… | Trailing off |
| ……真是的 | …참… | Both work |

**Rule:** 6 dots → 3 dots in standard Korean usage. 6 dots are also acceptable but less common.

### Rule 6.7: Chinese interpunct · → Korean interpunct ·

| Chinese | Korean | Notes |
|---------|--------|-------|
| 玛丽·史密斯 | 메리·스미스 | Foreign name middle dot |
| 9·11事件 | 9·11 테러 | Date separator |
| 《诗经·国风》 | 《시경·국풍》 | Section separator in titles |

**Rule:** The middle dot `·` (가운뎃점) is used identically in both languages.

### Rule 6.8: Chinese colon ： → Korean colon :

Chinese uses fullwidth `：`. Korean uses halfwidth `:`.

| Chinese | Korean | Example |
|---------|--------|---------|
| 如下： | 다음과 같습니다: | Before list |
| 注意： | 주의: | Caution/note |
| 例如： | 예: | Example |
| 包括： | 포함: | Including |

**Rule:** Fullwidth `：` → halfwidth `:`. No space before colon in Korean.

### Rule 6.9: Chinese semicolon ； → Korean ;

| Chinese | Korean | Usage |
|---------|--------|-------|
| 第一，…；第二，… | 첫째, …; 둘째, … | Parallel clauses |
| 一方面…；另一方面… | 한편으로…; 다른 한편으로… | Contrast |

**Rule:** Same usage. Korean prefers halfwidth `;`.

### Rule 6.10: Chinese parentheses （） → Korean ()

| Chinese | Korean | Example |
|---------|--------|---------|
| （以下简称甲方） | (이하 "갑"이라 함) | Definition note |
| （共10页） | (총 10페이지) | Page note |
| （注） | (주) / (참고) | Annotation |
| （一）（二）（三） | (가) (나) (다) | Ordered sub-items |

**Rule:** Fullwidth `（）` → halfwidth `()`. For Chinese numbered sub-items (一)(二)(三), Korean uses (가)(나)(다).

### Rule 6.11: Chinese tilde ～ → Korean ~

| Chinese | Korean | Example |
|---------|--------|---------|
| 5～10天 | 5~10일 | Time range |
| 3～5年 | 3~5년 | Year range |
| 10～20人 | 10~20명 | Quantity range |

**Rule:** ～ → ~. Same meaning. Korean leans toward halfwidth.

### Rule 6.12: Korean spacing rules (띄어쓰기) vs. Chinese

Chinese has no word spacing. Korean has strict word spacing rules.

| Chinese (no spaces) | Korean (with spaces) | Rule |
|--------------------|---------------------|------|
| 这是一个例子 | 이것은 예입니다 | Particle attached to word |
| 非常重要 | 매우 중요하다 | Adverb + verb = spaced |
| 认真学习 | 열심히 공부하다 | Adverb + verb = spaced |
| 参加会议 | 회의에 참석하다 | Object + verb = spaced |
| 公司员工 | 회사 직원 | Noun + noun = spaced |
| 不能接受 | 받아들일 수 없다 | Multi-verb construction |
| 新规定 | 새로운 규정 | Adjective + noun = spaced |
| 在北京 | 베이징에서 | Particle attached |

**Critical rule:** Always apply Korean spacing rules (띄어쓰기). Particles (은/는, 이/가, 을/를, 에, 에서, 의, etc.) are attached directly to the preceding word without a space. Other words are separated by spaces.

### Rule 6.13: Korean spacing after punctuation

| Context | Chinese | Korean | Rule |
|---------|---------|--------|------|
| After period | 。no space | .[space] | Space after period in Korean |
| After question | ？no space | ?[space] | Space after question mark |
| After exclamation | ！no space | ![space] | Space after exclamation |
| After comma | ，no space | ,[space] | Space after comma |
| After colon | ：no space | :[space] | Space after colon |

### Rule 6.14: Chinese question/exclamation frequency

Korean uses question marks `?` and exclamation marks `!` less aggressively than Chinese.

| Chinese | Korean | Notes |
|---------|--------|-------|
| 真的吗？！ | 정말입니까? / 진짜예요? | Single question mark |
| 太好了！ | 좋습니다! | Single exclamation |
| 怎么会这样？！ | 어떻게 이럴 수가 있습니까? | Single question mark |

**Rule:** Reduce multiple punctuation (`？！`, `!!`, `??`) to single marks. Korean formal text prefers `?` over `!`.

### Rule 6.15: Chinese → Korean special legal/document formatting

| Element | Chinese convention | Korean convention |
|---------|-------------------|-------------------|
| Article numbering | 第X条 | 제X조 |
| Chapter | 第X章 | 제X장 |
| Section | 第X节 | 제X절 |
| Item (numbered) | (一)(二)(三) | (가)(나)(다) |
| Item (alphabetic) | (A)(B)(C) | (A)(B)(C) [keep] |
| Sub-item | 1. 2. 3. | 1. 2. 3. |
| Paragraph | 第一款 | 제1항 |
| Annex | 附件 | 부록 / 첨부 |
| Effective date clause | 自X年X月X日起施行 | X년 X월 X일부터 시행 |

**Summary of all punctuation conversions:**

| Chinese | Korean | Conversion rule |
|---------|--------|-----------------|
| 。 | . | Fullwidth period → halfwidth period |
| 、 | , | Enumeration comma → comma |
| "" | "" | Keep (usage differs) |
| ： | : | Fullwidth colon → halfwidth |
| ； | ; | Fullwidth semicolon → halfwidth |
| （） | () | Fullwidth parentheses → halfwidth |
| —— | — | Length may differ |
| …… | … | 6 dots → 3 dots (preferred) |
| ～ | ~ | Fullwidth tilde → halfwidth |
| · | · | Keep interpunct |
| spacing | apply spacing | Add Korean word spacing |
| No Oxford comma | No Oxford comma | Same |
| No em space | No em space | Same |
