# English to Korean Adaptation Guide

> Target language: Korean (한국어 / 韓國語)  
> Source language: English (English)  
> Purpose: Standardized adaptation rules for English-to-Korean document translation  
> Last updated: 2026-05-25

---

## 1. Formality: English Address and Politeness to Korean 존댓말

### Overview

English has minimal grammatical formality marking (primarily lexical choices like "please" vs. modal verbs). Korean has an elaborate honorific system (경어법) that requires the translator to infer social relationships from context and encode them through speech levels, honorific infixes, and lexical choices.

### Rule 1.1: English "you" → Korean address avoidance

English uses "you" freely for all addressees. Korean avoids the second-person pronoun in polite contexts, preferring titles, names, or omission.

| English | Korean | Strategy |
|---------|--------|----------|
| "You" (to customer) | 고객님 (customer-nim) | Title + 님 |
| "You" (to superior) | ~님 / 직함 | Honorific title |
| "You" (in manual) | 사용자 / 이용자 | User (generic) |
| "You" (formal letter) | 귀하 | Very formal |
| "You" (equal, friend) | 당신 | Context-dependent; can be confrontational |
| "You" (book/formal) | 독자 | Reader (for books) |
| "your company" | 귀사 / 귀 측 | Formal company reference |
| "your request" | 요청하신 사항 | Honorific verb |
| "you said" | 말씀하신 | Honorific verb |
| "you (plural)" | 여러분 | Polite plural |

**Rule:** Avoid `당신` in most formal documents (it can sound confrontational or overly intimate). Use `귀하` for very formal letters, `고객님` for customer-facing text, or embed respect through verb endings (`-시-`) without an explicit pronoun.

### Rule 1.2: English "Mr./Ms./Mrs./Dr./Prof." → Korean title system

English honorifics precede the name. Korean honorifics follow the name or replace the name entirely.

| English | Korean | Notes |
|---------|--------|-------|
| Mr. Kim | 김 선생님 / 김 씨 | 선생님 (respectful), 씨 (neutral, less common) |
| Ms. Lee | 이 선생님 / 이 씨 | Same pattern |
| Mrs. Park | 박 선생님 | Korean does not distinguish marital status |
| Dr. Smith | 스미스 박사님 | 박사님 more formal |
| Prof. Johnson | 존슨 교수님 | 교수님 standard |
| President (title) | 사장님 / 회장님 | Corporate president |
| CEO | CEO / 대표이사님 | Both used |
| Sir (general) | 선생님 / ~님 | Context-dependent |
| Madam | 귀하 / 사모님 | Very formal / specific |
| Mr. President | 대통령님 | Head of state |

**Rule:** English `Title + Name` → Korean `Name + Title + 님`. Korean does not mark marital status in titles. `님` is the standard honorific suffix attached to all titles.

### Rule 1.3: English "please" → Korean request forms

English "please" is a single word. Korean has a spectrum of request forms.

| English | Korean | Context |
|---------|--------|---------|
| "Please confirm." | 확인해 주시기 바랍니다. | Very formal (document) |
| "Please wait." | 잠시만 기다려 주십시오. | Formal |
| "Please send the file." | 파일을 보내 주세요. | Polite |
| "Please note..." | 참고하시기 바랍니다. | Formal notice |
| "Please feel free to..." | 부담 없이 ~하시기 바랍니다. | Polite permission |
| "Please be advised..." | 알려드립니다. / 양지하시기 바랍니다. | Very formal |
| "Please do not..." | ~하지 말아 주십시오. | Prohibition (polite) |
| "Please let me know..." | 알려 주시기 바랍니다. | Request for info |

**Rule:** Never translate "please" as 제발 (desperate/pleading). Use `-주십시오`, `-주세요`, or `-시기 바랍니다` instead. "Please" is embedded in the verb ending, not rendered as a separate word.

### Rule 1.4: English "I/we" → Korean humble forms

| English | Korean | Context |
|---------|--------|---------|
| "I" (formal) | 저 | Humble form |
| "I" (informal) | 나 | Casual |
| "We" (company) | 저희 / 당사 | Humble corporate "we" |
| "We" (inclusive) | 우리 | Includes reader |
| "We" (editorial) | 필자 / 본인 | Writer/author |
| "our company" | 당사 / 저희 회사 | Humble reference |
| "I think" | ~라고 생각합니다 / ~라고 사료됩니다 | Very formal for "I think" |
| "in my opinion" | 제 의견으로는 / 본인의 의견으로는 | Formal opinion |

### Rule 1.5: English passive voice → Korean active voice

English uses passive voice extensively in formal writing. Korean prefers active voice.

| English (passive) | Korean (active) | Notes |
|-------------------|-----------------|-------|
| "It is agreed that..." | 다음과 같이 합의합니다/합의한다 | Active agreement |
| "The report was prepared by..." | (보고서 작성자)가 보고서를 작성했다 | Subject + active verb |
| "It is understood that..." | ~하는 것으로 한다 | Convert to active |
| "Payment shall be made by..." | (주체)가 지급한다 | Active payment |
| "It is prohibited to..." | ~을(를) 금지한다 | Active prohibition |
| "The data is collected..." | 데이터를 수집한다 | Active collection |
| "Permission is required." | 허가가 필요하다 | Keep as passive if natural |
| "The contract is signed..." | 계약이 체결된다 | Korean `-되다` passive sometimes natural |

**Rule:** Convert English passive to Korean active where possible. Korean has a passive form (`-되다`, `-당하다`, `-아/어지다`) but uses it less frequently than English.

### Rule 1.6: English "Dear" → Korean salutation

English formal letters begin with "Dear". Korean has no direct equivalent.

| English | Korean | Notes |
|---------|--------|-------|
| "Dear Sir/Madam" | 귀중 / 귀하 | Very formal, no direct equivalent |
| "Dear Mr. Kim" | 김 선생님께 | Formal letter opening |
| "Dear Customers" | 고객님께 / 고객 여러분께 | Customer address |
| "Dear All" | 여러분께 / 관계자 여러분께 | Group address |
| "Dear Dr. Lee" | 이 박사님께 | Academic |
| "To whom it may concern" | 귀중 / 관계자 여러분께 | Generic formal |

**Rule:** Korean often omits the salutation entirely in formal documents. When used, `께` after the addressee is the standard equivalent.

### Rule 1.7: English first-name basis → Korean formal address

| English style | Korean adaptation | Notes |
|--------------|------------------|-------|
| "Dear John" | 존 님 / 존 씨께 | Use surname or full name + 님 |
| "Hi Mike" | 마이크 님 / 마이크 씨 | Korean may add 님 |
| "John and Mary" | 존과 메리 / 존 님과 메리 님 | Respectful in formal context |
| "Dear Team" | 팀원 여러분 | Group address |

**Rule:** Korean formal text does not use first-name-only address. Always add `님` or `씨` even when English uses first name only.

### Rule 1.8: English closing formulas → Korean

| English | Korean | Formality |
|---------|--------|-----------|
| "Sincerely" | 드림 / 올림 | Formal letter closing |
| "Best regards" | 감사합니다 / 고맙습니다 | Common business closing |
| "Yours faithfully" | 올림 / 드림 | Very formal |
| "Respectfully" | 삼가 드림 / 올림 | Highest formality |
| "Cheers" (informal) | 안녕 / 이만 | Informal |
| "Best" | 감사합니다 | Common email closing |
| "Warm regards" | 따뜻한 마음을 전하며 | Less common |
| "Cordially" | 정중히 | Rare |

### Rule 1.9: English "shall" / "must" / "should" → Korean modals

| English modal | Korean | Strength |
|---------------|--------|----------|
| "shall" (legal) | 한다 / 하여야 한다 | Legal obligation |
| "must" | 반드시 ~해야 한다 | Strong obligation |
| "should" (recommendation) | 하는 것이 좋다 / 함이 바람직하다 | Recommendation |
| "may" (permission) | 할 수 있다 | Permission |
| "may" (possibility) | ~일 수도 있다 / ~할 수도 있다 | Possibility |
| "will" (future) | ~할 것이다 / ~겠다 | Future |
| "would" | ~일 것이다 / ~하겠다 | Polite conditional |
| "could" | ~할 수도 있다 | Possibility |
| "might" | ~할지도 모른다 | Weak possibility |

### Rule 1.10: English "please" vs. Korean formality hierarchy

| English | Korean | Level |
|---------|--------|-------|
| "Please submit the form." | 서식을 제출해 주시기 바랍니다. | Highest (하십시오체) |
| "Please submit the form." | 서식을 제출해 주십시오. | High (하십시오체) |
| "Please submit the form." | 서식을 제출해 주세요. | Mid (해요체) |
| "Submit the form." | 서식을 제출하세요. | Mid (해요체, direct) |
| "Submit the form." | 서식을 제출해. | Low (해체, inform) |

## 2. Date, Time, Number: English Conventions to Korean

### Overview

English and Korean differ fundamentally in date format, time notation, number system, and measurement units. English uses MM/DD/YYYY (US) or DD/MM/YYYY (UK); Korean uses YYYY년 MM월 DD일. English uses AM/PM; Korean uses 오전/오후. English uses Western thousands grouping; Korean uses 만/억/조 4-digit grouping.

### Rule 2.1: English date format → Korean date format

| English (US) | English (UK) | Korean | Notes |
|-------------|-------------|--------|-------|
| March 15, 2024 | 15 March 2024 | 2024년 3월 15일 | Standard conversion |
| 03/15/2024 | 15/03/2024 | 2024. 3. 15. | Alternative format |
| Jan. 1, 2024 | 1 Jan. 2024 | 2024년 1월 1일 | Month abbreviation expanded |
| 2024-03-15 | 2024-03-15 | 2024-03-15 | ISO format can be kept |
| March 2024 | March 2024 | 2024년 3월 | Month+year only |
| the 1990s | the 1990s | 1990년대 | Decades |
| Q1 2024 | Q1 2024 | 2024년 1분기 | Quarters |
| FY2024 | FY2024 | 2024 회계연도 | Fiscal year |
| H1 2024 | H1 2024 | 2024년 상반기 | Half year |
| Week of Mar 1 | — | 3월 1일 주 | Week notation |

**Rule:** Convert all date formats to `YYYY년 M월 D일`. English month names → Korean month numbers (March = 3월). Remove commas used in English dates.

### Rule 2.2: English days of week → Korean 요일

| English | Korean | Notes |
|---------|--------|-------|
| Monday | 월요일 (月曜日) | Hanja-based |
| Tuesday | 화요일 (火曜日) | Hanja-based |
| Wednesday | 수요일 (水曜日) | Hanja-based |
| Thursday | 목요일 (木曜日) | Hanja-based |
| Friday | 금요일 (金曜日) | Hanja-based |
| Saturday | 토요일 (土曜日) | Hanja-based |
| Sunday | 일요일 (日曜日) | Hanja-based |
| this Monday | 이번 주 월요일 | Current week |
| next Friday | 다음 주 금요일 | Next week |
| last Wednesday | 지난 주 수요일 | Previous week |
| every Monday | 매주 월요일 | Weekly recurrence |
| by Friday | 금요일까지 | Deadline |

### Rule 2.3: English AM/PM → Korean 오전/오후

| English | Korean | Notes |
|---------|--------|-------|
| 9:00 AM | 오전 9시 / 오전 9:00 | Before noon |
| 3:00 PM | 오후 3시 / 오후 3:00 | After noon |
| 12:00 PM (noon) | 낮 12시 / 정오 12시 | Noon |
| 12:00 AM (midnight) | 밤 12시 / 자정 | Midnight |
| 3:30 PM | 오후 3시 30분 | With minutes |
| 8:00 AM to 5:00 PM | 오전 8시부터 오후 5시까지 | Business hours |
| 10:15 AM | 오전 10시 15분 | Quarter past |
| 6:45 PM | 오후 6시 45분 | Quarter to |

**Rule:** AM → 오전, PM → 오후. Korean uses 12-hour clock with 오전/오후 in daily contexts, but 24-hour clock in official/scheduled contexts.

### Rule 2.4: English time expressions → Korean

| English | Korean | Notes |
|---------|--------|-------|
| half past two | 2시 30분 | Korean does not use "half past" |
| quarter to three | 2시 45분 / 3시 15분 전 | Korean uses minutes |
| a quarter past | 15분 | |
| ten to four | 4시 10분 전 / 3시 50분 | |
| five after seven | 7시 5분 | |
| around 3 o'clock | 3시쯤 / 3시 경 | Approximation |
| sharp / exactly 3 | 정확히 3시 / 3시 정각 | Exact time |
| 24/7 | 24시간 / 연중무휴 | |
| 9-to-5 | 오전 9시~오후 5시 | |
| noon | 정오 / 낮 12시 | |
| midnight | 자정 / 밤 12시 | |

| English | Korean | Notes |
|---------|--------|-------|

| yesterday | 어제 | |

### Rule 2.6: English ordinal numbers → Korean

| English | Korean | Context |
|---------|--------|---------|
| 1st | 첫 번째 / 제1 | General / formal |
| 2nd | 두 번째 / 제2 | General / formal |
| 3rd | 세 번째 / 제3 | General / formal |
| 4th | 네 번째 / 제4 | General / formal |
| 10th | 열 번째 / 제10 | General / formal |
| 21st | 스물한 번째 / 제21 | General / formal |
| Chapter 1 | 제1장 / 1장 | Sections |
| Article 3 | 제3조 | Legal articles |
| Page 10 | 10페이지 | Pages |
| Part II | 제2부 / 2부 | Parts |

**Rule:** English ordinal suffix (-st, -nd, -rd, -th) is removed. Use `제` + number for formal designations, `~번째` for general ordinal.

### Rule 2.7: English number grouping → Korean 4-digit grouping

| English | Korean | Notes |
|---------|--------|-------|
| 10,000 | 1만 / 10,000 | 만 unit introduced |
| 100,000 | 10만 / 100,000 | 만 unit |
| 1,000,000 | 100만 / 1,000,000 | 만 unit |
| 10,000,000 | 1,000만 / 10,000,000 | 만 unit |
| 100,000,000 | 1억 / 100,000,000 | 억 unit |
| 1,000,000,000 | 10억 / 1,000,000,000 | 억 unit |
| 1,000,000,000,000 | 1조 / 1,000,000,000,000 | 조 unit |

**Rule:** Prefer Korean 4-digit grouping (만/억/조) for readability in Korean text, especially for large numbers. Keep Western comma grouping as a secondary notation.

### Rule 2.8: English measurement units → Korean

| English (imperial) | Korean (metric) | Notes |
|--------------------|-----------------|-------|
| mile | 마일 (keep) / km (convert) | Conversion optional |
| inch | 인치 (keep) / cm (convert) | |
| foot/feet | 피트 (keep) / m (convert) | |
| pound (lb) | 파운드 (keep) / kg (convert) | |
| ounce (oz) | 온스 (keep) / g (convert) | |
| gallon | 갤런 (keep) / L (convert) | |
| Fahrenheit (°F) | 화씨 (°F) (keep) | Optionally add 섭씨 conversion |
| yard | 야드 (keep) / m (convert) | |
| acre | 에이커 (keep) | |
| pint | 파인트 (keep) | |

**Rule:** Korean uses the metric system. Imperial units may be kept as transliterated loanwords but should ideally be converted to metric with the original in parentheses.

### Rule 2.9: English currency → Korean

| English | Korean | Notes |
|---------|--------|-------|
| $1,000 | 1,000달러 / 미화 1,000달러 | Dollar amount |
| $10 million | 1,000만 달러 | Large dollar amounts |
| ￡500 | 500파운드 | British pound |
| EUR 1,000 | 1,000유로 | Euro |
| ￥10,000 | 1만 엔 / 10,000엔 | Japanese yen |
| $19.99 | 19.99달러 | Decimal amounts |
| 50 cents | 50센트 | Cents |

**Rule:** Currency symbol ($, ￡, etc.) → Korean transliteration + amount. Use default exchange rate context if needed.

### Rule 2.10: English percentage → Korean

| English | Korean | Notes |
|---------|--------|-------|
| 10% | 10% | Same notation |
| 0.5% | 0.5% | Decimal fraction |
| 50 percent | 50% / 50퍼센트 | Full word |
| a 5% increase | 5% 증가 | Growth rate |
| interest rate of 3% | 3% 금리 | Interest rate |
| 10 basis points | 10bp / 0.1% | Financial terms |

### Rule 2.11: English century/era → Korean

| English | Korean | Notes |
|---------|--------|-------|
| 21st century | 21세기 / 21세기 | Same |
| BC 500 | 기원전 500년 | Before Christ |
| AD 1066 | 서기 1066년 / 기원후 1066년 | Anno Domini |
| the 1800s | 1800년대 | Century reference |
| circa 1500 | 약 1500년경 | Approximate year |
| 400 BCE | 기원전 400년 | Alternative notation |
| the present century | 금세기 | Current century |

**Rule:** BC→기원전, AD→서기/기원후. Korean uses both the Western calendar (서기) and the Korean era (단기, rarely used in modern context).

## 3. Cultural: Western Cultural References to Korean

### Overview

English text frequently contains Western cultural references, idioms, religious allusions, and cultural concepts unfamiliar to Korean readers. Adaptation strategies range from finding Korean equivalents to providing explanatory translations.

### Rule 3.1: English idioms → Korean equivalents

| English idiom | Korean equivalent | Strategy |
|---------------|-------------------|----------|
| "Kill two birds with one stone" | 일석이조 (一石二鳥) / 한 번에 두 마리 토끼를 잡다 | 사자성어 + native |
| "The ball is in your court" | 네 차례다 / 이제 네가 결정할 차례다 | Plain translation |
| "It's raining cats and dogs" | 비가 억수같이 내린다 | Korean idiom |
| "Once in a blue moon" | 아주 가끔 / 매우 드물게 | Plain equivalent |
| "A piece of cake" | 누워서 떡 먹기 | Korean equivalent idiom |
| "Break the ice" | 어색한 분위기를 깨다 | Plain translation |
| "The elephant in the room" | 뻔한 문제를 외면하다 | Explanatory |
| "Bite the bullet" | 각오하다 / 이를 악물고 하다 | Plain translation |
| "Don't count your chickens" | 김칫국부터 마시지 마라 | Korean specific idiom |
| "When in Rome, do as the Romans do" | 로마에 가면 로마법을 따르라 | Direct translation (understood) |
| "Curiosity killed the cat" | 궁금증이 화를 부른다 / 필요 이상의 호기심은 금물 | Explanatory |
| "Hit the nail on the head" | 정곡을 찌르다 | Korean equivalent |
| "Let the cat out of the bag" | 비밀이 새다 / 실수로 누설하다 | Plain translation |
| "Steal someone's thunder" | 남의 공로를 가로채다 / 주인공 자리를 빼앗다 | Explanatory |
| "Add insult to injury" | 설상가상 (雪上加霜) / 엎친 데 덮치다 | 사자성어 + native |

### Rule 3.2: Western proverbs/sayings → Korean

| English | Korean equivalent | Notes |
|---------|-------------------|-------|
| "All that glitters is not gold" | 번지르르하다고 다 금은 아니다 | Direct translation works |
| "Don't judge a book by its cover" | 겉모습으로 판단하지 마라 | Direct translation |
| "Actions speak louder than words" | 말보다 행동이 중요하다 | Direct equivalent |
| "Better late than never" | 늦더라도 안 하는 것보다 낫다 / 망양보뢰 | Common equivalent |
| "Practice makes perfect" | 연습이 완벽을 만든다 | Direct translation |
| "Where there's a will, there's a way" | 뜻이 있는 곳에 길이 있다 | Direct translation |
| "Honesty is the best policy" | 정직이 최선의 정책이다 | Direct translation |
| "The early bird catches the worm" | 일찍 일어나는 새가 벌레를 잡는다 | Direct translation (well understood) |
| "Absence makes the heart grow fonder" | 떠나 있어야 정이 더 든다 | Direct equivalent |
| "Every cloud has a silver lining" | 모든 어둠에 빛이 있다 / 새옹지마 | 사자성어 equivalent |

### Rule 3.3: Biblical/references → Korean

The Bible is widely known in Korea (Christianity is a major religion), but references may need adaptation.

| English reference | Korean rendering | Notes |
|-------------------|-----------------|-------|
| "Garden of Eden" | 에덴 동산 | Standard term |
| "Forbidden fruit" | 금단의 열매 | Standard term |
| "Good Samaritan" | 선한 사마리아인 | Well-known in Korea |
| "Prodigal son" | 탕자 (蕩子) | Standard term |
| "David and Goliath" | 다윗과 골리앗 | Well-known |
| "Armageddon" | 아마겟돈 | Standard term |
| "Job's patience" | 욥의 인내 / 욥의 참을성 | Well-known |
| "Writing on the wall" | 벽에 쓰인 글씨 / 불길한 징조 | Explanatory |
| "Doubting Thomas" | 의심하는 도마 | Understandable |
| "Casting pearls before swine" | 돼지 앞에 진주 / 대우탄금 | Both understood |

**Rule:** Biblical references are generally well-understood in Korea due to the large Christian population. Transliterate names in standard Korean Bible convention. For obscure references, add explanation.

### Rule 3.4: Greek/Roman mythology → Korean

| English reference | Korean rendering | Notes |
|-------------------|-----------------|-------|
| "Pandora's box" | 판도라의 상자 | Well-known |

## 5. Administrative/Legal Style: English Official Style to Korean

### Overview

English administrative and legal writing features long sentences, passive voice, nominalization, Latin-derived terminology, and specific legal formulas. Korean administrative/legal style prefers active voice, shorter clauses, Sino-Korean legal terms, and different formulaic expressions.

### Rule 5.1: English legal "shall" → Korean obligation forms

| English | Korean | Notes |
|---------|--------|-------|
| "The Lessee shall..." | 임차인은 ... 한다 / ... 하여야 한다 | Legal obligation |
| "Payment shall be made..." | 대금은 ... 지급한다 | Active equivalent |
| "The Company shall indemnify..." | 회사는 ... 보상한다 / 보상하여야 한다 | Legal obligation |
| "This Agreement shall be governed by..." | 본 계약은 ...에 따른다 | Governing law |
| "The term shall commence on..." | 기간은 ...부터 시작된다 | Effective date |

**Rule:** "Shall" in legal English → `-한다` (declarative) or `-하여야 한다` (obligation). Korean legal writing uses the plain declarative `-한다` for normative statements, not a separate modal verb.

### Rule 5.2: English legal "hereby/herein/hereunder" → Korean

| English legal adverb | Korean | Notes |
|---------------------|--------|-------|
| "hereby agrees" | 이에 동의한다 | 이에 = by this |
| "herein contained" | 본 (本) / 이에 포함된 | 본 = this |
| "hereinafter referred to as" | 이하 "..."라 한다 | 이하 = hereinafter |
| "hereunder" | 이에 따라 / 본 ...에 따라 | |
| "hereto" | 본 ...에 / 이에 | |
| "hereof" | 본 ...의 / 이의 | |
| "herewith" | 이에 / 이와 함께 | |
| "thereof" | 그 / 그것의 / 이의 | |
| "therein" | 그 안에 / 그 중에 | |
| "thereunder" | 그에 따라 | |
| "thereto" | 거기에 / 그것에 | |
| "therefor" | 그것에 대한 / 이에 대한 | |

**Rule:** English compound adverbs (hereby, thereof, etc.) → Korean `이/그 + postposition`. Many can be simplified to plain Korean.

### Rule 5.3: English passive → Korean active in legal context

| English (passive) | Korean (active/neutral) | Notes |
|-------------------|------------------------|-------|
| "It is hereby agreed that..." | 이에 다음과 같이 합의한다 | Active |
| "This Agreement is entered into..." | 본 계약은 ... 체결된다 | Korean -되다 passive |
| "The following terms are defined..." | 다음 용어는 ... 정의한다 | Active |
| "Payment shall be made..." | 대금은 ... 지급한다 | Active style |
| "The data is collected..." | 데이터는 ... 수집된다 | -되다 passive acceptable |
| "It is understood that..." | ...하는 것으로 한다 | Legal formula |
| "Notice shall be deemed given..." | 통지는 ... 송달된 것으로 간주한다 | Legal formula |

### Rule 5.4: English long sentences → Korean clause structure

English legal sentences can be 100+ words. Korean prefers shorter clauses.

| English style | Korean adaptation |
|---------------|------------------|
| Single long sentence with multiple subordinate clauses | Break into 2-3 shorter sentences |
| "Subject to the terms and conditions of this Agreement, the Company may..." | 본 계약의 조건에 따라, 회사는 ...할 수 있다 |
| "In the event that the Buyer fails to make payment within 30 days..." | 매수인이 30일 이내에 대금을 지급하지 않는 경우 |
| "Notwithstanding anything to the contrary contained herein..." | 본 계약에 반대되는 규정이 있음에도 불구하고 |
| "Unless otherwise specified in writing by the Party..." | 당사자가 서면으로 달리 명시하지 않는 한 |
| Multiple "and/or" chains | Break into enumerated list (가, 나, 다) |

### Rule 5.5: English legal conditionals → Korean

| English | Korean | Structure |
|---------|--------|-----------|
| "If... / In the event that..." | ...하는 경우 / ...하면 | Conditional clause |
| "Provided that..." | 단, ... / ...하는 조건으로 | Proviso |
| "Subject to..." | ...에 따라 / ...을 조건으로 | Condition precedent |
| "Notwithstanding..." | ...에도 불구하고 | Despite clause |
| "Unless..." | ...하지 않는 한 / ...외에는 | Negative condition |
| "Where..." (legal) | ...하는 경우에는 | Legal conditional |
| "When..." | ...하는 때에는 / ...하면 | Temporal conditional |

### Rule 5.6: English administrative formulas → Korean

| English | Korean | Context |
|---------|--------|---------|
| "Pursuant to Section 5..." | 제5조에 따라 / 제5조에 의거하여 | Authority reference |

### Rule 5.10: English-Korean legal sample comparison

**English original (Service Agreement clause):**
"Pursuant to Section 5.2 of this Agreement, the Service Provider shall, within thirty (30) days of receipt of a valid invoice from the Client, remit payment in full for all Services rendered during the preceding calendar month. Any amounts not paid when due shall bear interest at the rate of 1.5% per month."

**Korean translation (표준 계약 번역):**
본 계약 제5.2조에 따라, 서비스 제공자는 고객으로부터 유효한 청구서를 수령한 날로부터 30일 이내에 전월에 제공된 모든 서비스에 대한 대금 전액을 지급하여야 한다. 기한 내에 지급되지 않은 금액에는 월 1.5%의 이자가 발생한다.

**Key conversion points:**
- "Pursuant to" → 조에 따라
- "shall" → 하여야 한다 (obligation)
- "within thirty (30) days" → 30일 이내에
- "a valid invoice" → 유효한 청구서
- "remit payment in full" → 대금 전액을 지급하다 (active verbal style)
- "rendered during" → 제공된 (Korean past attributive)
- "the preceding calendar month" → 전월 (more concise)
- "bear interest" → 이자가 발생하다 (natural Korean financial expression)

## 6. Punctuation: English Punctuation to Korean

### Overview

English and Korean share many punctuation marks (period, comma, question mark, exclamation mark, quotation marks) but differ in spacing rules, quotation conventions, and usage frequency. Korean also has unique conventions for book title marks and range notation.

### Rule 6.1: English period . → Korean period .

Both English and Korean use `.` as the sentence-ending period (마침표 / matkimpyo). However, Korean usage differs in spacing.

| English | Korean | Notes |
|---------|--------|-------|
| "Thank you." | "감사합니다." | Same period |
| "Please submit by Friday." | 금요일까지 제출해 주십시오. | Same period |
| "The results are conclusive." | 결과는 확정적입니다. | Same period |

**Rule:** Unlike Chinese (which uses 。), Korean and English both use `.`. However, Korean Korean text places a space after the period (same as English). Ensure consistency.

### Rule 6.2: English comma , → Korean comma ,

English comma usage is similar to Korean, but Korean uses the comma less frequently, preferring grammatical particles and spacing to indicate relationships.

| English comma usage | Korean adaptation | Notes |
|--------------------|-------------------|-------|
| "apples, oranges, bananas" | 사과, 오렌지, 바나나 | Same: listing |
| "However, we cannot..." | 그러나, 저희는 ...할 수 없습니다 | Similar: conjunctive |
| "March 15, 2024" | 2024년 3월 15일 | Remove comma in date |
| "New York, NY" | 뉴욕주, 뉴욕시 | Similar: location |
| "Therefore, the result is..." | 따라서, 결과는 ...입니다 | Similar |
| "The company, which was founded in 2000, is..." | 2000년에 설립된 회사는 ... | Korean uses relative clause without commas |
| Oxford comma (serial comma) | Not used in Korean | Remove Oxford comma |
| "Dear Mr. Smith," | 김 선생님께 | Korean does not use comma after salutation |
| "1,000,000" (number) | 1,000,000 | Same comma grouping |

**Critical difference:** Korean does NOT use the Oxford comma (serial comma before "and"). Remove it in translation.

### Rule 6.3: English quotation marks "" → Korean ""

Both use `""` (큰따옴표 / keunddaompyo) for direct quotes and `''` (작은따옴표 / jageunddaompyo) for quotes within quotes.

| English | Korean | Notes |
|---------|--------|-------|
| He said, "I'll come." | 그는 "갈게요."라고 말했다. | Korean no colon before quote |
| She asked, "Why?" | 그녀가 "왜요?"라고 물었다. | Korean uses 라고 after quote |
| The term "cloud" refers to... | "클라우드"라는 용어는 ...을 의미한다 | Terminology quotes |
| He said, "She said, 'No.'" | 그는 "그녀가 '아니요'라고 말했어."라고 말했다. | Nested quotes: '' inside "" |
| "Chapter 3: Results" | "제3장: 결과" | Section titles |

**Key difference:** English places a comma before a quote ("He said, ..."). Korean attaches `라고` directly to the quote without a comma: `"...라고 말했다`.

### Rule 6.4: English quotation end punctuation → Korean

English puts periods inside closing quotation marks (US style). Korean period goes outside.

| English (US) | Korean | Notes |
|-------------|--------|-------|
| He said, "It's done." | 그는 "끝났어요."라고 말했다. | Period inside in KO too (but convention varies) |
| The word "cat." | "고양이."라는 단어 | Period placement varies |
| "Please read this." | "이것을 읽어 주십시오." | Generally same |

**Note:** Korean convention on period placement with quotes is less rigid than English. Both inside and outside occur.

### Rule 6.5: English em dash — → Korean

| English | Korean | Notes |
|---------|--------|-------|
| "The results — which were unexpected — show..." | 결과는 — 예상치 못한 — ...을 보여준다 | Same usage |
| "Three items — a pen, paper, and a ruler." | 세 가지 물품 — 펜, 종이, 자 | Similar |
| Range: "2010–2020" | 2010~2020 | Korean prefers ~ for ranges |
| "Pages 10–15" | 10~15페이지 | Range notation |

**Rule:** Em dash `—` for parenthetical breaks is the same in both languages. For ranges, Korean uses `~` instead of en dash `–`.

### Rule 6.6: English hyphen - → Korean

| English | Korean |
|---------|--------|
| "well-known" | 잘 알려진 (space instead of hyphen) |
| "up-to-date" | 최신의 |
| "state-of-the-art" | 최첨단의 |
| "part-time" | 파트타임 / 시간제 |
| "three-year-old" | 세 살짜리 |
| phone numbers: 555-1234 | 555-1234 (same) |

**Rule:** Hyphenated compound adjectives in English are typically rendered as separate words or single words in Korean without hyphens.

### Rule 6.7: English colon : → Korean colon :

Both use colons similarly, but usage frequency differs.

| English | Korean | Notes |
|---------|--------|-------|
| "The following items:" | 다음 항목: | Same |
| "Note: Please read carefully." | 참고: 주의 깊게 읽어 주십시오. | Same |
| "Time: 3:00 PM" | 시간: 오후 3시 | Same |
| "Ratio of 3:1" | 3:1 비율 | Same for ratios |

**Rule:** Generally the same. Korean uses halfwidth `:` (English style).

### Rule 6.8: English semicolon ; → Korean

Semicolons are less common in Korean than in English.

| English | Korean | Notes |
|---------|--------|-------|
| "First, ...; second, ..." | 첫째, ...; 둘째, ... | Semicolon retained for parallel lists |
| "Some went left; others went right." | 일부는 왼쪽으로 갔고, 다른 이들은 오른쪽으로 갔다 | Korean prefers separate clauses |

**Rule:** Semicolons are acceptable in Korean but can often be replaced with periods (new sentence) or commas + conjunctions.

### Rule 6.9: English apostrophe ' → Korean

English uses apostrophes for possessives and contractions. Korean does not use apostrophes.

| English | Korean rendering | Strategy |
|---------|------------------|----------|
| "the company's policy" | 회사의 정책 | Possessive: `의` particle |
| "John's book" | 존의 책 | Possessive: `의` |
| "the students' grades" | 학생들의 성적 | Possessive: `의` |
| "don't" | 하지 마라 / 하지 않다 | No contraction in KO |
| "it's" | 그것은 / 그것이 | Expand contraction |
| "can't" | 할 수 없다 | Expand |
| "won't" | 않을 것이다 | Expand |
| "I'm" | 저는 / 나는 | Expand |
| "they're" | 그들은 | Expand |
| "O'Clock" | 시 (時) | Replace with Korean time marker |

**Rule:** Remove all apostrophes. Possessives → `의` particle. Contractions → expanded forms.

### Rule 6.10: English ellipsis … → Korean ellipsis

| English | Korean | Notes |
|---------|--------|-------|
| "And so on..." | 등등... / 기타... | 3 dots (both) |
| "To be continued..." | 계속... | 3 dots |
| "I'm not sure if..." | 확실하지 않은데... | 3 or 6 dots acceptable |

**Rule:** English 3-dot ellipsis → Korean 3-dot or 6-dot ellipsis. 3 dots are standard, 6 dots also used for stronger pause.

### Rule 6.11: English slash / → Korean

| English | Korean | Notes |
|---------|--------|-------|
| "and/or" | 및/또는 | Same slash usage |
| "input/output" | 입력/출력 | Same |
| "2019/2020" | 2019/2020 | Same for fiscal years |
| "he/she" | 그/그녀 | Same (but avoid in KO if possible) |
| "Yes/No" | 예/아니오 | Same |

**Rule:** Slash usage is the same. In formal Korean, `및/또는` is preferred over `and/or`.

### Rule 6.12: English ampersand & → Korean

| English | Korean | Context |
|---------|--------|---------|
| "Research & Development" | 연구개발(R&D) | Formal: use full term |
| "Law & Economics" | 법률 및 경제 | Formal: use 및 |
| "AT&T" | AT&T (keep) | Proper name: keep |
| "P&L" | 손익 (損益) | Translate the meaning |
| "R&D" | 연구개발 (R&D) | Abbreviation kept |

**Rule:** `&` in running text → `및` (formal) or `와/과`. `&` in proper names → keep as is.

### Rule 6.13: English spacing rules → Korean spacing

| Rule | English | Korean | Notes |
|------|---------|--------|-------|
| After period | 1 space | 1 space | Same |
| After comma | 1 space | 1 space | Same |
| Before colon | 0 space | 0 space | Same |
| After colon | 1 space | 1 space | Same |
| After opening quote | 0 space | 0 space | Same |
| Before closing quote | 0 space | 0 space | Same |
| Parentheses inner | 0 space | 0 space | Same |
| Em dash spacing | 1 space each side | 1 space each side | Similar |
| Number + unit | space (sometimes) | NO space (KO convention) | **Different** |
| Slash | no space | no space | Same |
| Percent sign | 10 % (space varies) | 10% (no space) | No space in KO |

**Critical difference:** English often spaces between number and unit (10 km, 5 kg). Korean does NOT space: `10km`, `5kg`.

### Rule 6.14: English italics/bold → Korean

English uses italics for emphasis, foreign words, and titles. Korean Hangul text cannot be italicized.

| English formatting | Korean rendering | Strategy |
|-------------------|-----------------|----------|
| *italic emphasis* | Use bold or quotation marks | Alternative formatting |
| *Foreign word* (e.g., *ad hoc*) | "임시" (ad hoc) | Quote or keep in parentheses |
| ***Bold italic*** | Bold only (if supported) | Remove italics |
| *Book title* | "책 제목" or 《》 | Use quotation marks |
| *Emphasis in text* | Emphasis through word order | Structural emphasis |
| *Latin/scientific name* (e.g., *Homo sapiens*) | 호모 사피엔스 (Homo sapiens) | Keep Latin in roman |

**Rule:** Italics cannot be rendered in Hangul. Use bold, quotation marks, or plain text as alternatives.

### Rule 6.15: English capitalization → Korean

English uses capitalization for proper nouns, sentence beginnings, titles, etc. Korean has no capitalization.

| English capitalization | Korean adaptation |
|------------------------|------------------|
| "Apple Inc." | 애플 주식회사 (no caps) |
| "New York" | 뉴욕 (no caps) |
| "Chapter 5" | 제5장 (no caps) |
| "The President" | 대통령 (no caps) |
| "AI (Artificial Intelligence)" | 인공지능(AI) (no caps) |
| "GDP (Gross Domestic Product)" | 국내총생산(GDP) |
| "John Smith" | 존 스미스 (but proper nouns may keep caps in mixed text) |
| ALL CAPS (shouting) | Use bold or plain text |

**Rule:** Hangul script has no capitalization. Proper nouns are not capitalized. English abbreviations (AI, GDP, USA) may retain capitalization.

### Rule 6.16: English number range notation → Korean

| English | Korean | Notes |
|---------|--------|-------|
| "pages 10-15" | 10~15페이지 | Use ~ |
| "2010-2020" | 2010~2020년 | Use ~ |
| "$10-$20" | 10~20달러 | Use ~ |
| "Monday-Friday" | 월요일~금요일 | Use ~ |
| "10-15 people" | 10~15명 | Use ~ |
| "3-5 years" | 3~5년 | Use ~ |

**Rule:** English en dash or hyphen for ranges → Korean tilde `~`.
