# 中译泰适应规则 (zh-CN to th-TH Adaptation Rules)

## 1. Formality and Address — รูปแบบภาษาและคำเรียกขาน (敬语与称谓)

### 概述

汉语没有严格的敬语体系（仅靠"您/你"区分），而泰语拥有高度复杂的敬语系统，包括人称代词的选择、结尾礼貌词（คำลงท้าย）、以及王室用语（ราชาศัพท์）体系。中译泰时必须根据说话者的性别、双方社会关系、场合正式程度等因素，选择正确的表达层级。

### 具体规则

#### 1.1 "您"与"你"的翻译

| 场景 | 中文原文 | 泰语译文 | 说明 |
|------|----------|----------|------|
| 正式公文/对上级 | "您" | ท่าน (than) | 用于正式场合、尊称长辈或上级 |
| 一般礼貌场合 | "您" | คุณ (khun) | 通用礼貌称呼，使用频率最高 |
| 对朋友/平辈 | "你" | คุณ (khun) / เธอ (thoe) | คุณ 更礼貌，เธอ 较亲密 |
| 对下级/晚辈 | "你" | 直接使用名字或称号 | 泰语常避免直接使用"你" |
| 对顾客 | "您" | คุณลูกค้า / คุณ (khun) | 用 คุณลูกค้า（顾客您）表示尊重 |
| 客服对话 | "您" | ท่าน (than) | 极度礼貌 |
| 亲密关系 | "你" | ใช้ชื่อเล่น (用昵称) | 泰语常直接用名字或小名 |
| 对陌生人 | "你" | คุณ (khun) | 最安全的通用选择 |
| 对僧侣 | "您" | ท่าน (than) / พระคุณเจ้า | 须用专门敬语 |
| 对王室成员 | "您" | ใต้ฝ่าละอองธุลีพระบาท | 须用王室用语（ราชาศัพท์） |

#### 1.2 结尾礼貌词（คำลงท้าย）

| 说话者性别 | 陈述句 | 疑问句 | 祈使句 |
|-----------|--------|--------|--------|
| 男性 | ครับ (khráp) | ครับ (khráp) / ไหมครับ | ครับ (khráp) / เถอะครับ |
| 女性 | ค่ะ (khâ) | คะ (khá) | ค่ะ (khâ) / เถอะค่ะ |
| 男性(极正式) | ครับผม (khráp-phǒm) | ครับผม | ครับผม |
| 儿童/亲密 | จ้า (jâ) / ฮะ (há) | จ๊ะ (já) | จ้า (jâ) |

**示例对比**：

| 中文原文 | 泰语译文(男) | 泰语译文(女) |
|----------|-------------|-------------|
| "谢谢你。" | ขอบคุณครับ | ขอบคุณค่ะ |
| "请问，这是哪里？" | ขอโทษครับ ที่นี่ที่ไหนครับ | ขอโทษคะ ที่นี่ที่ไหนคะ |
| "请坐。" | เชิญนั่งครับ | เชิญนั่งค่ะ |
| "我明白了。" | เข้าใจแล้วครับ | เข้าใจแล้วค่ะ |
| "是的。" | ใช่ครับ | ใช่ค่ะ |
| "不是。" | ไม่ใช่ครับ | ไม่ใช่ค่ะ |
| "请稍等。" | รอสักครู่ครับ | รอสักครู่ค่ะ |
| "你好吗？" | สบายดีไหมครับ | สบายดีไหมคะ |
| "再见。" | ลาก่อนครับ / แล้วเจอกันครับ | ลาก่อนค่ะ / แล้วเจอกันค่ะ |
| "对不起。" | ขอโทษครับ | ขอโทษค่ะ |

#### 1.3 自称词的选择

| 中文 | 泰语(男) | 泰语(女) | 正式度 |
|------|---------|---------|--------|
| "我"(一般) | ผม (phǒm) | ดิฉัน (dì-chǎn) | 正式 |
| "我"(非正式) | ฉัน (chǎn) | ฉัน (chǎn) | 中性/女性常用 |
| "我"(极正式) | กระผม (grà-phǒm) | ดิฉัน (dì-chǎn) | 极正式 |
| "我"(亲密) | กู (guu) | เรา (rao) / ฉัน (chǎn) | 非正式 |
| "我们"(包括) | พวกเรา (phûak-rao) | พวกเรา (phûak-rao) | 通用 |
| "我们"(排除) | พวกผม (phûak-phǒm) | พวกดิฉัน (phûak-dì-chǎn) | 正式 |

#### 1.4 王室用语（ราชาศัพท์）基本原则

| 概念 | 普通用语 | 王室用语 | 说明 |
|------|---------|---------|------|
| 吃 | กิน (kin) | เสวย (sà-wǒey) | 王室专用 |
| 说 | พูด (phûut) | ตรัส (tràt) | 王室专用 |
| 去 | ไป (bpai) | เสด็จ (sà-dèt) | 王室专用 |
| 给 | ให้ (hâi) | ถวาย (thà-wǎai) | 给僧侣/王室 |
| 睡 | นอน (nɔɔn) | บรรทม (ban-thom) | 王室专用 |
| 生病 | ป่วย (pùay) | ประชวร (prà-chuan) | 王室专用 |
| 死 | ตาย (taai) | สวรรคต (sà-wǎn-khót) | 王室专用 |

**关键规则**：
1. 凡涉及泰国王室成员的文本，必须使用ราชาศัพท์专用词汇
2. 涉及佛教僧侣（พระสงฆ์）时，使用仅次于王室用语的敬语层级
3. 翻译中国政治人物讲话时，无需使用泰国王室用语，但应使用礼貌语体
4. 结尾词的选择取决于说话者性别，不取决于被称呼者性别

## 2. Number and Date Conventions — ตัวเลขและการบอกวันเดือนปี (数字与日期)

### 概述

中文和泰语在数字表达上有重大差异：中文使用"万"（10⁴）和"亿"（10⁸）作为数字单位，而泰语使用"หมื่น"（10⁴）和"ล้าน"（10⁶）。日期方面，泰历（พุทธศักราช, พ.ศ.）比公历（คริสต์ศักราช, ค.ศ.）早543年。此外，泰语拥有复杂的名词分类词（ลักษณนาม）系统。

### 具体规则

#### 2.1 数字单位转换

| 中文数字 | 数值 | 泰语表达 | 说明 |
|---------|------|---------|------|
| 万 | 10,000 | หนึ่งหมื่น (nʉ̀ng-mʉ̀ʉn) / 10,000 | 泰语无"万"概念，用"十千" |
| 十万 | 100,000 | หนึ่งแสน (nʉ̀ng-sɛ̌ɛn) / 100,000 | 十万 → แสน |
| 百万 | 1,000,000 | หนึ่งล้าน (nʉ̀ng-láan) / 1,000,000 | 百万 → ล้าน |
| 千万 | 10,000,000 | สิบล้าน (sìp-láan) | 千万用"十百万" |
| 亿 | 100,000,000 | ร้อยล้าน (rɔ́ɔy-láan) / 100,000,000 | 亿用"百百万" |
| 十亿 | 1,000,000,000 | หนึ่งพันล้าน (nʉ̀ng-phan-láan) | 十亿用"千百万" |
| 百亿 | 10,000,000,000 | หนึ่งหมื่นล้าน (nʉ̀ng-mʉ̀ʉn-láan) | 百亿用"万百万" |
| 千亿 | 100,000,000,000 | หนึ่งแสนล้าน (nʉ̀ng-sɛ̌ɛn-láan) | 千亿用"十万百万" |
| 万亿 | 1,000,000,000,000 | หนึ่งล้านล้าน (nʉ̀ng-láan-láan) | 万亿用"百万百万" |

**示例**：

| 中文原文 | 泰语译文 | 逐词说明 |
|----------|---------|---------|
| "人口约14亿" | ประชากรประมาณ 1,400 ล้านคน | 14亿 → 1.4 พันล้าน (1,400 ล้าน) |
| "投资50万元" | ลงทุน 500,000 หยวน / เงินลงทุนห้าแสนหยวน | 50万 → ห้าแสน |
| "3万5千人" | 35,000 คน / สามหมื่นห้าพันคน | 拆分表达 |
| "年产值2亿" | มูลค่าผลผลิตปีละ 200 ล้าน | 2亿 → 200 ล้าน |
| "数亿人口" | ประชากรหลายร้อยล้านคน | "数亿" → หลายร้อยล้าน |

#### 2.2 泰历 (พุทธศักราช) 年份转换

**公式**：พ.ศ. = ค.ศ. + 543  หรือ  พ.ศ. = 中文年份（公元）+ 543

| 公元年份 | 泰历年份 | 说明 |
|---------|---------|------|
| 2024 | พ.ศ. 2567 | +543 |
| 2025 | พ.ศ. 2568 | +543 |
| 1990 | พ.ศ. 2533 | +543 |
| 1949 | พ.ศ. 2492 | +543 |
| 2000 | พ.ศ. 2543 | +543 |
| 1782 | พ.ศ. 2325 | 曼谷王朝建立 |

**中文→泰语年代表达示例**：

| 中文原文 | 泰语译文 | 说明 |
|----------|---------|------|
| "2024年3月15日" | 15 มีนาคม พ.ศ. 2567 | 日+月+年(泰历) |
| "2024年3月15日" | วันที่ 15 มีนาคม 2567 | 可省略 พ.ศ. 如果上下文清楚 |
| "2024年3月" | มีนาคม พ.ศ. 2567 | 月份+年份 |
| "2024年" | พ.ศ. 2567 | 单独年份加 พ.ศ. |
| "2024-2025年" | พ.ศ. 2567-2568 | 年份范围 |
| "上世纪90年代" | ทศวรรษ 1990 / ช่วงปี ค.ศ. 1990 | 中国传统表达需意译 |

## 3. Cultural References — การปรับเปลี่ยนวัฒนธรรม (文化概念适应)

### 概述

中文文本中包含大量中国特有的文化概念、成语典故、节日、饮食名称、政治制度等。翻译为泰语时，需要根据泰国读者的文化背景进行适应化处理：或保留并加注，或替换为泰语对应概念，或用中性解释性翻译。

### 具体规则

#### 3.1 中国传统节日表达

| 中文 | 泰语 | 说明 |
|------|------|------|
| 春节 | วันตรุษจีน (wan dtrùt jiin) / เทศกาลตรุษจีน | 泰国华裔广泛庆祝 |
| 元宵节 | วันเทียนซี่ / วันขึ้น 15 ค่ำ เดือน 3 ของจีน | 泰国华裔称"เทียนซี่" |
| 端午节 | วันไหว้บ๊ะจ่าง (wan wâi bá-jàang) | 泰国华裔称"วันไหว้บ๊ะจ่าง"（粽子节） |
| 中秋节 | วันไหว้พระจันทร์ (wan wâi phrá jan) | 直译"拜月节" |
| 清明节 | วันเชงเม้ง (wan cheeng-méng) | 泰语直接音译潮州话"清明" |
| 重阳节 | วันต้งจิ๋ว / วันขึ้น 9 ค่ำ เดือน 9 | 泰国华裔庆祝 |
| 七夕 | วันแห่งความรักของจีน / เทศกาลชีซี | 需解释"中国情人节" |
| 腊八节 | วันลาบ้า | 泰国华裔较少庆祝 |
| 冬至 | วันต้งจิ๋ว (วันเหมายันของจีน) | 泰国华裔称"ต้งจิ๋ว" |
| 中元节 | วันสารทจีน (wan sàat jiin) | 泰国华裔称为"วันสารทจีน" |

## 4. Source-specific Terminology — คำศัพท์เฉพาะทาง (专业术语)

### 概述

中文和泰语在专业术语上存在大量"假朋友"（表面上相似的词汇但含义不同）和体系性差异。中译泰时必须使用泰国对应领域的标准术语，避免字面直译。

## 5. Administrative/Legal Style — รูปแบบภาษาในเอกสารทางราชการ (行政/法律文体)

### 概述

中文行政和法律法规文本具有特定的句式和用语习惯，翻译为泰语时需要转换为泰语行政法律文书的惯用表达方式。泰语的行政法律语言（ภาษาไทยในเอกสารราชการ）高度程式化，使用大量梵语-巴利语源词汇和固定句式。

### 具体规则

#### 5.1 "根据"的多种翻译

| 语境 | 中文原文 | 泰语译文 | 说明 |
|------|----------|---------|------|
| 法律依据 | "根据《……》第X条" | ตาม (taam) มาตรา…… แห่ง…… | 最常用 |
| 引用法规 | "根据……的规定" | ตามที่ (taam-thîi) กำหนดใน…… | 稍正式 |
| 授权依据 | "根据……授予的权力" | โดยอาศัยอำนาจตาม (dooy-àa-sǎi-am-nâat-taam)…… | 极正式，法律用语 |
| 行政决定 | "根据……情况" | ตาม (taam) สถานการณ์ | 一般用法 |
| 合同依据 | "根据本合同" | ตามสัญญานี้ / เป็นไปตามสัญญานี้ | 商务用语 |

#### 5.6 法律条文实例对比

**中文原文**：
"根据《中华人民共和国个人信息保护法》，处理个人信息应当遵循合法、正当、必要和诚信原则，不得通过误导、欺诈、胁迫等方式处理个人信息。"

**泰语译文**：
ตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคลแห่งสาธารณรัฐประชาชนจีน การประมวลผลข้อมูลส่วนบุคคลต้องเป็นไปตามหลักความชอบด้วยกฎหมาย หลักความถูกต้องเหมาะสม หลักความจำเป็น และหลักสุจริต ห้ามประมวลผลข้อมูลส่วนบุคคลโดยวิธีการที่ทำให้เข้าใจผิด ฉ้อโกง หรือข่มขู่

**要点分析**：
- "根据" → ตาม
- "应当" → ต้อง (义务)
- "不得" → ห้าม (禁止)
- "合法、正当、必要和诚信原则" → ใช้วลี "หลัก" + 各原则
- "通过……等方式" → โดยวิธีการ……

## 6. Punctuation Conversion — การแปลงเครื่องหมายวรรคตอน (标点符号转换)

### 概述

中文和泰语在标点符号使用上存在系统性差异。中文使用全角标点，泰语传统上几乎不使用标点符号（以空格分隔句子和段落），但现代泰语受西方影响逐渐引入部分标点。中译泰时必须进行全面的标点转换以符合泰语书写规范。

### 具体规则

#### 6.1 句号转换

| 中文 | 泰语 | 说明 |
|------|------|------|
| 。 | (无) / 空格 | 传统泰语不在句尾使用句点 |
| "今天天气很好。" | อากาศวันนี้ดีมาก (space) | 句尾无句点，以空格分隔句子 |
| "请确认。" | กรุณายืนยัน (space) | 句尾无句点 |
| "特此通知。" | จึงแจ้งให้ทราบ (space) | 公文同样不加句点 |
| "再见。" | ลาก่อน (space) | 日常用语 |
| "明白了。" | เข้าใจแล้ว (space) | 句尾无句点 |

**注意**：现代泰语中，部分受英语影响的文本可能使用句点(.)，但传统和标准泰语不在句尾使用句点。句与句之间用空格（วรรค）分隔。

#### 6.2 顿号/逗号转换

| 中文 | 泰语 | 说明 |
|------|------|------|
| 、 | , (逗号) | 泰语无顿号，用逗号 |
| "苹果、香蕉、橘子" | แอปเปิล กล้วย ส้ม | 泰语常用空格分隔，也可用逗号 |
| "第一条、第二条、第三条" | ข้อ 1, ข้อ 2, ข้อ 3 | 正式文本可用逗号 |
| "北京、上海、广州" | ปักกิ่ง เซี่ยงไฮ้ กว่างโจว | 空格分隔更常见 |
| "研究、开发、生产" | วิจัย พัฒนา ผลิต | 空格或逗号均可 |
| "一、二、三、" | 1, 2, 3 / หนึ่ง สอง สาม | 数字列举 |

**关键规则**：泰语列举时倾向使用空格而不是逗号，逗号主要用于正式/法律文本中的列举。翻译时优先使用空格分隔列举项。

#### 6.3 引号转换

| 中文 | 泰语 | 说明 |
|------|------|------|
| " " | " " / " " | 现代泰语通用双引号 |
| ' ' | ' ' | 单引号用于嵌套引用 |
| 《》 | " " / / / " " | 泰语无书名号，用双引号或斜线 |
| 「」 | " " | 泰语不使用日式引号 |
| "他说："好的。"" | เขาพูดว่า "ดี" | 泰语冒号不用于引语前 |
| 书名号《三国演义》 | "สามก๊ก" / นิยายเรื่องสามก๊ก | 书名用双引号或说明 |
| 《中华人民共和国宪法》 | รัฐธรรมนูญแห่งสาธารณรัฐประชาชนจีน | 法律名不用书名号 |

**泰语引号使用规范**：
- 直接引语使用双引号 " "
- 嵌套引语使用单引号 ' '
- 书名、作品名可用双引号或斜线 / /
- 法律名称不用书名号，直接写全称

#### 6.4 泰语特有符号

| 符号 | 名称 | 功能 | 中文对应 |
|------|------|------|---------|
| ๆ | ไม้ยมก (máai-yá-mók) | 重复前一个词 | 无（类似中文"々"） |
| ฯ | ไปยาลน้อย (bpai-yaan-nɔ́ɔy) | 缩写标记 | ……等 |
| ฯลฯ | ไปยาลใหญ่ (bpai-yaan-yài) | 等等 | ……等等 |
| ฯ | (用于地名) | 短语缩写 | 无 |
| ๏ | ฟองมัน (fɔng-man) | 段落开始（古语） | 现代已少用 |
| ฯ | อังคั่นเดี่ยว (ang-khan-dîaw) | 分隔诗句 | 无 |

**示例**：

| 中文原文 | 泰语译文 | 说明 |
|----------|---------|------|
| "小孩" | เด็ก ๆ (dèk-dèk) | 重复表示复数 |
| "慢慢来" | ช้า ๆ (cháa-cháa) | 重复表示程度 |
| "等等" | ฯลฯ (bpai-yaan-yài) | 列举省略 |
| "中华人民共和国" | จีน / สาธารณรัฐประชาชนจีน | 可缩写为 จีน |
| "很好很好" | ดี ๆ (dii-dii) | 重复表示强调 |

#### 6.5 其他标点差异

| 中文 | 泰语 | 说明 |
|------|------|------|
| ： | : | 冒号用法基本一致，泰语用冒号表示比例、时间等 |
| ； | ; | 分号在泰语中使用较少，多用空格或句号 |
| …… | ... / ฯลฯ | 省略号可用三点或等词 |
| —— | — | 破折号用法一致 |
| · (间隔号) | · | 泰语间隔号用法类似 |
| ？ | ? | 问号一致 |
| ！ | ! | 感叹号一致 |
| （） | () | 括号用法一致 |
| —— (书名号替代) | — | 泰语不用双破折号做书名 |

#### 6.6 空格规则

| 规则 | 中文 | 泰语 |
|------|------|------|
| 词间空格 | 无 | 无（泰语词之间也不空格） |
| 句子间隔 | 无/逗号/句号 | 一个空格（วรรค） |
| 段落间隔 | 换行/缩进 | 换行（可加空格缩进） |
| 数字与单位 | 无空格 | 无空格或视情况 |
| 数量与单位 | 无空格 | 无空格（如 3คน） |
| 时间表示 | 无空格 | 无特定空格规则 |

**泰语空格规则汇总**：
1. 泰语没有词间空格（词连写）
2. 句与句之间使用一个空格分隔（相当于句号功能）
3. 列举项之间使用空格或逗号
4. 数字与单位之间通常不加空格
5. 名称与姓氏之间可加空格
6. 敬称与名字之间不加空格（คุณพ่อ, คุณแม่）

#### 6.7 特殊转换场景汇总

| 场景 | 中文原文 | 泰语译文 | 说明 |
|------|----------|---------|------|
| 列举 | 第一、第二、第三 | หนึ่ง สอง สาม / 1 2 3 | 空格分隔 |
| 百分比 | 50% | 50% | 一致 |
| 温度 | 30°C | 30°C | 一致 |
| 时间 | 3:30 | 3:30 น. | 加 น. (นาฬิกา) |
| 日期范围 | 2024-2025年 | พ.ศ. 2567-2568 | 连接号保留 |
| 缩写 | 中国 | จีน (缩写) | 可省略缩写符号 |
| 书名 | 《西游记》 | "ไซอิ๋ว" | 双引号或意译 |
| 法律名 | 《合同法》 | กฎหมายสัญญา / พระราชบัญญัติสัญญา | 不用书名号 |
| 邮件地址 | user@example.com | user@example.com | 保留 |
| 网址 | https://example.com | https://example.com | 保留 |
| 电话号码 | 010-1234-5678 | 010-1234-5678 | 保留格式 |
| 标号 | 第1号 | หมายเลข 1 / เลขที่ 1 | 泰语表达 |
| 法律条款引用 | 第3条第2款 | มาตรา 3 วรรค 2 | 标准引用 |
| 序号 | (一)(二)(三) | (1) (2) (3) | 阿拉伯数字更常见 |

**格式规则汇总**：
1. 中文句号 。→ 删除，用空格代替
2. 中文顿号 、→ 泰语逗号 , 或空格
3. 引号 " " 保留使用
4. 书名号《》→ 双引号 " " 或直接写名称
5. 泰语特有符号 ๆ (重复)和ฯลฯ (等等)适当使用
6. 列举项用空格而非逗号
7. 保持全文标点风格统一
8. 句间使用一个空格分隔
