# 英译泰适应规则 (en to th-TH Adaptation Rules)

## 1. Formality and Address — รูปแบบภาษาและคำสุภาพ (敬语与称谓)

### 概述

英语没有敬语体系，一个"you"可用于所有人，而泰语拥有高度复杂的敬语系统——包含不同层级的代词、结尾礼貌词（คำลงท้าย）、以及基于社会关系的称呼体系。英译泰时必须根据说话者性别、双方关系、场合正式程度选择正确的语体层级。核心挑战在于英语中隐含的社会关系需要在泰语中显式表达。

### 具体规则

#### 1.1 礼貌层级选择

| 英语原文 | 泰语译文 | 正式度 | 适用场景 |
|----------|---------|--------|---------|
| "Please be informed that..." | ขอเรียนให้ทราบว่า…… | 极正式 | 公文/正式通知 |
| "Dear Sir/Madam" | เรียน ท่านผู้มีเกียรติ | 极正式 | 正式信函抬头 |
| "Dear Mr./Ms. ..." | เรียน คุณ…… | 正式 | 商务邮件 |
| "Thank you very much." | ขอบคุณอย่างยิ่งครับ/ค่ะ | 正式 | 感谢表达 |
| "You are cordially invited..." | ขอเรียนเชิญท่าน…… | 极正式 | 邀请函 |
| "Welcome to..." | ยินดีต้อนรับสู่…… | 正式 | 欢迎语 |
| "No smoking." | ห้ามสูบบุหรี่ | 中性（命令式） | 告示/标语 |
| "Please queue here." | กรุณาต่อแถวที่นี่ | 礼貌 | 公共告示 |
| "How are you?" | สบายดีไหมครับ/คะ | 礼貌 | 日常问候 |
| "See you later." | แล้วเจอกันครับ/ค่ะ | 礼貌友好 | 日常告别 |
| "Hello." | สวัสดีครับ/ค่ะ | 礼貌 | 通用问候 |
| "Good morning." | สวัสดีตอนเช้าครับ/ค่ะ | 礼貌 | 早上问候 |
| "Excuse me." | ขอโทษครับ/คะ | 礼貌 | 引起注意 |
| "Sorry." | ขอโทษครับ/ค่ะ | 礼貌 | 道歉 |
| "Yes." | ใช่ครับ/ค่ะ | 礼貌 | 肯定回应 |

#### 1.2 "You" 的泰语翻译策略

| 英语场景 | 泰语对应 | 说明 |
|----------|---------|------|
| "You" (customer/client) | คุณลูกค้า / ท่าน | 用尊称代替直接称呼"คุณ" |
| "You" (superior/boss) | ท่าน (than) | 对上级用最高敬语 |
| "You" (colleague) | คุณ (khun) + 名字 | 最安全的选择 |
| "You" (friend) | เธอ (thəə) / 名字 | 较亲密的称呼 |
| "You" (child) | หนู (nǔu) / 名字 | 对儿童使用 |
| "You" (subordinate) | ใช้ชื่อ / คุณ | 可用名字直接称呼 |
| "You" (royalty) | ใต้ฝ่าละอองธุลีพระบาท | 须用王室用语 |
| "You" (monk) | ท่าน / พระคุณเจ้า | 僧侣专用敬语 |
| "You" (in instructions) | (省略) | 祈使句可省略主语 |
| "You" (general public) | ท่าน / ประชาชน | 公共告示 |

**重要原则**：泰语文化中，直接使用"คุณ"在某些情况仍显生硬。更好的策略是：
- 用头衔代替：อาจารย์（老师）, หมอ（医生）, พี่（哥/姐）
- 用关系词：คุณลูกค้า（顾客您）, ท่านผู้ชม（观众您）
- 用名字/昵称
- 礼貌性省略主语

#### 1.3 结尾礼貌词规则（คำลงท้าย）

| 说话者性别 | 陈述句尾 | 疑问句尾 | 祈使句尾 | 语体 |
|-----------|---------|---------|---------|------|
| 男性 | ครับ (kráp) | ครับ (kráp) | ครับ / เถอะครับ | 礼貌 |
| 女性 | ค่ะ (khâ) | คะ (khá) | ค่ะ | 礼貌 |
| 男性 | ครับผม (kráp-phǒm) | ครับผม | ครับผม | 极正式 |
| 女性 | ค่ะ (khâ) | คะ (khá) | ค่ะ / เถอะค่ะ | 正式 |
| 男性 | จ้า (jâa) | ไหม / จ๊ะ | จ้า | 非正式 |
| 女性 | จ้า (jâa) | ไหม / จ๊ะ | จ้า | 非正式 |
| 任何 | ฮะ (há) | ฮะ | ฮะ | 亲密/口语 |

**完整对话示例**：

| 英语原文 | 泰语（男） | 泰语（女） |
|----------|-----------|-----------|
| "Hello, how are you?" | สวัสดีครับ สบายดีไหมครับ | สวัสดีค่ะ สบายดีไหมคะ |
| "I'm fine, thank you." | สบายดีครับ ขอบคุณครับ | สบายดีค่ะ ขอบคุณค่ะ |
| "What is your name?" | คุณชื่ออะไรครับ | คุณชื่ออะไรคะ |
| "My name is John." | ผมชื่อจอห์นครับ | ดิฉันชื่อจอห์นค่ะ |
| "Nice to meet you." | ยินดีที่ได้รู้จักครับ | ยินดีที่ได้รู้จักค่ะ |
| "Where are you from?" | คุณมาจากไหนครับ | คุณมาจากไหนคะ |
| "I am from the USA." | ผมมาจากสหรัฐอเมริกาครับ | ดิฉันมาจากสหรัฐอเมริกาค่ะ |
| "Goodbye." | ลาก่อนครับ / แล้วเจอกันครับ | ลาก่อนค่ะ / แล้วเจอกันค่ะ |

#### 1.4 "Thank you" 和 "Please" 的翻译

| 英语 | 泰语 | 说明 |
|------|------|------|
| "Thank you." | ขอบคุณครับ/ค่ะ | 通用感谢 |
| "Thank you very much." | ขอบคุณมากครับ/ค่ะ | 加强感谢 |
| "Thank you for your help." | ขอบคุณสำหรับความช่วยเหลือครับ/ค่ะ | 具体感谢 |
| "Thank you for your understanding." | ขอบคุณสำหรับความเข้าใจครับ/ค่ะ | 理解感谢 |
| "We appreciate your business." | ขอบคุณที่ใช้บริการครับ/ค่ะ | 商务感谢 |
| "Thank you for your kind consideration." | ขอบคุณที่กรุณาพิจารณาครับ/ค่ะ | 正式感谢 |
| "Please" (request) | กรุณา (gà-rú-naa) | 正式请求 |
| "Please" (offer) | เชิญ (chəən) | 邀请/请便 |
| "Please wait." | กรุณารอสักครู่ครับ/ค่ะ | 礼貌请求 |
| "Please confirm." | กรุณายืนยันครับ/ค่ะ | 请求确认 |
| "Please note that..." | โปรดทราบว่า…… | 正式提示 |
| "Please feel free to..." | กรุณาอย่าเกรงใจที่จะ…… | 请随意 |

**关键原则**：英语的"please"不能机械翻译为泰语的"ได้โปรด"（太过夸张）。正式场合用"กรุณา"，一般礼貌用动词+ครับ/ค่ะ即可。

#### 1.5 头衔/称呼体系

| 英语 | 泰语 | 说明 |
|------|------|------|
| Mr. / Mrs. / Ms. | คุณ (khun) | 通用礼貌头衔 |
| Dr. (medical) | หมอ (mɔ̌ɔ) / คุณหมอ | 对医生 |
| Dr. (academic) | ดร. / อาจารย์ | 对博士 |
| Professor | ศาสตราจารย์ (sàat-sà-dtraa-jaan) | 正式头衔 |
| Teacher | อาจารย์ (aa-jaan) | 对教师（极受尊敬） |
| Mr. President | ท่านประธานาธิบดี | 总统阁下 |
| Your Excellency | ฯพณฯ (phá-ná) / ท่าน | 对大使/部长等 |
| Elder sibling | พี่ (phîi) | 对年长者/前辈 |
| Younger sibling | น้อง (nɔ́ɔng) | 对年幼者/后辈 |
| Manager | ผู้จัดการ / คุณผู้จัดการ | 对经理 |
| Boss | เจ้านาย / หัวหน้า | 对上级 |
| Khun (name) | คุณ + 名字 | 日常称呼最常用 |

## 2. Dates and Numbers — วันเดือนปีและตัวเลข (日期与数字)

### 概述

英语使用月/日/年顺序（美国式）和公历纪元，而泰语使用日/月/年顺序和泰历（พุทธศักราช, พ.ศ.）。英语的12小时制需转换为泰式时间表达。此外，泰语数字表达使用不同的数量级单位和分类词系统。

### 具体规则

#### 2.1 日期格式转换

| 英语原文 | 泰语译文 | 说明 |
|----------|---------|------|
| March 15, 2024 | 15 มีนาคม พ.ศ. 2567 | 月日年 → 日月年 +543年 |
| 03/15/2024 | 15/03/2567 | 美国格式→日/月/泰历年 |
| 03/15/2024 | 15 มีนาคม 2567 | 正式泰语写法 |
| 15 March 2024 | 15 มีนาคม พ.ศ. 2567 | 日月年可保留但加543 |
| March 2024 | มีนาคม พ.ศ. 2567 | 年月格式 |
| 2024-03-15 | 2024-03-15 | ISO格式可保留（国际场合） |
| the 1990s | ทศวรรษ 1990 / ช่วงปี ค.ศ. 1990 | 公历年代用ค.ศ. |
| this Wednesday | วันพุธนี้ (wan-phút-níi) | 泰语星期表达 |
| today | วันนี้ (wan-níi) | 一致 |
| yesterday | เมื่อวานนี้ (mʉ̂ʉa-waan-níi) | 一致 |
| tomorrow | พรุ่งนี้ (phrûng-níi) | 一致 |
| next week | สัปดาห์หน้า (sàp-daa-nâa) | 一致 |
| last month | เดือนที่แล้ว (dʉʉan-thîi-lɛ́ɛo) | 一致 |
| Q1 2024 | ไตรมาส 1 ปี 2567 | 季度表达+泰历年 |
| FY2024 | ปีงบประมาณ 2567 | 会计年度 |

| 英语 | 泰语 | 罗马化 |
|------|------|--------|
| January | มกราคม | mók-gà-raa-khom |
| February | กุมภาพันธ์ | gum-phaa-phan |
| March | มีนาคม | mii-naa-khom |
| April | เมษายน | mee-sǎa-yon |
| May | พฤษภาคม | phrʉ́t-sà-phaa-khom |
| June | มิถุนายน | mí-thù-naa-yon |
| July | กรกฎาคม | gà-rá-gà-daa-khom |
| August | สิงหาคม | sǐng-hǎa-khom |
| September | กันยายน | gan-yaa-yon |
| October | ตุลาคม | dù-laa-khom |
| November | พฤศจิกายน | phrʉ́t-sà-jì-gaa-yon |
| December | ธันวาคม | than-waa-khom |

#### 2.3 时间格式

| 英语原文 | 泰语译文 | 说明 |
|----------|---------|------|
| 3:00 PM | 15:00 น. / บ่าย 3 โมง | 24小时制或说"午后3时" |
| 9:30 AM | 09:30 น. / 9 โมงครึ่ง | 上午9:30 |
| 12:00 PM | 12:00 น. / เที่ยง (thîang) | 中午 |
| 12:00 AM | 00:00 น. / เที่ยงคืน (thîang-khʉʉn) | 午夜 |
| 6:00 AM | 06:00 น. / 6 โมงเช้า | 早上6点 |
| 6:00 PM | 18:00 น. / 6 โมงเย็น | 下午6点 |
| 1 hour 30 min | 1 ชั่วโมง 30 นาที | 时间单位 |
| 24/7 | 24 ชั่วโมง / ตลอด 24 ชั่วโมง | 全天候 |
| 9 to 5 | 09:00-17:00 น. / เก้าโมงถึงห้าโมงเย็น | 工作时间 |
| quarter past two | 02:15 น. / บ่าย 2 โมง 15 นาที | 泰语不说"quarter past" |
| half past two | 02:30 น. / บ่าย 2 โมงครึ่ง | ครึ่ง = ครึ่งชั่วโมง |
| quarter to three | 02:45 น. / บ่าย 2 โมง 45 นาที | 泰语不说"quarter to" |

**时间表达关键规则**：
- 用 น. (นาฬิกา) 表示"点钟"，置于时间后（如 15:00 น.）
- ครึ่ง (khrʉ̂ʉng) = "半"，用于半点表达
- โมง (moong) = 日常口语中的"点钟"
- ทุ่ม (thûm) = 下午的时点（如 1 ทุ่ม = 19:00）
- ตี (dtii) = 凌晨的时点（如 ตี 3 = 03:00）

#### 2.4 数字单位转换

| 英语 | 数值 | 泰语 | 说明 |
|------|------|------|------|
| hundred | 100 | ร้อย (rɔ́ɔy) | 一致 |
| thousand | 1,000 | พัน (phan) | 一致 |
| ten thousand | 10,000 | หมื่น (mʉ̀ʉn) | 万→หมื่น |
| hundred thousand | 100,000 | แสน (sɛ̌ɛn) | 十万→แสน |
| million | 1,000,000 | ล้าน (láan) | 百万→ล้าน |
| ten million | 10,000,000 | สิบล้าน (sìp-láan) | 千万→十ล้าน |
| hundred million | 100,000,000 | ร้อยล้าน (rɔ́ɔy-láan) | 亿→百ล้าน |
| billion | 1,000,000,000 | พันล้าน (phan-láan) | 十亿→千ล้าน |
| trillion | 1,000,000,000,000 | ล้านล้าน (láan-láan) | 万亿→ล้านล้าน |

**示例**：

| 英语原文 | 泰语译文 |
|----------|---------|
| "50,000 people" | 50,000 คน / ห้าหมื่นคน |
| "2 million dollars" | 2 ล้านดอลลาร์ |
| "1.5 billion" | 1.5 พันล้าน / หนึ่งพันห้าร้อยล้าน |
| "a hundred thousand" | หนึ่งแสน (hundred thousand) |
| "ten thousand" | หนึ่งหมื่น (ten thousand) |

## 3. Cultural References — การปรับเปลี่ยนวัฒนธรรม (文化概念适应)

### 概述

英语文本中包含大量西方文化概念、宗教典故、习语谚语、节日名称、历史事件等。翻译为泰语时需要根据泰国读者的文化背景和目标语境进行适应化处理。

### 具体规则

#### 3.1 西方节日的泰语表达

| 英语原文 | 泰语译文 | 说明 |
|----------|---------|------|
| Christmas | วันคริสต์มาส (wan khrít-mâat) / คริสต์มาส | 泰国广泛庆祝 |
| Christmas Day | วันคริสต์มาส (25 ธันวาคม) | 全国通用 |
| New Year's Day | วันขึ้นปีใหม่ (1 มกราคม) | 一月一日 |
| New Year's Eve | วันส่งท้ายปีเก่า (31 ธันวาคม) | 12月31日 |
| Easter | วันอีสเตอร์ (wan îis-dtə̂ə) | 基督教节日 |
| Halloween | วันฮาโลวีน (wan haa-loo-wiin) | 近年泰国年轻人也开始庆祝 |
| Thanksgiving | วันขอบคุณพระเจ้า | 泰国无此节日，需解释 |
| Valentine's Day | วันวาเลนไทน์ (wan waa-leen-thai) | 泰国广泛庆祝 |
| Mother's Day | วันแม่ (wan mɛ̂ɛ) | 泰国以8月12日为母亲节 |
| Father's Day | วันพ่อ (wan phɔ̂ɔ) | 泰国以12月5日为父亲节 |
| April Fools' Day | วันเมษาหน้าโง่ (wan mee-sǎa nâa ngôo) / วันเอพริลฟูลส์ | 前者直译"四月笨脸"日 |
| Saint Patrick's Day | วันเซนต์แพทริก | 需解释 |
| Boxing Day | วันบ็อกซิ่งเดย์ (26 ธันวาคม) | 泰国无此节日 |
| Independence Day | วันประกาศอิสรภาพ | 需加国家名称 |
| Thanksgiving (US) | วันขอบคุณพระเจ้า (ของอเมริกา) | 需加"ของอเมริกา"限定 |

#### 3.2 英语习语→泰语对应

| 英语原文 | 泰语译文 | 策略 |
|----------|---------|------|
| "Kill two birds with one stone" | ยิงปืนนัดเดียวได้นกสองตัว | 泰语有完全对应"一枪打两鸟" |
| "The ball is in your court" | ถึงตาคุณแล้ว / ขึ้นอยู่กับคุณแล้ว | 意译"轮到你了" |
| "It's raining cats and dogs" | ฝนตกหนักมาก / ฝนเทลงมาไม่ขาดสาย | 意译 |
| "Once in a blue moon" | นานๆ ครั้ง / ไม่ค่อยจะเกิดขึ้น | 意译"很久才一次" |
| "When in Rome, do as the Romans do" | เข้าเมืองตาหลิ่วต้องหลิ่วตาตาม | 泰语完全对应习语 |
| "Bite the bullet" | อดทน / จำยอม / ทนรับสถานการณ์ | 意译 |
| "Break the ice" | ทำลายความเงียบ / เริ่มพูดคุย | 意译"打破沉默" |
| "The elephant in the room" | ปัญหาที่ทุกคนรู้แต่ไม่กล้าพูดถึง | 描述性翻译 |

## 4. Source-specific Terminology — คำศัพท์เฉพาะทาง (专业术语)

### 概述

英语文本中的专业术语大量源自拉丁语和希腊语，翻译为泰语时需要使用泰国对应领域的标准术语。泰语专业术语体系呈现三大来源：（1）梵语-巴利语源正式词，（2）英语直接借词（ทับศัพท์），（3）泰语本族词。

| account | บัญชี (ban-chii) / แอ็กเคานต์ | 泰语词更推荐 |
| notification | การแจ้งเตือน (gaan-jɛ̂ɛng-dtʉʉan) | 泰语词更推荐 |

## 5. Administrative/Legal Style — รูปแบบภาษาในเอกสารทางราชการ (行政/法律文体)

### 概述

英语行政和法律文本的特点是长句、被动语态、名词化结构、古体词汇（hereby, whereas, etc.）以及独特的句式结构。翻译为泰语时需要转换为泰语行政法律文书的惯用表达方式——泰语正式公文使用的词汇大量源自梵语-巴利语，句式趋向简洁主动。

### 具体规则

#### 5.1 被动语态转换

| 英语原文 | 泰语译文 | 说明 |
|----------|---------|------|
| "It is agreed that..." | คู่สัญญาตกลงกันดังต่อไปนี้…… | 英语被动→泰语主动 |
| "The report was prepared by..." | ……เป็นผู้จัดทำรายงาน | 英语被动→泰语主动 |
| "It is understood that..." | เป็นที่เข้าใจกันว่า…… | 泰语系表结构 |
| "The contract is signed..." | ได้มีการลงนามในสัญญาแล้ว | 泰语可用"ได้มีการ"表被动含义 |
| "Payment shall be made..." | ให้ชำระเงิน…… / การชำระเงินให้กระทำ…… | 两种翻译均可 |
| "It should be noted that..." | ควรสังเกตว่า…… / โปรดทราบว่า…… | 转换句式 |
| "Data is collected by..." | มีการเก็บรวบรวมข้อมูลโดย…… | 主动结构 |
| "It is prohibited to..." | ห้าม…… / การ……เป็นสิ่งต้องห้าม | 主动更简洁 |
| "Permission is required." | ต้องได้รับอนุญาต / จำเป็นต้องขออนุญาต | 主动更自然 |
| "The form must be submitted by..." | ต้องยื่นแบบฟอร์มภายใน…… | 主动句式 |

#### 5.2 英语法律古体词的泰语对应

| 英语古体词 | 泰语对应 | 说明 |
|-----------|---------|------|
| "Hereby" (hereby agrees) | โดยนี้ (โดยนี้ตกลง) / ด้วย | 法律正式用语 |
| "Thereof" | ของมัน / ดังกล่าว | 指代前述 |
| "Thereto" | แก่สิ่งนั้น / ดังกล่าว | 指前述事项 |
| "Therein" | ในนั้น / ในเอกสารดังกล่าว | 指在前文中 |
| "Thereunder" | ภายใต้นั้น / ตามนั้น | 依据前述 |
| "Whereof" | ซึ่ง / โดยที่ | 关联词 |
| "Whereby" | โดยซึ่ง / ซึ่งโดยวิธีนี้ | 手段关系 |
| "Aforesaid" / "Aforementioned" | ดังกล่าวข้างต้น (dang-glàaw-khâang-dtôn) | 法律标准用语 |
| "Hereunder" / "hereinafter" | ภายใต้สัญญานี้ / ต่อไปนี้ | 在下文/以下 |
| "Now, therefore" | ดังนั้น / อาศัยเหตุนี้ | 过渡用语 |
| "Witnesseth" | คู่สัญญาตกลงกันดังต่อไปนี้ | 合同开场句 |
| "Know all men by these presents" | สัญญานี้ได้ทำขึ้น…… | 旧式用语 |
| "In witness whereof" | เพื่อเป็นพยานหลักฐาน | 证明条款 |

#### 5.3 法律英语固定句式

| 英语原文 | 泰语译文 | 说明 |
|----------|---------|------|
| "This Agreement is entered into as of..." | สัญญานี้ทำขึ้นเมื่อวันที่…… | 合同开头 |
| "Pursuant to Article..." | ตามมาตรา…… / โดยอาศัยอำนาจตาม…… | 法律依据 |
| "Subject to the terms and conditions..." | ภายใต้เงื่อนไขและข้อกำหนด…… | 前置条件 |
| "Notwithstanding the foregoing..." | อย่างไรก็ตาม / ทั้งนี้มิได้เป็นการยกเว้น…… | 尽管有前述 |
| "In the event that..." | ในกรณีที่…… / หาก…… | 假设条件 |
| "Unless otherwise specified..." | เว้นแต่จะระบุไว้เป็นอย่างอื่น | 排除条件 |
| "Without prejudice to..." | โดยไม่กระทบต่อ…… / โดยไม่เป็นการเสื่อมเสียต่อ…… | 不影响条款 |
| "For the purpose of this Agreement..." | เพื่อประโยชน์แห่งสัญญานี้…… | 定义范围 |
| "Save as expressly provided..." | เว้นแต่ที่ได้กำหนดไว้โดยชัดแจ้ง…… | 除外条款 |
| "By virtue of..." | โดยอาศัยอำนาจตาม…… | 授权依据 |

#### 5.4 泰语行政公文体式

| 句式功能 | 泰语 | 使用场景 |
|---------|------|---------|
| 必须句式 | ต้อง…… / จะต้อง…… / จำเป็นต้อง…… | 义务规定 |
| 禁止句式 | ห้าม…… / ต้องไม่…… | 禁止规定 |
| 许可句式 | สามารถ……ได้ / อาจ……ได้ | 权利/许可 |
| 免除句式 | ได้รับการยกเว้น…… | 免除条款 |
| 定义句式 | ……หมายความว่า…… (mǎai-khwaam-wâa) | 定义条款 |
| 引用句式 | ตาม…… / ตามที่…… / โดยอาศัยอำนาจตาม…… | 法律依据 |
| 除外句式 | เว้นแต่…… (wén-dtɛ̀ɛ) / ยกเว้น…… | 除外条款 |
| 生效句式 | มีผลบังคับใช้ตั้งแต่…… | 生效条款 |
| 处罚句式 | ต้องระวางโทษ…… | 罚则条款 |
| 委任句式 | มอบอำนาจให้…… | 授权条款 |

#### 5.5 正式公文句式对比

**英语原文（合同）**：
"This Agreement is entered into as of March 15, 2024, by and between ABC Company (hereinafter referred to as 'Party A') and XYZ Corporation (hereinafter referred to as 'Party B'), pursuant to the laws of the Kingdom of Thailand."

**泰语译文**：
สัญญานี้ทำขึ้นเมื่อวันที่ 15 มีนาคม พ.ศ. 2567 ระหว่าง บริษัท ABC จำกัด (ซึ่งต่อไปนี้เรียกว่า "คู่สัญญาฝ่ายที่หนึ่ง") กับ บริษัท XYZ จำกัด (ซึ่งต่อไปนี้เรียกว่า "คู่สัญญาฝ่ายที่สอง") โดยอาศัยอำนาจตามกฎหมายแห่งราชอาณาจักรไทย

**要点分析**：
- "is entered into" → ทำขึ้น (主动)
- "as of March 15, 2024" → เมื่อวันที่ 15 มีนาคม พ.ศ. 2567 (泰历)
- "by and between" → ระหว่าง……กับ…… (固定公式)
- "hereinafter referred to as" → ซึ่งต่อไปนี้เรียกว่า…… (固定用语)
- "Party A/Party B" → คู่สัญญาฝ่ายที่หนึ่ง/ที่สอง
- "pursuant to" → โดยอาศัยอำนาจตาม
- "the laws of the Kingdom of Thailand" → กฎหมายแห่งราชอาณาจักรไทย

**英语原文（法规）**：
"No person shall collect, use, or disclose personal information without the prior consent of the data subject, except as otherwise provided by law."

**泰语译文**：
ห้ามมิให้ผู้ใดเก็บรวบรวม ใช้ หรือเปิดเผยข้อมูลส่วนบุคคลโดยไม่ได้รับความยินยอมจากเจ้าของข้อมูลส่วนบุคคลก่อน เว้นแต่ที่กฎหมายบัญญัติไว้เป็นอย่างอื่น

**要点分析**：
- "No person shall" → ห้ามมิให้ผู้ใด…… (法律禁止句式)
- "collect, use, or disclose" → เก็บรวบรวม ใช้ หรือเปิดเผย
- "prior consent" → ความยินยอม……ก่อน
- "data subject" → เจ้าของข้อมูลส่วนบุคคล (标准法律术语)
- "except as otherwise provided by law" → เว้นแต่ที่กฎหมายบัญญัติไว้เป็นอย่างอื่น (标准法律例外句式)

## 6. Punctuation Conversion — การแปลงเครื่องหมายวรรคตอน (标点符号转换)

### 概述

英语使用半角标点并有严格的句法标点规则，而泰语拥有独特且相对简约的标点体系。传统泰语几乎不使用句尾标点，句子之间以空格分隔。现代泰语受英语影响引入了部分标点，但使用规则与英语存在显著差异。

### 具体规则

#### 6.1 句尾标点

| 英语 | 泰语 | 说明 |
|------|------|------|
| . (period) | (无) / 空格 | 传统泰语不在句尾加句点 |
| "I am going to school." | ฉันไปโรงเรียน (space) | 句尾无句点，用空格隔句 |
| "The meeting is at 3 PM." | การประชุมเวลา 15:00 น. (space) | 句尾无句点 |
| "Thank you very much." | ขอบคุณมากครับ (space) | 句尾无句点 |
| "Please confirm receipt." | กรุณายืนยันการรับ (space) | 公文也无句点 |

**注意**：现代泰语文本（尤其是网络和科技文档）有时借用英文句点，但标准泰语书写规范中句尾不加句点。在正式翻译中，应删除英语句尾句点，改用空格分隔句子。

#### 6.2 引号转换

| 英语 | 泰语 | 说明 |
|------|------|------|
| " " | " " | 现代泰语普遍使用双引号 |
| ' ' | ' ' | 嵌套引语使用单引号 |
| "He said, 'Hello.'" | เขาพูดว่า "สวัสดีครับ" | 英语引号前逗号→泰语不用 |
| "He said, 'I'm tired.'" | เขาพูดว่า "ฉันเหนื่อย" | 引号前不用逗号或冒号 |
| 'Chapter 1' | "บทที่ 1" | 书名/标题用双引号 |
| "for example," | เช่น | 英语引用的短语需要适应 |

**关键差异**：
- 英语引号前用逗号：He said, "..." → 泰语无需逗号：เขาพูดว่า "..."
- 英语句点在引号内 → 泰语倾向于句点在引号外或省略
- 泰语没有书名号《》，书名用双引号 " " 表示
- 泰语正式文本中引用法律条文常用「」或 " "

#### 6.3 逗号/空格规则

| 英语原文 | 泰语译文 | 说明 |
|----------|---------|------|
| "apples, oranges, bananas" | แอปเปิล กล้วย ส้ม | 泰语常用空格代替逗号列举 |
| "However, ..." | อย่างไรก็ตาม …… | 逗号保留或删除均可 |
| "March 15, 2024" | 15 มีนาคม 2567 | 日期中的逗号删除 |
| "New York, NY" | นิวยอร์ก รัฐนิวยอร์ก | 逗号改为空格 |
| "She is tall, beautiful, and smart." | เธอสูง สวย และฉลาด | 列举用空格，and译为และ |
| "First, ..." | ประการแรก …… | 逗号可保留 |
| "Inc.," | บริษัท……จำกัด | Inc.不直接对应 |

**泰语空格与列举规则**：
1. 列举时倾向使用空格而非逗号
2. 句与句之间使用一个空格
3. 正式和法律文本中列举使用逗号
4. 数字日期中的逗号删除

#### 6.4 冒号与分号

| 英语 | 泰语 | 说明 |
|------|------|------|
| "The following items: a, b, c" | รายการดังต่อไปนี้: ก, ข, ค | 冒号用法基本一致 |
| "Time: 3:00 PM" | เวลา: 15:00 น. | 冒号常用 |
| "Therefore; however" | ดังนั้น อย่างไรก็ตาม | 分号在泰语使用较少 |

**注意**：泰语中冒号常见于正式文本和列举，分号使用频率远低于英语，通常可改为句号或空格。

#### 6.5 其他标点

| 英语 | 泰语 | 说明 |
|------|------|------|
| - (hyphen) | - | 连字符一致 |
| — (dash/em-dash) | — | 破折号一致 |
| ... / … (ellipsis) | ... / … | 省略号一致 |
| / (slash) | / | 斜线一致，表示"或" |
| & | และ / & | 正式文本用และ，商标名保留& |
| ' (apostrophe) | (无直接对应) | 所有格需转换为 ของ+名词 结构 |
| % | % | 百分号一致 |
| @ | @ | 邮箱中一致 |
| * | * | 星号注释一致 |
| ? | ? | 问号一致 |
| ! | ! | 感叹号一致 |
| ( ) | ( ) | 括号一致 |

**所有格转换示例**：

| 英语 | 泰语 |
|------|------|
| "the company's policy" | นโยบายของบริษัท |
| "John's book" | หนังสือของจอห์น |
| "the government's decision" | การตัดสินใจของรัฐบาล |
| "children's education" | การศึกษาของเด็ก |
| "today's meeting" | การประชุมวันนี้ |

#### 6.6 泰语特有标点符号

| 符号 | 泰语名称 | 功能 | 使用场景 |
|------|---------|------|---------|
| ๆ | ไม้ยมก (máai-yá-mók) | 重复前一个词或短语 | เด็ก ๆ (孩子们), ช้า ๆ (慢慢地) |
| ฯ | ไปยาลน้อย (bpai-yaan-nɔ́ɔy) | 省略缩写 | กรุงเทพฯ (曼谷缩写) |
| ฯลฯ | ไปยาลใหญ่ (bpai-yaan-yài) | "等等" | รองเท้า เสื้อ ฯลฯ (鞋、衣服等) |
| ฯ | อังคั่นเดี่ยว | 分隔诗句/短语 | 较少使用 |
| ๏ | ฟองมัน (fɔng-man) | 段落标记（古语） | 现代基本不用 |

**示例**：

| 英语 | 泰语 | 说明 |
|------|------|------|
| "children" | เด็ก ๆ (dèk-dèk) | 重复表复数 |
| "slowly" | ช้า ๆ (cháa-cháa) | 重复表程度/方式 |
| "Bangkok" | กรุงเทพฯ (grung-thêep) | 省称敬语 |
| "etc." | ฯลฯ | 列举省略 |

#### 6.7 数字/时间标点

| 英语原文 | 泰语译文 | 说明 |
|----------|---------|------|
| $1,000 | 1,000 ดอลลาร์ / ๑,๐๐๐ ดอลลาร์ | 千位分隔符保留 |
| 3.14 | 3.14 | 小数点一致 |
| 50% | 50% | 百分号一致 |
| 3:00 PM | 15:00 น. / 3:00 น. | 时间表示 |
| 30°C | 30°C / 30 องศาเซลเซียส | 温度一致 |
| pages 10-15 | หน้าที่ 10-15 / หน้า 10 ถึง 15 | 范围表示 |
| 10%-20% | 10%-20% | 范围一致 |
| 1/2 | 1/2 / ครึ่ง (half) | 分数表达 |
| +66 2 123 4567 | +66 2 123 4567 | 电话号码保留 |
| https://example.com | https://example.com | URL保留 |
| user@example.com | user@example.com | 邮箱保留 |

#### 6.8 空格与间距规则汇总

| 规则 | 英语 | 泰语 |
|------|------|------|
| 句尾句点后 | 1个空格 | 不适用（泰语无句尾句点） |
| 句与句之间 | 句点+1空格 | 1个空格（วรรค） |
| 逗号后 | 1个空格 | 0~1个空格 |
| 冒号/分号前 | 0空格 | 0空格 |
| 冒号/分号后 | 1空格 | 0~1空格 |
| 括号与文字 | 0空格 | 0空格 |
| 引号与文字 | 0空格 | 0空格 |
| 词语之间 | 1空格 | 0空格（泰语词间无空格） |
| 数字与单位 | 0~1空格 | 0空格 |
| 姓与名之间 | 1空格 | 0~1空格（通常有空格） |
| 敬称与名字 | 1空格 | 0空格（คุณพ่อ嫁，但คุณ พ่อ 可有空格） |

**标点转换核心规则**：
1. 英语句点 . → 删除，用泰语空格（วรรค）隔句
2. 英语逗号在列举中 → 可用空格代替
3. 英语日期中的逗号 → 删除
4. 所有格 's → ของ + 名词 结构
5. 英语引号 " " 保留，引语前不用逗号
6. 书名不用斜体，用双引号 " "
7. 分号使用频率降低，可改为句号或空格
8. 数字格式（千位分隔、小数点）保留
9. 适当使用泰语特有符号 ๆ（重复）和 ฯลฯ（等等）
10. 保持全文标点风格统一
