# Thai Administrative Domain (บริหาร/ราชการ) — Domain Knowledge for Thai Government Document Translation

---

## Reader Model

### Primary Reader: Thai Government Official (ข้าราชการ)

| Attribute | Description |
|-----------|-------------|
| **Title** | ข้าราชการ (Civil Servant), เจ้าหน้าที่ของรัฐ (Government Officer), ปลัดกระทรวง (Permanent Secretary) |
| **Education** | Typically ปริญญาตรีขึ้นไป; civil service exam for permanent positions |
| **Key Concern** | Strict adherence to official formats (แบบฟอร์มหนังสือราชการ); correct use of honorifics, titles, and formal register; proper routing through bureaucratic hierarchy |
| **Work Context** | Central government (ส่วนกลาง), provincial government (ส่วนภูมิภาค), local administration (ส่วนท้องถิ่น) |
| **Expectations** | Precise document classification; correct addressee titles; proper official stamps; adherence to ระเบียบสํานักนายกรัฐมนตรีว่าด้วยงานสารบรรณ |

### Secondary Reader: Member of the Public (ผู้ยื่นคําขอ / ประชาชน)

| Attribute | Description |
|-----------|-------------|
| **Role** | Submits applications to government; receives notifications |
| **Need** | Clear communication; documents must not obscure meaning with jargon |

## Decision Framework

### 1. Types of Official Documents (หนังสือราชการ)

| Thai | English | Purpose |
|------|---------|---------|
| หนังสือภายนอก | External Official Letter | Between agencies |
| หนังสือภายใน | Internal Official Letter | Within same agency |
| หนังสือประทับตรา | Stamped Letter | Routine matters |
| คําสั่ง | Administrative Order | Superior to subordinate |
| ประกาศ | Announcement / Notification | Public notice |
| ระเบียบ | Regulation | Binding internal rules |
| บันทึกข้อความ | Memorandum | Internal memo |
| รายงานการประชุม | Minutes of Meeting | Meeting record |
| หนังสือรับรอง | Certificate | Verification |
| หนังสือมอบอํานาจ | Power of Attorney | Authorization |
| ใบอนุญาต | License / Permit | Permission |
| หนังสือเวียน | Circulated Letter | Mass distribution |

### 2. External Letter Format (หนังสือภายนอก)

Per ระเบียบสํานักนายกรัฐมนตรีว่าด้วยงานสารบรรณ พ.ศ. 2526:

```
ส่วนราชการ [Agency Name]
ที่ [Ref No.]                วันที่ [Date]
เรื่อง [Subject]
เรียน [Addressee]

[อ้างถึง = Reference paragraph]
[สิ่งที่ส่งมาด้วย = Enclosures]
[Body]

จึงเรียนมาเพื่อโปรดทราบ/พิจารณา

(ลงชื่อ) [Signature]
([Full Name])
[Position]

[Seal]
[Division]
โทร. [Phone]
```

**Translation mapping**: ที่ = Ref. No. / เรื่อง = Re: / เรียน = To / สิ่งที่ส่งมาด้วย = Enclosure(s) / อ้างถึง = Reference

### 3. Internal Memo — หนังสือภายใน (บันทึกข้อความ)

```
บันทึกข้อความ
ส่วนราชการ [Division]
ที่ [Ref No.]              วันที่ [Date]
เรื่อง [Subject]
เรียน [Addressee]
[Body]
(ลงชื่อ) [Signature]
([Name])
[Position]
```

**Note**: Usually printed on pre-printed form (แบบพิมพ์) with "บันทึกข้อความ" header.

### 4. หนังสือประทับตรา (Stamped Letter)

- No signature line — official stamp replaces the signature
- Used for routine correspondence only
- Contains: letterhead, Ref, date, subject, addressee, body, official stamp

### 10. Official Seals (ตรา)

| Thai | English | Usage |
|------|---------|-------|
| ตรากลาง | Official Seal (Central) | Government agencies |
| ครุฑ | Garuda seal | Highest government symbol |
| ตรากระทรวง | Ministry seal | Specific ministry |
| ตราจังหวัด | Provincial seal | Provincial |
| ตราราชการ / ตรายาง | Agency rubber stamp | General use |

**Garuda (ครุฑ)**: Highest govt symbol; appears on all agency letterhead.

**Seal placement**: Left of signature, slightly overlapping line.

### 11. Signature Conventions

- (ลงชื่อ) = (Signed) / (Signature)
- (นาย/นาง/นางสาว [Name]) = name in parentheses
- ผู้มีอํานาจลงนาม = Authorized signatory
- ลงนามแทน = Signing on behalf of
- ลงนามกํากับทุกหน้า = Initial every page (multi-page docs)
- ประทับตราสําคัญ = Affix official seal

**Block**:
```
ลงชื่อ .........................................
(นายสมชาย ใจดี)
ตําแหน่ง อธิบดี...
วันที่ ........................
ประทับตรา
```

### 12. How to Address Officials

| Addressee | Opening | Closing |
|-----------|---------|---------|
| PM (นายกฯ) | กราบเรียนท่านนายกรัฐมนตรี | ขอแสดงความนับถืออย่างยิ่ง |
| Minister | เรียนท่านรัฐมนตรี | ขอแสดงความนับถืออย่างสูง |
| Perm. Sec. / DG | เรียนท่านปลัด/อธิบดี | ขอแสดงความนับถือ |
| Governor | เรียนท่านผู้ว่าราชการจังหวัด | ขอแสดงความนับถือ |
| General officer | เรียน [Name/Title] | ขอแสดงความนับถือ |
| Royalty | ขอพระราชทานกราบทูล | Royal language |

**Rule**: Never shorten royal salutations. "กราบเรียน" vs "เรียน" marks prostration-level respect.

### 13. Reference Number Structure (ที่)

**Format**: ที่ [Agency Abbr]/[Dept Code]/[BE Year]/[Serial No.]
**Example**: ที่ กค 0301/2565/1234

**Common ministry abbreviations**:
- กค = การคลัง (Finance)
- กต = ต่างประเทศ (Foreign Affairs)
- คม = คมนาคม (Transport)
- ศธ = ศึกษาธิการ (Education)
- สธ = สาธารณสุข (Public Health)
- ยธ = ยุติธรรม (Justice)
- มท = มหาดไทย (Interior)
- พณ = พาณิชย์ (Commerce)

### 14. Bureaucratic Register Features

| Feature | Description |
|---------|-------------|
| Nominalization | การ + verb: การดําเนินการ, การพิจารณา |
| Pali/Sanskrit | Formal lexicon: บุคลากร, ราชการ, ปฏิบัติ |
| Doublets | Paired synonyms: ครบถ้วนสมบูรณ์, ถูกต้องและเป็นธรรม |
| Passive | Indirect: "จึงขอแจ้งให้ทราบว่า" not "เราบอกว่า" |
| Hedges | Softening: เห็นควร, สมควร, น่าจะ |
| Fixed phrases | อาศัยอํานาจตามความในมาตรา... |
| Honorifics | ท่าน, กรุณา, โปรด |

**Translation approach**: Thai bureaucratic language is much more formal than standard English. Target standard formal English register.

## Standard Conventions

### 1. Document Classification Stamping

Classification (ชั้นความลับ) and urgency (ชั้นความเร็ว) stamped at top-center and bottom-center of every page. Color-coded per table above.

### 2. Attachment Notation (สิ่งที่ส่งมาด้วย)

```
สิ่งที่ส่งมาด้วย
1. [Document name] จํานวน 1 ชุด (3 แผ่น)
2. [Document name] จํานวน 2 ชุด (10 แผ่น)
```

### 3. Document Registration

Every document registered in:
- ทะเบียนหนังสือรับ = Incoming register
- ทะเบียนหนังสือส่ง = Outgoing register
- Fields: เลขทะเบียน, วันเดือนปี, เรื่อง, การปฏิบัติ

### 4. Page Layout

| Element | Thai Standard |
|---------|---------------|
| Paper | A4 (210 x 297 mm) |
| Top/Left margin | 40 mm (1.5 inch) |
| Right/Bottom margin | 20 mm (0.75 inch) |
| Font | Angsana New or Thonburi, size 16 |
| Spacing | 1 or 1.5 lines; block paragraphs, no indent |

### 5. Language Register by Document Type

| Document | Thai Register | English Equivalent |
|----------|---------------|--------------------|
| Statute / Royal Decree | แบบแผน (highest formal) | Formal legislative English |
| Ministerial Regulation | กึ่งแบบแผน | Formal regulatory English |
| External letter | แผน (formal) | Standard official English |
| Internal memo | กึ่งทางการ (semi-formal) | Standard memo English |
| Public notice | ทางการ (formal plain) | Clear accessible English |
| Certificate | แบบแผน (formal) | Formal English |

**Golden rule**: Never translate Thai bureaucratic language into casual English.

### 6. Document Submission Cover Letter Format

```
ที่ [Ref.]
วันที่ [Date]
เรื่อง ขอส่ง [Document]
เรียน [Agency Head]
ตาม [reference] ... จึงขอส่ง [docs] พร้อมรายละเอียดดังนี้
1. ...
2. ...
จึงเรียนมาเพื่อโปรด [action]
ขอแสดงความนับถือ
(ลงชื่อ) [...]
([Name])
[Position]
```

### 7. Certification and Witness Conventions

```
ข้าพเจ้าขอรับรองว่า ได้ตรวจสอบเอกสารถูกต้องตามต้นฉบับทุกประการ
I hereby certify that the foregoing is a true copy.

ลงชื่อ .........................................
([Name])
[Title]
วันที่ ..........
ประทับตรา
```

- Minimum 1 witness (usually 2 for important docs)
- Witness format: "ลงชื่อ (พยาน)"

### 8. House Registration (ทะเบียนบ้าน — ท.ร.13)

Fundamental identity document. Key fields:
- บ้านเลขที่ = House number
- หมู่บ้าน / ซอย / ถนน = Village / Lane / Road
- ตําบล / อําเภอ / จังหวัด = Sub-district / District / Province

### 9. ID Card Number Format

**Format**: X XXXX XXXXXX XX X
**Example**: 1-1234-56789-12-3
**Translation**: "Identification Card No.: 1-1234-56789-12-3"

### 12. Document Attestation / Legalization

Documents for foreign use require:
1. Certified translation (คําแปลรับรอง) by authorized translator
2. Department of Consular Affairs (กรมการกงสุล) attestation
3. Ministry of Foreign Affairs legalization
4. Destination country's embassy certification
5. Apostille (since 2024 when Thailand acceded to the Convention)
