# Domain: Administrative (Administrasi) — Indonesian

## 1. Reader Model: How to Read Indonesian Administrative Documents

### 1.1 Macro-Level Architecture

Indonesian administrative documents — particularly government correspondence (surat dinas) and official records (dokumen administrasi pemerintahan) — follow rigidly codified formats prescribed by the Ministry of Administrative and Bureaucratic Reform (KemenPAN-RB), the National Archives (ANRI — Arsip Nasional Republik Indonesia), and internal departmental instructions. The language is referred to as **bahasa resmi** (official language) or **bahasa Indonesia baku** (standard Indonesian).

**Standard government letter format (Surat Dinas — Format Baku):**

```
    [KOP SURAT / Letterhead] — Instansi, alamat, telepon, fax, website, email
        → [Nomor / Reference Number] — Format: Nomor: B-XXX/INSTANSI/YYYY
            → [Sifat / Classification] — Biasa (Routine), Rahasia (Secret), Sangat Rahasia (Top Secret)
                → [Lampiran / Attachments] — "Lampiran: X (kata) berkas"
                    → [Hal / Subject] — "Hal: [subject]" (bold, ALL CAPS, or underlined)
                        → [Tanggal / Date] — Kota, X Bulan YYYY (capitalized)
                            → [Alamat Tujuan / Addressee] — Yth. [Jabatan/Nama]
                                → [Isi Surat / Body] — Format standar paragraf
                                    → [Salam Penutup / Closing] — (optional in government letters)
                                        → [Tanda Tangan / Signature]
                                            → [Nama Jelas / Full Name]
                                                → [NIP / Civil Service ID]
                                                    → [Tembusan / Distribution List]
```

**Typical official memorandum (Nota Dinas):**

```
    NOTA DINAS
        → Kepada Yth.: [Pejabat Tujuan]
            → Dari: [Pejabat Pengirim]
                → Nomor: ND-XXX/INSTANSI/YYYY
                    → Tanggal: X Bulan YYYY
                        → Sifat: [Biasa / Segera / Rahasia]
                            → Hal: [Subject]
                                → [Isi Nota / Body]
                                    → [Tanda Tangan]
```

**Typical official decree (Keputusan/Surat Keputusan):**

```
    KEPUTUSAN [PEJABAT] NOMOR X TAHUN YYYY
        → [Judul / Title]
            → [Menimbang / Considering] — numbered considerations
                → [Mengingat / In View Of] — legal bases
                    → [MEMUTUSKAN / DECIDES]
                        → [Menetapkan / To Stipulate]
                            → [Pasal 1, 2, 3... / Articles]
                                → [Pasal Akhir / Final Article]
                                    → [Penutup / Closing]
                                        → [Ditetapkan di [kota], pada tanggal [tanggal]]
                                            → [Tanda Tangan dan NIP]

```

### 1.2 Meso-Level Patterns

**Move 1: Opening Formula — Standard Three-Line Header**
Every surat dinas opens with three mandatory lines:
- **Nomor:** `B-XXX/INSTANSI/YYYY` — the reference number encodes: letter type/department/sequential number/year
- **Sifat:** Classification (Biasa/Penting/Segera/Rahasia/Sangat Rahasia)
- **Lampiran:** Number of attachments (or "—" if none)
- **Hal:** Subject line (bold, concise)

**Move 2: Addressee Block**
- "Yth." (Yang Terhormat) always precedes the recipient's title
- Goes directly to the position, not the person: "Yth. Kepala Dinas Pendidikan Provinsi DKI Jakarta"
- Never use personal names in the addressee block of formal government letters (except for direct personal correspondence)

**Move 3: Pronominal Reference in Government Writing**
- Government documents rarely use "saya" (I). Use:
  - "kami" = we (institutional voice, representing the agency)
  - "saya" = I (only in personal capacity letters, rare)
  - "Bapak/Ibu" = you (polite, not "Anda" which is less formal in government context)

**Move 4: Closing Formula**
- Increasingly minimal in modern surat dinas — often just the signature block
- When used: "Demikianlah surat ini disampaikan untuk dapat dipergunakan sebagaimana mestinya."
  → "Thus this letter is submitted for appropriate use as necessary."

### 1.3 Micro-Level Patterns

**Formal register markers:**
- Use of prefix "ber-" + noun: "bertempat tinggal di" (residing at) instead of "tinggal di"
- Use of circumfix "ke-an": "ketidaksesuaian" (non-conformity), "ketidakhadiran" (absence)
- Avoidance of colloquial contractions: "tidak" not "nggak", "ingin" not "mau"
- Complete sentences with no slang, no acronyms without first-use definition

**NIP (Nomor Induk Pegawai):**
Format: `XXXXXXXXXX XXXXXX` (18 digits in two groups). Preserve exactly.

**NIK (Nomor Induk Kependudukan):**
Format: 16 digits. Preserve exactly.

**Standard date format in official documents:**
- "Jakarta, 15 Januari 2024" — spelled-out month, no "th" or ordinal suffix
- All letters: "Jakarta, 15 Januari 2024" NOT "Jakarta, 15 Jan 2024" or "Jakarta, 15/01/2024"

## 2. Decision Framework

### Table 1: Government Letter Reference Number Format

| Indonesian Format | Meaning | Translation Treatment |
|---|---|---|
| Nomor: B-1234/UN.10.F3/HK.00.01/2024 | Letter type / institution code / unit code / subject code / year | Preserve entirely — do not translate |
| Nomor: ND-45/BPPT/III/2024 | Memo number / institution / month (Roman) / year | Preserve entirely |
| Nomor: S-567/KN.2/2024 | Letter code "S" default / office / year | Preserve entirely |
| Nomor: KU.01.01/1/2024 | Functional code / sequential / year | Preserve entirely |
| Lampiran: Satu berkas | One file / one bundle | Translate |
| Lampiran: —
 | No attachments | Keep dash |
| Sifat: Biasa | Classification: Routine | Translate |
| Sifat: Segera | Classification: Urgent | Translate |
| Sifat: Rahasia | Classification: Confidential | Translate |
| Sifat: Sangat Rahasia | Classification: Top Secret / Classified | Translate |

**Decision rule:** Reference numbers are never translated. They encode administrative metadata. Preserve them exactly as written, including punctuation, digits, and letters.

### Table 2: Salutation and Addressing Protocol

| Indonesian | Register/Context | English Translation |
|---|---|---|
| Yth. Bapak Menteri... | Very formal / Government minister | Dear Minister... / To the Honorable Minister... |
| Yth. Ibu Kepala Dinas... | Formal / Head of agency | Dear Head of Agency... / To the Esteemed Head of... |
| Yth. Saudara Direktur... | Formal / Director | Dear Director... |
| Yth. Pimpinan Perusahaan... | Formal / Company management | Dear Management of... / To the Management of... |
| Yth. Bapak/Ibu/Saudara [Nama] | Formal but personal | Dear Mr./Ms. [Name] |
| Bapak/Ibu yang terhormat, | Very formal | The Esteemed Sir/Madam, |
| Kepada Yth.: | Formal | To the Esteemed: |
| Assalamu'alaikum Wr. Wb. | Islamic greeting (very formal) | May peace be upon you (preserve Arabic or gloss) |

**Decision rule:** "Yth." should generally be rendered as "Dear" in English correspondence. For very high-ranking officials (Minister, Governor), "The Honorable" may be appropriate. The Islamic greeting "Assalamu'alaikum Wr. Wb." should be presented in its full Arabic form with an English gloss in brackets on first occurrence, or simply preserved with a footnote.

| Materai cukup | Sufficiently stamped / Duly stamped | Indicates correct stamp duty paid |

**Decision rule:** For official stamps and signatures, note the presence of materai in brackets but do not replicate the stamp visually. "NIP" can be kept as-is or translated as "Civil Service ID." In signature blocks, keep the structure: signature line → name → NIP.

## 3. Standards

### 3.1 Regulatory Basis for Administrative Language

The use of bahasa resmi (official Indonesian) in government correspondence is governed by:

1. **Peraturan Presiden No. 63 Tahun 2019** tentang Penggunaan Bahasa Indonesia — mandates correct and standard Indonesian in official documents
2. **Peraturan Menteri PAN-RB No. 80 Tahun 2012** — Pedoman Tata Naskah Dinas (Official Document Formatting Guidelines)
3. **Peraturan Kepala ANRI No. 2 Tahun 2014** — Tata Naskah Dinas di Lingkungan Arsip Nasional (Archival Standards)
4. **Pedoman Umum Ejaan Bahasa Indonesia (PUEBI)** — General Guidelines for Indonesian Spelling (now superseded by **Ejaan Bahasa Indonesia yang Disempurnakan / EYD Edisi V**, 2022)
5. **KBBI (Kamus Besar Bahasa Indonesia)** — The official dictionary of Indonesian

### 3.2 Official Document Formatting Standards (PerMenPAN-RB 80/2012)

Key formatting rules for surat dinas:

- **Paper size:** A4 (210 mm x 297 mm) — same as international
- **Margins:** Top 3 cm, Left 3.5 cm, Right 2.5 cm, Bottom 2.5 cm
- **Font:** Times New Roman 12 pt (standard), or Arial 11 pt
- **Spacing:** 1.15 or 1.5 lines
- **Letterhead:** Must include: government agency logo (Garuda), agency name, address, telephone, fax, website, email
- **Date format:** City name, DD Month YYYY (no comma after city in some modern formats)
- **Reference number location:** Left-aligned under letterhead
- **Subject/Perihal:** Bold, ALL CAPS or title case, right-aligned or left-aligned below reference number
- **Salutation:** Left-aligned, not followed by colon in some formats
- **Body:** Justified (rata kanan-kiri)
- **Closing formula:** Right-aligned
- **Signature block:** Right-aligned, 3-4 line spaces for signature, then name, then NIP
- **Distribution list (Tembusan):** Left-aligned at bottom

### 3.4 Language Register Features of Bahasa Resmi

| Feature | Example (Baku) | Informal Equivalent |
|---|---|---|
| Prefix meN- | melakukan | ngelakuin |
| Prefix ber- | bertempat tinggal | tinggal |
| Suffix -kan | menyampaikan | ngasih tau |
| Circumfix ke-an | ketidaksesuaian | gak cocok |
| Avoidance of clipped words | terima kasih | makasih |
| Avoidance of foreign loanwords (where native exists) | undangan | invitation (avoid unless necessary) |
| Complete structure | Yang bertanda tangan di bawah ini | Saya yang tanda tangan |
| Standard word order | Sesuai dengan peraturan yang berlaku | Menurut peraturan |

**Decision rule:** In administrative translation into English, do not attempt to replicate the formal register's "heaviness" through archaic English. Instead, aim for precise, clear, formal English that mirrors the original's specificity and completeness without being stilted. Use standard formal English conventions (e.g., "we," "the undersigned," "in accordance with," "pursuant to" as appropriate).

### 3.5 Correspondence Hierarchy and Flow

Indonesian administrative correspondence observes strict hierarchy:

- **From lower to higher:** Use "memohon" (to request), "melaporkan" (to report), "Mohon petunjuk" (Request guidance)
- **Equal level:** Use "menyampaikan" (to convey), "memberitahukan" (to inform)
- **From higher to lower:** Use "memerintahkan" (to order), "menugaskan" (to assign), "menginstruksikan" (to instruct)

Translation posture: Preserve the hierarchical direction. If a letter is from a Camat to a Bupati, use respectful language in the translation (e.g., "We respectfully report," not "We inform you").

### 3.6 Digital Administrative Transformation (SPBE)

Indonesia has been implementing **Sistem Pemerintahan Berbasis Elektronik (SPBE)** — Electronic-Based Government System:

- **Surat elektronik (e-office):** Digital correspondence systems
- **Tanda tangan elektronik (TTE):** Electronic signatures (certified per UU ITE)
- **Naskah dinas elektronik:** Electronic official documents
- **Sistem Informasi Kearsipan:** Digital archiving

Translation note: These digital equivalents follow the same structural and linguistic conventions as paper-based documents.
