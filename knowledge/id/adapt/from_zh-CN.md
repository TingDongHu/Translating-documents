# Adaptasi dari Bahasa Mandarin (Zhongwen) ke Bahasa Indonesia

Panduan adaptasi saat menerjemahkan dari bahasa Mandarin (Zhongwen) ke bahasa Indonesia.

---

## 1. Formalitas (形式)

| Mandarin | Indonesia | Keterangan |
|---|---|---|
| 您 (nín) | Anda / Bapak / Ibu | Pilih berdasarkan konteks dan jabatan lawan bicara |
| 你 (nǐ) | kamu / Anda | 你 ke teman sebaya/setara; gunakan **Anda** jika ragu |
| 你好 | Halo / Selamat pagi/siang/sore | Tidak ada padanan langsung; gunakan salam waktu |
| 大家好 | Halo semuanya / Salam semua | Padanan natural dalam konteks formal |
| 先生 (xiānsheng) | Bapak / Tuan | 先生 = suami juga, bedakan dengan teliti |
| 女士 (nǚshì) | Ibu / Nyonya | Cocok untuk konteks formal |
| 小姐 (xiǎojiě) | Nona / Saudari | Makna bergeser; hati-hati penggunaannya |
| 同志们 (tóngzhìmen) | rekan-rekan / hadirin | 同志 tidak lazim di ID; cari padanan netral |
| 请 (qǐng) | silakan / mohon | Tergantung konteks: ajakan vs permohonan |
| 谢谢 (xièxie) | terima kasih | Langsung, tidak perlu perubahan |
| 对不起 (duìbuqǐ) | maaf | Sama, langsung |
| 贵 (guì) + kata benda | — | Awalan hormat; adaptasi: **Perusahaan Anda** (贵公司) |
| 令 (lìng) + kata benda | — | Awalan hormat untuk kerabat; **ayah Anda** (令尊) |
| 敝 (bì) + kata benda | — | Awalan rendah hati; **perusahaan kami** (敝公司) |
| 尊敬的 (zūnjìng de) | Yang Terhormat / Yth. | Cocok di surat resmi |
| 亲爱的 (qīn'ài de) | Yang terkasih / Sayang | Hanya untuk hubungan sangat dekat; hindari di surat resmi |
| 为盼 (wèi pàn) | — | Frasa penutup surat; adaptasi: **Atas perhatiannya, kami ucapkan terima kasih** |
| 此致 (cǐ zhì) | Hormat kami / Demikian | Padanan penutup surat formal |

## 2. Angka dan Tanggal (数字与日期)

| Mandarin | Indonesia | Keterangan |
|---|---|---|
| 万 (wàn) | 10.000 | Satuan berubah; 三万 = 30.000 |
| 亿 (yì) | 100.000.000 | 一亿 = 100.000.000 |
| 万亿 (wànyì) | 1.000.000.000.000 (triliun) | Konversi hati-hati;万亿 = triliun dalam sistem ID |
| 十万 | 100.000 | Gunakan titik pemisah ribuan |
| 百万 (bǎiwàn) | 1.000.000 | juta |
| 千万 (qiānwàn) | 10.000.000 | sepuluh juta |
| 兆 (zhào) | triliun | Beda makna di IT (megabyte); konfirmasi konteks |
| 百分之... (bǎi fēn zhī) | ... persen | 百分之五十 = 50% |
| 小数点 (xiǎoshù diǎn) | koma desimal | Tanda: **,** (koma) di ID |
| 日期格式 | DD/MM/YYYY | 2025/03/15 → 15/03/2025 |
| 2025年3月15日 | 15 Maret 2025 | Urutan: tanggal-bulan-tahun |
| 2025年 | tahun 2025 | Tambahkan "tahun" di depan |
| 一季度 (dì yī jìdù) | triwulan I / kuartal I | Q1 |
| 二零二五 | dua ribu dua puluh lima | Angka bertulis; tulis dalam angka Arab |
| 第1条 | Pasal 1 / Ayat 1 | Penomoran hukum |
| 包括但不限于 | termasuk namun tidak terbatas pada | Padanan baku kontrak |
| 左右 (zuǒyòu) | sekitar / kurang lebih | 五十人左右 = sekitar 50 orang |
| 余 (yú) | lebih dari / sekian | 五十余人 = lebih dari 50 orang |

## 6. Tanda Baca (标点符号)

| Mandarin | Indonesia | Keterangan |
|---|---|---|
| 。(句号) | . (titik) | Tanda titik, bukan lingkaran |
| ，(逗号) | , (koma) | Koma biasa, bukan koma penuh |
| 、(顿号) | , (koma) | Tanda baca enumerasi CN → koma ID |
| ""(引号) | ""(tanda kutip) | Gunakan tanda kutip Indonesia "" |
| 《》(书名号) | ""(tanda kutip) atau miring | 《论语》→ "Analek" atau *Analek* |
| ( )(括号) | ( ) (tanda kurung) | Sama, tidak perlu perubahan |
| ——(破折号) | — (tanda pisah) | Panjang em-dash Indonesia lebih pendek |
| ……(省略号) | ... (tiga titik) | Enam titik CN → tiga titik ID |
| ·(间隔号) | tanda titik tengah / spasi | 卡尔·马克思 → Karl Marx |
| ：(冒号) | : (titik dua) | Sama |
| ；(分号) | ; (titik koma) | Sama |
| ?(问号) | ? (tanda tanya) | Sama |
| !(感叹号) | ! (tanda seru) | Sama |
| —(连接号) | — (tanda hubung) | Bisa pakai tanda pisah atau strip |
| 、(列举号) | , atau ; | Dalam enumerasi panjang, bisa pakai titik koma |
| ( ) dan 【 】 | ( ) | Gunakan kurung biasa; 【 】 tidak lazim di ID |
| 空格 (spasi) | spasi | Sama, tidak perlu perubahan |

> **Catatan Umum**: Setelah adaptasi tanda baca, selalu periksa bahwa tanda kutip, titik, dan koma sudah sesuai Ejaan Bahasa Indonesia yang Disempurnakan (EYD V).
