# Adaptasi dari Bahasa Inggris (English) ke Bahasa Indonesia

Panduan adaptasi saat menerjemahkan dari bahasa Inggris ke bahasa Indonesia.

---

## 1. Formalitas (Formality)

| English | Indonesia | Keterangan |
|---|---|---|
| You (singular, formal) | Anda / Bapak / Ibu | Pilih berdasarkan hubungan dan konteks |
| You (singular, informal) | kamu / engkau | Hanya untuk hubungan akrab; **Anda** lebih aman |
| You (plural) | kalian / Anda sekalian | **Anda sekalian** untuk formal; **kalian** untuk informal |
| Your (formal) | ... Anda / ... Bapak / ... Ibu | your report → laporan Anda |
| My (humble) | ... saya | Lebih sopan daripada **aku** dalam konteks formal |
| I / me | saya / aku | **Saya** standar; **aku** hanya untuk informal/puisi |
| We / us (inclusive) | kita | Termasuk lawan bicara |
| We / us (exclusive) | kami | Tidak termasuk lawan bicara |
| Mr. | Bapak / Pak | **Pak** untuk informal, **Bapak** untuk formal |
| Mrs. | Ibu / Bu | **Bu** untuk informal, **Ibu** untuk formal |
| Ms. | Saudari / Ibu | Gunakan **Ibu** jika status pernikahan tidak diketahui |
| Sir | Pak / Tuan | Konteks militer/pelayanan |
| Madam | Bu / Nyonya | Konteks formal |
| Ladies and Gentlemen | Hadirin sekalian / Bapak-Ibu | Padanan natural |
| Dear Sir/Madam | Dengan hormat | Pembuka surat resmi bahasa Indonesia |
| Sincerely | Hormat saya | Penutup surat resmi |
| passive voice (e.g. "is done") | di- (dilakukan) | Contoh: "The report is prepared" → "Laporan disiapkan" |
| passive voice (by agent) | di- + oleh | Contoh: "prepared by the team" → "disiapkan oleh tim" |
| imperative (polite) | mohon / harap | "Please submit" → "Mohon disampaikan" |
| imperative (direct) | ... -lah | "Do it now" → "Lakukanlah sekarang" |

## 2. Tanggal dan Waktu (Dates & Time)

| English | Indonesia | Keterangan |
|---|---|---|
| MM/DD/YYYY | DD/MM/YYYY | 03/15/2025 → 15/03/2025 |
| March 15, 2025 | 15 Maret 2025 | Urutan: tanggal-bulan-tahun |
| 2025-03-15 | 15 Maret 2025 | Format ISO juga perlu diadaptasi |
| 3/15 | 15/3 | Hari-bulan, bukan bulan-hari |
| AM / am | pagi | 9:00 AM → 09.00 pagi |
| PM / pm | siang / sore / malam | Bedakan: 12.00 siang, 15.00 sore, 20.00 malam |
| 12:00 AM | 00.00 / tengah malam | Jangan tulis "12:00 pagi" |
| 12:00 PM | 12.00 / tengah hari | Jangan tulis "12:00 siang" saja jika ragu |
| 3:00 PM | 15.00 | Gunakan format 24 jam |
| 9:30 AM | 09.30 | Format 24 jam; gunakan titik sebagai pemisah (09.30 bukan 09:30) |
| 5:45 PM | 17.45 | Sore hari; konsisten format 24 jam |
| o'clock | pukul / jam | "at 3 o'clock" → "pukul 15.00" |
| half past | setengah | "half past three" → "setengah empat" (bukan "setengah tiga") |
| quarter past | lewat seperempat | "quarter past two" → "pukul 02.15 / dua lewat seperempat" |
| quarter to | kurang seperempat | "quarter to three" → "pukul 02.45 / tiga kurang seperempat" |
| today | hari ini | Langsung |
| tomorrow | besok | Langsung |
| yesterday | kemarin | Langsung |
| next week | minggu depan | Langsung; perhatikan penggunaan "pekan" vs "minggu" |
| last month | bulan lalu | Langsung |
| FY (Fiscal Year) | TA (Tahun Anggaran) | FY 2025 → TA 2025 |
| Q1, Q2, Q3, Q4 | TW I, TW II, TW III, TW IV | atau Kuartal I, II, III, IV |

| English | Indonesia | Keterangan |
|---|---|---|
| shall | harus / wajib | **Wajib** untuk kewajiban hukum; **harus** lebih umum |
| shall not | dilarang / tidak boleh | Larangan mutlak |
| must | harus / wajib | Sama dengan shall |
| may | dapat / boleh | Opsional, bukan kewajiban |
| may not | tidak boleh / tidak dapat | Larangan opsional |
| will | akan | Tidak sekaku **shall** |
| pursuant to | berdasarkan / sesuai dengan | Istilah kontrak baku |
| in accordance with | sesuai dengan / sejalan dengan |  sesuai dengan ketentuan |
| by virtue of | berdasarkan / demi | Berdasarkan kekuasaan hukum |
| under this Agreement | berdasarkan Perjanjian ini | / menurut Perjanjian ini |
| subject to | dengan tunduk pada / terikat pada | Frasa hukum penting |
| notwithstanding | meskipun / tanpa mengesampingkan | **Notwithstanding** → "Tanpa mengurangi berlakunya" |
| without prejudice to | tanpa mengesampingkan | Istilah hukum teknis |
| hereby | dengan ini |  hereby certify → dengan ini menyatakan |
| thereof | daripadanya / tersebut | "or any part thereof" → "atau bagian daripadanya" |
| therein | di dalamnya | "as set forth therein" → "sebagaimana diatur di dalamnya" |
| hereinafter | selanjutnya / berikut ini | "hereinafter referred to" → "selanjutnya disebut" |
| whereas | menimbang bahwa | Pembuka konsiderans |
| in witness whereof | sebagai bukti / demi saksi | Penutup akta |
| effective date | tanggal berlaku | Tanggal efektif |

| force majeure | keadaan kahar | Padanan baku |
| dispute | sengketa | Istilah hukum |

## 6. Tanda Baca (Punctuation)

| English | Indonesia | Keterangan |
|---|---|---|
| "" (quotation marks) | ""(tanda kutip) | Gunakan tanda kutip Indonesia; tidak ada perubahan |
| '' (single quotes) | ''(tanda kutip tunggal) | Sama, gunakan untuk kutipan di dalam kutipan |
| . (period / full stop) | . (titik) | Sama |
| , (comma) | , (koma) | Sama — **tetapi** |
| Oxford comma (serial comma) | **JANGAN** gunakan | Indonesia tidak menggunakan koma sebelum "dan/atau" dalam enumerasi |
| Example: a, b, and c | a, b, dan c | Tanpa koma sebelum "dan" |
| : (colon) | : (titik dua) | Sama; spasi sebelum titik dua di ID? Tergantung gaya |
| ; (semicolon) | ; (titik koma) | Sama; gunakan dalam enumerasi kompleks |
| - (hyphen) | - (tanda hubung) | Sama; kata ulang ID: **berulang-ulang** |
| -- / — (em dash) | — (tanda pisah) | Tanda pisah Indonesia; spasi sebelum dan sesudah |
| ... (ellipsis) | ... (tiga titik) | Sama, tiga titik |
| ! (exclamation) | ! (tanda seru) | Sama |
| ? (question mark) | ? (tanda tanya) | Sama |
| ( ) (parentheses) | ( ) (tanda kurung) | Sama |
| [ ] (brackets) | [ ] (tanda kurung siku) | Sama |
| / (slash) | / (garis miring) | Sama; dan/atau → lazim di ID |
| & (ampersand) | dan | Jangan gunakan & dalam teks formal; tulis "dan" |
| ' (apostrophe) | — | Apostrof tidak lazim di ID; **John's book** → **buku John** |
| % (percent) | % (persen) | Sama; tulis **50%** bukan **50 %%** |
| # (number sign) | No. / Nomor | #1 → No. 1 |
| @ (at sign) | @ (tanda at) | @ dalam surel; "at" jika dibaca: **john@mail.com** |

> **Catatan Umum**: Pastikan hasil terjemahan mengikuti Pedoman Umum Ejaan Bahasa Indonesia (PUEBI / EYD V). Perhatikan konsistensi penggunaan istilah dan tanda baca di seluruh dokumen.
