## Títulos e saudações

### Correspondência formal
- «Madame, Monsieur,» → «Prezado(a) Senhor(a),» / «A quem possa interessar,»
- «Monsieur [Nom],» → «Prezado Sr. [Sobrenome],»
- «Madame [Nom],» → «Prezada Sra. [Sobrenome],»
- «Veuillez agréer, Madame, Monsieur, l'expression de mes salutations distinguées.» → «Atenciosamente,» / «Respeitosamente,»
- «Cordialement,» → «Cordialmente,»

### Títulos profissionais
- «Directeur général» → «Diretor-Geral» / «CEO»
- «Directeur financier» → «Diretor Financeiro» / «CFO»
- «Président» → «Presidente»
- «Gérant» → «Gerente» / «Diretor»
- «Chef de service» → «Chefe de departamento»
- «Chargé de mission» → «Responsável de projeto»

## Estilo e registro

### Diferenças de tom
- O francês formal é mais elaborado que o português brasileiro
- Simplificar fórmulas longas quando apropriado
- «Nous avons l'honneur de vous informer» → «Temos a honra de informar» / «Informamos»
- «Conformément à votre demande» → «Conforme solicitado»
- «Ci-joint» → «Segue em anexo»

### Concordância de gênero
- O francês tem concordância de gênero mais estrita que o português
- «Cher/Chère» → «Prezado/Prezada»
- «Attaché/Attachée» → «Responsável» (forma neutra)

### Artigos
- O francês usa artigos definidos mais frequentemente que o português
- «La recherche montre...» → «A pesquisa mostra...»
- «L'accès aux données» → «O acesso aos dados»

## Expressões idiomáticas

### Expressões temporais
- «Dans les meilleurs délais» → «O mais breve possível»
- «En temps utile» → «Em tempo oportuno»
- «Dès que possible» → «Assim que possível»
- «Avec effet immédiat» → «Com efeito imediato»

### Expressões comerciais
- «N'hésitez pas à nous contacter» → «Não hesite em entrar em contato»
- «Dans l'attente de votre réponse» → «Aguardamos seu retorno»
- «Veuillez trouver ci-joint» → «Segue em anexo»

## Estruturas gramaticais

### Condicionais
- «Si vous avez des questions» → «Caso tenha dúvidas» / «Em caso de dúvidas»
- «Dans le cas où» → «No caso em que» / «Caso»

### Voz passiva
- «Il est recommandé de...» → «Recomenda-se que...» / «É recomendável que...»
- «Le rapport a été préparé par...» → «... preparou o relatório» (voz ativa quando possível)

### Subjuntivo
- «Il est essentiel que cela soit fait» → «É essencial que isto seja feito»
- «Nous insistons pour qu'il soit présent» → «Insistimos que ele compareça»

## Referências culturais

### Instituições
- «Parlement» → «Congresso Nacional»
- «Cour de cassation» → «Superior Tribunal de Justiça (STJ)»
- «Conseil d'État» → «Conselho de Estado» / «Tribunal Superior»
- «Direction générale des Finances publiques» → «Receita Federal do Brasil»
- «ANSM» → «ANVISA»

### Unidades e medidas
- França usa sistema métrico — compatível com o Brasil
- «kilomètres» → «quilômetros»
- «kilogrammes» → «quilogramas»
- «Celsius» → «Celsius»

### Moeda
- «€» → «R$» (se contexto brasileiro) ou manter a moeda de origem
- «euros» → «euros» / «reais» (dependendo do contexto)
- Formato: R$ 1.234,56 (ponto para milhares, vírgula para decimais)

## Nomes próprios e entidades

### Nomes de pessoas
- Transliterar prenomes quando existe forma portuguesa: «Jean» → «João»
- Manter sobrenomes inalterados
- «Pierre» → «Pedro» (se tradicional), caso contrário manter «Pierre»

### Nomes de empresas
- Manter nomes de empresas inalterados (L'Oréal, Total, etc.)
- Traduzir nomes descritivos se apropriado: «Société Nationale des Chemins de fer» → «Companhia Nacional de Ferrovias»

## Sistema jurídico

### Conceitos específicos do direito francês
- «SARL» → «Ltda.» (Sociedade Limitada)
- «SA» → «S.A.» (Sociedade Anônima)
- «SNC» → «Sociedade em Nome Coletivo»
- «SCI» → «Sociedade Civil Imobiliária» (sem equivalente exato — explicar)
- «SCOP» → «Cooperativa de Trabalho» (sem equivalente exato)

### Documentos
- «Kbis» → «Cartão CNPJ» (equivalente funcional)
- «Statuts» → «Contrato Social» / «Estatuto Social»
- «Procès-verbal» → «Ata» / «Auto de Infração»
- «Récépissé» → «Recibo» / «Comprovante»

## Sistema fiscal

### Impostos
- «TVA» → «ICMS» / «IVA» (dependendo do contexto)
- «Impôt sur le revenu» → «Imposto de Renda»
- «Impôt sur les sociétés» → «Imposto de Renda Pessoa Jurídica (IRPJ)»
- «Contribution sociale» → «Contribuição Social»
- «URSSAF» → «INSS» (equivalente funcional)

### Documentos fiscais
- «Déclaration fiscale» → «Declaração de Imposto de Renda»
- «Avis d'imposition» → «Aviso de Lançamento»
- «Bulletin de salaire» → «Holerite» / «Contracheque»

## Sistema social

### Previdência
- «Sécurité sociale» → «Previdência Social» / «INSS»
- «Assurance maladie» → «Plano de Saúde» / «SUS»
- «Assurance chômage» → «Seguro-Desemprego»
- «Retraite» → «Aposentadoria»
- «Allocations familiales» → «Auxílio-família» / «Benefício Familiar»

### Emprego
- «Contrat de travail» → «Contrato de Trabalho»
- «Licenciement» → «Demissão» / «Rescisão»
- «Préavis» → «Aviso prévio»
- «Période d'essai» → «Período de experiência»
- «Congés payés» → «Férias remuneradas»
- «Heures supplémentaires» → «Horas extras»
