## Formalidade (Formalidade)

### Pronomes de tratamento

| Chinês (zh-CN) | Português (BR) | Contexto |
|---|---|---|
| 您 | o senhor / a senhora | correspondência formal, documentos oficiais |
| 您 | você | semi-formal, cada vez mais comum no Brasil |
| 你 | você | informal, familiar, conversas cotidianas |
| 你 | tu | Portugal (PT-PT), informal; Brasil: uso regional (Sul) |
| 您们 | vocês / os senhores | plural de 您 (informal; não padronizado em chinês) |
| 诸位 | todos / ilustres presentes / senhoras e senhores | muito formal, discursos |
| 各位 | prezados(as) / senhores(as) | formal, abertura de reuniões e eventos |
| 大家 | todos / pessoal | semi-formal a informal |
| 先生 | Senhor (Sr.) | formal, cortesia |
| 女士 | Senhora (Sra.) | formal, cortesia |
| 小姐 | senhorita / moça | formal (atenção: contexto brasileiro pode ser delicado) |
| 同志 | camarada | contexto político-partidário |
| 亲爱的 | querido(a) / caro(a) | cartas pessoais, e-mails informais |
| 尊敬的 | prezado(a) / ilustre / excelentíssimo(a) | muito formal, autoridades |
| 贵公司 | essa conceituada empresa / vossa empresa / essa empresa | formal, tratamento indireto a empresas |
| 贵方 | Vossa Senhoria / V.Sa. | formal em documentos (arcaico no Brasil) |
| 您贵姓? | Qual é o seu nome? / Como se chama? | formal, pedido de identificação |

### Contexto de uso

- O chinês distingue formalidade principalmente por pronomes (您 vs. 你); o português tem um sistema mais rico de formas de tratamento
- No PT-BR, «você» é usado amplamente mesmo em contextos semi-formais; «o senhor / a senhora» é reservado para situações formais ou hierarquia
- No PT-PT, «você» pode soar mais distante; «tu» é o padrão informal
- Escolha o tratamento com base no contexto do documento, não traduza literalmente

### Hierarquia e sociedade

- «老板» → «chefe» / «patrão» (literalmente «patrão»; contexto-dependente)
- «领导» → «superior» / «gestor» / «direção» (literalmente «líder»; usar o cargo específico se possível)
- «老师» → «professor(a)» / «mestre» (literalmente «professor»; também usado como título de respeito para artistas, médicos)
- «师傅» → «mestre» / «senhor» (tratamento respeitoso para trabalhadores experientes; sem equivalente direto)
- 大哥/大姐 → «senhor» / «senhora» (tratamento respeitoso informal, geralmente substituir por «senhor(a)»)

### Tabela de conversão

|---|---|---|

| ， | , (vírgula) | sem espaço antes |
| 、 | , (vírgula) | 、 é usado para enumerações → converter para vírgula |
| ； | ; (ponto e vírgula) | uso semelhante |
| ： | : (dois-pontos) | sem espaço antes |
| ？ | ? (ponto de interrogação) | sem espaço antes |
| ！ | ! (ponto de exclamação) | sem espaço antes |
| “ ” | " " (aspas curvas duplas) | PT-BR prefere aspas curvas " " |
| 『』 | ' ' (aspas simples) | ou « » (aspas angulares, PT-PT) |
| 《》 | « » (aspas angulares) OU itálico | títulos de livros, filmes, obras |
| —— | — (travessão) | travessão (—) sem espaços; em PT-PT: « » |
| …… | ... (reticências) | 3 pontos em PT; 6 pontos em chinês |

| / | / (barra) | uso semelhante |

| ·· | " " | aspas chinesas ··texto·· → "texto" |
| # | n.º / nº | número |

### Regras especiais

- Espaçamento: português NÃO usa espaços antes de sinais de pontuação (diferente do francês)
- Vírgula serial (Oxford comma): NÃO usar em PT-BR; não se coloca vírgula antes de «e» em enumerações simples
- Aspas internas: usar aspas simples ' ' dentro de aspas duplas " "
- Títulos de obras: em PT-BR, usar itálico para títulos de livros, filmes e obras; se não houver itálico, usar "aspas"
- Travessão: substituir —— chinês por — (travessão simples); em PT-PT, diálogos usam travessão também
- Reticências: reduzir de 6 para 3 pontos
- Ordem dos sinais: em chinês, as aspas de fechamento vêm antes dos sinais finais (。」) → em PT, a pontuação final fica DENTRO das aspas apenas se fizer parte da citação
