## Fórmulas de cortesia

### Correspondência
- «尊敬的» → «Prezado(a) Senhor(a),» / «Prezados(as),»
- «此致敬礼» → «Atenciosamente,» / «Cordialmente,»
- «顺祝商祺» → «Respeitosamente,»
- «敬上» → «Atenciosamente,»

### Contexto formal
- O chinês é frequentemente mais formal e indireto que o português brasileiro
- Adaptar o registro ao contexto brasileiro (administrativo, comercial, etc.)
- As fórmulas de cortesia chinesas são frequentemente mais longas — condensar em português

## Números e formatos

### Números cardinais
- Chinês: 一万 (10.000) → português: dez mil
- Chinês: 一亿 (100.000.000) → português: cem milhões
- Atenção às ordens de grandeza diferentes

### Datas
- Chinês: 2026年5月18日 → português: 18 de maio de 2026
- Formato: DD/MM/AAAA em português (não AAAA/MM/DD)

### Números de telefone
- Chinês: +86 10 1234 5678 → manter o formato internacional
- Brasileiro: +55 11 1234-5678

### Endereços
- Chinês: ordem do maior para o menor (省→市→区→街→号)
- Brasileiro: ordem do menor para o maior (número→rua→cidade→CEP)
- Adaptar a ordem dos elementos

## Termos administrativos chineses

### Instituições
- «公安局» → «Departamento de Segurança Pública» / «Polícia»
- «工商局» → «Administração de Indústria e Comércio»
- «税务局» → «Administração Fiscal» / «Receita Federal»
- «人社局» → «Departamento de Recursos Humanos e Segurança Social»
- «人民法院» → «Tribunal Popular» / «Tribunal»
- «人民检察院» → «Ministério Público Popular» / «Ministério Público»
- «司法局» → «Departamento de Justiça»

### Documentos oficiais
- «营业执照» → «Licença Comercial» / «Cartão CNPJ»
- «身份证» → «Carteira de Identidade» / «RG»
- «户口本» → «Registro de Residência» (sem equivalente direto — explicar se necessário)
- «暂住证» → «Autorização de Residência Temporária»
- «学历证书» → «Diploma» / «Certificado de Conclusão»
- «无犯罪记录证明» → «Certidão de Antecedentes Criminais»

### Empresas
- «有限公司» → «Sociedade Limitada (Ltda.)»
- «股份有限公司» → «Sociedade Anônima (S.A.)»
- «个体工商户» → «Empresário Individual» / «MEI»
- «合伙企业» → «Sociedade Simples»
- «法人代表» → «Representante Legal»
- «注册资本» → «Capital Social»

## Expressões idiomáticas

### Expressões cotidianas
- «谢谢» → «Obrigado(a)» / «Agradeço»
- «不好意思» → «Com licença» / «Desculpe»
- «麻烦您了» → «Desculpe o incômodo.»
- «请多关照» → «Agradeço a atenção.» (formal)

### Expressões comerciais
- «恭喜发财» → «Feliz Ano Novo e prosperidade!» (contexto Ano Novo Chinês)
- «生意兴隆» → «Bons negócios!»
- «合作愉快» → «Parceria de sucesso.»

## Referências culturais

### Festas e datas
- «春节» → «Ano Novo Chinês» / «Festa da Primavera»
- «中秋节» → «Festa da Lua» / «Festa do Meio do Outono»
- «国庆节» → «Festa Nacional Chinesa» (1º de outubro)
- «端午节» → «Festa dos Barcos de Dragão»
- «清明节» → «Festa dos Mortos» / «Qingming»

### História e geografia
- Manter nomes de lugares chineses com romanização pinyin
- «北京» → «Pequim» (uso tradicional brasileiro) / «Beijing»
- «上海» → «Xangai» / «Shanghai»
- «香港» → «Hong Kong»
- «台湾» → «Taiwan» / «Formosa»

### Sistema educacional
- «高考» → «Exame Nacional de Admissão à Universidade (Gaokao)»
- «本科» → «Graduação» (4 anos)
- «硕士» → «Mestrado» (2-3 anos)
- «博士» → «Doutorado» (3-4 anos)

## Estilo e sintaxe

### Frases
- O chinês usa frases mais longas e coordenadas; o português prefere frases mais curtas e subordinadas
- Dividir frases longas chinesas em frases portuguesas mais curtas
- O chinês omite frequentemente o sujeito; o português o exige

### Conectores
- «因此» → «Portanto,» / «Por conseguinte,»
- «然而» → «No entanto,» / «Contudo,»
- «此外» → «Além disso,» / «Ademais,»
- «总之» → «Em resumo,» / «Em conclusão,»

### Medidas de cortesia
- O chinês usa frequentemente títulos (李总、王经理); o português usa «Senhor/Senhora + Sobrenome»
- «请» → «Por favor» / «Favor de»
- «您» → «Você» / «O senhor / A senhora» (tratamento formal)

## Domínios específicos

### Jurídico
- Os sistemas jurídicos chinês e brasileiro diferem fundamentalmente
- Conceitos do direito chinês (法律、法规、规章) nem sempre têm equivalente direto
- «法律» → «Lei»
- «法规» → «Regulamento» / «Decreto»
- «规章» → «Portaria» / «Regulamento Interno»

### Médico
- «中医» → «Medicina Tradicional Chinesa (MTC)»
- «针灸» → «Acupuntura»
- «中药» → «Fitoterapia chinesa» / «Plantas medicinais chinesas»

### Tecnológico
- «微信» → «WeChat»
- «支付宝» → «Alipay»
- «淘宝» → «Taobao»
- «人工智能» → «Inteligência Artificial (IA)»
- «大数据» → «Big Data» / «Dados massivos»
- «区块链» → «Blockchain»
