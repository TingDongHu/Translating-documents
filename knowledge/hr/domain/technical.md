# Technical Documents — Croatian

## 1. Reader Model

### 1.1 Croatian Technical Standards System

Croatia uses the **Croatian Institute for Standardization (HZN)** system, aligned with European (EN) and international (ISO) standards.

| Organization | Croatian Name | Function |
|--------------|---------------|----------|
| Croatian Institute for Standardization | Hrvatski zavod za norme (HZN) | National standards body |
| Croatian Regulatory Agency for Network Industries | Hrvatska regulatorna agencija za mrežne djelatnosti (HAKOM) | Network/telecom regulation |
| Croatian Agency for Small Business, Innovation and Investments | HAMAG-BICRO | Innovation and investment |
| Croatian Chamber of Economy | Hrvatska gospodarska komora (HGK) | Business standards and certification |
| State Inspectorate | Državni inspektorat | Safety and compliance inspections |

## 2. Decision Framework

### 2.1 Croatian Technical Writing Style

| Feature | Croatian Approach | Notes |
|---------|-------------------|-------|
| Formality | Technical, precise | Use standard terminology |
| Tone | Direct, instructional | Clear, unambiguous |
| Pronoun | Impersonal or "Vi" | Formal address in manuals |
| Passive Voice | Sometimes used | For objectivity |
| Active Voice | Preferred | For instructions |
| Sentence Length | Short to medium | Avoid complex sentences |
| Lists | Numbered for steps | Bulleted for features |

### 2.2 Croatian Technical Translation Principles

| Principle | Croatian Approach | Notes |
|-----------|-------------------|-------|
| Terminology | Use HZN/ISO standard terms | Consult HZN for Croatian terms |
| Units | SI units (metric) | Use Croatian unit names |
| Numbers | Period thousands, comma decimal | `1.000,50` not `1,000.50` |
| Abbreviations | Define on first use | Use Croatian expansion |
| Cross-references | Use § (stavak) | Follow Croatian convention |
| Warnings | Use standard symbols | Follow ISO 3864 |
| Instructions | Use imperative mood | "Povežite" (Connect) |
