# Chinese (Simplified) to Malay Adaptation Guide

> Target language: Malay (Bahasa Melayu / Bahasa Malaysia)
> Source language: Chinese Simplified (简体中文)
> Purpose: Standardized adaptation rules for Chinese-to-Malay document translation
> Last updated: 2026-05-27

---

## 1. Formality: Chinese Honorifics to Malay Bahasa Baku

### Overview

Chinese has a limited honorific system (primarily 您 and a few polite particles), while Malay has a formality system based on vocabulary choice, address terms, and sentence structure. The primary formal level used in formal document translation is **Bahasa Baku** (standard formal Malay). This section covers how to map Chinese formality cues into the Malay formal system.

### Rule 1.1: Chinese 您 → Malay formal address

The Chinese honorific pronoun 您 has no single Malay equivalent. Malay expresses respect through title-based address rather than a separate pronoun.

| Chinese | Malay (Formal) | Malay (Semi-formal) | Notes |
|---------|---------------|---------------------|-------|
| 您请坐 | Sila duduk. | Sila duduk. | Formal seating request |
| 您要什么？ | Apa yang tuan/puan hajatkan? | Awak nak apa? | Question with title |
| 您先走 | Sila pergi dahulu. | Awak pergi dulu. | Direction with formality |
| 您贵姓？ | Boleh saya tahu nama tuan/puan? | Awak nama siapa? | Name inquiry (formal) |
| 您好 | Selamat sejahtera. / Salam. | Hai / Apa khabar? | Standard greeting |

**Rule:** Never translate 您 as `awak` or `kamu` in formal contexts (these are informal). Use title-based address (`Tuan/Puan`) or third-person references (`pengguna`, `pelanggan`) instead.

### Rule 1.2: Chinese 请 → Malay request forms

Chinese 请 is a versatile politeness marker. Malay distinguishes multiple levels of request depending on formality.

| Chinese | Malay (Formal) | Malay (Informal) | Context |
|---------|---------------|-----------------|---------|
| 请稍等 | Sila tunggu sebentar. | Tunggu jap. | Wait request |
| 请进 | Sila masuk. | Masuk lah. | Entry invitation |
| 请坐 | Sila duduk. | Duduk lah. | Seating |
| 请告诉我 | Sila maklumkan kepada saya. | Bagitahu saya. | Request for info |
| 请问 | Boleh saya tanya? | Boleh tanya sikit? | "May I ask" |
| 请确认 | Sila sahkan. | Tolong sahkan. | Confirmation request |
| 请参考附件 | Sila rujuk lampiran. | Rujuk lampiran. | Attachment reference |
| 请勿吸烟 | Dilarang merokok. | Jangan merokok. | Prohibition |
| 请在这里签字 | Sila tanda tangan di sini. | Tandatangan sini. | Signature request |
| 请尽快回复 | Sila balas secepat mungkin. | Balas cepat sikit. | Urgency |

**Rule:** Use `sila` for formal requests, `tolong` for informal requests, and `dilarang` for prohibitions. Never use `tolong` in formal documents — it implies personal favor rather than official instruction.

### Rule 1.3: Chinese address nouns → Malay equivalents

Chinese uses specific honorific nouns for the addressee's affiliations and family members.

| Chinese | Malay | Notes |
|---------|-------|-------|
| 贵公司 | Syarikat tuan/puan / Pihak syarikat | Formal company address |
| 贵校 | Sekolah/universiti tuan/puan | Formal school address |
| 贵国 | Negara tuan/puan | Formal country address |
| 贵方 | Pihak tuan/puan | Formal counterparty |
| 阁下 | Tuan/Puan / Yang Berhormat | Very formal, for dignitaries |
| 先生 | Tuan / Encik | General respectful address |
| 女士 | Puan / Cik | Context-dependent |
| 令尊 | Bapa tuan/puan | "Your father" (formal) |
| 令堂 | Ibu tuan/puan | "Your mother" (formal) |
| 令郎 | Anak lelaki tuan/puan | "Your son" (formal) |
| 令爱 | Anak perempuan tuan/puan | "Your daughter" (formal) |
| 您老 | Tuan / Orang yang dihormati | For elderly persons |

**Rule:** Malay does not have an equivalent of the elaborate Chinese family honorific system (令尊, 令堂, etc.). Use simple descriptive phrases: `bapa tuan/puan` (your father), `ibu tuan/puan` (your mother).

### Rule 1.4: Chinese humble self-reference → Malay humble forms

Chinese uses humble self-references to show respect to the addressee. Malay has a parallel but simpler system.

| Chinese | Malay | Notes |
|---------|-------|-------|
| 本人 (formal I) | Saya / Pihak ini | Official self-reference |
| 我 (I) | Saya | Standard humble form |
| 我们 (we) | Kami | Humble form (excludes listener) |
| 敝公司 (our company) | Syarikat kami / Pihak kami | Formal humble reference |
| 敝校 (our school) | Sekolah kami | Formal humble reference |
| 拙作 (my work) | Karya saya / Hasil karya saya | Humble reference to own work |
| 愚见 (my opinion) | Pendapat saya / Pandangan saya | Humble opinion |
| 寒舍 (our home) | Rumah kami | Humble home reference |
| 贱内 (my wife) | Isteri saya | Standard (no longer "humble") |
| 小弟 (I, junior) | Saya | Junior's self-reference |

**Rule:** Malay does not have an elaborate self-deprecation system like Chinese. Use `saya` (I) and `kami` (we) as standard humble forms. Do not invent humble forms that do not exist in Malay.

### Rule 1.5: Chinese formal sentence endings → Malay formal endings

| Chinese | Malay | Context |
|---------|-------|---------|
| 此致敬礼 (letter closing) | Sekian, terima kasih. / Dengan segala hormatnya. | Formal letter close |
| 顺祝商祺 (business closing) | Semoga maju jaya. / Dengan ini. | Business letter close |
| 以上请审阅 (review request) | Sila semak dan review. / Sila teliti perkara di atas. | Document review |
| 特此通知 (hereby notify) | Dengan ini dimaklumkan bahawa... | Formal notification |
| 特此证明 (hereby certify) | Dengan ini saya mengesahkan bahawa... | Formal certification |
| 请予以批准 (please approve) | Sila luluskan permohonan ini. | Approval request |
| 为盼 (hoping for positive response) | Berharap mendapat pertimbangan yang baik. | Formal hope |

**Rule:** Chinese formal closing formulas (此致敬礼, 顺祝商祺) have no direct Malay equivalent. Use established Malay administrative closings: `Sekian, terima kasih` (thank you) or `Dengan segala hormatnya` (with all respect).

## 2. Dates/Numbers: Chinese Numeric Conventions to Malay

### Overview

Chinese numbers use a different counting system (万/亿 large number markers) and different date format (年月日 order). Malay uses comma as thousands separator and dot as decimal separator, which differs from Chinese conventions.

### Rule 2.1: Number system conversion

| Chinese | Malay | Notes |
|---------|-------|-------|
| 一万 (10,000) | 10,000 / sepuluh ribu | Malay uses comma thousands |
| 十万 (100,000) | 100,000 / seratus ribu | Same pattern |
| 一百万 (1,000,000) | 1,000,000 / satu juta | Malay `juta` = Chinese `百万` |
| 一亿 (100,000,000) | 100,000,000 / seratus juta | Malay `juta` = 10^6, not 10^8 |
| 十亿 (1,000,000,000) | 1,000,000,000 / satu bilion | Malay `bilion` = 10^9 |

**Critical difference**: Chinese `亿` (yì) = 100 million, but Malay `juta` = 1 million. `一亿` = `100 juta` (not `1 juta`). Similarly, Chinese `万` (wàn) = 10,000; Malay `ribu` = 1,000. `一万` = `10 ribu`.

### Rule 2.2: Date format conversion

| Chinese | Malay | Notes |
|---------|-------|-------|
| 2026年1月27日 | 27 Januari 2026 | Convert year-first to day-first |
| 2026年01月27日 | 27/01/2026 | DD/MM/YYYY format |
| 2026-01-27 (ISO) | 27/01/2026 or 2026-01-27 | Both acceptable |
| 一月二十七日 | 27 Januari | Remove Chinese month names |

**Rule:** Chinese date format is YYYY年MM月DD日 (year-first). Malay uses DD/MM/YYYY (day-first). Always convert the order. Month names change from Chinese to Malay: 一月→Januari, 二月→Februari, etc.

### Rule 2.3: Time format conversion

| Chinese | Malay (Formal) | Malay (Informal) |
|---------|---------------|-----------------|
| 下午2:30 | 14:30 | 2:30 petang |
| 上午9:00 | 09:00 | 9:00 pagi |
| 晚上11:45 | 23:45 | 11:45 malam |
| 中午12:00 | 12:00 | tengah hari |
| 凌晨0:00 | 00:00 | tengah malam |

### Rule 2.4: Currency conversion

| Chinese | Malay | Notes |
|---------|-------|-------|
| ¥100.00 | ¥100.00 (JPY) or RM100.00 | Check if JPY or CNY |
| 100元 | RM100.00 or 100 Yuan | Convert currency name |
| 人民币 (RMB) | Yuan Cina / RMB | Currency name |
| 美元 (USD) | Dolar Amerika Syarikat / USD | Currency name |
| 100美元 | $100.00 or RM100.00 | Keep original or convert |

### Rule 2.5: Chinese measure words → Malay

Chinese requires measure words (量词) between numbers and nouns. Malay does NOT use measure words in the same way.

| Chinese | Malay | Notes |
|---------|-------|-------|
| 一本书 | satu buah buku / sebuah buku | `buah` is a general classifier, optional |
| 三个人 | tiga orang | `orang` is the counter for people |
| 两台电脑 | dua komputer / dua unit komputer | No measure word needed |
| 五条新闻 | lima berita | No measure word for abstract nouns |
| 十架飞机 | sepuluh buah kapal terbang | `buah` for vehicles (optional) |
| 一间房间 | satu bilik | No measure word needed |

**Rule:** Malay does NOT require measure words between numbers and nouns in most contexts. Remove Chinese measure words during translation. Use counter words (`orang` for people, `buah` for large objects) only when natural in Malay.

### Rule 2.6: Chinese number idioms

| Chinese | Malay | Notes |
|---------|-------|-------|
| 三三两两 | Berkelompok-kelompok kecil | "In small groups" |
| 七上八下 | Berdebar-debar / Gelisah | "Anxious" |
| 十全十美 | Sempurna | "Perfect" |
| 一五一十 | Dengan terperinci | "In detail" |
| 九牛一毛 | Setitik air di lautan | "A drop in the ocean" |

## 3. Cultural References: Chinese Cultural References to Malaysian Context

### Overview

Malaysia has a significant Chinese-Malaysian community (approximately 23% of the population). Chinese cultural references that are common in mainland China may need adaptation for the Malaysian Chinese context, which has its own distinct cultural identity shaped by generations of local development.

### Rule 3.1: Cultural context differences

| Topic | Mainland Chinese Context | Malaysian Chinese Context | Adaptation |
|---|---|---|---|
| Festivals | Spring Festival, Mid-Autumn | Chinese New Year, Mid-Autumn Festival | Use Malaysian terms |
| Food references | Peking duck, xiao long bao | Dim sum, char kway teow | Use Malaysian Chinese food names |
| Place names | Beijing, Shanghai, Guangzhou | Beijing, Shanghai, Guangzhou | Same (preserved) |
| Historical figures | Mao Zedong, Sun Yat-sen | Same, plus local figures | Preserve names |
| Political system | Communist Party, PRC | Democratic system | Neutral, factual language |
| Language reference | 普通话 (Mandarin) | Bahasa Cina / 中文 | Malaysian Chinese use both terms |

### Rule 3.3: Chinese philosophical/literary references

| Chinese | Malay | Notes |

**Rule:** Classical Chinese texts are generally preserved in their romanized form in Malay, as they are considered foreign cultural references. Do not attempt to translate classical Chinese titles into Malay.

### Rule 3.4: Chinese idioms (成语) → Malay

| Chinese | Literal Meaning | Malay Equivalent | Notes |
|---------|----------------|-----------------|-------|

| 狐假虎威 | Fox borrow tiger's power | "Sokong orang besar" | Bullying through authority |

**Rule:** Do not translate Chinese idioms (成语) literally into Malay. Use equivalent Malay proverbs (peribahasa) or paraphrase. Literal translations of Chinese idioms are meaningless in Malay.

## 6. Punctuation: Chinese Punctuation to Malay

### Overview

Chinese punctuation uses full-width characters (，。！？etc.) while Malay uses half-width characters (, . ! ? etc.). Chinese also uses different quotation marks and has specific spacing rules that differ from Malay.

### Rule 6.1: Full-width to half-width conversion

| Chinese | Malay | Notes |
|---------|-------|-------|
| ，(full-width comma) | , (half-width comma) | Convert all full-width punctuation |
| 。(full-width period) | . (half-width period) | Convert all full-width punctuation |
| ！(full-width exclamation) | ! (half-width exclamation) | Convert |
| ？(full-width question) | ? (half-width question) | Convert |
| ：(full-width colon) | : (half-width colon) | Convert |
| ；(full-width semicolon) | ; (half-width semicolon) | Convert |
| （）(full-width parentheses) | () (half-width parentheses) | Convert |

**Rule:** ALL Chinese full-width punctuation marks must be converted to their half-width Malay equivalents during translation. Full-width punctuation is a formatting error in Malay text.

### Rule 6.2: Quotation marks

| Chinese | Malay | Notes |
|---------|-------|-------|
| "" (Chinese quotes) | "" (straight double quotes) | Primary quotes |
| '' (Chinese single quotes) | '' (straight single quotes) | Quotes within quotes |
| 《》 (book title marks) | *...* (italic) or "..." | Book/work titles |

**Rule:** Chinese uses special quotation marks (「」, 『』, "", ''). Convert to standard Malay straight quotes ("", '') or guillemets («») for formal/academic text.

### Rule 6.3: Spacing rules

| Element | Chinese | Malay | Notes |
|---|---|---|---|
| No space before/after Chinese punctuation | No spaces | No spaces (same for most) | Same |
| Space before % | No space | Space required: `50 %` | Critical difference |
| Currency symbol spacing | ¥100 (no space) | RM100 (no space) | Same |
| Number-unit spacing | 10公斤 (no space) | 10 kg (space required) | Critical difference |
| Space between Chinese characters | No spaces between characters | Standard word spacing | Different |

**Rule:** Chinese text has no spaces between characters. Malay uses standard Latin word spacing. When translating, add appropriate word spaces between Malay words.

### Rule 6.4: Chinese-specific punctuation with no Malay equivalent

| Chinese | Malay Handling | Notes |
|---------|---------------|-------|
| 、(dunhao, list comma) | , (regular comma) | Convert to standard comma |
| 〈〉(angle brackets) | < > or ( ) | Convert based on context |
| ——(double em dash) | — (single em dash) | Convert to single dash |
| ……(Chinese ellipsis) | ... (three dots) | Convert to standard ellipsis |
| ·(middle dot for lists) | - (hyphen) or • (bullet) | Convert to standard list marker |

### Rule 6.5: Chinese text direction and formatting

| Element | Chinese | Malay | Notes |
|---|---|---|---|
| Text direction | Horizontal (modern) | Horizontal (same) | Same in modern documents |
| Vertical text | Sometimes used | Never | Convert vertical to horizontal |
| Bold/italic usage | Less common | Standard | May need to add formatting |
| Paragraph indentation | 2 characters | Optional (first-line indent) | Adapt to Malay convention |

## 7. Grammar Adaptation: Chinese Structures to Malay

### Overview

Chinese and Malay have fundamentally different grammar systems. Chinese is an isolating language (no affixation), while Malay is agglutinative (uses prefixes, suffixes, infixes). Chinese uses measure words extensively; Malay does not. Chinese uses topic-comment structure; Malay uses SVO. This section covers the most common structural adaptations.

### Rule 7.1: Chinese topic-comment structure → Malay SVO

Chinese frequently uses a topic-comment structure where the topic is stated first, then commented upon. Malay prefers standard SVO order.

| Chinese | Malay | Notes |
|---------|-------|-------|
| 这本书，我已经读完了。 | Saya sudah selesai membaca buku ini. | SVO order |
| 这个问题，我们需要讨论。 | Kita perlu membincangkan masalah ini. | SVO order |
| 明天的会议，我不能参加。 | Saya tidak dapat menghadiri mesyuarat esok. | SVO order |

**Rule:** When translating Chinese topic-comment sentences, restructure to Malay SVO order. Do not preserve the Chinese topic-first structure.

### Rule 7.2: Chinese 把 (bǎ) construction → Malay

The Chinese 把 construction moves the object before the verb. Malay does not have this construction.

| Chinese | Malay | Notes |
|---------|-------|-------|
| 请把文件发给我。 | Sila hantar fail tersebut kepada saya. | Standard SVO |
| 他把门关上了。 | Dia menutup pintu itu. | Standard SVO |
| 我把报告写完了。 | Saya telah selesai menulis laporan. | Standard SVO |

**Rule:** The Chinese 把 construction has no Malay equivalent. Convert to standard Malay SVO order: Subject + Verb + Object.

### Rule 7.3: Chinese 了 (le) aspect → Malay tense markers

Chinese uses 了 to mark completed action. Malay uses `telah`, `sudah`, or `-lah` suffix.

| Chinese | Malay | Notes |
|---------|-------|-------|
| 我吃了。 | Saya sudah makan. / Saya telah makan. | Completed action |
| 他走了。 | Dia sudah pergi. / Dia telah pergi. | Completed action |
| 下雨了。 | Hujan sudah turun. / Hujan telah turun. | Change of state |

### Rule 7.4: Chinese 是...的 emphasis structure → Malay

Chinese uses 是...的 to emphasize details of a past event. Malay uses different emphasis strategies.

| Chinese | Malay | Notes |
|---------|-------|-------|
| 我是昨天来的。 | Saya datang semalam. | Time adverb placement |
| 他是在吉隆坡出生的。 | Dia dilahirkan di Kuala Lumpur. | Passive + location |
| 这本书是在2020年出版的。 | Buku ini diterbitkan pada tahun 2020. | Passive + date |

### Rule 7.5: Chinese reduplication → Malay

Chinese uses reduplication for various grammatical functions. Malay also uses reduplication but with different patterns.

| Chinese | Malay | Notes |
|---------|-------|-------|
| 看看 (try looking) | Lihat / Lihat-lihat | Optional reduplication |
| 人人 (everyone) | Setiap orang / Semua orang | Different structure |
| 天天 (every day) | Setiap hari | No reduplication needed |
| 高高兴兴 (happily) | Dengan gembira | No reduplication needed |
| 研究研究 (discuss) | Bincangkan / Kaji semula | No reduplication needed |

**Rule:** Chinese reduplication patterns do not map directly to Malay reduplication. Malay reduplication is primarily for plurality (`buku-buku` = books) and does not serve the same grammatical functions as Chinese reduplication.

## 8. Chinese-Malaysian Community Context

### Overview

Malaysia's Chinese community has maintained its cultural identity while developing a distinct Malaysian Chinese culture. Translators must be aware of these differences when translating from Simplified Chinese (PRC standard) to Malay for a Malaysian audience.

### Rule 8.1: Simplified vs Traditional Chinese

| Context | Guidance |
|---------|----------|
| Source is Simplified Chinese (PRC) | Translate directly to Malay; no need to convert to Traditional |
| Source contains Traditional Chinese terms | Preserve the term and provide Malay translation |
| Malaysian Chinese audience | May read both Simplified and Traditional; Malay translation is primary |
| Place names in PRC | Use standard Malay forms: Beijing (not Peking), Shanghai, Guangzhou |

### Rule 8.2: PRC-specific terms that need adaptation

| PRC Term | Malaysian Context | Adaptation |
|----------|-------------------|------------|
| 同志 (comrade) | Not used in Malaysia | Translate as `rakan sejawat` or `kawan` |
| 人民 (the people) | Not used in Malaysian Malay | Translate as `rakyat` |
| 干部 (cadre) | Not used in Malaysia | Translate as `pegawai` or `kakitangan` |
| 革命 (revolution) | Context-dependent | Translate based on actual meaning |
| 群众 (masses) | Not used in Malaysian Malay | Translate as `orang ramai` |
| 组织 (organization, in party context) | Context-dependent | Translate as `pertubuhan` or `organisasi` |

### Rule 8.3: Chinese business etiquette → Malaysian

| Chinese Practice | Malaysian Adaptation | Notes |
|-----------------|---------------------|-------|
| Business card exchange with both hands | Similar in Malay culture | Both cultures value respect |
| Gift giving | Similar; avoid alcohol/pork for Malay recipients | Cultural sensitivity needed |
| Seating hierarchy | Similar; host sits furthest from door | Both cultures respect hierarchy |
| Dining etiquette | Similar; halal considerations for Malay hosts | Important for multicultural settings |
| Address by surname + title | Same pattern: `Tuan Ahmad` | Both use title-based address |
