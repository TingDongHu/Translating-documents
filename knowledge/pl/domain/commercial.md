# Polish Commercial Domain: Reader Model + Decision Framework + Standards

---

## PART 1: READER MODEL

### 1.1 Target Reader Profile

| Attribute | Description |
|-----------|-------------|
| **Primary audience** | Polish business professionals, export-import managers, sales representatives, purchasing officers, accountants (glowni ksiegowi), entrepreneurs (przedsiebiorcy), business correspondence specialists |
| **Language variant** | Polski biznesowy -- formal yet functional register of Standard Polish. Mix of neutral formality (for external partners) and efficiency (for internal). Heavy use of business-specific collocations |
| **Proficiency required** | B2+ in general Polish + familiarity with Polish commercial correspondence conventions, invoice formats, and regulatory terms |
| **Expected text types** | Business letters (pisma handlowe), emails, offers (oferty), contracts (umowy handlowe), invoices (faktury), pro-forma invoices, delivery notes (WZ / dokumenty wydania), payment reminders (wezwania do zaplaty), minutes of meetings (protokoly), tenders (przetargi), terms and conditions (regulaminy / OWU) |
| **Reader priorities** | Politeness markers (Szanowni Panstwo / Z powazaniem), precise commercial terms, unambiguous numbers (PLN, decimal comma), compliance with VAT regulations |

### 1.2 Reader Expectations

| Expectation | Manifestation |
|-------------|---------------|
| **Formal address** | "Pan" / "Pani" / "Panstwo" always capitalized: "Szanowni Panstwo", "Szanowny Panie", "Szanowna Pani" |
| **Closing formulas** | "Z powazaniem" (standard), "Z wyrazami szacunku" (more respectful), "Lacze pozdrowienia" (less formal, internal) |
| **Currency format** | "PLN" or "zl" after amount, decimal comma: "1 234,56 PLN" / "1 234,56 zl" |
| **Date format** | "dnia 15 marca 2024 r." or "15.03.2024 r." |
| **Nominal politeness** | "Z uprzejma prosba o..." rather than direct imperative |
| **VAT treatment** | Net/gross always distinguished: "netto" / "brutto" / "VAT" |
| **Number separation** | Amounts: "1 000,00" (space as thousand separator, comma as decimal) |

## PART 2: DECISION FRAMEWORK

### 2.1 Polish Commercial Orthography

#### Table C1: Capitalization in Business Polish

| Context | Rule | Example |
|---------|------|---------|
| Formal titles in salutation | Always capitalize | "Szanowny Panie Dyrektorze", "Szanowna Pani Prezes" |
| Company names | Full names capitalized as registered | "ABC Spolka z ograniczona odpowiedzialnoscia" |
| Product names | Brand names keep original | "System ERP Optima", "Program Fakturownia" |
| Names of documents | Sentence case | "Faktura VAT nr 123/2024" |
| Job titles in running text | Lowercase | "dyrektor sprzedazy", "prezes zarzadu" |
| Legal forms of companies | Lowercase the form type | "spolka z o.o.", "spolka akcyjna" |
| Currency names | Lowercase | "zloty", "euro", "dolar" |
| Month names | Lowercase | "10 czerwca 2024 r." |
| "R." after year | Lowercase, with period and space | "2024 r." |
| Trademark symbols | Kept as original | "Marka X(TM)", "Produkt Y(R)" |

#### Table C2: Punctuation in Commercial Documents

| Context | Rule | Example |
|---------|------|---------|
| Decimal separator | Comma (,) | "Cena: 1 234,56 zl" |
| Thousands separator | Space or period | "1 000 sztuk" or "1.000 sztuk" |
| Currency code placement | After amount, with space | "500,00 PLN" (NOT PLN 500,00) |
| Percentage | "procent" / "%" -- space before % | "23%" or "23 procent" |
| VAT rates | %, no space in technical context | "23% VAT" in tables |
| Date in running text | "dnia [date] roku" | "dnia 15 lutego 2024 roku" |
| Date in headers | "dd.mm.rrrr" or "dd-mm-rrrr" | "15.02.2024" |
| Time | "godz." followed by time | "godz. 14:30" |
| Bullet list introduction | Colon before list | "Zamowienie obejmuje:" |
| Comma in enumerations | Typical Polish comma use | "towar A, towar B oraz towar C" |
| Subject line (dotyczy) | "Dotyczy:" with colon | "Dotyczy: Faktura VAT nr 123/2024" |
| Reference markers | "Nasz numer:" / "Wasz numer:" | "Nasz numer: FW/2024/088" |
