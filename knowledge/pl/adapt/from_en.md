# Polish Adaptation: English → Polish

## 1. Formality

| English | Polish | Notes |
|---------|--------|-------|
| You (singular formal) | Pan (m.) / Pani (f.) | Capitalize in letters/emails |
| You (plural formal) | Państwo | Mixed or unknown gender group |
| You (singular informal) | ty | Lowercase; close friends, family, children |
| You (plural informal) | wy | Rare in formal writing |
| Mr. / Mr | Pan | "Mr. Smith" → Pan Kowalski |
| Mrs. / Ms. / Miss | Pani | Polish uses one form `Pani` for all |
| Dear Mr. Smith | Szanowny Panie Kowalski | Vocative case required |
| Dear Sir/Madam | Szanowni Państwo | Standard letter opening |
| To whom it may concern | Do wiadomości zainteresowanych | Formal/official |
| Sincerely, | Z poważaniem, | Standard letter closing |
| Best regards, | Pozdrawiam / Z wyrazami szacunku | `Pozdrawiam` = semi-formal; `Z wyrazami szacunku` = formal |
| Please | Proszę | Always with infinitive: "proszę usiąść" not "proszę siadać" |
| Thank you | Dziękuję | `Dziękuję Panu/Pani` to address formally |
| You're welcome | Proszę bardzo / Nie ma za co | |
| Hey / Hi | Cześć / Hej | Informal only |
| Passive: "It is believed that…" | Uważa się, że… / Sądzi się, że… | Replace EN passive with `się` construction |
| Passive: "The report was submitted" | Raport został złożony / Raport złożono | Two options: `został` + passive participle OR impersonal -no/-to form |
| Passive: "The door was opened" | Drzwi zostały otwarte / Drzwi otwarto | Impersonal form preferred in formal PL |
| "One should…" | Należy… / Powinno się… | Impersonal obligation |

## 2. Dates, Times & Numbers

| English | Polish | Notes |
|---------|--------|-------|
| 05/25/2025 | 25.05.2025 | DD.MM.YYYY — never MM/DD/YYYY |
| May 25, 2025 | 25 maja 2025 roku / r. | Month names are lowercase |
| January 1 | 1 stycznia | Genitive case after date: "1 stycznia" |
| 2025 | rok 2025 / r. 2025 | `r.` = abbreviation of `roku` |
| 2025-2026 | 2025–2026 | En-dash, no spaces |
| 10:00 AM | 10:00 | 24-hour clock, no AM/PM |
| 3:30 PM | 15:30 | |
| 12:00 PM (noon) | 12:00 / południe | |
| 12:00 AM (midnight) | 0:00 / północ | "24:00" also used for midnight |
| 9:15 | 9:15 / kwadrans po 9 | Written format: 9:15 |
| 10,000 | 10 000 | Space as thousands separator, never comma |
| 3.14 | 3,14 | Comma as decimal separator |
| $100 | 100 dolarów / 100 USD | Currency before or after number |
| 50% | 50% / 50 procent | |
| Number 1 (No. 1) | nr 1 / numer 1 | `nr` or `Nr` abbreviation |
| Phone: (312) 555-1234 | +48 123 456 789 / 123-456-789 | Native phone format for target locale |
| $1.5 million | 1,5 mln dolarów / 1,5 miliona dolarów | Note comma as decimal |

## 4. Terminology

| English | Polish | Notes |
|---------|--------|-------|
| computer | komputer | False friend trap: none here, but watch for faux amis |
| laptop | laptop / notebook | Both used in PL; `laptop` more common |
| software | oprogramowanie | Never "softwar" — respell as PL noun |
| hardware | sprzęt komputerowy / hardware | `Sprzęt` preferred |
| email | e-mail / mail / wiadomość e-mail | Hyphenated `e-mail` standard in PL |
| internet | internet | Lowercase in PL (but often capitalized in EN) |
| website | strona internetowa / witryna / serwis | `Serwis` for portal; `strona` for single page |
| username | nazwa użytkownika | |
| password | hasło | `Hasło` also means "slogan/motto" |
| account | konto | Neuter noun: "to konto" (not "ta konto") |
| login | logowanie / login | `Login` = masculine noun in PL |
| logout | wylogowanie | |
| data | dane | Always plural: "dane są" not "dane jest" |
| file | plik | |
| folder | folder / katalog | `Katalog` = directory; `folder` = GUI folder |
| server | serwer | |
| network | sieć | |
| database | baza danych | |
| cloud | chmura / usługa w chmurze | "Chmura" for cloud computing |
| application | aplikacja / program | `Aplikacja` = mobile/app; `program` = desktop |
| algorithm | algorytm | |
| interface | interfejs | |
| compliance | zgodność / przestrzeganie przepisów | No single PL equivalent; context-dependent |
| deliverable | rezultat / wynik / dostarczony produkt | `Produkt` not `dostarczalnik` (avoid calque) |
| stakeholder | interesariusz / strona zainteresowana | OK in EU/legal PL; `zainteresowany` in less formal texts |

|-------------|-------------------|-----------|------------|
| eventually | eventualnie | `ewentualnie` = possibly | ostatecznie / w końcu |
| actually | aktualnie | `aktualnie` = currently | właściwie / faktycznie |
| fabric | fabryka | `fabryka` = factory | tkanina / materiał |
| chef | szef | `szef` = boss | kucharz / szef kuchni |
| preservative | prezerwatywa | `prezerwatywa` = condom | konserwant / środek konserwujący |
| sympathetic | sympatyczny | `sympatyczny` = nice/likable | współczujący / pełen zrozumienia |
| novel | nowela | `nowela` = short story | powieść |

| path | pas | `pas` = belt | ścieżka / droga |

## 6. Punctuation

| English | Polish | Notes |
|---------|--------|-------|
| "quotes" | „quotes" | Opening „ (U+201E), closing " (U+201D) |
| 'single quotes' | ‚single quotes' | Opening ‚ (U+201A), closing ' (U+2019) |
| "quote, "inner"" | „quote, ‚inner'" | Nested: outer „", inner ‚' (reverse of EN) |
| Period inside quotes: "word." | Period outside quotes: „word". or "word." | Both possible in PL, but outside is traditional |
| Comma before "and" in list (Oxford comma) | No comma before "i" in list | "A, B i C" (never "A, B, i C") |
| Comma before "that" | Comma before "że" | Subordinate clauses always take comma: "Wiedziałem, że przyjdziesz." |
| "I know that…" | "Wiem, że …" | Comma required before subordinate conjunction |
| "He said that…" | "Powiedział, że …" | |
| "If…, then…" | "Jeśli …, to …" | Comma after conditional clause |
| "Although…" | "Chociaż …," / "Mimo że …," | Comma after introductory subordinate clause |
| Space after . ? ! | Same in PL | ✓ |
| No space between number and % | Space preferred: "50 %" | Both accepted, space more traditional |
| Ellipsis: three dots ... | Ellipsis: three dots ... | Same, but PL often uses spaces around: "… tak …" |
| Apostrophe in possessives: John's | Apostrophe never used for possession in PL | "Jana" not "John'a" |
| Colon + capital letter | Colon + lowercase letter | Unless proper noun follows |
| "e.g." and "i.e." | "np." and "tj." | "np." = na przykład; "tj." = to jest |
| "etc." | "itd." / "itp." | "itd." = i tak dalej; "itp." = i tym podobne |
| "vs." | "w." / "vs" / "kontra" | "w." = wobec/wersus |
| "cf." | "por." | "por." = porównaj |
| Bullet lists: capitalize first word | Bullet lists: lowercase first word | PL convention: lowercase after bullet (unless full sentence) |
| Bullet list punctuation: period at end | Bullet list: semicolons between items | Formal PL uses `;` between items, final `.` |
| Comma in numbers: 1,000 | Space in numbers: 1 000 | |
