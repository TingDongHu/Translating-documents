# Polish Adaptation: Chinese (zh-CN) → Polish (pl)

## 1. Formality

| Chinese | Polish | Notes |
|---------|--------|-------|
| 您 | Pan/Pani (formal) | Capitalize in correspondence. Pan for male, Pani for female. |
| 你 | ty (informal) | Lowercase; used with friends, family, children. |
| 你们 | wy / Państwo / Panie / Panowie | `Państwo` = mixed/formal group; `Panie` = women only; `Panowie` = men only. |
| 先生 | Pan | E.g., "王先生" → Pan Wang |
| 女士 | Pani | E.g., "李女士" → Pani Li |
| 老师 | Pan Profesor / Pani Profesor / nauczyciel | `老师` as title → Pan Profesor; as occupation → nauczyciel |

| 您好 | Dzień dobry / Szanowny Panie/Pani | Formal greeting |
| 再见 | Do widzenia (formal) / Do zobaczenia (informal) | |
| 谢谢 | Dziękuję (formal) / Dzięki (informal) | |
| 请 | Proszę | Universal polite marker |
| 抱歉 | Przepraszam | Used both for apology and attention-getting |
| 经理先生 | Szanowny Panie Dyrektorze | Title + name in vocative |

| 组织 | organizacja | ✓ Direct equivalent works |
| 经济 | ekonomia / gospodarka | `Gospodarka` = economy; `ekonomiczny` = economic |
| 社会 | społeczeństwo | `Socjalny` ≠ social in the same sense |
| 政治 | polityka | ✓ |

| 法律 | prawo | `Prawo` = law; `ustawa` = statute/act |

| 部长 | minister | ✓ |
| 委员会 | komisja / komitet | `Komisja` = commission; `komitet` = committee |
| 关于 | dotyczący / odnośnie / w sprawie | |
| 基于 | na podstawie / oparty na / bazujący na | |
| 下列 | następujący / poniższy | "Jak następuje" in legal context |

## 6. Punctuation

| Chinese | Polish | Notes |
|---------|--------|-------|
| 。 | . (kropka) | Period: full stop; no spacing change |
| ， | , (przecinek) | Comma: same usage generally; Polish has stricter rules for subordinate clauses |
| 、 | , (przecinek) | Enumeration comma: Chinese `、` → Polish comma + space |
| ； | ; (średnik) | Semicolon: same usage |
| ： | : (dwukropek) | Colon: no space before, lowercase after (unless proper noun) |
| ？ | ? (pytajnik) | Question mark: same |
| ！ | ! (wykrzyknik) | Exclamation mark: same |
| "" | „" (cudzysłów) | Polish uses „..." („ = U+201E, " = U+201D). No space inside. |
| '' | ‚' (cudzysłów pojedynczy) | Single quotes: ‚...' (‚ = U+201A, ' = U+2019) |
| 《》 | „" (cudzysłów) or brak | Book/film titles: either Polish quotes or italics/none |
| （） | ( ) (nawiasy) | Parentheses: same usage, spaces as in EN |
| —— | — (pauza myślnik) | Chinese em-dash (2×) → Polish pauza (—, U+2014) |
| · · · | ... (wielokropek) | Chinese uses ··· (middle dots); Polish uses ... (three dots) |
| · (间隔号) | (omit or space) | Chinese middle dot in foreign names → omit or transliterate |
| 《 》in titles | Kursywa / brak / „" | Chinese book/film title brackets → Polish italics or quotes |
| No space after 。 | Space after . | Always add space after period before next sentence |
| No space after 、 | Space after , | Always add space after comma |
| Space around Chinese brackets | No space around Polish brackets | Remove spaces: (text) not ( text ) |
| Chinese comma inside quotes | Polish comma outside „ | "text", not „text", |
| 第……条 | § / art. ___ | "Article X" format: § 5 / art. 5. |
