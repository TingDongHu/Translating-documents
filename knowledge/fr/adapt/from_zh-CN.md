# Adaptation francophone depuis le chinois simplifié (zh-CN)

## 1. Formule de politesse et registre

| Chinois (zh-CN) | Francais (FR) | Notes |
|---|---|---|
| 您 | vous | Toujours utiliser "vous" pour le vouvoiement. |
| 你 | tu | Réserver "tu" au tutoiement familier ou aux enfants. |
| 尊敬的客户 | Cher client / Madame, Monsieur | Éviter les calques ; "尊重的" ne se traduit pas litteralement. |
| 贵公司 | votre entreprise / votre societe | "贵" marque la deference ; en FR, simple possessif poli. |
| 本公司 / 我公司 | notre entreprise / notre societe | La modestie chinoise ("notre humble") disparait en FR. |
| 敝公司 | notre entreprise | Idem : "敝" (humble) ne se traduit pas. |
| 请 | veuillez / merci de | "请" suivi d'un imperatif peut devenir "veuillez + inf." |
| 谢谢 | merci / je vous remercie | "谢谢" seul convient ; "Je vous remercie" plus formel. |
| 不好意思 | desole / pardon | Moins fort que "Je suis desole" ; adapter le registre. |
| 抱歉 | toutes mes excuses / veuillez m'excuser | Plus formel que "不好意思". |
| 你好 | Bonjour | Systematique ; ne pas traduire par "Bonjour vous". |
| 大家好 | Bonjour a tous | Inclusif. |
| 先生 | Monsieur | Abrege en "M." dans les courriers. |
| 女士 | Madame | Abrege en "Mme" ou "Mme". |
| 小姐 | Mademoiselle | Usage de plus en plus rare ; preferer "Madame". |
| 同志们 | Chers collegues / Messieurs | Eviter "camarades" sauf contexte historique. |
| 致 | A l'attention de / Madame, Monsieur | En tete de lettre. |
| 此致 | Veuillez agreer, Madame, Monsieur, l'expression... | Formule de politesse de cloture standard. |
| 顺颂商祺 | Cordialement / Sinceres salutations | Formule commerciale ; ne pas traduire litteralement. |
| 敬礼 | Cordialement | "Salut respectueux" est obsolete en FR moderne. |
| 拜上 | Bien cordialement | Archaisme chinois ; choix d'une formule FR standard. |

## 2. Nombres, dates et heures

| Chinois (zh-CN) | Francais (FR) | Notes |
|---|---|---|
| 万 | 10 000 | Jamais "dix milliers". Separateur : espace insécable. |
| 亿 | 100 000 000 | Ne pas confondre avec "milliard" (十亿). |
| 万亿 | 1 000 000 000 000 (mille milliards / billion) | Attention au billion francais qui vaut 10^12. |
| 一百二十 | 120 | Le "un" initial est obligatoire en FR. |
| 二百 | 200 | Pas de "deux cents" avec "de" ; "deux cent" sans "s" devant un nom. |
| 两千零二十 | 2 020 | Le "零" marque le zero intermediaire. |
| 五分之一 | 1/5 (un cinquieme) | Fraction : numerateur + "sur" ou ordinal. |
| 百分之五 | 5 % | "Pour cent" en toutes lettres. |
| 50% | 50 % | Espace avant le signe % en FR (norme typographique). |
| 2024年5月25日 | 25/05/2024 | Ordre : jour/mois/annee. Pas de "annee/mois/jour". |
| 2024年5月25日 | 25 mai 2024 | En toutes lettres : pas de virgule entre mois et annee. |
| 2024年5月 | mai 2024 | Mois en minuscule, pas de majuscule en FR. |
| 2024年第一季度 | premier trimestre 2024 | T1 2024 acceptable en contexte technique. |
| 公元前206年 | 206 av. J.-C. | "avant Jesus-Christ". |
| 公元2024年 | 2024 apr. J.-C. | "apres Jesus-Christ". Habituellement omis. |
| 上午8点 | 8 h 00 / 8 h du matin | Format 24h recommande (8 h 00). |
| 下午3点 | 15 h 00 / 15 h | Le 24h est la norme en FR. |
| 下午3点30分 | 15 h 30 | "30分" devient ",30" ou " h 30". |
| 第1条 | Article 1er / premier | "1er" pour le premier element. |
| 第2条 | Article 2 | "2e" ou "2eme" ; "2nd" rare en FR. |
| 五十六个民族 | cinquante-six ethnies | Le classificateur "个" n'a pas d'equivalent. |
| 十余年 | une dizaine d'annees / plus de dix ans | Approximatif. |
| 三四天 | trois ou quatre jours | Intervalle approximatif. |

## 6. Ponctuation et typographie

| Chinois (zh-CN) | Francais (FR) | Regle |
|---|---|---|
| 。 | . | Point final : identique visuellement, attention aux confusions. |
| ， | , | Virgule chinoise d'encadrement -> virgule FR. |
| 、 | , | Le "dun hao" chinois (separateur d'enumeration) devient la virgule en FR. Jamais de 、en FR. |
| "" | « » | Guillemets francais avec espaces insécables (« »). |
| '' | « » | Idem ; les guillemets simples chinois deviennent des guillemets FR. |
| 《》 | « » | Guillemets de titre chinois -> guillemets FR (pas de chevrons simples). |
| （） | ( ) | Parentheses : identiques, mais verifier l'espacement. |
| 【】 | [ ] / ( ) | Crochets chinois -> crochets ou parentheses selon le contexte. |
| —— | — | Tiret cadratin (—) en FR ; le double-tiret chinois est un tiret simple. |
| …… | ... / ... | Points de suspension : 3 ou 6 points selon la norme ; les points chinois (6) deviennent 3 en FR moderne. |
| ； | ; | Point-virgule : identique, veiller a l'espace avant. |
| ： | : | Deux-points : ajouter une espace insécable avant. |
| ？ | ? | Point d'interrogation : espace insécable avant en FR. |
| ！ | ! | Point d'exclamation : espace insécable avant en FR ; eviter la surabondance chinoise. |
| · | - / · | Point median chinois (separateur de noms) -> trait d'union ou conserver. |
| ~ | a / - / environ | Tilde chinois d'intervalle -> "a" ou tiret en FR. |
| # | n° / numero | "Dieze" pour numero en FR (n°). |
| % | % | Espace insécable avant le signe % en FR. |
| / | / | Pas d'espace autour de la barre oblique en FR. |
| 1. | 1. | Listes numerotees : identiques. |
| - | - | Listes a puces : utiliser le tiret ou le point. |

### Regles typographiques francaises essentielles

- **Espaces insécables** : toujours avant `:`, `;`, `?`, `!`, `%`, `«` et apres `»`.
- **Guillemets** : utiliser `«` (AltGr + 7) et `»` (AltGr + Shift + 7) avec espaces insécables.
- **Apostrophe** : toujours `'` (apostrophe courbe) ; jamais l'apostrophe droite dactylographique `'`.
- **Majuscules accentuees** : `É`, `È`, `Ê`, `Ë`, `À`, `Ù`, `Î`, `Ô`, `Ç` sont obligatoires.
- **Nombres** : espace insécable tous les trois chiffres (10 000, 1 000 000).
- **Abreviations** : `M.` (Monsieur), `Mme` (Madame), `Mlle` (Mademoiselle) -- pas de point pour Mme/Mlle dans l'usage moderne.
- **Heure** : format 24h (14 h 30), espace avant "h".
- **Date** : `25 mai 2024` (pas de virgule). `1er mai` pour le premier.
