# Adaptation francophone depuis l'anglais (en)

## 1. Formule de politesse et registre

| Anglais (en) | Francais (FR) | Notes |
|---|---|---|
| You | vous / tu, on | "Vous" par defaut ; "tu" reserve au tutoiement. "On" peut remplacer "you" impersonnel. |
| Dear Sir | Monsieur, | Formule standard ; pas de "Cher Monsieur" trop affecte. |
| Dear Madam | Madame, | Idem. |
| Dear Sir or Madam | Madame, Monsieur, | Ordre fixe : Madame avant Monsieur. |
| Dear Mr. Smith | Monsieur Smith, | Pas de "Cher" devant "Monsieur" + nom. |
| Dear Mrs. Smith | Madame Smith, | Idem. |
| Dear Dr. Jones | Monsieur le Docteur Jones / Docteur, | Selon le contexte. |
| To whom it may concern | A qui de droit / A l'attention de | Equivalent direct. |
| Sincerely, | Cordialement, / Je vous prie d'agreer... | Formule standard anglaise -> formule FR variable. |
| Best regards, | Cordialement, | Equivalent courant. |
| Yours faithfully, | Veuillez agreer, Madame, Monsieur, l'expression... | Formule tres formelle de cloture. |
| Yours sincerely, | Je vous prie d'agreer, Madame, Monsieur... | Formule formelle. |
| Hi / Hello | Bonjour | "Salut" est trop familier pour la correspondance pro. |
| Hey | Salut / Bonjour | Familier ; reserver aux contextes informels. |
| Passive voice | Voix active ou "on" | L'anglais abuse du passif ; le FR prefere l'actif ou "on". |
| Please | Veuillez / Merci de / S'il vous plait | "Please" est moins fort en EN ; "veuillez" plus poli en FR. |
| Thank you for... | Nous vous remercions de... / Merci pour... | Forme impersonnelle souvent preferee en FR pro. |
| We are pleased to... | Nous avons le plaisir de... / Nous sommes heureux de... | Equivalent standard. |
| I look forward to... | Dans l'attente de... / Je me rejouis de... | Formule de cloture. |
| Please do not hesitate to... | N'hesitez pas a... | Equivalent direct, acceptable. |
| Could you please... | Pourriez-vous... / Merci de... | Formule de requete polie. |

## 2. Dates, heures et nombres

| Anglais (en) | Francais (FR) | Notes |
|---|---|---|
| May 25, 2024 | 25 mai 2024 | Inverser mois et jour ; supprimer la virgule. |
| 05/25/2024 | 25/05/2024 | MM/JJ/AAAA -> JJ/MM/AAAA. |
| 5/25/2024 | 25/5/2024 | Idem. |
| 2024-05-25 | 25/05/2024 | Format ISO -> format FR si contexte non technique. |
| May 2024 | mai 2024 | Mois en minuscule en FR ; jamais de majuscule. |
| the 1st of May | le 1er mai | "1er" (premier) pour le premier jour du mois. |
| the 2nd of May | le 2 mai | "2" sans suffixe en FR (pas de "2nd"). |
| 10:00 AM | 10 h 00 / 10 h | Format 24h obligatoire : 10 h 00 du matin. |
| 3:30 PM | 15 h 30 | Convertir en 24h ; "3:30 PM" n'est pas standard en FR. |
| 12:00 PM / noon | 12 h 00 / midi | "Midi" prefere. |
| 12:00 AM / midnight | 0 h 00 / minuit | "Minuit" prefere. |
| 10:45 AM | 10 h 45 | Pas de "AM/PM". |
| 2 hours | 2 heures / 2 h | Abreviation "h" pour les durees. |
| 2h30 | 2 h 30 | Espace avant "h" (sauf usages tres techniques). |
| $1,000.50 | 1 000,50 $ / 1 000,50 USD | Virgule -> espace ; point -> virgule decimale ; signe apres le nombre. |
| $1.5 million | 1,5 million de dollars | Virgule decimale ; "million" singulier. |
| 1,000 | 1 000 | Separateur : espace insécable, pas de virgule. |
| 1,000,000 | 1 000 000 | Idem. |
| 0.5 | 0,5 | Virgule decimale (jamais le point en FR). |
| 2.5% | 2,5 % | Virgule decimale + espace avant %. |
| 10 °C | 10 °C | Espace entre nombre et symbole. |
| 10 x 20 cm | 10 x 20 cm / 10 × 20 cm | "x" ou signe multiplication. |

## 5. Formules administratives et juridiques

| Anglais (en) | Francais (FR) | Contexte |

| Shall | doit / est tenu de / est oblige de | "Shall" exprime une obligation legale ; ne pas traduire par le futur simple. |
| Shall not | ne peut / ne doit pas / il est interdit de | Interdiction formelle. |
| Must | doit / est tenu de | Obligation. |
| May | peut / est autorise a / a le droit de | Permission. |
| May not | ne peut pas / n'est pas autorise a | Interdiction moins forte que "shall not". |
| Should | devrait / il est recommande de | Recommendation, pas obligation. |
| Pursuant to | conformement a / en vertu de / aux termes de | Locution juridique. |
| In accordance with | conformement a / selon | Equivalent. |
| Subject to | sous reserve de / sous condition de | Condition suspensive. |
| Notwithstanding | nonobstant / malgre les dispositions de | "Nonobstant" est le terme juridique exact. |
| Whereas | considerant que / attendu que | Preambule de contrat. |
| Hereby | par les presentes / par la presente | Adverbe juridique. |
| Hereinafter | ci-apres | Renvoi dans le texte. |
| Herein | ci-inclus / dans le present document | "Aux presentes" dans un contrat. |

### Regles typographiques francaises essentielles

- **Espaces insécables** (AltGr + Maj + Espace sur AZERTY) avant `:`, `;`, `?`, `!`, `%`, `«` et apres `»`.
- **Guillemets** : toujours utiliser `«` et `»` avec espaces insécables. Ne jamais utiliser les guillemets droits anglais seuls.
- **Apostrophe** : utiliser l'apostrophe courbe `'` (AltGr + 4 sur AZERTY) ; pas d'espace avant ou apres.
- **Majuscules accentuees** : obligatoires (`É`, `È`, `Ê`, `Ë`, `À`, `Û`, `Î`, `Ô`, `Ç`). "Etat" -> "État". "Ecouter" -> "Écouter".
- **Virgule d'Oxford** : ne jamais l'utiliser en FR. "rouge, vert et bleu" (pas de virgule avant "et").
- **Nombres** : espace insécable tous les trois chiffres. Virgule comme séparateur decimal.
- **Abreviations** : `M.` (Monsieur), `Mme` (Madame), `Mlle` (Mademoiselle). `MM.` pour plusieurs messieurs, `Mmes` pour plusieurs dames.
- **Heure** : format 24h obligatoire. Espace avant et apres le "h" (`14 h 30`). Les formats `2:00 PM` sont inacceptables.
- **Date** : `25 mai 2024` (pas de virgule). `1er` pour le premier du mois.
- **Unites de mesure** : conversion au systeme metrique obligatoire (pieds, miles, Fahrenheit -> metres, km, Celsius).
