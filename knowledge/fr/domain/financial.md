## Principes généraux

- Tolérance zéro pour les valeurs numériques : aucune arrondi, aucune modification
- Utiliser la terminologie comptable française standard (PCG)
- Conserver la précision décimale (100,00 € pas 100 €)
- Les montants négatifs : utiliser le signe moins ou les parenthèses

## Conventions numériques

- Format français : 1 234,56 € (espace insécable pour les milliers, virgule pour les décimales)
- Pourcentages : 15,5 % (avec espace avant le signe)
- Ratios : 1,5:1

## Termes des états financiers

- Bilan : actif / passif
- Compte de résultat : produits / charges
- Tableau des flux de trésorerie : flux opérationnels / investissement / financement

## Termes fiscaux

- TVA : taxe sur la valeur ajoutée
- IS : impôt sur les sociétés
- IR : impôt sur le revenu
- URSSAF : Union de recouvrement des cotisations de sécurité sociale et d'allocations familiales

## Termes de facturation

- Facture : document comptable officiel
- Bon de commande : document préalable à la facture
- Devis : estimation de prix
- Avoir : note de crédit

## Rapports d'audit

- Rapport de gestion : rapport des dirigeants
- Commissaire aux comptes : auditeur externe
- Certification des comptes : opinion d'audit
- Réserve : opinion avec réserves
- Lettre d'observation : lettre de recommandation

## Fusions et acquisitions

- Due diligence : vérification préalable
- Valorisation : évaluation
- Cession de titres : cession d'actions
- Fusion : opération de rapprochement
- Acquisition : rachat

## Différences IFRS / PCG

- Le PCG (Plan Comptable Général) français diffère des IFRS
- « Produits constatés d'avance » → produits différés
- « Charges constatées d'avance » → charges différées
- « Écart de conversion » → différence de change
- Indiquer si les états financiers suivent le PCG ou les IFRS
