# =============================================================================
# MODELE DE LECTURE / READER MODEL
# =============================================================================

Ce modele decrit comment lire et interpreter un document administratif source
en vue de sa traduction en francais. Il identifie les registres de la langue
administrative francaise, les references juridiques et les formules consacrees.

## 1. Identification du type de document administratif

| Type de document                  | Marqueurs source                            | Registre              |
|-----------------------------------|---------------------------------------------|------------------------|
| Arrete                            | "Vu... Vu... Sur proposition... Arrete"     | Reglementaire          |
| Decret                            | "Vu la Constitution... Vu... Decrete"       | Reglementaire          |
| Circulaire                        | "La ministre a [...] l'honneur de [...]"    | Interpretatif          |
| Avis                              | "Avis de vacance" / "Avis d'appel public"   | Informatif             |
| Decision individuelle             | "Decision portant attribution de..."        | Individuel             |
| Lettre administrative             | "J'ai l'honneur de vous informer"           | Formel officiel        |
| Note de service                   | "La direction rappelle que..."              | Interne                |
| Rapport administratif             | "En application de la loi du..."            | Explicatif             |
| Compte rendu administratif        | "Il a ete expose que..."                    | Factuel                |
| Certificat administratif          | "Je soussigne certifie que..."              | Attestatif             |
| Formulaire administratif          | "Nom : ... Prenom : ..."                   | Structured             |
| Procès-verbal                    | "L'an deux mille vingt-six, le [...]"       | Solennel               |
| Marche public                     | "Entre l'Etat represente par... et..."      | Contractuel            |
| Convention                        | "Entre les soussignes..."                   | Contractuel admin.     |
| Arret de la juridiction admin.   | "Vu la requete... Considerant... DECIDE"    | Motive / juridique     |
| Deliberation                      | "Le conseil municipal, deliberant..."       | Decision collegiale    |
| Arrete prefectoral                | "Le prefet du departement... Vu... Arrete"  | Reglementaire local    |
| Avis de la CADA                   | "La commission est competente..."           | Consultatif            |

## 2. Registres de la langue administrative

| Registre             | Usage                                       | Caracteristiques                               |
|----------------------|---------------------------------------------|------------------------------------------------|
| Solennel             | Actes reglementaires, decrets, arretes      | "Le President de la Republique... Vu... Decrete" |
| Officiel             | Lettres administratives, notifications      | "J'ai l'honneur de vous informer..."            |
| Impersonnel          | Avis publics, annonces officielles          | "Il est porte a la connaissance du public..."  |
| Motivant             | Decisions, arrets, jugements                | "Considerant que... En droit... Par ces motifs" |
| Descriptif           | Rapports, bilans                            | "Il a ete constate que..."                      |
| Informatif           | Notes de service, circulaires               | "La direction rappelle que..."                  |

## 4. Formules de cloture administratives

| Contexte                                    | Formule de cloture                                                      |
|---------------------------------------------|-------------------------------------------------------------------------|
| Lettre a un ministre                        | "Je vous prie d'agreer, Monsieur le Ministre, l'expression de ma haute consideration." |
| Lettre a un prefet                          | "Je vous prie d'agreer, Monsieur le Prefet, l'expression de ma consideration distinguee." |
| Lettre a un maire                           | "Veuillez agreer, Monsieur le Maire, l'expression de mes salutations respectueuses." |
| Lettre a un fonctionnaire                   | "Je vous prie d'agreer, Monsieur le Directeur, l'expression de mes salutations distinguees." |
| Lettre administrative standard (demandeur)  | "Je vous prie d'agreer, Madame, Monsieur, l'expression de mes salutations distinguees." |
| Lettre administrative standard (expediteur) | "Veuillez agreer, Madame, Monsieur, l'expression de mes salutations distinguees." |
| Note de service                             | Sans formule de cloture / "Merci de votre attention."                   |
| Communication officielle courte             | "Cordialement" (rare dans l'administration)                             |
| Reponse a une reclamation                   | "Je vous prie d'agreer, Madame, Monsieur, l'assurance de ma consideration." |
| Lettre de refus                             | "Je vous prie d'agreer, Madame, Monsieur, l'expression de mes salutations." (sans "distinguees") |

## 5. Structure typique d'une lettre administrative

1. **En-tete officiel** : Republique Francaise / Ministere / Direction / Service
2. **Reference** : "Ref : V/L du [date] / N/Ref : [code dossier]"
3. **Objet** : "Objet : Demande de..."
4. **Visa** (documents reglementaires) : "Vu le code... Vu la loi du..."
5. **Corps** :
   - Introduction : "J'ai l'honneur de vous informer que..."
   - Developpement : expose des motifs et des faits
   - Conclusion : decision ou demande d'action
6. **Formule de politesse**
7. **Signature** : nom, prenom, qualite, cachet
8. **Annexes** : "P.J. : ..."

## Codes des relations entre le public et l'administration (CRPA)

Le CRPA, entree en vigueur le 1er janvier 2016, codifie les regles applicables
aux relations entre l'administration et les usagers. Points cles :

- Principe de legalite : toute decision administrative doit etre fondee sur un texte
- Principe du contradictoire : l'administre doit etre informe des mesures envisagees
- Delai d'instruction : sauf texte contraire, le silence de l'administration vaut
  acceptation (loi du 12 novembre 2013) - "silence vaut accord"
- Communication des documents administratifs : droit d'acces sous reserve des
  secrets proteges par la loi (secret de la defense, medical, etc.)
- Recours administratif prealable obligatoire (RAPO) dans certains domaines
- Motivation des decisions defavorables (loi du 11 juillet 1979)

## Regles de la communication administrative

1. L'administration repond au nom de l'Etat, pas en son nom propre.
   Utiliser "Nous" ou formes impersonnelles, jamais "Je" sauf pour certaines
   correspondances signees personnellement.

2. Les formules de politesse administratives sont plus longues et plus
   ceremonieuses que les formules commerciales.

3. Tout refus doit etre motive en droit et en fait (CRPA, art. L. 211-2).

4. Les decisions defavorables doivent mentionner les voies et delais de recours
   (CRPA, art. L. 211-2).

5. Les visas (enumeration des textes applicables) sont obligatoires dans les
   actes reglementaires et les decisions individuelles.

6. L'objet doit etre formule de maniere succincte mais complete, commencant
   par un nom : "Objet : Demande de permis de construire" (pas "Objet : Je
   souhaite obtenir...").

7. Les delais de recours contentieux sont de 2 mois a compter de la
   notification de la decision.

8. Les recommandes avec accuse de reception (RAR) sont la norme pour les
   notifications officielles.

9. Les sigles administratifs doivent etre explicites a leur premiere occurrence.
   Exemple : "Direction regionale de l'environnement, de l'amenagement et du
   logement (DREAL)".

10. Les decisions individuelles peuvent etre attaquees par la voie du recours
    pour exces de pouvoir (REP) devant le tribunal administratif.
