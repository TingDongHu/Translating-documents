# =============================================================================
# MODELE DE LECTURE / READER MODEL
# =============================================================================

Ce modele decrit comment lire et interpreter un document juridique source en
vue de sa traduction en francais. Il identifie les registres de langue,
les structures argumentatives et les marqueurs de force obligatoire.

## 1. Identification du type de document juridique

| Type de document                 | Marqueurs francais                         | Registre            |
|----------------------------------|--------------------------------------------|----------------------|
| Contrat                          | "Les parties conviennent de ce qui suit"   | Obligation / volonte |
| Convention collective            | "Est convenu ce qui suit"                   | Normatif negocie     |
| Traite                           | "Les Etats parties au present Traite"       | Solennel             |
| Loi / Code                       | "Art. 1er. - Est interdit..."               | Imperatif legislatif |
| Arret de cour                    | "Attendu que... / Aux motifs que..."         | Motive               |
| Arrete ministeriel               | "Vu le code..., Vu le decret..., Arrete"    | Reglementaire        |
| Decret                           | "Vu la Constitution..., Le Conseil d'Etat entendu, Decrete" | Reglementaire  |
| Acte notarie                     | "Par devant Me [nom], notaire..."           | Authentique          |
| Statut d'association             | "Il est constitue entre..."                 | Conventionnel        |
| Contrat de bail                  | "Monsieur X donne a bail a Monsieur Y..."   | Synallagmatique      |
| Contrat de travail               | "Est engage en qualite de..."               | Salarial             |
| Pacte d'actionnaires             | "Entre les soussignes, il a ete convenu"   | Conventionnel        |
| Mandat / Procuration             | "Je soussigne donne pouvoir a..."          | Unilateral           |
| Assignation en justice           | "L'an deux mille vingt-six et le..."        | Procedural           |
| Conclusions                      | "Plaise au Tribunal de..."                  | Judiciaire           |
| Consultation juridique           | "En l'etat du droit positif..."             | Doctrinal            |

## 2. Registres de langue juridique

| Registre           | Usage                               | Caracteristiques linguistiques                     |
|--------------------|--------------------------------------|-----------------------------------------------------|
| Solennel           | Traites, preambules, serments        | "Nous, [autorite], proclamons..." Formules longues  |
| Administratif      | Arretes, decrets, circulaires        | "Vu... Vu... Considerant... Sur proposition de..."  |
| Judiciaire         | Decisions, arrets, jugements         | "Attendu que... En droit... Par ces motifs..."       |
| Contractuel        | Contrats, conventions, accords       | "Les parties conviennent de ce qui suit..."          |
| Notarial           | Actes authentiques, testaments       | "Par devant le notaire soussigne..."                |
| Procedural         | Assignations, conclusions, memoires  | "L'an... A la requete de... Plaise au Tribunal..."  |

| Element anglo-americain        | Equivalent francais                    | Decision                                   |
|--------------------------------|----------------------------------------|--------------------------------------------|
| "Parties"                      | "Les soussignes" / "Entre les parties" | Conserver la mention des parties            |
| "Recitals / Whereas clauses"   | "Expose prealable" / "Considerant"     | Traduire en "Considerant que..."            |
| "Definitions"                  | "Definitions"                          | Conserver la structure                      |

| "Covenants"                    | "Engagements"                          | Traduire par "engagements"                  |

| "Conditions Subsequent"       | "Conditions resolutoires"              | Concept identique                           |
| "Indemnification"              | "Garantie / Obligation d'indemnisation" | Necessite d'adaptation                     |

| "Termination"                  | "Resiliation"                          | "Resiliation" (pas "terminaison")           |
| "Governing Law"               | "Droit applicable" / "Lex contractus"  | Clause standard                             |
| "Dispute Resolution"          | "Reglement des litiges"                | Arbitrage ou tribunaux                      |
| "Force Majeure"               | "Force majeure"                        | Definition plus restrictive en droit francais |
| "Entire Agreement"            | "Accord integral" / "Integration"      | Concept connu mais moins systematique       |

| "Notices"                     | "Notifications"                         | Conserver                                   |
| "Signatures"                  | "Signatures"                           | Forme : "Fait a [ville], le [date]"          |
| "Schedules / Exhibits"        | "Annexes"                              | "Annexe" (pas "emploi du temps")            |

## Format d'un contrat en droit francais

1. **Intitule** : "CONTRAT DE [type]"
2. **Entre les soussignes** : identification des parties :
   - Personne physique : Nom, prenom, date et lieu de naissance, nationalite, domicile
   - Personne morale : Denomination sociale, forme juridique, capital social, siege social, RCS, SIRET, representant legal
3. **Expose prealable** (Considerants) : contexte et motivations
4. **Definitions** : termes definis dans le contrat
5. **Corps du contrat** : clauses organisees par thematique
6. **Annexes** : documents contractuels joints
7. **Signature** : "Fait a [ville], le [date]", en [nombre] exemplaires originaux

## Formules de politesse juridiques

- "Plaise au Tribunal de..." (formule de requete judiciaire)
- "En foi de quoi les parties ont signe les presentes"
- "Lu et approuve" (formule avant signature dans les actes sous seing prive)
- "Bon pour accord" (formule abregee de signature)
- "Bon pour pouvoirs" (signature par mandataire)
- "De convention expresse entre les parties"

## Principes d'interpretation des contrats (Code civil)

- Article 1188 : Le contrat s'interprete d'apres la commune intention des parties
- Article 1189 : Toutes les clauses s'interprettent les unes par rapport aux autres
- Article 1190 : En cas de doute, le contrat s'interprete contre le creancier et en faveur du debiteur
- Article 1191 : Les termes generaux ne couvrent que ce qui est dans l'intention commune

## Mentions obligatoires dans les contrats commerciaux

- Conditions generales de vente (CGV) obligatoires pour les professionnels
- Delai de livraison mentionne explicitement
- Prix TTC et HT
- Conditions de paiement, penalites de retard (taux d'interet legal)
- Clause de reserve de propriete (optionnelle mais recommandee)
- Renseignements sur les garanties legales (conformite, vices caches)
- Tribunal competent en cas de litige
