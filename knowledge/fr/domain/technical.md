## Principes généraux

- Utiliser la terminologie technique standard (normes ISO, AFNOR, NF)
- Conserver les valeurs mesurées exactes (pas d'arrondi)
- Utiliser le système d'unités international (SI)
- Conserver la cohérence terminologique dans tout le document

## Style technique

- Utiliser l'impératif pour les instructions (« Appuyez sur le bouton »)
- Utiliser la voix passive pour les spécifications (« La pièce est usinée... »)
- Phrases courtes et précises
- Listes numérotées pour les procédures séquentielles

## Unités et mesures

- Conserver les unités d'origine ; ajouter la conversion SI si demandé
- Température : degrés Celsius (°C), kelvin (K)
- Pression : pascal (Pa), bar, mégapascal (MPa)
- Énergie : joule (J), kilowattheure (kWh)

## Normes et certifications

- Conserver les numéros de normes exactement (ISO 9001, CE, NF, etc.)
- « CE marking » → « marquage CE »
- « ISO certification » → « certification ISO »
- Normes françaises : NF (Norme Française)
