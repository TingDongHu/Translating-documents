## Formality and Address

- **English "you" → Bengali আপনি / তুমি / তুই distinction**
  - Source: "You are required to..." → Target: "আপনাকে এটি করতে হবে..." (formal আপনি)
  - Source: "You may proceed" → Target: "আপনি এগিয়ে যেতে পারেন" (formal register for business/tech)
  - Source: "You (singular, intimate)" → Target: "তুই" (only for very close relationships)
  - Source: "You all" → Target: "আপনারা" (formal plural)
  - Context: Business, official, and technical documents always use আপনি — never তুমি or তুই

- **English "your" → Bengali আপনার / তোমার / তোদ**
  - Source: "Your application" → Target: "আপনার আবেদন" (formal)
  - Source: "Your order" → Target: "আপনার অর্ডার"
  - Source: "Yours sincerely" → Target: "বিশ্বাসী," (letter closing)

- **English "Dear Sir / Dear Madam" → Bengali equivalent**
  - Source: "Dear Sir" → Target: "সম্মানিত মহোদয়" or "জনাব"
  - Source: "Dear Madam" → Target: "সম্মানিত মহোদয়া" or "জনাবা"
  - Source: "Dear Sir or Madam" → Target: "সম্মানিত মহোদয়/মহোদয়া"
  - Note: Bengali formal letters use সম্মানিত (respected) or প্রিয় (dear) — সম্মানিত is more formal

- **English "Dear Mr. / Ms." → Bengali জনাব / জনাবা**
  - Source: "Dear Mr. Rahman" → Target: "সম্মানিত জনাব রহমান"
  - Source: "Dear Ms. Begum" → Target: "সম্মানিত জনাবা বেগম"
  - Note: Bengali uses জনাব (Mr.) and জনাবা (Ms./Mrs.) as standard formal address

- **English "To whom it may concern" → Bengali equivalent**
  - Source: "To whom it may concern" → Target: "সংশ্লিষ্ট ব্যক্তি/কর্তৃপক্ষের জন্য"

- **English title "Mr." → Bengali জনাব**
  - Source: "Mr. Rahman" → Target: "জনাব রহমান" (never "মিস্টার রহমান" in formal Bengali)

- **English title "Mrs." → Bengali জনাবা**
  - Source: "Mrs. Begum" → Target: "জনাবা বেগম"

- **English title "Ms." → Bengali জনাবা**
  - Source: "Ms. Akter" → Target: "জনাবা আক্তার"

- **English "Dr." → Bengali ড.** (same, used as prefix)
  - Source: "Dr. Khan" → Target: "ড. খান" or "ডক্টর খান"

- **English "Professor" → Bengali প্রফেসর** (or অধ্যাপক in formal contexts)
  - Source: "Professor Das" → Target: "প্রফেসর দাস" / "অধ্যাপক দাস"

- **English imperative mood → Bengali formal imperative (আপনি-form)**
  - Source: "Click here" → Target: "এখানে ক্লিক করুন" (formal imperative)
  - Source: "Enter your password" → Target: "আপনার পাসওয়ার্ড লিখুন"
  - Source: "Please wait" → Target: "অনুগ্রহ করে অপেক্ষা করুন"
  - Note: Bengali technical docs use আপনি-form (formal) by default

- **English "please" → Bengali অনুগ্রহ করে / দয়া করে**
  - Source: "Please find attached..." → Target: "সংলগ্ন দেখুন..." / "সংলগ্ন রয়েছে..."
  - Source: "Please confirm receipt" → Target: "অনুগ্রহ করে প্রাপ্তির নিশ্চিত করুন"
  - Note: অনুগ্রহ করে is more frequent in formal Bengali than "please" in formal English

- **English "we" → Bengali আমরা / আমাদের**
  - Source: "We are pleased to inform you" → Target: "আমরা আপনাকে জানাতে পেরে আনন্দিত"
  - Source: "We hereby confirm" → Target: "আমরা এতদ্বারা নিশ্চিত করছি"

- **English gender-neutral language → Bengali (Bengali is less gender-marked)**
  - Source: "The applicant must submit his/her form" → Target: "আবেদনকারীকে তার/তার ফর্ম জমা দিতে হবে" (Bengali uses তার for both genders)
  - Source: "Each employee should check his email" → Target: "প্রত্যেক কর্মচারীকে তার ইমেল পরীক্ষা করা উচিত"
  - Note: Bengali তার/তার is gender-neutral for possessive — no need to specify gender for inanimate objects

- **English "he/she" → Bengali তিনি (formal) / সে (informal)**
  - Source: "He is the manager" → Target: "তিনি ব্যবস্থাপক" (formal, respectful)
  - Source: "She is my friend" → Target: "সে আমার বন্ধু" (informal) or "তিনি আমার বন্ধু" (formal)
  - Note: Bengali তিনি is gender-neutral and respectful. সে is informal and also gender-neutral.

- **English "his/her" → Bengali তাঁর (formal) / তার (informal)**
  - Source: "His office" → Target: "তাঁর অফিস" (formal)
  - Source: "Her name" → Target: "তার নাম" (informal) or "তাঁর নাম" (formal)
  - Note: Bengali possessive pronouns are gender-neutral

- **English "him/her" → Bengali তাঁকে (formal) / তাকে (informal)**
  - Source: "I told him" → Target: "আমি তাঁকে বলেছি" (formal)
  - Source: "I saw her" → Target: "আমি তাকে দেখেছি" (informal)

- **English "himself/herself" → Bengali নিজে (same for both genders)**
  - Source: "He did it himself" → Target: "তিনি নিজে করেছেন"
  - Source: "She did it herself" → Target: "সে নিজে করেছে"
  - Note: Bengali নিজে is gender-neutral

- **English "they/them" (singular, gender-neutral) → Bengali তিনি / সে**
  - Source: "They approved the document" → Target: "তিনি/সে নথিটি অনুমোদন করেছেন"
  - Note: Bengali তিনি/সে are already gender-neutral — no special handling needed

- **English "it" → Bengali এটি / সেটি / ওটি**
  - Source: "It is important" → Target: "এটি গুরুত্বপূর্ণ"
  - Source: "It is a book" → Target: "এটি একটি বই"
  - Note: Bengali has three demonstrative pronouns: এ (proximal), স (medial), ও (distal)

## Number and Date Conventions

- **English decimal point (.) → Bengali decimal point (.)** — same, NO change needed
  - Source: "3.14" → Target: "3.14" (Bengali uses the same decimal convention as English)
  - Important: Bengali does NOT use comma as decimal separator

- **English thousands separator (,) → Indian comma format (lakh/crore grouping)**
  - Source: "12,345,678" → Target: "1,23,45,678" (Indian grouping)
  - Source: "1,000,000" → Target: "10,00,000"
  - Source: "100,000" → Target: "1,00,000"
  - Note: Indian system groups the first three digits from right, then every two digits

- **English "billion" (1,000,000,000) → Bengali অরব / বিলিয়ন**
  - Source: "$5 billion" → Target: "5 অরব ডলার" or "US$5 অরব"
  - Note: Bengali system: 1,00,00,00,000 = 1 অরব (same as English billion, different grouping)

- **English "million" (1,000,000) → Bengali মিলিয়ন**
  - Source: "$5 million" → Target: "50 লাখ ডলার" or "US$50,00,000"
  - Source: "2.5 million" → Target: "25 লাখ" (25,00,000)

- **English "trillion" (10¹²) → Bengali ট্রিলিয়ন / খরব**
  - Source: "$2 trillion" → Target: "2 ট্রিলিয়ন ডলার" (1 খরব = 1,00,00,00,00,000)

- **English "lakh/crore" → Bengali লাখ/কোটি** (these terms are already used in Bengali)
  - Source: "5 lakh" → Target: "5 লাখ"
  - Source: "2 crore" → Target: "2 কোটি"

- **English date format (MM/DD/YYYY or DD/MM/YYYY) → Bengali DD/MM/YYYY**
  - Source: "05/27/2026" (US format) → Target: "27/05/2026" (Bengali format)
  - Source: "May 27, 2026" → Target: "২৭ মে ২০২৬"
  - Source: "27 May 2026" → Target: "২৭ মে ২০২৬"
  - Note: Bengali uses DD/MM/YYYY, not US MM/DD/YYYY

- **English "January" → Bengali জানুয়ারি**
  - All English month names have standard Bengali equivalents (see base rules)

- **English time format (12-hour with AM/PM) → Bengali 12-hour with time-of-day terms**
  - Source: "2:30 PM" → Target: "বিকাল ২:৩০" or "14:30"
  - Source: "10:00 AM" → Target: "সকাল ১০:০০"
  - Source: "8:00 PM" → Target: "রাত ৮:০০"
  - Note: Bengali uses time-of-day terms: সকাল (morning), দুপুর (afternoon), বিকাল (late afternoon), রাত (night)

- **English day names → Bengali day names**
  - Source: "Monday" → Target: "সোমবার"
  - Source: "Friday" → Target: "শুক্রবার"
  - Note: Bengali day names end in -বার (day)

- **English week/month references → Bengali**
  - Source: "next Monday" → Target: "আসন্ন সোমবার"
  - Source: "last Friday" → Target: "গত শুক্রবার"
  - Source: "this month" → Target: "এই মাসে"
  - Source: "next year" → Target: "আগামী বছর"

- **English ordinal numbers → Bengali ordinal suffixes**
  - Source: "1st" → Target: "1ম" or "প্রথম"
  - Source: "2nd" → Target: "2য়" or "দ্বিতীয়"
  - Source: "3rd" → Target: "3য়" or "তৃতীয়"
  - Note: Bengali ordinals use suffixes -তম or -ম/-য়/-ং

- **English "percent" → Bengali শতাংশ**
  - Source: "50 percent" → Target: "50 শতাংশ" or "50%"
  - Source: "5%" → Target: "5%" or "5 শতাংশ"

- **English fractions → Bengali**
  - Source: "1/2" → Target: "1/2" or "এক-দ্বিতীয়াংশ"
  - Source: "3/4" → Target: "3/4" or "তিন-চতুর্থাংশ"

- **English date ranges → Bengali**
  - Source: "January to March" → Target: "জানুয়ারি থেকে মার্চ"
  - Source: "from 2020 to 2025" → Target: "২০২০ থেকে ২০২৫"
  - Source: "between 10 and 15" → Target: "১০ এবং ১৫ এর মধ্যে"

## Cultural References

- **English idioms → Bengali idioms of equivalent meaning**
  - Source: "burning the midnight oil" → Target: "রাত-দিন পরিশ্রম করা" (work day and night)
  - Source: "the ball is in your court" → Target: "এখন আপনার পালা" (now it's your turn)
  - Source: "break the ice" → Target: "প্রথম পদক্ষেপ নেওয়া" (take the first step)
  - Source: "kill two birds with one stone" → Target: "এক ঢিলে দুই পাখি" (one stone two birds — similar in Bengali)
  - Source: "a blessing in disguise" → Target: "অভাগার ভাগ্য" (misfortune's fortune)
  - Note: Do NOT translate idioms literally — find the Bengali equivalent

- **English proverbs → Bengali proverbs**
  - Source: "Time is money" → Target: "সময়ই অর্থ"
  - Source: "Actions speak louder than words" → Target: "কথার চেয়ে কাজ বেশি" (work speaks more than words)
  - Source: "Practice makes perfect" → Target: "অনুশীলনে দক্ষতা আসে"

- **English cultural references → Bengali adaptation**
  - Source: References to Western holidays (Christmas, Thanksgiving) → Add brief explanation if context requires
  - Source: References to Western legal/business practices → Adapt to Bengali equivalents where possible
  - Note: Preserve cultural references that are understood globally (e.g., "Silicon Valley")

- **English "etc." → Bengali ইত্যাদি**
  - Source: "apples, bananas, etc." → Target: "কলা, আপেল, ইত্যাদি"

- **English "for example" → Bengali যেমন**
  - Source: "For example, ..." → Target: "যেমন, ..."

- **English "i.e." → Bengali অর্থাৎ**
  - Source: "i.e., that is" → Target: "অর্থাৎ"

- **English "e.g." → Bengali যেমন**
  - Source: "e.g., for example" → Target: "যেমন"

- **English "vs." → Bengali বনাম**
  - Source: "Company A vs. Company B" → Target: "কোম্পানি A বনাম কোম্পানি B"

- **English "approximately" → Bengali প্রায় / আনুমানিক**
  - Source: "approximately 100" → Target: "প্রায় 100" or "আনুমানিক 100"

- **English "respectively" → Bengali যথাক্রমে**
  - Source: "A and B are 10 and 20 respectively" → Target: "A এবং B যথাক্রমে 10 এবং 20"

- **English "however" → Bengali তবে / যাইহোক**
  - Source: "However, ..." → Target: "তবে, ..." or "যাইহোক, ..."

- **English "therefore" → Bengali সুতরাং / তাই**
  - Source: "Therefore, ..." → Target: "সুতরাং, ..." or "তাই, ..."

- **English "furthermore/moreover" → Bengali এছাড়াও / তদুপরি**
  - Source: "Furthermore, ..." → Target: "এছাড়াও, ..." or "তদুপরি, ..."

- **English "in addition" → Bengali এছাড়া**
  - Source: "In addition, ..." → Target: "এছাড়া, ..."

- **English "nevertheless" → Bengali তথাপি / তবুও**
  - Source: "Nevertheless, ..." → Target: "তথাপি, ..." or "তবুও, ..."

- **English "hence" → Bengali সুতরাং / তাই**
  - Source: "Hence, ..." → Target: "সুতরাং, ..."

- **English "namely" → Bengali অর্থাৎ / অর্থাৎ**
  - Source: "namely, A, B, and C" → Target: "অর্থাৎ, A, B, এবং C"

## Terminology

- **Technical terms: preserve English where no established Bengali equivalent exists**
  - Source: "software" → Target: "সফটওয়্যার" (preserve)
  - Source: "hardware" → Target: "হার্ডওয়্যার" (preserve)
  - Source: "internet" → Target: "ইন্টারনেট" (preserve)
  - Source: "artificial intelligence" → Target: "কৃত্রিম বুদ্ধিমত্তা" (Bengali equivalent exists)
  - Note: When in doubt, preserve the English term and add Bengali explanation on first use

- **Legal terms: use established Bengali legal terminology**
  - Source: "contract" → Target: "চুক্তি"
  - Source: "agreement" → Target: "চুক্তি" or "সমঝোতা"
  - Source: "liability" → Target: "দায়বদ্ধতা"
  - Source: "negligence" → Target: "অবহেলা"
  - Source: "damages" → Target: "ক্ষতিপূরণ"
  - Note: Legal terms have specific Bengali equivalents — do not translate literally

- **Financial terms: use established Bengali financial terminology**
  - Source: "revenue" → Target: "রাজস্ব"
  - Source: "expense" → Target: "ব্যয়"
  - Source: "profit" → Target: "লাভ"
  - Source: "loss" → Target: "ক্ষতি"
  - Source: "assets" → Target: "সম্পদ"
  - Source: "liabilities" → Target: "দায়"

- **Government terms: use established Bengali administrative terminology**
  - Source: "minister" → Target: "মন্ত্রী"
  - Source: "secretary" → Target: "সচিব"
  - Source: "director" → Target: "পরিচালক"
  - Note: Government titles have specific Bengali equivalents — never translate literally

- **Brand names: NEVER translate**
  - Source: "iPhone" → Target: "iPhone" (preserve)
  - Source: "Google" → Target: "Google" (preserve)
  - Source: "Samsung" → Target: "Samsung" (preserve)

- **Measurement units: preserve or convert as appropriate**
  - Source: "5 miles" → Target: "8 কিলোমিটার" (convert to metric) or "5 মাইল" (preserve if context requires)
  - Source: "5 kg" → Target: "5 কেজি" (same)
  - Source: "5 feet" → Target: "1.5 মিটার" (convert) or "5 ফুট" (preserve)
  - Source: "100°F" → Target: "37.8°C" (convert to Celsius) or "100°F" (preserve)
  - Note: Bengali uses metric system. Convert imperial units unless preserving original is necessary.

- **English "please note" → Bengali দয়া করে লক্ষ করুন**
  - Source: "Please note that..." → Target: "দয়া করে লক্ষ করুন যে..."
  - Source: "Please note" → Target: "দয়া করে লক্ষ করুন"

- **English "as mentioned above/below" → Bengali উপরে/নীচে উল্লিখিত অনুযায়ী**
  - Source: "As mentioned above" → Target: "উপরে উল্লিখিত অনুযায়ী"
  - Source: "As described below" → Target: "নীচে বর্ণিত অনুযায়ী"

- **English "in accordance with" → Bengali অনুযায়ী**
  - Source: "In accordance with the policy" → Target: "নীতি অনুযায়ী"

## Administrative and Legal

- **English "shall" → Bengali obligation equivalents**
  - Source: "The Company shall..." → Target: "কোম্পানি করতে হবে..."
  - Source: "You shall not..." → Target: "আপনি করতে পারবেন না..."
  - Note: Bengali does not have a single modal equivalent of "shall" — use appropriate construction

- **English "hereby" → Bengali এতদ্বারা**
  - Source: "We hereby confirm" → Target: "আমরা এতদ্বারা নিশ্চিত করছি"

- **English "herein" → Bengali এতে / এই চুক্তিতে**
  - Source: "as described herein" → Target: "এতে বর্ণিত অনুযায়ী"

- **English "whereas" → Bengali যেহেতু**
  - Source: "Whereas the parties agree..." → Target: "যেহেতু পক্ষগণ সম্মত হন যে..."

- **English "notwithstanding" → Bengali তথাপি / এর বাইরেও**
  - Source: "Notwithstanding the above..." → Target: "উপরের সত্ত্বেও..."

- **English "in lieu of" → Bengali পরিবর্তে**
  - Source: "in lieu of payment" → Target: "পেমেন্টের পরিবর্তে"

- **English "with respect to / in respect of" → Bengali সংক্রান্ত / সম্পর্কে**
  - Source: "with respect to this matter" → Target: "এই বিষয় সংক্রান্ত"

- **English "for the purpose of" → Bengali উদ্দেশ্যে**
  - Source: "for the purpose of this agreement" → Target: "এই চুক্তির উদ্দেশ্যে"

- **English "in the event that" → Bengali যদি / পরিস্থিতিতে**
  - Source: "in the event that..." → Target: "যদি..." or "এই পরিস্থিতিতে..."

- **English "prior to" → Bengali পূর্বে**
  - Source: "prior to the meeting" → Target: "সভার পূর্বে"

- **English "subsequent to" → Bengali পরে / পরবর্তীতে**

## Punctuation and Formatting

- **English period (.) → Bengali dari (।) or period (.)**
  - Formal/literary Bengali uses dari (।) as sentence terminator
  - Modern, technical, and business Bengali commonly uses period (.)
  - Choose based on document type and maintain consistency

- **English quotation marks ("...") → Bengali same ("...")**
  - Bengali uses the same double quotation marks as English
  - No conversion needed

- **English comma (,) → Bengali same (,)**
  - Bengali uses commas in the same way as English
  - No conversion needed

- **English colon (:) → Bengali same (:)**
  - Bengali uses colons in the same way as English
  - No conversion needed

- **English semicolon (;) → Bengali same (;)**
  - Bengali uses semicolons in the same way as English
  - No conversion needed

- **English hyphen (-) → Bengali same (-)**
  - Bengali uses hyphens in the same way as English
  - No conversion needed

- **English em dash (—) → Bengali same (—)**
  - Bengali uses em dashes in the same way as English
  - No conversion needed

- **English parentheses () → Bengali same ()**
  - Bengali uses parentheses in the same way as English
  - No conversion needed

- **English apostrophe (') → Bengali context-dependent**
  - In English loanwords: preserve (e.g., "ওয়ার্ক'স" is not standard — just use "ওয়ার্ক")
  - In possessives: Bengali does not use apostrophes for possession
  - Note: Bengali does NOT use apostrophe as a grammatical marker

- **English percentage (%) → Bengali same (%)**
  - Bengali uses the same percent symbol
  - No conversion needed

- **English currency symbols → Bengali equivalents**
  - Source: "$100" → Target: "$100" or "100 ডলার" (context-dependent)
  - Source: "€100" → Target: "€100" or "100 ইউরো"
  - Source: "£100" → Target: "£100" or "100 পাউন্ড"

- **English bullet points → Bengali same**
  - Bengali uses the same bullet point style as English
  - No conversion needed

- **English numbered lists → Bengali same**
  - Bengali uses the same numbered list style as English
  - No conversion needed

- **English "note:" → Bengali টীকা:**
  - Source: "Note:" → Target: "টীকা:" or "দ্রষ্টব্য:"
  - Source: "Important note:" → Target: "গুরুত্বপূর্ণ টীকা:"

- **English "warning:" → Bengali সতর্কতা:**
  - Source: "Warning:" → Target: "সতর্কতা:"

- **English "tip:" → Bengali পরামর্শ:**
  - Source: "Tip:" → Target: "পরামর্শ:"

- **English "see also" → Bengali দেখুন এছাড়াও**
  - Source: "See also: ..." → Target: "দেখুন এছাড়াও: ..."

- **English "ref." → Bengali রেফ. / রেফারেন্স**

  - Source: "Smith et al." → Target: "স্মিথ প্রমুখ" or "স্মিথ ও অন্যান্য"
