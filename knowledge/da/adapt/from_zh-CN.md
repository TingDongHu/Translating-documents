# Danish Adaptation Guide: Chinese → Danish

This document provides comprehensive adaptation rules for translating content from
Chinese (Simplified, zh-CN) to Danish (da). It is intended as a reference for
professional translators and reviewers working with the Chinese-to-Danish language pair.

---

## 1. Formality and Address

### 1.1 Address System Comparison

Chinese and Danish have fundamentally different address systems. Chinese uses
a complex system of titles, honorifics, and social hierarchy markers. Danish
is comparatively egalitarian, with a declining formal "De" pronoun and minimal
title usage.

**Chinese Address Elements → Danish Mapping:**

| Chinese | Pinyin | Danish | Notes |
|---|---|---|---|
| 先生 | xiānsheng | Hr. / herre | "Hr." is the standard abbreviation |
| 女士 | nǚshì | Fru / kvinde | "Fru" for married; no distinct "Ms." equivalent |
| 小姐 | xiǎojiě | Frue / kvinde | Historically "Miss"; now less common in Chinese too |
| 老师 | lǎoshī | Lærer / hr./fru [Navn] | Title used in education contexts |
| 教授 | jiàoshòu | Professor [Navn] | Academic title preserved |
| 医生 | yīshēng | Læge / doktor | "Læge" for doctor; "doktor" for title |
| 律师 | lǜshī | Advokat | Professional title |
| 经理 | jīnglǐ | Chef / direktør | "Chef" for manager; "direktør" for director |
| 总裁 | zǒngcái | Administrerende direktør / CEO | "CEO" also used in Danish |
| 董事长 | dǒngshìzhǎng | Formand for bestyrelsen | Board chairman |

### 1.2 Formality Levels

| Chinese Context | Danish Adaptation | Formality Level |
|---|---|---|
| 尊敬的客户 (Dear valued customer) | Kære kunde | Semi-formal |
| 亲爱的用户 (Dear user) | Kære bruger | Semi-formal |
| 各位领导 (Dear leaders) | Kære ledere / Kære [Navn] | Formal |
| 致有关人士 (To whom it may concern) | Til vedkommende | Formal |
| 同志们 (Comrades) | Kollegaer / Damer og herrer | Formal (historical Chinese context) |

### 1.3 Recommendation for Chinese-to-Danish Translation

When translating Chinese business or formal content into Danish:

1. **Reduce formality by one level**: Chinese formal → Danish semi-formal
2. **Use "du" by default** unless the context is clearly legal/government
3. **Avoid overly stiff language** — Danish values directness and clarity
4. **Remove excessive politeness markers** that would sound unnatural in Danish
5. **Preserve professional titles** where they carry meaningful information

### 1.4 Chinese Honorifics and Politeness Markers

Chinese uses many politeness markers that have no direct Danish equivalent:

| Chinese | Function | Danish Adaptation |
|---|---|---|
| 请 (qǐng) | Polite request | "venligst" or omit (Danish is less formal) |
| 谢谢 (xièxie) | Thank you | "tak" |
| 麻烦您 (máfan nín) | Apologetic request | "Vær venlig at..." / "Kunne du..." |
| 拜托 (bàituō) | Please (earnest) | "Vær venlig" / "Jeg beder dig om" |
| 辛苦了 (xīnkǔ le) | Acknowledgement of effort | "Tak for indsatsen" / "Godt arbejde" |
| 多谢 (duōxiè) | Many thanks | "Mange tak" |
| 麻烦 (máfan) | Troublesome | "besværlig" / "træls" (informal) |

**Rule**: Chinese politeness is often excessive when translated literally into
Danish. Reduce to natural Danish levels — "tak" and "venlig" are sufficient
for most professional contexts.

## 2. Dates, Times, and Numbers

### 2.1 Date Format Conversion

Chinese dates follow the large-to-small order (年月日), while Danish uses
small-to-large (dag.måned.år). This requires complete restructuring.

| Chinese | Danish | Notes |
|---|---|---|
| 2026年5月27日 | 27. maj 2026 | Reorder: day first |
| 2026年5月 | maj 2026 | Month-year format |
| 5月27日 | 27. maj | Day-month format |
| 二〇二六年五月二十七日 | 27. maj 2026 | Written-out Chinese → numeric Danish |
| 农历 (lunar calendar) | Den kinesiske kalender | Clarify if lunar calendar used |

**Key conversion rule**: Reverse the order completely. Chinese年月日 → Danish dag.måned.år.

### 2.2 Time Format Conversion

Chinese uses a 12-hour system with 时/分/秒 markers, while Danish uses 24-hour format.

| Chinese | Danish | Notes |
|---|---|---|
| 上午9点 (shàngwǔ 9 diǎn) | 09:00 | Morning = AM |
| 下午3点30分 (xiàwǔ 3 diǎn 30 fēn) | 15:30 | Afternoon = +12 hours |
| 晚上8点 (wǎnshàng 8 diǎn) | 20:00 | Evening = PM + 12 |
| 中午12点 (zhōngwǔ 12 diǎn) | 12:00 | Noon |
| 半夜12点 (bànyè 12 diǎn) | 00:00 / 24:00 | Midnight |
| 几点? (jǐ diǎn?) | Hvad er klokken? | How to ask time |

**Conversion rule**: For PM times, add 12 to the hour. Remove 时/分/秒 markers
and use colon separator.

### 2.3 Number Format Conversion

Chinese uses the same decimal comma convention as Danish (both use comma for
decimal), but Chinese may use 万/亿 for large numbers, which need conversion.

| Chinese | Danish | Notes |
|---|---|---|
| 1,234.56 | 1.234,56 | Swap comma and period |
| 一千万 (10,000,000) | 10.000.000 | 万 = 10,000 |
| 一亿 (100,000,000) | 100.000.000 | 亿 = 100,000 |
| 百分之二十五 (25%) | 25 % | Danish format |
| 一月 (January) | januar | Lowercase in Danish |

**Chinese numeric system conversion:**

| Chinese Unit | Value | Danish Equivalent |
|---|---|---|
| 十 (shí) | 10 | ti |
| 百 (bǎi) | 100 | hundrede |
| 千 (qiān) | 1.000 | tusind |
| 万 (wàn) | 10.000 | ti tusind |
| 亿 (yì) | 100.000.000 | hundrede millioner |

**Rule**: When Chinese text uses 万/亿, convert to Danish numeric notation.
For example: 1.5万 → 15.000; 3.2亿 → 320.000.000.

### 2.4 Currency Conversion

| Chinese | Danish | Notes |
|---|---|---|
| ¥100 / 100元 | 100,00 kr. / 100,00 CNY | Convert to DKK or keep CNY |
| $100 | 100,00 $ | Same convention |
| €100 | 100,00 € | Same convention |
| 人民币 (RMB) | yuan / CNY | "Yuan" or "CNY" in Danish |

**Rule**: Chinese currency symbols precede the amount (¥100), while Danish
symbols follow (100 kr.). Always adjust position when translating.

## 3. Cultural References

### 3.1 Measurement Units

Chinese uses the metric system (SI units), same as Danish. No conversion needed
for most units. However, some Chinese-specific units need conversion:

| Chinese Unit | Danish Equivalent | Conversion |
|---|---|---|
| 里 (lǐ) | kilometer (km) | 1 里 ≈ 0,5 km |
| 斤 (jīn) | kilogram (kg) | 1 斤 = 0,5 kg |
| 两 (liǎng) | gram (g) | 1 两 = 50 g |
| 尺 (chǐ) | meter (m) | 1 尺 ≈ 0,333 m |
| 亩 (mǔ) | hektar (ha) | 1 亩 ≈ 0,0667 ha |

### 3.2 Chinese Holidays → Danish Context

|---|---|---|
| 春节 (Spring Festival) | Kinesisk nytår / forårsfest | Preserve cultural reference |
| 中秋节 (Mid-Autumn Festival) | Mefestival / Kinesisk høstfest | Preserve cultural reference |

| 清明节 (Qingming Festival) | Qingming-festival / Gravsoigningsdag | Preserve or translate |
| 七夕 (Qixi) | Kinesisk valentinsdag | Adapt to Danish concept |

### 3.3 Chinese Proverbs and Idioms

Chinese chengyu (成语) and proverbs require careful adaptation:

| Chinese | Literal | Danish Adaptation | Notes |
|---|---|---|---|
| 画蛇添足 | Draw legs on a snake | "At gøre noget overflødigt" | Danish equivalent expression |
| 对牛弹琴 | Play music to a cow | "At tale for døve øren" | Danish: "to talk to deaf ears" |
| 班门弄斧 | Show off axes before Lu Ban | "At sælge sko til en skomager" | Danish: "to sell shoes to a shoemaker" |
| 入乡随俗 | When entering a village, follow customs | "Når man er i Rom, gør man som romerne" | Danish: "When in Rome..." |
| 一石二鸟 | Kill two birds with one stone | "Slå to fluer med ét smæk" | Danish: "Kill two flies with one swat" |

**Rule**: Never translate Chinese idioms literally into Danish. Use equivalent
Danish expressions that convey the same meaning.

## 4. Terminology

### 4.1 Structural Differences: Chinese vs. Danish Grammar

| Feature | Chinese | Danish | Implication |
|---|---|---|---|
| Word order | SVO (same as Danish) | SVO with V2 rule | Main clause: verb must be second constituent |
| Measure words (量词) | Required (个, 本, 张, etc.) | Not used | Remove measure words in translation |
| Particles | 吗, 呢, 吧, 的 | N/A | Convert to intonation/punctuation |
| Aspect markers | 了, 过, 着 | Tense system | Convert to Danish verb tenses |
| Topic-prominence | Common | Subject-prominent | Restructure topic-comment to subject-predicate |
| Pro-drop | Common | Not allowed (usually) | Add explicit subject |

### 4.2 Measure Word Removal

Chinese requires measure words (量词) between numerals and nouns. Danish does
not use measure words — they must be removed entirely.

| Chinese | Incorrect Danish (literal) | Correct Danish |
|---|---|---|
| 一个人 (yí ge rén) | "én person" (correct — no measure word) | en person |
| 三本书 (sān běn shū) | "tre bog" | tre bøger |
| 两张纸 (liǎng zhāng zhǐ) | "to papir" | to stykker papir / to ark papir |
| 五条鱼 (wǔ tiáo yú) | " fem fisk" | fem fisk |
| 一匹马 (yì pǐ mǎ) | "én hest" | en hest |

**Rule**: Simply remove the measure word. Do NOT try to find a Danish equivalent
of the Chinese measure word — Danish does not have this grammatical category.

### 4.3 Particle Conversion

Chinese particles (吗, 呢, 吧, 的, 了, 过) have no direct Danish equivalents.
They must be converted to Danish grammar structures:

| Chinese Particle | Function | Danish Conversion |
|---|---|---|
| 吗 (ma) | Question marker | Invert word order or use intonation: "Du kommer?" |
| 呢 (ne) | Follow-up question | "Hvad med..." / "Og dig?" |
| 吧 (ba) | Suggestion/supposition | "Ikke sandt?" / "Vel?" |
| 的 (de) | Possessive/attributive | Danish suffix "-s" or adjective form |
| 了 (le) | Completed action | Past tense verb form |
| 过 (guò) | Experiential aspect | "har...før" / past tense |
| 着 (zhe) | Progressive aspect | Present tense or "er ved at" |
| 被 (bèi) | Passive marker | Danish passive voice with "blive" or "-s" |

### 4.4 Chinese Connector Words

| Chinese | Danish | Notes |
|---|---|---|
| 但是 (dànshì) | men / dog | "men" is standard; "dog" for contrast |
| 而且 (érqiě) | og / desuden | "og" for addition; "desuden" for moreover |
| 因为 (yīnwèi) | fordi / da | "fordi" for because; "da" for since |
| 所以 (suǒyǐ) | derfor / så | "derfor" for therefore; "så" for so |
| 如果 (rúguǒ) | hvis | Standard conditional |
| 虽然 (suīrán) | selvom / although | "selvom" is Danish |
| 例如 (lìrú) | for eksempel / f.eks. | "f.eks." is common abbreviation |
| 总之 (zǒngzhī) | kort sagt / overordnet | "kort sagt" for in short |
| 然而 (rán'ér) | imidlertid / dog | "imidlertid" is formal |
| 此外 (cǐwài) / 另外 (lìngwài) | desuden / yderligere | Both acceptable |
| 一般来说 (yìbān lái shuō) | generelt set / overordnet | For general statements |
| 事实上 (shìshí shàng) | faktisk / i virkeligheden | "faktisk" is standard |

Some Chinese-origin words are used in Danish (mostly food and culture):

| Chinese | Danish | Notes |
|---|---|---|

## 6. Punctuation

### 6.1 Punctuation Differences

| Element | Chinese | Danish | Conversion |
|---|---|---|---|
| Period | 。 | . | Replace with Western period |
| Comma | ， | , | Replace with Western comma |
| Question mark | ？ | ? | Replace with Western question mark |
| Exclamation | ！ | ! | Replace with Western exclamation |
| Semicolon | ； | ; | Replace with Western semicolon |
| Colon | ： | : | Replace with Western colon |
| Quotation marks | "" '' | "" '' | Replace with Western quotes |
| Parentheses | （） | () | Replace with Western parentheses |
| Ellipsis | …… | ... | Replace with Western ellipsis (three dots) |

**Rule**: Chinese uses full-width punctuation. Danish uses half-width Western
punctuation. ALWAYS convert punctuation width and style.

### 6.2 Chinese-Specific Punctuation Features

| Chinese Feature | Danish Adaptation | Notes |
|---|---|---|
| 顿号 (、) — list comma | Comma (,) | Chinese uses 顿号 for lists; Danish uses comma |
| 书名号 (《》) — book title marks | Italic or quotation marks | Danish uses italic for titles |
| 破折号 (——) — em dash | Em dash (—) | With spaces in Danish |
| 间隔号 (·) — middle dot | Hyphen (-) or nothing | Depends on context |
| 专名号 (___) — proper name underline | Not used | Remove; capitalisation suffices |

### 6.3 Chinese Sentence Structure Adaptation

Chinese topic-comment structure must be reorganised into Danish SVO structure:

| Chinese Pattern | Danish Adaptation |
|---|---|
| Topic + comment: "这本书，我已经读了。" | Subject-verb: "Jeg har allerede læst denne bog." |
| 被动句: "报告被提交了。" | Danish passive: "Rapporten blev indsendt." |
| 把字句: "请把门关上。" | Direct object: "Luk venligst døren." |
| 连动句: "他去商店买东西。" | Single verb phrase: "Han går i butikken for at købe ind." |

### 6.4 Formatting Conventions

| Element | Chinese | Danish |
|---|---|---|
| Title capitalisation | No capitalisation (Chinese has no case) | Sentence case (only first word capitalised) |
| Lists | Numerical or bullet | Same conventions as English |
| Paragraph spacing | Often none | Space between paragraphs |
| Numbers in text | 一, 二, 三 (sometimes) | en, to, tre (words for small numbers) |
| Phone numbers | Grouped by 3-4-4 | Grouped by 2-2-2-2: +45 12 34 56 78 |
| Addresses | Province → City → District → Street (large to small) | Street → Postal code + City (small to large) |

### 6.5 Common Chinese-to-Danish Translation Errors

| Error Type | Incorrect (literal) | Correct (adapted) | Explanation |
|---|---|---|---|
| Measure word | "én bog" from "一本书" | "en bog" | Remove measure word |
| Date order | "2026年5月27日" kept | "27. maj 2026" | Reverse to Danish order |
| Number format | "1,000.00" | "1.000,00" | Swap separators |
| Topic-comment | "This topic, we have discussed." | "Vi har diskuteret dette emne." | Convert to SVO |
| Particle | Literal 吗 → "?" | Invert verb: "Gør du det?" | Convert to Danish question form |
| Formality | Overly formal Chinese | Natural Danish register | Reduce formality level |
| Full-width punctuation | "，。；" | ", . ;" | Convert to half-width |
| Currency position | "¥100" | "100,00 kr." | Move symbol after amount |
