## 1. Reader Model (Модель читателя)

### Целевая аудитория

Техническая документация на русском языке ориентирована на следующие категории читателей:

| Категория | Уровень подготовки | Ожидания |
|-----------|-------------------|----------|
| Инженер-конструктор | Высшее техническое образование, знание ЕСКД | Точность терминологии, соблюдение ГОСТ, однозначность чертежей и спецификаций |
| Техник-механик / оператор | Среднее специальное образование | Понятные пошаговые инструкции, повелительное наклонение, минимум абстракции |
| Инженер по охране труда | Знание ССБТ (ГОСТ 12.х.ххх) | Чёткая классификация опасностей, ссылки на нормативные документы |
| Специалист по сертификации | Знание ТР ТС / ТР ЕАЭС | Точное соответствие терминологии Техническим регламентам |

### Факторы доверия

- **Соблюдение ГОСТ** — любое отклонение от стандартных терминов ЕСКД подрывает доверие специалиста
- **Единообразие единиц измерения** — обязательное применение системных единиц по ГОСТ 8.417
- **Правильная классификация опасностей** — неверный уровень предупреждения (ОСТОРОЖНО вместо ОПАСНО) недопустим
- **Точность ссылок** — каждая ссылка на ГОСТ/ТУ должна быть воспроизводима

### Что разрушает доверие

- Путаница в уровнях опасности (ОСТОРОЖНО / ВНИМАНИЕ / ОПАСНО)
- Использование внесистемных единиц без указания пересчёта
- Непоследовательное применение терминов (например, «задвижка» и «шибер» для одного устройства)
- Отсутствие ссылок на действующие нормативные документы

## 2. Decision Framework (Система принятия решений)

### D1: Единицы измерения и метрология

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит единицу длины (фут, дюйм) | Перевести в метрическую систему по ГОСТ 8.417; в скобках может быть указан исходный размер | `12 inches → 305 мм (12 дюймов)` |
| Исходный текст содержит единицу массы (фунт, унция) | Перевести в килограммы/граммы по ГОСТ 8.417; в скобках — исходное значение | `5 lbs → 2,27 кг (5 фунтов)` |
| Исходный текст содержит единицу давления (psi, bar) | Перевести в Паскали (кПа, МПа) по ГОСТ 8.417 | `150 psi → 1,03 МПа (150 psi)` |
| Исходный текст содержит единицу температуры (°F) | Перевести в °C; указать пересчёт | `212 °F → 100 °C` |
| Исходный текст содержит единицу скорости (mph, fps) | Перевести в м/с или км/ч по ГОСТ 8.417 | `60 mph → 96,6 км/ч` |
| Исходный текст содержит единицу объёма (галлон, баррель) | Перевести в литры или м³ по ГОСТ 8.417 | `1 gal (US) → 3,785 л` |
| Исходный текст содержит внесистемную единицу, допущенную к применению | Оставить с обязательным пересчётом в системную единицу | `1 кВт·ч (допущенная внесистемная единица)` |
| Обозначение единицы стоит после числового значения | Ставить пробел между числом и обозначением: `10 мм`, `5 кг`, `100 кПа` | `10mm → 10 мм` |
| Числовое значение с десятичной дробью | Использовать запятую как десятичный разделитель | `3.14 → 3,14` |
| Многоразрядное число (более 4 знаков) | Группировать цифры по три справа, разделяя неразрывным пробелом | `1000000 → 1 000 000` |

### D2: Уровни предупреждения об опасности (по ГОСТ 12.2.003-91)

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит Danger / DANGER — непосредственная угроза жизни | **ОПАСНО!** (прописные, восклицательный знак, красный цвет при возможности) | `DANGER: High voltage` → `ОПАСНО! Высокое напряжение` |
| Исходный текст содержит Warning / WARNING — потенциально опасная ситуация | **ВНИМАНИЕ!** (прописные, восклицательный знак) | `WARNING: Hot surface` → `ВНИМАНИЕ! Горячая поверхность` |
| Исходный текст содержит Caution / CAUTION — ситуация с возможным повреждением оборудования | **ОСТОРОЖНО!** (прописные, восклицательный знак) | `CAUTION: Electrostatic sensitive device` → `ОСТОРОЖНО! Чувствительное к электростатике устройство` |
| Исходный текст содержит Notice / NOTE — информационное примечание без опасности | **Примечание.** (без восклицательного знака, с точкой) | `NOTE: Use original cable` → `Примечание. Используйте оригинальный кабель` |
| Исходный текст содержит Important / IMPORTANT | **Важно!** (без восклицательного знака, с восклицательным) | `IMPORTANT: Back up data first` → `Важно! Сначала создайте резервную копию` |
| Многоуровневое предупреждение (Danger + consequence) | ОПАСНО! + указание на последствие + способ предотвращения | `DANGER: Risk of electric shock. Disconnect power before servicing.` → `ОПАСНО! Опасность поражения электрическим током. Перед обслуживанием отключите питание.` |

### D3: Оформление перекрёстных ссылок

| Условие | Решение | Пример |
|---------|---------|--------|
| Ссылка на раздел внутри документа | Использовать «см. раздел» или «в соответствии с разделом» | `See Section 3.1` → `см. раздел 3.1` |
| Ссылка на рисунок | Использовать «см. рисунок» (строчные), номер с точкой | `See Figure 5` → `см. рисунок 5` |
| Ссылка на таблицу | Использовать «см. таблицу» (строчные) | `See Table 2.1` → `см. таблицу 2.1` |
| Ссылка на формулу | Использовать «см. формулу (1)» в скобках | `See Equation (3)` → `см. формулу (3)` |
| Ссылка на приложение | Использовать «см. приложение» (строчные) | `See Appendix A` → `см. приложение А` |
| Ссылка на ГОСТ/стандарт | Полное обозначение с номером без сокращения | `per ISO 2768-1` → `в соответствии с ГОСТ 30893.1-2002 (ISO 2768-1:1989)` |
| Ссылка на раздел в другом документе | Полное обозначение документа + раздел | `Per IEC 60950-1, clause 2.5` → `в соответствии с IEC 60950-1, пункт 2.5` |

### D4: Формат кодов неисправностей (коды ошибок)

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный код ошибки содержит буквенно-цифровой код | Сохранить код без изменений; перевод описания | `Error 0xE4: Paper jam` → `Ошибка 0xE4: Замятие бумаги` |
| Исходный код представлен в формате E-NNN | Сохранить формат E-XXX (латиница), описание перевести | `E-001: Motor failure` → `E-001: Неисправность двигателя` |
| Код неисправности — комбинация символов | Код оставить латиницей; перевести только пояснительный текст | `Fault code F0123` → `Код неисправности F0123` |
| LED-индикация ошибки (мигание, цвет) | Описать режим работы индикатора русскими терминами | `LED blinks red 3 times` → `Светодиод мигает красным 3 раза` |

### D5: Подписи к иллюстрациям (рисунки и схемы)

| Условие | Решение | Пример |
|---------|---------|--------|
| Подпись к рисунку | Формат: «Рисунок N — Название» (тире, без знака №) | `Figure 1 — System architecture` → `Рисунок 1 — Архитектура системы` |
| Подпись к чертежу | Полное наименование по ЕСКД, прописные буквы в начале | `Drawing 12.345.678` → `Чертёж 12.345.678` |
| Позиционные обозначения на схеме (поз. 1, 2, 3...) | Сохранить нумерацию поз.; термины элементов перевести | `Item 5: Bearing` → `Поз. 5: Подшипник` |
| Текст внутри схемы/блок-схемы | Перевести, сохраняя краткость (1-3 слова, без артиклей) | `Start / Input / Process / End` → `Начало / Ввод / Обработка / Конец` |
| Подписи осей на графиках | Полный перевод с сохранением единиц измерения | `Time (ms) → Время (мс); Amplitude (V) → Амплитуда (В)` |
| Легенда / условные обозначения | «Условные обозначения:» с последующим перечнем | `Legend: 1 — Valve, 2 — Pump` → `Условные обозначения: 1 — Вентиль, 2 — Насос` |

### D6: Заголовки таблиц

| Условие | Решение | Пример |
|---------|---------|--------|
| Заголовок таблицы | Формат: «Таблица N — Название» (без знака №, тире) | `Table 5 — Specifications` → `Таблица 5 — Технические характеристики` |
| Продолжение таблицы на новой странице | «Продолжение таблицы N» с повтором шапки | (над продолжением) → `Продолжение таблицы 5` |
| Заголовки столбцов в таблице | Перевод, именительный падеж, прописная буква; единицы — в скобках | `Voltage (V)` → `Напряжение (В)` |
| Ячейка таблицы с текстом на нескольких языках | Каждый язык оформляется своим шрифтом; перевод — на русском | `Type / Тип` → только `Тип` для русской версии |
| Числовые данные в таблице | Десятичная запятая; выравнивание по правому краю | `3.14 → 3,14` |
| Единица измерения общая для всех данных | Вынести в заголовок столбца или строки, не повторять в ячейках | `Длина, мм` над столбцом |

### D7: Таблицы технических параметров

| Условие | Решение | Пример |
|---------|---------|--------|
| Технические характеристики (Specifications) | Термин: «Технические характеристики» | `Specifications` → `Технические характеристики` |
| Параметр и его значение | Двоеточие после параметра или два столбца: Параметр / Значение | `Voltage: 220V` → `Напряжение: 220 В` |
| Диапазон значений | Тире между числами; предлог «от... до...» для текстового описания | `10-20 mm` → `10–20 мм` или `от 10 до 20 мм` |
| Допуски и предельные отклонения | ±, формат по ГОСТ 25346 | `±0.1 mm` → `±0,1 мм` |
| Условия эксплуатации | «Условия эксплуатации» — диапазоны с «от... до...» | `Operating temp: -20~60°C` → `Рабочая температура: от –20 до 60 °C` |
| Степень защиты IP | Оставить латиницей IPXX с переводом описания | `IP67 — Dust-tight, waterproof` → `IP67 — Пыленепроницаемое, водозащищённое` |

### D8: Пошаговые инструкции / операции

| Условие | Решение | Пример |
|---------|---------|--------|
| Инструкция в повелительном наклонении (англ. imperative) | Сохранить повелительное наклонение, форма «Вы» опускается | `Press the button` → `Нажмите кнопку` |
| Последовательность шагов | Нумерованный список; каждый шаг — глагол в повелительном наклонении | `1. Turn off power. 2. Open cover.` → `1. Отключите питание. 2. Откройте крышку.` |
| Условное действие (if... then...) | «Если..., то...» с глаголом в изъявительном наклонении | `If alarm sounds, stop the machine` → `Если прозвучит сигнал тревоги, остановите станок` |
| Предостережение внутри шага | Двоеточие или отдельное предупреждение перед шагом | `Disconnect power before servicing` → `Перед обслуживанием отключите питание` |
| Отрицательная инструкция (do not...) | «Не» + глагол в повелительном наклонении | `Do not touch the rotating parts` → `Не прикасайтесь к вращающимся частям` |
| Примечание к шагу | Курсив или скобки; «Примечание:» с новой строки | `Note: Repeat if necessary.` → `Примечание. При необходимости повторите.` |

### D9: Математические формулы и обозначения

| Условие | Решение | Пример |
|---------|---------|--------|
| Переменные в формулах | Оставить латиницу; перевести пояснительный текст | `where: F — force, m — mass, a — acceleration` → `где: F — сила, m — масса, a — ускорение` |
| Название формулы | «Формула N — Название» | `Formula 1 — Ohm's law` → `Формула 1 — Закон Ома` |
| Пояснения к формуле | «где:» (строчные, с новой строки, двоеточие) | `where:` → `где:` |
| Индексы в формулах | Расшифровать на русском; индекс оставить латиницей | `max → макс, min → мин` или оставить `T_max` с русским пояснением |

### D10: Терминология и сокращения

| Условие | Решение | Пример |
|---------|---------|--------|
| Английская аббревиатура, имеющая русский аналог по ГОСТ | Использовать русскую аббревиатуру | `CAD (Computer-Aided Design)` → `САПР (система автоматизированного проектирования)` |
| Английская аббревиатура без русского аналога | Сохранить латиницу; расшифровать на русском при первом упоминании | `PLC (Programmable Logic Controller)` → `ПЛК (программируемый логический контроллер)` |
| Технический термин с несколькими русскими соответствиями | Выбрать по ГОСТ/отраслевому стандарту; составить глоссарий | `valve` → `клапан` (не «вентиль» для общего случая) |
| Патентованные названия / торговые марки | Оставить без перевода, латиницей, с заглавной буквы | `Teflon` → `Teflon (тефлон)` — допускается устоявшаяся транслитерация |
| Единица измерения, названная в честь учёного | Прописная буква в обозначении, строчная — в названии | `Pascal → Па (паскаль), Watt → Вт (ватт)` |

### Правила оформления ссылок на стандарты в тексте

- При первом упоминании — полное обозначение: `ГОСТ 2.105-95 «Единая система конструкторской документации. Общие требования к текстовым документам»`

- Для международных стандартов, принятых в РФ: `ГОСТ Р ИСО 9001-2015 (ISO 9001:2015)`
- Для ссылок на пункты стандарта: `ГОСТ 12.2.003-91, пункт 4.5`

## 4. Language-Specific Conventions (Языковые особенности технического перевода)

### Использование падежей

| Конструкция | Падеж | Пример |
|-------------|-------|--------|
| В соответствии с + noun | Творительный падеж (instrumental) | `Согласно ГОСТ 2.105` (дат. — `согласно ГОСТу` — допустимо, но `в соответствии с ГОСТом` — тв.) |
| Состоять из + noun | Родительный падеж | `consists of 3 parts` → `состоит из 3 частей` |
| Управляться + noun | Творительный падеж | `controlled by PLC` → `управляется ПЛК` |
| Предназначен для + noun | Родительный падеж | `designed for cutting` → `предназначен для резки` |

### Использование предлогов

| Английская конструкция | Русский эквивалент | Пример |
|-----------------------|-------------------|--------|
| by (agency) | с помощью + род. п. / посредством + род. п. | `Driven by motor` → `Приводится с помощью электродвигателя` |
| for (purpose) | для + род. п. | `for calibration` → `для калибровки` |
| with (feature) | с + тв. п. | `with automatic control` → `с автоматическим управлением` |
| without | без + род. п. | `without load` → `без нагрузки` |

### Числовой формат

- Десятичный разделитель: **запятая** (`,`) — `3,14` не `3.14`
- Разделитель разрядов: **неразрывный пробел** — `1 234 567` не `1,234,567`
- Знак минуса: тире или символ минуса `−` (U+2212) — `−10 °C`
- Диапазон: тире — `10–20 мм`
- Дробные числа: `1/2 → ½` (при возможности), `1,5`

### Фирменные наименования и логотипы

- Товарные знаки и названия брендов не переводятся: `Microsoft Windows`, `Siemens S7-1200`
- Типы и модели сохраняются латиницей: `Type A-100`, `Model XT-2000`
- Продуктовые термины при наличии локализации: `AutoCAD → AutoCAD` (название не переводится)
