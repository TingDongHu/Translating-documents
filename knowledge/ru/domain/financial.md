## 1. Reader Model (Модель читателя)

### Целевая аудитория

Финансовые документы на русском языке ориентированы на следующие категории читателей:

| Категория | Уровень подготовки | Ожидания |
|-----------|-------------------|----------|
| Главный бухгалтер | Высшее бухгалтерское / экономическое образование, знание ФСБУ, МСФО | Точное соответствие плану счетов, корректная корреспонденция, правильное отражение операций |
| Финансовый директор (CFO) | Высшее финансовое образование, MBA | Понятность отчётности, соответствие стандартам, прозрачность статей |
| Аудитор | Квалификационный аттестат аудитора (Минфин РФ) | Соблюдение ФСБУ / МСФО, надлежащая документация, обоснованность оценок |
| Инвестор / аналитик | Знание рынка, финансовой отчётности | Сопоставимость данных, чёткое выделение курсовых разниц, инфляционные корректировки |
| Налоговый инспектор | Высшее юридическое / экономическое | Полнота отражения доходов/расходов, правильность расчёта налогов, соответствие НК РФ |
| Финансовый контролёр | Профессиональный опыт | Сверка сальдо, контрольные суммы, аудиторские следы |

### Факторы доверия

- **Корректная валюта** — правильный код валюты (RUB, USD, EUR), соответствие ОКВ
- **Правильная цифровая группировка** — 1 234 567,89 руб. (не 1,234,567.89)
- **Соблюдение ФСБУ** — русские стандарты бухгалтерского учёта имеют существенные отличия от МСФО
- **Чёткое разделение валютных и рублёвых сумм** — обязательное выделение курсовых разниц

### Что разрушает доверие

- Смешение дебета и кредита (актив/пассив)
- Неправильное выделение НДС / налогов
- Игнорирование плана счетов РФ (счета 01–99)
- Путаница между кассовым методом и методом начисления

## 2. Decision Framework (Система принятия решений)

### D1: Формат денежных сумм и чисел

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит сумму в долларах США | **доллар США / долларов США**, код USD, две цифры после запятой | `USD 1,500.00` → `1 500,00 долларов США` |
| Исходный текст содержит сумму в евро | **евро** (несклоняемое), код EUR | `EUR 10,000.00` → `10 000,00 евро` |
| Исходный текст содержит рублёвую сумму | **рубль / рублей**, код RUB (в отчётности — в рублях без копеек для целых чисел) | `RUB 1,200.50` → `1 200,50 руб.` |
| Десятичные числа в финансовых документах | Десятичная запятая; две цифры после запятой для денежных сумм | `3.14 → 3,14`; для денег: `1,250.00 → 1 250,00` |
| Разряды чисел (группировка) | Неразрывный пробел каждые 3 разряда | `1000000 → 1 000 000` |
| Отрицательные суммы | Круглые скобки в отчётности; знак минуса в тексте | `(5 000 руб.)` или `–5 000 руб.` |
| Проценты в финансовых текстах | Цифра + пробел + % (неразрывный пробел) | `10.5% → 10,5 %` |
| Крупные миллионные/миллиардные суммы | В отчётности — цифрами; в тексте допускается сокращение млн / млрд | `5,000,000 → 5 000 000` или `5 млн руб.` |

### D2: Ключевые бухгалтерские термины

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "assets" | **активы** | `Total assets: USD 1,000,000` → `Активы, всего: 1 000 000 долларов США` |
| Исходный текст содержит "liabilities" | **обязательства** (пассивы) | `Current liabilities` → `Краткосрочные обязательства` |
| Исходный текст содержит "equity" | **капитал / собственный капитал** | `Shareholders' equity` → `Капитал акционеров` |
| Исходный текст содержит "revenue / income" | **выручка / доход** | `Revenue from operations` → `Выручка от основной деятельности` |
| Исходный текст содержит "profit / loss" | **прибыль / убыток** | `Net profit for the period` → `Чистая прибыль за период` |
| Исходный текст содержит "accounts receivable" | **дебиторская задолженность** | `Trade accounts receivable` → `Дебиторская задолженность покупателей` |
| Исходный текст содержит "accounts payable" | **кредиторская задолженность** | `Accounts payable to suppliers` → `Кредиторская задолженность перед поставщиками` |
| Исходный текст содержит "depreciation" | **амортизация** | `Depreciation of fixed assets` → `Амортизация основных средств` |
| Исходный текст содержит "accrual" | **начисление (метод начисления)** | `Accrual basis accounting` → `Учёт по методу начисления` |
| Исходный текст содержит "goodwill" | **деловая репутация (гудвилл)** | `Goodwill arising on acquisition` → `Деловая репутация, возникшая при приобретении` |
| Исходный текст содержит "intangible assets" | **нематериальные активы (НМА)** | `Intangible assets recognition` → `Признание нематериальных активов` |
| Исходный текст содержит "provision / reserve" | **резерв / оценочное обязательство** | `Provision for doubtful debts` → `Резерв по сомнительным долгам` |

### D3: Формы финансовой отчётности

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "Balance Sheet" | **Бухгалтерский баланс** (форма № 1 по ОКУД 0710001) | `Balance Sheet as at 31.12.2024` → `Бухгалтерский баланс на 31 декабря 2024 г.` |
| Исходный текст содержит "Income Statement / P&L" | **Отчёт о финансовых результатах** (форма № 2, ОКУД 0710002) | `Statement of Profit or Loss for the year ended 31.12.2024` → `Отчёт о финансовых результатах за 2024 год` |
| Исходный текст содержит "Statement of Cash Flows" | **Отчёт о движении денежных средств** (форма № 4, ОКУД 0710004) | `Cash Flow Statement` → `Отчёт о движении денежных средств` |
| Исходный текст содержит "Statement of Changes in Equity" | **Отчёт об изменениях капитала** (форма № 3, ОКУД 0710003) | `Statement of Changes in Shareholders' Equity` → `Отчёт об изменениях капитала` |
| Исходный текст содержит "Notes to Financial Statements" | **Пояснения к бухгалтерскому балансу и отчёту о финансовых результатах** | `Notes to the Financial Statements` → `Пояснения к бухгалтерской (финансовой) отчётности` |
| Исходный текст содержит "Auditor's Report" | **Аудиторское заключение** | `Independent Auditor's Report` → `Аудиторское заключение независимого аудитора` |

### D4: Статьи бухгалтерского баланса (актив)

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "current assets" | **оборотные активы** | `Current assets` → `II. Оборотные активы` |
| Исходный текст содержит "non-current assets" | **внеоборотные активы** | `Non-current assets` → `I. Внеоборотные активы` |
| Исходный текст содержит "fixed assets" | **основные средства** | `Property, plant and equipment` → `Основные средства` |
| Исходный текст содержит "inventory / stock" | **запасы** | `Inventories` → `Запасы` |
| Исходный текст содержит "cash and cash equivalents" | **денежные средства и их эквиваленты** | `Cash and cash equivalents` → `Денежные средства и денежные эквиваленты` |
| Исходный текст содержит "prepaid expenses" | **расходы будущих периодов** | `Prepayments` → `Расходы будущих периодов` |

### D5: Статьи пассива и отчёта о прибылях и убытках

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "long-term debt" | **долгосрочные обязательства** | `Long-term borrowings` → `Долгосрочные заёмные средства` |
| Исходный текст содержит "short-term debt" | **краткосрочные обязательства** | `Short-term borrowings` → `Краткосрочные заёмные средства` |
| Исходный текст содержит "retained earnings" | **нераспределённая прибыль** | `Retained earnings (accumulated deficit)` → `Нераспределённая прибыль (непокрытый убыток)` |
| Исходный текст содержит "cost of goods sold (COGS)" | **себестоимость продаж** | `Cost of goods sold` → `Себестоимость продаж` |
| Исходный текст содержит "gross profit" | **валовая прибыль (убыток)** | `Gross profit margin 35%` → `Валовая рентабельность 35%` |
| Исходный текст содержит "operating expenses" | **коммерческие и управленческие расходы** | `Selling, general and administrative expenses` → `Коммерческие и управленческие расходы` |
| Исходный текст содержит "EBITDA" | **EBITDA** (оставить латиницей с пояснением) | `EBITDA for the period` → `Показатель EBITDA за период` |
| Исходный текст содержит "earnings per share (EPS)" | **прибыль на акцию** | `Basic EPS: 2.50` → `Базовая прибыль на акцию: 2,50` |

### D6: Курсы валют и курсовые разницы

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "exchange rate" | **обменный курс / курс валюты** | `The exchange rate of EUR/RUB as at 31.12.2024` → `Курс евро к рублю по состоянию на 31.12.2024` |
| Исходный текст содержит "foreign exchange gain/loss" | **курсовая разница** (положительная / отрицательная) | `Foreign exchange loss: USD 5,000` → `Отрицательная курсовая разница: 5 000 долларов США` |
| Исходный текст содержит "translation of foreign operations" | **пересчёт показателей зарубежных операций** | `Translation differences from foreign operations` → `Курсовые разницы от пересчёта зарубежных операций` |
| Исходный текст содержит "functional currency" | **функциональная валюта** | `Functional currency: RUB` → `Функциональная валюта: российский рубль` |
| Исходный текст содержит "presentation currency" | **валюта представления отчётности** | `The financial statements are presented in USD` → `Финансовая отчётность представлена в долларах США` |
| Исходный текст содержит "spot rate" | **текущий курс (спот-курс)** | `Spot exchange rate at transaction date` → `Текущий курс на дату совершения операции` |

### D7: Налоговые термины

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "corporate income tax" | **налог на прибыль организаций** | `Corporate income tax rate: 20%` → `Ставка налога на прибыль организаций: 20%` |
| Исходный текст содержит "value added tax (VAT)" | **налог на добавленную стоимость (НДС)** | `VAT rate: 20% / 10% / 0%` → `Ставка НДС: 20% / 10% / 0%` |
| Исходный текст содержит "withholding tax" | **налог у источника (НДФЛ для физлиц, налог у источника для юрлиц)** | `Withholding tax on dividends: 15%` → `Налог у источника на дивиденды: 15%` |
| Исходный текст содержит "deferred tax asset / liability" | **отложенный налоговый актив (ОНА) / отложенное налоговое обязательство (ОНО)** | `Deferred tax asset recognised` → `Признан отложенный налоговый актив` |
| Исходный текст содержит "tax base / tax basis" | **налоговая база** | `Tax base of the asset` → `Налоговая база актива` |
| Исходный текст содержит "permanent difference" | **постоянная разница** | `Permanent differences for tax purposes` → `Постоянные разницы для целей налогообложения` |
| Исходный текст содержит "temporary difference" | **временная разница** | `Temporary differences giving rise to deferred tax` → `Временные разницы, приводящие к возникновению отложенного налога` |

### D8: Финансовые коэффициенты и показатели

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "liquidity ratio" | **коэффициент ликвидности** | `Current ratio (liquidity)` → `Коэффициент текущей ликвидности` |
| Исходный текст содержит "profitability ratio" | **коэффициент рентабельности** | `Return on equity (ROE)` → `Рентабельность собственного капитала (ROE)` |
| Исходный текст содержит "debt ratio / leverage" | **коэффициент финансового рычага / долговая нагрузка** | `Debt-to-equity ratio` → `Отношение заёмного капитала к собственному` |
| Исходный текст содержит "turnover ratio" | **коэффициент оборачиваемости** | `Inventory turnover ratio` → `Коэффициент оборачиваемости запасов` |
| Исходный текст содержит "working capital" | **оборотный капитал** | `Net working capital` → `Чистый оборотный капитал` |
| Исходный текст содержит "dividend" | **дивиденд** | `Dividend per share` → `Дивиденд на акцию` |
| Исходный текст содержит "interest coverage" | **коэффициент покрытия процентов** | `Interest coverage ratio (ICR)` → `Коэффициент покрытия процентов` |
| Исходный текст содержит "amortization" (intangible) | **амортизация нематериальных активов** | `Amortization of intangible assets` → `Амортизация НМА` |

## 3. Standards (Стандарты и нормативные документы)

### Российские стандарты бухгалтерского учёта (ФСБУ)

| Стандарт | Название | Примечание |
|----------|----------|------------|
| **ФСБУ 5/2019** | Запасы | Учёт товаров, материалов, готовой продукции |
| **ФСБУ 6/2020** | Основные средства | Учёт внеоборотных активов, амортизация |
| **ФСБУ 14/2022** | Нематериальные активы | Учёт НМА, деловая репутация |
| **ФСБУ 25/2018** | Аренда | Учёт арендованных активов |
| **ФСБУ 26/2020** | Капитальные вложения | Учёт затрат на приобретение/создание ОС |
| **ФСБУ 27/2021** | Документы и документооборот | Первичные учётные документы |
| **ПБУ 18/02** (до замены на ФСБУ 18/2022) | Расчёты по налогу на прибыль | Учёт ОНА/ОНО |
| **ПБУ 3/2006** (до замены на ФСБУ 3/2023) | Активы и обязательства в иностранной валюте | Курсовые разницы, пересчёт валют |

| Стандарт | Русское название | Описание |
|----------|-----------------|----------|
| **IFRS 1** | МСФО (IFRS) 1 | Первое применение МСФО |
| **IFRS 9** | МСФО (IFRS) 9 | Финансовые инструменты |
| **IFRS 15** | МСФО (IFRS) 15 | Выручка по договорам с покупателями |
| **IFRS 16** | МСФО (IFRS) 16 | Аренда |
| **IAS 1** | МСФО (IAS) 1 | Представление финансовой отчётности |
| **IAS 7** | МСФО (IAS) 7 | Отчёт о движении денежных средств |
| **IAS 12** | МСФО (IAS) 12 | Налоги на прибыль |
| **IAS 16** | МСФО (IAS) 16 | Основные средства |

### План счетов бухгалтерского учёта РФ

Типовой план счетов, утверждённый Приказом Минфина РФ от 31.10.2000 № 94н.

| Раздел | Счета | Название |
|--------|-------|----------|
| I | 01–09 | Внеоборотные активы |
| II | 10–19 | Производственные запасы |
| III | 20–29 | Затраты на производство |
| IV | 40–46 | Готовая продукция и товары |
| V | 50–59 | Денежные средства |
| VI | 60–69 | Расчёты |
| VII | 70–79 | Оплата труда |
| VIII | 80–89 | Капитал |
| IX | 90–99 | Финансовые результаты |

| Код ОКУД | Форма | Назначение |
|----------|-------|------------|
| 0710001 | Бухгалтерский баланс | Основная форма отчётности |
| 0710002 | Отчёт о финансовых результатах | Прибыли и убытки |
| 0710003 | Отчёт об изменениях капитала | Движение капитала |
| 0710004 | Отчёт о движении денежных средств | Денежные потоки |
| 0710005 | Отчёт о целевом использовании средств | Для некоммерческих организаций |

### Правила оформления дат в финансовых документах

| Контекст | Формат | Пример |
|----------|--------|--------|
| Заголовок отчётности | «за [год] год» / «на [дата]» | `за 2024 год` или `на 31 декабря 2024 г.` |
| Период | «за [период] [год] года» | `за 9 месяцев 2024 года` |
| Дата в таблице | ДД.ММ.ГГГГ | `31.12.2024` |
| Дата прописью в тексте | [число] [месяц] [год] года | `31 декабря 2024 года` |

### Особенности перевода МСФО → ФСБУ / РПБУ

| МСФО термин | Русский (МСФО-отчётность) | Русский (РПБУ) |
|-------------|--------------------------|-----------------|
| Revenue | Выручка | Выручка |
| Finance costs | Финансовые расходы | Проценты к уплате |
| Property, plant and equipment | Основные средства | Основные средства |
| Investment property | Инвестиционная недвижимость | Доходные вложения в материальные ценности |
| Current tax | Текущий налог | Текущий налог на прибыль |
| Other comprehensive income | Прочий совокупный доход | (отсутствует в РПБУ) |
