## 1. Reader Model (Модель читателя)

### Целевая аудитория

Административные документы на русском языке ориентированы на следующие категории читателей:

| Категория | Уровень подготовки | Ожидания |
|-----------|-------------------|----------|
| Руководитель организации / директор | Высшее образование, опыт управления | Лаконичность, чёткость, ясность резолюции, правовые последствия |
| Секретарь / делопроизводитель | Специализированное образование (делопроизводство) | Соблюдение ГОСТ Р 7.0.97-2016, правильность реквизитов, нумерация |
| Сотрудник / исполнитель | Среднее/высшее образование | Понятный порядок действий, сроки исполнения, ответственные лица |
| Государственный служащий | Знание регламентов | Строгая форма, соответствие административному регламенту |
| Контролёр / надзорный орган | Юридическое / специальное образование | Полнота реквизитов, соответствие закону, идентификация подписей |

### Факторы доверия

- **Полнота реквизитов** — каждый документ должен иметь регистрационный номер, дату, подпись
- **Соблюдение официально-делового стиля** — клишированные конструкции, отсутствие эмоциональной окраски
- **Единообразие форм** — шаблонная структура, унифицированные бланки
- **Правильный порядок оформления** — реквизиты в строгой последовательности по ГОСТ Р 7.0.97-2016

### Что разрушает доверие

- Неправильное оформление реквизитов (отсутствие даты, номера, подписи)
- Разговорный / неформальный стиль в официальной переписке
- Нарушение иерархии адресации (генеральному директору → от рядового сотрудника напрямую)
- Отсутствие резолюции, виз согласования на внутренних документах

## 2. Decision Framework (Система принятия решений)

### D1: Структура официального письма (реквизиты)

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит адресата (кому) | **Кому:** / **Генеральному директору...** в дательном падеже | `To: Mr. Ivanov` → `Генеральному директору ООО «Пример» И.И. Иванову` |
| Исходный текст содержит отправителя (от кого) | **От:** в родительном падеже с указанием должности | `From: John Smith, Sales Manager` → `От: Менеджера по продажам П.П. Петрова` |
| Исходный текст содержит тему письма | **О:** / **Касательно:** / **Re:** → «О ...» (предложный падеж) | `Re: Delivery schedule` → `О графике поставок` |
| Исходный текст содержит дату письма | **Дата:** ДД.ММ.ГГГГ или «[число] [месяц] [год] г.» | `Date: 15.05.2024` → `15 мая 2024 г.` |
| Исходный текст содержит исходящий номер | **Исх. №** + номер + дата | `Ref. No. 123/45` → `Исх. № 123/45 от 15.05.2024` |
| Исходный текст содержит входящий номер | **Вх. №** + номер + дата | `Ref. No. 678/90` → `Вх. № 678/90 от 16.05.2024` |
| Исходный текст содержит приложение | **Приложение:** перечень на русском языке, пронумерованный | `Enclosures: 1. Copy of contract` → `Приложение: 1. Копия договора на 5 л. в 1 экз.` |
| Исходный текст содержит ссылку на предыдущую переписку | «В ответ на Ваше письмо от [дата] № [номер]» | `Further to your letter dated...` → `В ответ на Ваше письмо от 10.05.2024 № 45/67` |

### D2: Официально-деловые формулировки (вступительные обороты)

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "We hereby inform you" | **Настоящим уведомляем Вас / Сообщаем** | `We hereby inform you that...` → `Настоящим уведомляем Вас о том, что...` |
| Исходный текст содержит "We would like to bring to your attention" | **Обращаем Ваше внимание на то, что...** | `We would like to draw your attention to...` → `Обращаем Ваше внимание на...` |
| Исходный текст содержит "In response to your request" | **В ответ на Ваш запрос / обращение** | `In response to your request dated...` → `В ответ на Ваше обращение от...` |
| Исходный текст содержит "Please be advised" | **Ставим Вас в известность / Сообщаем** | `Please be advised that...` → `Сообщаем, что...` |
| Исходный текст содержит "We enclose / attached" | **Направляем Вам / Прилагаем к настоящему письму** | `We enclose the following documents` → `Прилагаем к настоящему письму следующие документы` |
| Исходный текст содержит "We acknowledge receipt" | **Подтверждаем получение** | `We acknowledge receipt of your letter of...` → `Подтверждаем получение Вашего письма от...` |

### D3: Заключительные формулы вежливости

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "Sincerely / Best regards" | **С уважением,** (официально-деловое письмо) | `Sincerely, John Smith` → `С уважением, П.П. Петров` |
| Исходный текст содержит "Yours faithfully" | **С уважением,** (если адресат неизвестен — также «С уважением») | `Yours faithfully, The Management` → `С уважением, Администрация` |
| Исходный текст содержит "Respectfully" | **С глубоким уважением,** / **С почтением,** | `Respectfully yours` → `С глубоким уважением,` |
| Исходный текст содержит "Thank you for your cooperation" | **Благодарим за сотрудничество** | `Thank you for your cooperation in this matter` → `Благодарим за содействие в данном вопросе` |
| Исходный текст содержит "Looking forward to your reply" | **С нетерпением ожидаем Вашего ответа** | `We look forward to your prompt response` → `Ожидаем Вашего скорейшего ответа` |
| Исходный текст содержит "Do not hesitate to contact us" | **При возникновении вопросов просим обращаться** | `Please do not hesitate to contact us` → `По всем вопросам просим обращаться` |

### D4: Внутренние документы — приказы и распоряжения

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "Order No." (приказ) | **Приказ №** | `Order No. 45-k` → `Приказ № 45-к` |
| Исходный текст содержит "Directive" | **Распоряжение №** | `Directive No. 12` → `Распоряжение № 12` |
| Исходный текст содержит "I hereby order" | **ПРИКАЗЫВАЮ:** (прописные, с двоеточием) | `I order that...` → `ПРИКАЗЫВАЮ:` |
| Исходный текст содержит "In order to improve / ensure" | **В целях:** + родительный падеж | `In order to improve efficiency` → `В целях повышения эффективности работы` |
| Исходный текст содержит "pursuant to / in accordance with" | **В соответствии с / На основании** + творительный падеж | `In accordance with the Labour Code` → `В соответствии с Трудовым кодексом РФ` |
| Исходный текст содержит "I propose / recommend" | **ПРЕДЛАГАЮ:** (прописные, с двоеточием) / **РЕКОМЕНДУЮ:** | `I propose to establish a working group` → `ПРЕДЛАГАЮ: создать рабочую группу...` |
| Исходный текст содержит "control over implementation" | **Контроль за исполнением возлагаю на** / **Контроль оставляю за собой** | `Control is assigned to the Deputy Director` → `Контроль за исполнением возлагаю на заместителя директора` |

### D5: Протоколы собраний / совещаний

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "Minutes of Meeting" | **Протокол №** / **Протокол собрания** | `Minutes of the General Meeting No. 1` → `Протокол № 1 общего собрания` |
| Исходный текст содержит "Present / Attendees" | **Присутствовали:** (именительный падеж, список) | `Present: Ivanov I.I., Petrov P.P.` → `Присутствовали: Иванов И.И., Петров П.П.` |
| Исходный текст содержит "Agenda" | **Повестка дня:** | `Agenda: 1. Approval of report` → `Повестка дня: 1. Утверждение отчёта` |
| Исходный текст содержит "Chairman / Secretary" | **Председатель:** / **Секретарь:** | `Chairman: I.I. Ivanov` → `Председатель: Иванов И.И.` |
| Исходный текст содержит "Heard / Reported" | **СЛУШАЛИ:** (прописные, двоеточие) | `Heard: report of the CEO` → `СЛУШАЛИ: доклад генерального директора` |
| Исходный текст содержит "Resolved / Decision" | **РЕШИЛИ:** (прописные, двоеточие) — для коллегиального решения | `Resolved: To approve...` → `РЕШИЛИ: Утвердить...` |
| Исходный текст содержит "Voting results" | **Голосовали:** — результаты: «За — 10, Против — 0, Воздержались — 2» | `Votes in favor: 10, against: 0, abstentions: 2` → `Голосовали: За — 10, Против — 0, Воздержались — 2` |
| Исходный текст содержит "Action items" | **Поручить:** / **Ответственный:** / **Срок:** | `Action: Petrov to prepare report by 01.06` → `Поручить: Петрову П.П. подготовить отчёт. Срок — до 01.06.2024` |

### D6: Служебные / докладные записки

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "Memorandum" | **Служебная записка** (внутренняя) / **Докладная записка** (вышестоящему) | `Memorandum to the CEO` → `Докладная записка генеральному директору` |
| Исходный текст содержит "Concerning / Regarding" | **Касательно:** / **О:** (предложный падеж) | `Concerning the server malfunction` → `Касательно неисправности сервера` |
| Исходный текст содержит "I would like to inform" | **Довожу до Вашего сведения / Сообщаю** | `I would like to inform you of a delay` → `Довожу до Вашего сведения о задержке` |
| Исходный текст содержит "I request your permission" | **Прошу Вашего разрешения / Прошу согласовать** | `I request your approval to purchase...` → `Прошу согласовать приобретение...` |
| Исходный текст содержит "For your approval / signature" | **На согласование** / **На подпись** | `Submitted for approval` → `Направляется на согласование` |
| Исходный текст содержит "I propose the following" | **В целях... предлагаю:** (со списком предложений) | `I propose the following corrective actions` → `В целях устранения неполадок предлагаю:` |

### D7: Оформление подписей, печатей, утверждения

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "Signature" | **Подпись** (расшифровка в скобках — инициалы + фамилия) | `Signature: _____________ /I.I. Ivanov/` → `Подпись: _____________ (И.И. Иванов)` |
| Исходный текст содержит "Seal / Stamp" | **М.П.** (место печати) или **Печать** | `(Seal)` → `М.П.` (для документов, требующих печати) |
| Исходный текст содержит "Approved by" | **УТВЕРЖДАЮ:** (прописные, с двоеточием) — начальник / директор | `Approved by: CEO` → `УТВЕРЖДАЮ: Генеральный директор _____________ (И.И. Иванов)` |
| Исходный текст содержит "Endorsed by / Agreed" | **СОГЛАСОВАНО:** (прописные, с двоеточием) | `Agreed: Chief Accountant` → `СОГЛАСОВАНО: Главный бухгалтер _____________ (О.А. Сидорова)` |
| Исходный текст содержит "Legalized by / Notarized" | **ЗАВЕРЕНО:** / **Удостоверяю:** (нотариус) | `Notarized: Notary Public` → `Свидетельствую подлинность подписи: Нотариус _____________ (М.И. Кузнецов)` |

### D8: Стилистические требования официально-делового стиля

| Условие | Решение | Пример |
|---------|---------|--------|
| Личное местоимение | Использовать «Мы» от имени организации, «Вы» как вежливое обращение, с прописной буквы | `We kindly ask you to...` → `Просим Вас...` |
| Глагольные конструкции | Широкое использование отглагольных существительных (номинализация) | `We confirm receipt` → `Подтверждаем получение` (не «Подтверждаем, что получили») |
| Сложные предлоги | Использовать официальные предлоги: «в соответствии», «в связи», «в целях», «по причине» | `Due to production delays` → `В связи с задержками производства` |
| Пассивные конструкции | Пассивный залог предпочтителен («нами рассмотрен», «документ подписан») | `The report was reviewed by the committee` → `Отчёт рассмотрен комиссией` |
| Клишированные сочетания | «в установленном порядке», «в кратчайшие сроки», «по согласованию сторон» | `as soon as possible` → `в кратчайшие сроки` |
| Стандартные формулировки | Не заменять клише на синонимы — это снижает формальность | `in due course` → `в установленном порядке` (не «своевременно» неформально) |
| Императивность | Повелительное наклонение в приказах; сослагательное — в просьбах | `I order...` → `ПРИКАЗЫВАЮ: ...` / `We ask...` → `Просим Вас...` |

## 3. Standards (Стандарты и нормативные документы)

### Нормативная база делопроизводства в РФ

| Стандарт / Документ | Описание | Область применения |
|---------------------|----------|-------------------|
| **ГОСТ Р 7.0.97-2016** | Система стандартов по информации, библиотечному и издательскому делу. Организационно-распорядительная документация. Требования к оформлению документов | Основной стандарт оформления реквизитов документов |
| **ГОСТ Р 7.0.8-2013** | Делопроизводство и архивное дело. Термины и определения | Терминология делопроизводства |
| **ГОСТ Р 6.30-2003** (заменён на 7.0.97-2016) | Унифицированная система организационно-распорядительной документации. Требования к оформлению документов | Предыдущий стандарт (может встречаться в старых переводах) |
| **ФЗ № 125-ФЗ** | Об архивном деле в Российской Федерации | Хранение и уничтожение документов |
| **ФЗ № 152-ФЗ** | О персональных данных | Обработка персональных данных в документах |
| **ФЗ № 63-ФЗ** | Об электронной подписи | Юридическая сила электронных документов |
| **ТК РФ** (главы, регламентирующие кадровое делопроизводство) | Трудовой кодекс | Приказы по личному составу, трудовые договоры |
| **Перечень типовых управленческих архивных документов** (Приказ Росархива) | Сроки хранения документов | Определение срока хранения |

### Типовые виды организационно-распорядительных документов (ОРД)

| Вид документа | Назначение | Типовая форма |
|---------------|------------|---------------|
| **Приказ** | Правовой акт руководителя, обязательный для исполнения | По основной деятельности / по личному составу |
| **Распоряжение** | Правовой акт по оперативным вопросам | Менее формальный, чем приказ |
| **Положение** | Нормативный акт, устанавливающий статус / порядок | Структура: Общие положения → Права → Ответственность |
| **Инструкция** | Описание порядка выполнения действий | Правила, регламенты |
| **Акт** | Подтверждение факта / события | Приём-передача, обследование, списание |
| **Протокол** | Фиксация хода собрания / заседания | СЛУШАЛИ — ВЫСТУПИЛИ — РЕШИЛИ |
| **Служебная записка** | Внутренняя коммуникация | Информационная / отчётная / инициативная |
| **Заявление** | Обращение сотрудника к руководству | Кадровое / материально-техническое |

### Правила оформления адресата

| Элемент | Формат | Пример |
|---------|--------|--------|
| Должность | Дательный падеж | `Генеральному директору` |
| Организация | Дательный падеж в середине | `ООО «Ромашка»` |
| ФИО | Дательный падеж, инициалы перед фамилией | `И.И. Иванову` |
| Полный адресат | Должность + Организация + ФИО | `Генеральному директору ООО «Ромашка» И.И. Иванову` |
| Несколько адресатов | Перечисление сверху вниз | `Руководителям структурных подразделений` |

### Правила оформления дат

| Элемент | Формат | Пример |
|---------|--------|--------|
| Дата в реквизитах | Цифровой: ДД.ММ.ГГГГ | `15.05.2024` |
| Дата в тексте письма | Словесно-цифровой: число + месяц + год | `15 мая 2024 г.` |
| Дата в приказе | Только цифровой | `15.05.2024` |
| Период в тексте | «с ДД.ММ.ГГГГ по ДД.ММ.ГГГГ» | `с 01.06.2024 по 30.06.2024` |
| Дата прописью в особо важных документах | Полностью словами | `Пятнадцатое мая две тысячи двадцать четвёртого года` |

### Распространённые ошибки при переводе административных документов

| Ошибка | Неправильно | Правильно |
|--------|-------------|-----------|
| Калька «Уважаемый господин» для русских имён | `Уважаемый господин Иванов` | `Уважаемый Иван Иванович!` |
| Отсутствие восклицательного знака после обращения | `Уважаемый Иван Иванович,` | `Уважаемый Иван Иванович!` |
| «С уважением» без запятой | `С уважением Петров П.П.` | `С уважением, П.П. Петров` |
| Перевод названия должности | `Менеджер по качеству` (калька `Quality Manager`) | `Менеджер по качеству` (если должность есть в штатном расписании) |
| Неправильное склонение фамилий | `Иванову И.И.` → `Иванов Иван` | `Иванову И.И.` |
