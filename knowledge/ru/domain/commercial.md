## 1. Reader Model (Модель читателя)

### Целевая аудитория

Коммерческие документы на русском языке ориентированы на следующие категории читателей:

| Категория | Уровень подготовки | Ожидания |
|-----------|-------------------|----------|
| Менеджер по продажам / коммерческий директор | Высшее экономическое / управленческое образование | Ясность коммерческих условий, сроки поставки и оплаты, ответственность сторон |
| Бухгалтер (главный бухгалтер) | Высшее бухгалтерское образование | Правильность оформления первичной документации, соответствие ФСБУ, выделение НДС |
| Логист / специалист по закупкам | Среднее специальное или высшее | Чёткие условия поставки (Инкотермс), спецификация товара, сроки |
| Руководитель организации | Управленческое образование | Экономическая эффективность, риски, рентабельность условий |
| Специалист по тендерам | Знание ФЗ-44, ФЗ-223 | Соответствие закупочной документации, правильность оформления заявки |

### Факторы доверия

- **Прозрачность ценообразования** — выделение НДС, точные суммы прописью
- **Корректное применение Инкотермс** — правильный перевод условий поставки
- **Идентичность валютных и налоговых формулировок** — буквальное соответствие законодательству РФ
- **Единообразие коммерческой терминологии** — последовательность в названиях товаров и услуг

### Что разрушает доверие

- Отсутствие указания НДС или неверная ставка
- Смешение терминов «счёт», «счёт-фактура», «инвойс», «накладная»
- Неправильное оформление валютных сумм (рубли → прописью с копейками)
- Условия оплаты, противоречащие российскому валютному законодательству

## 2. Decision Framework (Система принятия решений)

### D1: Коммерческие термины — основные понятия

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "invoice" | **счёт** (на оплату) / **счёт-фактура** (налоговый документ) | `Please send the invoice` → `Пожалуйста, направьте счёт на оплату` |
| Исходный текст содержит "proforma invoice" | **предварительный счёт (проформа-счёт)** | `Proforma Invoice for customs clearance` → `Предварительный счёт для таможенного оформления` |
| Исходный текст содержит "purchase order (PO)" | **заказ на поставку / заказ-наряд** | `PO Number` → `Номер заказа` |
| Исходный текст содержит "delivery note / packing list" | **товарная накладная / упаковочный лист** | `Packing List No. 123` → `Упаковочный лист № 123` |
| Исходный текст содержит "bill of lading" | **коносамент** | `Bill of Lading No. BL-12345` → `Коносамент № BL-12345` |
| Исходный текст содержит "contract" | **договор / контракт** (внешнеторговый — контракт) | `Supply Contract No. 2024/01` → `Контракт поставки № 2024/01` |
| Исходный текст содержит "agreement" | **соглашение / договор** | `License Agreement` → `Лицензионный договор` |
| Исходный текст содержит "addendum / amendment" | **дополнительное соглашение / приложение** | `Amendment No. 1 to the Agreement` → `Дополнительное соглашение № 1 к Договору` |

### D2: Цена и валюта

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит цену в иностранной валюте | Указать валюту (код ISO) и сумму; для договоров — сумму прописью | `USD 1,500.00` → `1 500,00 долларов США (Одна тысяча пятьсот долларов США 00 центов)` |
| Исходный текст содержит цену с НДС / без НДС | **в том числе НДС / без учета НДС** — обязательно указывать | `EUR 10,000.00 excl. VAT` → `10 000,00 евро, без учета НДС` |
| Исходный текст содержит "VAT" (налог на добавленную стоимость) | **НДС (налог на добавленную стоимость)** | `VAT 20%` → `НДС 20%` |
| Исходный текст содержит рублёвую цену | Сумма цифрами + прописью; копейки цифрами | `125 430,50 руб. (Сто двадцать пять тысяч четыреста тридцать рублей 50 копеек)` |
| Исходный текст содержит скидку (discount) | **скидка** | `5% volume discount` → `скидка 5% за объем` |
| Исходный текст содержит "price per unit" | **цена за единицу** | `Price per unit: USD 2.50` → `Цена за единицу: 2,50 доллара США` |
| Исходный текст содержит "total amount" | **общая сумма / итого** | `Total Amount: USD 25,000.00` → `Общая сумма: 25 000,00 долларов США` |

### D3: Условия оплаты

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "payment terms: net 30" | **оплата в течение 30 календарных дней с даты выставления счёта** | `Payment: net 30` → `Оплата производится в течение 30 (Тридцати) календарных дней с даты выставления счёта` |
| Исходный текст содержит "advance payment / prepayment" | **предоплата / авансовый платёж** (аванс) | `50% advance payment` → `предоплата в размере 50%` |
| Исходный текст содержит "down payment" | **первоначальный взнос / задаток** | `Down payment of 30% upon signing` → `Первоначальный взнос в размере 30% при подписании` |
| Исходный текст содержит "letter of credit (L/C)" | **аккредитив** | `Irrevocable Letter of Credit` → `Безотзывный аккредитив` |
| Исходный текст содержит "bank transfer / wire transfer" | **банковский перевод** | `Payment by bank transfer` → `Оплата банковским переводом` |
| Исходный текст содержит "installment payment" | **оплата частями / рассрочка** | `Payment in 3 installments` → `Оплата 3 равными частями` |
| Исходный текст содержит "late payment interest" | **проценты за просрочку платежа / пеня** | `Late payment interest: 0.1% per day` → `Пеня за просрочку: 0,1% за каждый день просрочки` |
| Исходный текст содержит "payment upon delivery" | **оплата по факту поставки** | `Cash on delivery` → `Оплата по факту поставки` |

### D4: Условия поставки (Инкотермс)

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "EXW (Ex Works)" | **EXW — Франко-завод (самовывоз)** | `Delivery: EXW, Shanghai` → `Условия поставки: EXW (франко-завод), Шанхай` |
| Исходный текст содержит "FOB (Free on Board)" | **FOB — Свободно на борту** | `FOB Shanghai port` → `FOB (свободно на борту), порт Шанхай` |
| Исходный текст содержит "CIF (Cost, Insurance, Freight)" | **CIF — Стоимость, страхование, фрахт** | `CIF Moscow` → `CIF (стоимость, страхование, фрахт), Москва` |
| Исходный текст содержит "DDP (Delivered Duty Paid)" | **DDP — Поставка с оплатой пошлин** | `DDP buyer's warehouse` → `DDP (поставка с оплатой таможенных пошлин), склад покупателя` |
| Исходный текст содержит "FCA (Free Carrier)" | **FCA — Франко-перевозчик** | `FCA warehouse of seller` → `FCA (франко-перевозчик), склад продавца` |
| Исходный текст содержит "CPT (Carriage Paid To)" | **CPT — Перевозка оплачена до** | `CPT Moscow` → `CPT (перевозка оплачена до), Москва` |

### D5: Описание товаров / спецификация

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит техническую спецификацию товара | Сохранить все технические параметры, перевести описательные поля | `Steel pipe, Ø 50 mm, length 6 m` → `Труба стальная, Ø 50 мм, длина 6 м` |
| Исходный текст содержит артикул / SKU | Сохранить без изменений | `SKU: ST-001-BLK` → `Артикул: ST-001-BLK` |
| Исходный текст содержит название товара | Перевести с сохранением ключевых характеристик | `Wireless Bluetooth Headphones` → `Беспроводные Bluetooth-наушники` |
| Исходный текст содержит единицу измерения товара | Перевести в метрическую систему при необходимости | `pcs → шт. (штуки); box → кор.; pallet → пал.; set → компл.` |
| Исходный текст содержит страну происхождения | **Страна происхождения:** указать полное наименование страны | `Country of origin: China` → `Страна происхождения: Китайская Народная Республика` |
| Исходный текст содержит бренд / производителя | Сохранить оригинальное наименование | `Manufacturer: Siemens AG` → `Производитель: Siemens AG` |

### D6: Коммерческое предложение и оферта

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "offer / quotation" | **коммерческое предложение / оферта** | `Commercial Offer dated 15.05.2024` → `Коммерческое предложение от 15.05.2024` |
| Исходный текст содержит "valid until / expiry" | **срок действия предложения / действительно до** | `The offer is valid until June 30, 2024` → `Предложение действительно до 30 июня 2024 г.` |
| Исходный текст содержит "firm offer" | **твёрдая оферта** (обязывающая) | `This is a firm offer` → `Настоящее предложение является твёрдой офертой` |
| Исходный текст содержит "non-binding offer" | **предварительное коммерческое предложение** (не обязывающее) | `This quotation is non-binding` → `Настоящее коммерческое предложение не является офертой` |
| Исходный текст содержит "framework agreement" | **рамочное соглашение / рамочный договор** | `Framework Supply Agreement` → `Рамочное соглашение о поставках` |

### D7: Налоговые и таможенные формулировки

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "customs duties" | **таможенные пошлины** | `All customs duties are borne by the Buyer` → `Все таможенные пошлины уплачиваются Покупателем` |
| Исходный текст содержит "customs clearance" | **таможенное оформление (растаможка)** | `Customs clearance by the Seller` → `Таможенное оформление осуществляется Продавцом` |
| Исходный текст содержит "excise duty" | **акцизный сбор / акциз** | `Goods subject to excise duty` → `Товары, облагаемые акцизом` |
| Исходный текст содержит "import / export license" | **лицензия на импорт / экспорт** | `The Buyer shall obtain the import license` → `Покупатель обязан получить импортную лицензию` |
| Исходный текст содержит "HS code / customs code" | **код ТН ВЭД** | `HS Code: 8471.30` → `Код ТН ВЭД: 8471.30` |
| Исходный текст содержит "certificate of origin" | **сертификат происхождения** | `Certificate of Origin Form A` → `Сертификат происхождения Форма А` |

### D8: Гарантии и рекламации

| Условие | Решение | Пример |
|---------|---------|--------|
| Исходный текст содержит "warranty period" | **гарантийный срок** | `Warranty: 12 months from the date of delivery` → `Гарантийный срок: 12 месяцев с даты поставки` |
| Исходный текст содержит "defective goods" | **товар ненадлежащего качества / бракованный товар** | `The Buyer shall notify of defective goods within 5 days` → `Покупатель обязан уведомить о товаре ненадлежащего качества в течение 5 дней` |
| Исходный текст содержит "claim / complaint" | **рекламация / претензия** | `The Buyer's claim shall be submitted in writing` → `Рекламация Покупателя подается в письменной форме` |
| Исходный текст содержит "replacement / refund" | **замена / возврат денежных средств** | `The Seller shall replace defective goods or provide a refund` → `Продавец осуществляет замену бракованного товара или возврат денежных средств` |
| Исходный текст содержит "liquidated damages" | **неустойка / штрафные санкции** | `Liquidated damages for delay: 0.5% per week` → `Неустойка за просрочку: 0,5% за каждую неделю` |
| Исходный текст содержит "confidentiality" | **конфиденциальность** | `Confidentiality obligations survive termination` → `Обязательства по конфиденциальности сохраняют силу после расторжения` |

## 3. Standards (Стандарты и нормативные документы)

### Законодательная база коммерческой деятельности в РФ

| Нормативный акт | Описание |
|-----------------|----------|
| **ГК РФ (главы 27–30)** | Общие положения о договорах, купля-продажа, поставка |
| **ФЗ № 381-ФЗ** | Об основах государственного регулирования торговой деятельности |
| **ФЗ № 44-ФЗ** | О контрактной системе в сфере закупок для государственных нужд |
| **ФЗ № 223-ФЗ** | О закупках товаров, работ, услуг отдельными видами юридических лиц |
| **ФЗ № 54-ФЗ** | О применении контрольно-кассовой техники |
| **НК РФ (часть 2, глава 21)** | НДС — ставки, порядок выделения, освобождения |
| **ФЗ № 173-ФЗ** | О валютном регулировании и валютном контроле (для внешнеторговых контрактов) |
| **ФСБУ 5/2019** | Учёт запасов (включая товары) |
| **ФСБУ 25/2018** | Учёт аренды (для договоров аренды/лизинга) |
| **Постановление Госкомстата № 132** | Унифицированные формы первичной учётной документации (ТОРГ-12, ТТН и др.) |

### Международные коммерческие стандарты, применяемые в РФ

| Стандарт | Применение |
|----------|------------|
| **Инкотермс 2020 (Incoterms 2020)** | Условия поставки — обязательны к указанию во внешнеторговых контрактах |
| **UCP 600** | Унифицированные правила для документарных аккредитивов |
| **SWIFT MT / ISO 20022** | Форматы банковских платежных сообщений |
| **UN/EDIFACT** | Электронный обмен коммерческими данными (EDI) |

### Типовые формы первичных документов в РФ

| Документ | Русское название | Назначение |
|----------|-----------------|------------|
| Invoice | Счёт на оплату | Запрос платежа от покупателя |
| Tax invoice | Счёт-фактура | Налоговый документ для вычета НДС |
| Delivery note (TORG-12) | Товарная накладная (ТОРГ-12) | Подтверждение передачи товара |
| Waybill (TTN) | Товарно-транспортная накладная (ТТН) | Перевозка груза |
| Acceptance certificate | Акт приёма-передачи | Подтверждение выполнения работ / поставки |
| Reconciliation report | Акт сверки взаиморасчетов | Сверка взаимных расчётов между сторонами |
| Universal transfer document (UPD) | Универсальный передаточный документ (УПД) | Совмещает счёт-фактуру и акт / накладную |

### Формат дат и подписей в коммерческих документах

| Элемент | Формат | Пример |
|---------|--------|--------|
| Дата договора | «[число] [месяц прописью] [год] года» | `15 мая 2024 года` |
| Дата счёта | `ДД.ММ.ГГГГ` | `15.05.2024` |
| Срок действия | «до [дата]» или «в течение [число] дней» | `до 30.06.2024` или `в течение 30 календарных дней` |
| Реквизиты подписи | **Должность** — **Подпись** — **Расшифровка** (И.О. Фамилия) | `Генеральный директор _____ И.И. Иванов` |
| Печать | «М.П.» (место печати) — актуально для организаций, имеющих печать | — |
