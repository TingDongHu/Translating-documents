# 行政文書 (Administrative Documents) — ja-JP

## Reader Model

**読者像：** 行政機関職員（国家公務員・地方公務員）、申請者・届出者（企業の法務・総務部門、個人事業主、外国人住民）、行政書士、司法書士、社会保険労務士、土地家屋調査士、自治体国際化協会（CLAIR）関係者、領事館職員、出入国在留管理庁職員、多言語情報提供の実務翻訳者。公文書としての形式的要件と実務的な手続きの正確性を両方重視する。具体的な期待と注意点：

- **形式的完全性**：公文書の書式・様式の遵守が最優先。申請書の様式番号、書式コード、管理番号の誤記は申請の受付拒否や無効につながる。翻訳文でも原文の様式情報を完全に保持しなければならない。
- **手続きの正確性**：行政手続きの各ステップ（申請→受理→審査→許可→通知）が翻訳で正確に伝達されていること。「申請」「届出」「許可」「認可」「承認」「認定」「登録」の法的効果の違いを認識した用語選択が必要。
- **公的証明書の翻訳精度**：戸籍謄本、住民票、納税証明書、印鑑登録証明書等の公的証明書の翻訳には、翻訳者の資格証明（翻訳証明書）が添付されることが多い。証明書のフォーマット情報（発行日、発行機関、証明番号、公印等）の完全な転記が要求される。
- **公用文特有の文体と語彙**：公用文で使用される定型表現（「…の定めるところにより」「…については、…を適用する」「…について必要な事項を定める」等）が正確に使用されていること。口語的表現やカタカナ英語の多用は行政文書としての信頼性を損なう。
- **やさしい日本語とのバランス**：外国人住民向けの多言語情報では「やさしい日本語」（簡易な日本語表現）と法令に基づく正確性のバランスが重要。対象読者（日本人・外国人・専門家・一般市民）に応じた文体選択が求められる。
- **印鑑・署名の取扱い**：日本の印鑑制度（実印・認印、銀行印、法人印等）の正確な理解と、外国の署名制度との対応付けが必要。記名押印と署名の法的効力の差異を認識した翻訳が求められる。
- **元号・西暦の表記**：行政文書では和暦（令和X年）表記が公式。西暦との併記が推奨される場合もある。文書全体での表記統一が必須。
- **外国人向け在留関連手続き**：在留資格（就労ビザ、留学、家族滞在等）、在留カード、特別永住者証明書等の出入国管理関連書類の翻訳では、入管法の定義に従った用語選択が必要。

## Decision Framework

### 行政機関名・部署名の正式名称使用
| Condition | Decision | Example |
|-----------|----------|---------|
| 省庁名の翻訳 | 日本の官公庁の正式日本語名称を正確に使用。略称（「経産省」「総務省」等）は正式文書では避け、正式名称を使用 | "Ministry of Economy, Trade and Industry" → 「経済産業省」（×「経産省」、×「経済産業省（METI）」は併記可） |
| 外局の名称 | 「○○庁」としての正式名称を使用。英語名が異なる場合もある | "Environmental Protection Agency"（米国）→ 「環境保護庁」（EPA）/ "Agency for Natural Resources and Energy" → 「資源エネルギー庁」 |
| 地方自治体名 | 「東京都」「大阪府」「北海道」「福岡県」「横浜市」等の正式名称。海外の都道府県相当は「○○州」「○○省」等と使い分け | "State of California" → 「カリフォルニア州」 / "Bavaria" → 「バイエルン州（ドイツ）」 / "Seine-Maritime" → 「セーヌ＝マリティーム県（フランス）」 |
| 部署名の階層 | 局→部→課→係の階層を正確に翻訳。英語のDepartment/Section/Divisionを日本の組織体系に合わせる | "Trade Control Department" → 「貿易管理課」文脈に応じて適切な階層語を選択 |
| 独立行政法人等 | 「独立行政法人○○」「国立研究開発法人○○」等の名称を正式に使用 | "National Institute of Advanced Industrial Science and Technology" → 「国立研究開発法人産業技術総合研究所（産総研）」 |
| 公的機関の英語名が複数存在 | 当該機関の公式英語表記を確認。省庁再編後の名称に更新 | 法務省：公式英語は"Ministry of Justice"。旧「法務省」→「MOJ」でなく正式英語名 |

### 様式番号・書式コードの不変保持
| Condition | Decision | Example |
|-----------|----------|---------|
| 申請書様式番号 | 番号・ハイフン・英数字を完全原文保持。翻訳・言い換え一切不可 | "Form No. 1-2" → 「Form No. 1-2」（変更不可。×「様式第1-2号」への変換は原則しない） |
| 管理番号・受付番号 | 番号体系を完全保持。ゼロパディングも含めて改変不可 | "Application No. 2025-0001234" → 「Application No. 2025-0001234」 |
| 行政コード | 都道府県コード、市区町村コード等の公的コードは保持 | "Municipal Code: 131016" → 「Municipal Code: 131016」（東京都千代田区のコード） |
| 管轄官庁の表示 | 所管官庁名は正式名称で記載。変更不可 | "Submitted to: Tokyo Regional Immigration Bureau" → 「提出先：東京出入国在留管理局」 |
| 帳票のバージョン/発行年 | 様式の改訂年・版数を保持 | "Form revised April 2024" → 「令和6年4月改訂」 |

### 行政手続き用語の法令準拠
| Condition | Decision | Example |
|-----------|----------|---------|
| "Application" | 「申請」行政手続法上の申請。許可等を求める行為 | "The applicant submitted an application for a business license." → 「申請者は、営業許可の申請を行った。」 |
| "Notification" / "Notice" | 「届出」行政機関への事実の通知。許可不要 | "The company filed a notification of change of address." → 「会社は、住所変更の届出を行った。」 |
| "Permit" / "Permission" | 「許可」一般的に禁止されている行為を特定の場合に解除 | "A permit for the handling of hazardous materials was obtained." → 「危険物取扱の許可を取得した。」 |
| "License" | 「免許」特定の行為を行うための資格付与 | "A driver's license is required." → 「運転免許証が必要である。」 |
| "Approval" | 「承認」一定の行為や計画を認めること | "Board approval is required for the merger." → 「合併には取締役会の承認が必要である。」 |
| "Authorization" | 「認可」行政庁が特定の行為を補充的に認めること | "Authorization by the regulatory authority is required." → 「規制当局の認可が必要である。」 |
| "Certification" | 「登録」「認定」「証明」文脈により使い分け | "Professional certification" → 「資格認定」 / "Business registration" → 「事業登録」 |
| "Designation" | 「指定」特定の機関・者を指定すること | "Designated quarantine facility" → 「指定検疫施設」 |
| "Accreditation" | 「認定（機関認定）」試験・検査機関等の認定 | "Accreditation by ISO/IEC 17025" → 「ISO/IEC 17025による認定」 |
| "Registration" | 「登録」公的簿冊への記載 | "Company registration" → 「会社登録（登記）」 |
| "Confirmation" | 「確認」行政庁による事実の確認行為 | "Confirmation of tax payment" → 「納税確認」 |

### 年月日・期間の日本書式適合
| Condition | Decision | Example |
|-----------|----------|---------|
| 和暦表記の基本 | 「令和X年」「平成X年」「昭和X年」を正式表記。文書全体で統一 | "April 1, 2026" → 「令和8年4月1日」 |
| 西暦併記の推奨 | 初出時は（西暦）を併記。外国人読者向けには西暦メイン | 「令和8年（2026年）4月1日」 |
| 元号の西暦換算 | 令和：西暦-2018 / 平成：西暦-1988 / 昭和：西暦-1925 | 令和7年 = 2025年 / 平成31年 = 2019年 / 昭和64年 = 1989年 |
| 期間の表記 | 「自 XX 至 XX」の形式。行政文書の定型 | "From April 1, 2025 to March 31, 2026" → 「自 令和7年4月1日 至 令和8年3月31日」 |
| 期限の表記 | 「XXまでに」「XXをもって」等の期限表現を正確に | "by March 31, 2026" → 「令和8年3月31日までに」 |
| 経過期間の表記 | 「XX以内」「XX以上」「XXを経過した後」等の時間表現 | "within 30 days of the effective date" → 「効力発生日から30日以内に」 |
| 営業日/暦日の区別 | "business day" / "calendar day" を明確に区別 | "within 10 business days" → 「10営業日以内に」 / "within 30 calendar days" → 「30日以内に（暦日）」 |

### パスポート・ID情報の取扱い
| Condition | Decision | Example |
|-----------|----------|---------|
| パスポート番号 | 番号体系を完全保持。英数字・記号の誤記は致命的 | "Passport No. AB1234567" → 「パスポート番号 AB1234567」 |
| 個人番号（マイナンバー） | 12桁の数字をそのまま保持。「マイナンバー」「個人番号」の表記 | "My Number: 1234-5678-9012" → 「マイナンバー（個人番号）：1234-5678-9012」 |
| 住民票コード | 11桁の英数字を保持 | "Resident Registration Code: ABCD1234EF" → 「住民票コード：ABCD1234EF」 |
| 氏名の表記 | パスポート表記に合わせる。姓・名の順序は日本の「姓 名」順に | "Taro YAMADA" → 「山田 太郎」（日本語表記）または「Taro Yamada」（外国籍者） |
| 住所の翻訳 | 日本の住所表記（郵便番号→都道府県→市区町村→町名→番地→建物名）に再構成 | "1-2-3 Marunouchi, Chiyoda-ku, Tokyo 100-0001, Japan" → 「〒100-0001 東京都千代田区丸の内1-2-3」 |
| 国籍の表記 | 「日本」「アメリカ合衆国」「大韓民国」等の正式国名 | "United States of America" → 「アメリカ合衆国」（×「米国」正式文書では「アメリカ合衆国」） |
| 性別の表記 | 「男」「女」または「男性」「女性」。文書により表記が異なるので原文に従う | "Gender: Male" → 「性別：男」または「性別：男性」 |
| 生年月日 | 和暦または西暦。文書の書式に従う。戸籍関連は和暦が基本 | "Date of Birth: April 1, 1980" → 「生年月日：昭和55年4月1日」 |

### 行政用語の平明化と正確性のバランス
| Condition | Decision | Example |
|-----------|----------|---------|
| 外国人住民向け案内文書 | 「やさしい日本語」の観点から、難しい漢語にはふりがなまたは簡潔な言い換えを併記。ただし法的効果のある用語は正確に | 「在留資格（ビザの種類）」 / 「許可（きょか）」 |
| 定義された法令用語 | 正確に使用。言い換え不可。ただし直後に簡潔な説明を付記可能 | 「届出（とどけで）」 / 「申請（もうしこみ）」 |
| 目的の明示 | 「…のため」で目的を簡潔に前置き。やさしい日本語の基本構造 | "For the purpose of preventing the spread of infectious diseases" → 「感染症のまん延を防ぐため」 |
| 根拠法令の明示 | 「XX法第X条に基づき」の形式で根拠を明示。原文の法令引用を省略しない | "pursuant to Article 5 of the Immigration Control Act" → 「出入国管理及び難民認定法第5条に基づき」 |
| 強制度の明確化 | 「しなければなりません」「できます」「してはいけません」で義務・許可・禁止を明確に | "must" → 「しなければなりません」 / "may" → 「できます」 / "must not" → 「してはいけません」 |
| 行政手続きのステップ説明 | 箇条書きと順序番号で手順を明確に。「申請→審査→通知」の流れを保持 | "Step 1: Prepare documents → Step 2: Submit to the counter..." → 「手順1：書類を準備する → 手順2：窓口に提出する…」 |
| 注意喚起の表現 | 「ご注意ください」「重要です」等、平明な注意表現 | "Important: Applications without supporting documents will be rejected." → 「重要：添付書類のない申請は受理されません。」 |
| 対象者の明示 | 誰が対象か（申請者/申請者以外）を明確に。条件の該当/非該当を明示 | "Those who are eligible for this program are..." → 「この制度の対象となる方は…」 |

### 通達・告示の書式と引用形式の維持
| Condition | Decision | Example |
|-----------|----------|---------|
| 通達番号の保持 | 「令和●年●月●日付け ●課長通知」等の形式を完全保持。追記改変不可 | "Director's Notice dated March 31, 2025" → 「令和7年3月31日付け ○○課長通知」 |
| 告示番号の保持 | 「総務省告示第XX号」「厚生労働省告示第XXX号」等の形式を保持 | "MIC Notice No. XXX" → 「総務省告示第XXX号」 |
| 通達内容の引用形式 | 「…については、●省令第X条に定めるところによる。」日本の法令引用形式で | "as provided in Article X of the relevant ministerial ordinance" → 「関係省令第X条に定めるところによる。」 |
| 法令引用の階層 | 法→政令→省令→告示の優先順位を示す引用形式を保持 | "pursuant to Article X of the Act and Article Y of the Cabinet Order" → 「法第X条及び政令第Y条の規定に基づき」 |
| 日付付き通知 | 発出日、適用日、失効日を明確に区別して表記 | "Effective from April 1, 2026" → 「令和8年4月1日から適用」 |

### 証明書類における翻訳認証文言の取扱い
| Condition | Decision | Example |
|-----------|----------|---------|
| "Certificate of Translation" | 「翻訳証明書」翻訳の正確性を証明する添付文書 | "I hereby certify that the foregoing is a true and accurate translation." → 「上記は正確な翻訳であることを証明する。」 |
| 翻訳証明書の必須項目 | 翻訳者氏名、住所、日付、署名、資格を必ず記載。原文の証明文言を正確に翻訳 | "Translator: Taro Yamada, Address: 1-2-3 Shinjuku, Tokyo, Date: May 25, 2026, Signature: [signed]" |
| 公印の取扱い | 原文にある公印（官公署の印）は「【公印省略】」「（印）」等と注記。印影の写しがある場合は「公印あり」と記載 | 公印が押されている場合：「東京都港区長 印」（印影の説明を付記） |
| "Certified Copy" | 「謄本」「正本」「認証謄本」文脈により選択 | "Certified copy of the diploma" → 「卒業証明書（認証謄本）」 |
| 戸籍関連の証明書 | 謄本（全部事項証明書）と抄本（個人事項証明書）を区別 | "Certified copy of family register" → 「戸籍全部事項証明書（戸籍謄本）」 / "Extract of family register" → 「戸籍個人事項証明書（戸籍抄本）」 |
| アポスティーユ | 「アポスティーユ（Apostille）」ハーグ条約に基づく公的文書認証。原語保持＋説明 | "Apostille Convention" → 「ハーグ条約（アポスティーユ条約）」 |
| 公証人手続き | "Notarization" → 「公証」または「公証人手続」 | "The document must be notarized." → 「本書類は公証人の認証を受ける必要がある。」 |
| "True Copy" | 「正確な写し」「正本と相違ない写し」 | "This is a true copy of the original." → 「原本と相違ないことを認証する。」 |

### 公用文の文体と表現
| Condition | Decision | Example |
|-----------|----------|---------|
| 公用文の文体 | 「である」調、文語調。口語・会話表現は不可 | 「ですます」調は原則不可（ガイドライン類では許容される場合あり） |
| 接続詞の使用 | 「したがって」「ただし」「なお」「また」「もっとも」等の文語的接続詞を使用 | "However" → 「ただし」 / "Furthermore" → 「なお」 / "In addition" → 「また」 |
| 句読点 | 公用文では「、。」を基本。JIS規格文書の「，．」ではなく行政文書は「、。」 | 横書き公用文でも「、。」を使用。カンマ・ピリオドは使用しない |
| 漢語の使用 | 和語よりも漢語を優先。公用文特有の語彙（「本件」「前記」「同」「当該」「該当」等） | "this matter" → 「本件」 / "the aforementioned" → 「前記の」 / "the same" → 「同」 |
| 禁止表現 | 「…してはならない」/「…を禁ずる」を使用。「…しちゃダメ」等の口語表現は不可 | "No person shall enter this area." → 「本区域に立ち入ってはならない。」 |
| 許可・許容表現 | 「…することができる」「…を認める」 | "Entry is permitted only with authorization." → 「許可を受けた場合に限り、入場することができる。」 |
| 義務表現 | 「…しなければならない」「…する義務を負う」 | "All residents must register their address." → 「全ての住民は、住所の登録をしなければならない。」 |
| 定義表現 | 「…とは、…をいう」「…と定義する」 | "The term 'resident' means a person who resides in Japan." → 「「居住者」とは、日本に居住する者をいう。」 |
| 目的表現 | 「…のため」、「…を目的として」 | "This law aims to promote the welfare of workers." → 「この法律は、労働者の福祉の増進を図ることを目的とする。」 |

### 本人確認と認証に関する翻訳
| Condition | Decision | Example |
|-----------|----------|---------|
| "Identification" / "ID" | 「身分証明書」「本人確認書類」 | "Please present a valid photo ID." → 「有効な写真付き身分証明書を提示してください。」 |
| "Seal" / "Personal Seal" | 「印鑑」「印章」「認印」「実印」の区別を正確に | "Registered seal (jitsuin)" → 「実印（登記印）」 / "Business seal" → 「法人印」 / "Personal seal" → 「認印」 |
| "Signature" | 「署名」自署による本人確認 | "Signed in the presence of a witness" → 「立会人の面前で署名」 |
| "Signature and Seal" | 「署名捺印」両方の完了を求める表現 | "Signed and sealed by the representative director" → 「代表取締役が署名捺印」 |
| "Power of Attorney" | 「委任状」代理人の権限を証明 | "Notarized power of attorney" → 「公証人認証付き委任状」 |
| "Affidavit" | 「宣誓供述書」「宣誓書」裁判外の宣誓陳述 | "The affiant swears that the foregoing is true." → 「宣誓者は、上記が真実であることを宣誓する。」 |

## Standard Conventions

### 日本の行政文書体系

**公文書の種類と階層：**
| 文書種別 | 説明 | 英語相当 |
|----------|------|----------|
| 法律 | 国会の議決を経て成立する法規範 | Act, Law |
| 政令 | 内閣が制定する命令 | Cabinet Order |
| 省令 | 各省大臣が制定する命令 | Ministerial Ordinance |
| 告示 | 行政機関が公に知らせる行為 | Public Notice, Notification |
| 通達 | 上級行政機関から下級機関への指示 | Directive, Circular Notice |
| 訓令 | 各省庁内の組織に対して発する命令 | Regulation |
| 指令 | 特定の案件に対する個別の指示 | Instruction |
| 公示 | 法令により公に示すこと | Official Announcement |

**行政手続きの主な書式：**
- 申請書（Application Form）
- 届出書（Notification Form）
- 申告書（Return Form）— 税務申告等
- 請求書（Claim Form）— 給付金等
- 報告書（Report Form）
- 証明書（Certificate）
- 通知書（Notice）
- 許可証（Permit / License）
- 認定証（Certificate of Approval）


### 公用文表記の基準

**公用文の句読点：**
- 横書き：句点「。」読点「、」を使用。カンマ「，」は使用しない
- 「なお」「ただし」「また」の後は読点を打つ
- 箇条書きの各項目の末尾は「。」で統一

**数字表記（公用文）：**
- 公用文の数字は原則として漢数字（「一、二、三」）を使用
- 横書きの増加に伴い、算用数字（1, 2, 3）も使用される
- ただし法令・条約等の条文番号は漢数字が正式
- 西暦年号の数字は算用数字（2026年）が近年標準
- 和暦年号の数字は漢数字（令和八年）または算用数字（令和8年）。原則として文書全体で統一

**公用文における注意点：**
| 項目 | 基準 | 例 |
|------|------|-----|
| 代名詞 | 「わたし」よりも「私」、平仮名よりも漢字を優先 | 「私」「当該」「本件」 |
| 副詞 | 漢字表記を優先 | 「既に」「即ち」「尚」「且つ」 |
| 助詞 | 平仮名表記 | 「は」「が」「を」「に」 |
| 接続詞 | 漢字または平仮名。常用漢字表に従う | 「ただし」「なお」「また」「したがって」 |
| 動詞の送り仮名 | 本則に従う | 「行う」「行った」「異なる」 |

**やさしい日本語のガイドライン（行政情報）：**
| 原文の難しい表現 | やさしい日本語への言い換え例 |
|------------------|------------------------------|
| 「手続きを行う」 | 「手続き（てつづき）をします」 |
| 「該当する」 | 「あてはまる」 |
| 「申請する」 | 「申し込む（もうしこむ）」または「申請（しんせい）する」 |
| 「収入を得る」 | 「収入（しゅうにゅう）をもらう」 |
| 「…を要する」 | 「…が必要（ひつよう）です」 |
| 「…の定めるところにより」 | 「…のルールにしたがって」 |
| 「…の場合を除く」 | 「…のときは、ちがいます」 |

### 印鑑・印章に関する制度

**日本の印鑑の種類：**
| 種類 | 用途 | 説明 |
|------|------|------|
| 実印 | 重要な契約・登記 | 市区町村に登録した印鑑。印鑑登録証明書と併用 |
| 認印 | 日常的な書類 | 登録不要の印鑑。銀行印と区別 |
| 銀行印 | 金融取引 | 銀行口座開設時に届け出る印鑑 |
| 法人印（会社印） | 会社の公式文書 | 登記所に登録する会社の代表印 |
| 角印 | 請求書等の事務文書 | 会社名を刻んだ角型の印鑑。社内・取引先用 |

**印鑑の表記：**
- 「押印」：印鑑を押す行為。記名（氏名を印刷）＋押印が「記名押印」
- 「署名」：自筆で氏名を書く行為。外国では署名が一般的
- 「記名押印」：氏名を印刷し、その上に印鑑を押す
- 日本の契約書・公文書では「記名押印」が標準
- 翻訳時に「sign」を一律「署名」と訳さず、文脈に応じて「記名押印」を選択
