# Persian Adaptation Guide: English → Persian

## 1. Formality

English uses _you_ for both formal and informal address. Persian has a distinction between _تو_ (tu, informal) and _شما_ (shomareh, formal). Persian strongly prefers active voice in everyday speech but uses passive constructions in formal/administrative writing.

| English | Persian | Notes |
|---|---|---|
| You (general/indefinite) | — (omit) / شخص / می‌توان | Impersonal "you" → use impersonal constructions |
| You (singular informal) | تو (Tu) | Use with close friends, family, children |
| You (singular formal) | شما (Shomareh) | Use in business, official, customer-facing contexts |
| You (plural) | شما (Shomareh) | Same as formal singular; context clarifies plurality |
| Please | لطفاً (Lotfan) | Standard polite marker |
| Thank you | متشکرم (Moteshakeram) / سپاسگزارم | "متشکرم" is common; "سپاسگزارم" is more formal |
| Dear [Name] | جناب آقای [Name] / سرکار خانم [Name] | Formal; "گرامی" (gerāmi) for semi-formal |
| Mr. / Mrs. / Ms. | آقای (Agha) / خانم (Khanom) | Used before name |
| Sir / Madam | جناب عالی / سرکار | Very formal address |
| Passive: "It is believed that..." | بر این باورند که... / تصور می‌شود که... | Persian uses active impersonal forms |
| Passive: "The system was designed..." | سیستم برای... طراحی شد | Convert to active or agentless passive |
| Passive: "It was decided that..." | تصمیم بر این شد که... | Use active: "تصمیم گرفته شد" |
| "One must consider..." | باید در نظر گرفت... | Impersonal construction |
| "It can be argued that..." | می‌توان استدلال کرد که... | Active impersonal works well |
| "The document is attached." | سند پیوست شد / سند در پیوست است | Turkish prefers direct statement |
| "It should be noted that..." | لازم به ذکر است که... | Standard formal phrase |
| "I was told that..." | به من گفته شد که... | Passive is acceptable in Persian formal |
| "It is recommended that..." | توصیه می‌شود که... | Impersonal active form |
| "He is considered to be..." | وی به عنوان... شناخته می‌شود | Acceptable impersonal form |

## 2. Dates, Times, and Numbers

English (US) uses MM/DD/YYYY and AM/PM. Persian uses the Solar Hijri calendar (YYYY/MM/DD) as primary, with 24-hour time format.

| English (US) | Persian | Notes |
|---|---|---|
| MM/DD/YYYY (e.g., 01/15/2025) | 1403/10/25 (Solar Hijri) | Convert Gregorian to Solar Hijri when possible |
| January 15, 2025 | ۲۵ دی ۱۴۰۳ | Solar Hijri month name, day before month |
| 1/15/2025 | ۱۴۰۳/۱۰/۲۵ | Solar Hijri format |
| 2025-01-15 (ISO) | ۱۴۰۳/۱۰/۲۵ | ISO → Solar Hijri |
| Week of 1/15 | هفته ۲۵ دی | First convert date |
| 9:00 AM | ۰۹:۰۰ | Convert to 24-hour format |
| 12:00 PM (noon) | ۱۲:۰۰ | 24-hour: midday is 12:00 |
| 12:00 AM (midnight) | ۰۰:۰۰ / ۲۴:۰۰ | 00:00 is standard |
| 3:30 PM | ۱۵:۳۰ | Add 12 hours to PM times |
| 1:00 PM | ۱۳:۰۰ | — |
| 11:45 PM | ۲۳:۴۵ | — |
| 6:00 AM | ۰۶:۰۰ | — |
| 10,000 (comma as thousands) | ۱۰,۰۰۰ (comma) | Persian uses comma for thousands separator |
| 1,000,000 (comma) | ۱,۰۰۰,۰۰۰ (comma) | Same rule |
| 3.14 (dot as decimal) | ۳.۱۴ (dot) | Persian uses dot for decimal separator |
| $100 (dollar) | ۱۰۰ دلار / ۱۰۰ $ | Indicate currency explicitly |
| 5'10" (height) | ۱.۷۸ متر / ۱۷۸ سانتی‌متر | Convert imperial to metric |
| 32°F (temperature) | ۰ درجه سانتی‌گراد | Convert Fahrenheit to Celsius |
| Phone: (555) 123-4567 | تلفن: ۰۲۱-۱۲۳۴۵۶۷ | Convert to Iranian format |
| Zip code: 90210 | کد پستی: ۱۲۳۴۵۶۷۸۹۰ | US 5-digit → Iranian 10-digit |
| Q1 2025 / FY2025 | سه‌ماهه اول ۱۴۰۳ / سال مالی ۱۴۰۳ | Quarter/fiscal year in Solar Hijri |
| 50% (no space before %) | ۵۰٪ | Persian places ٪ after the number |
| Page 1-5 | صفحه ۱ تا ۵ | Use "تا" (to) for ranges |

### Eastern Arabic vs. Persian Numerals
- English uses Western Arabic numerals (0-9).
- Persian may use either Western Arabic (0-9) or Persian-Eastern Arabic numerals (۰-۹).
- Persian numerals (۰۱۲۳۴۵۶۷۸۹) are preferred in literary and formal contexts.
- Western Arabic numerals are common in technical and financial texts.
- Be consistent within a document.

## 6. Punctuation

Persian punctuation differs from English in several key areas, particularly in comma, semicolon, question mark, and quotation mark usage.

| English | Persian | Rule |
|---|---|---|
| " " (double quotation marks) | « » (guillemets) | Persian primary quotation marks are guillemets; "" acceptable in modern texts |
| ' ' (single quotation marks) | ' ' or « » (nested) | Same usage for nested quotes |
| ... (ellipsis, 3 dots) | ... (3 dots) | Same |
| — (em dash) | — (em dash) | Same usage |
| – (en dash for ranges) | تا / – (en dash) | Persian often uses "تا" (to) for ranges |
| - (hyphen) | - (hyphen) | Same |
| , (comma) | ، (Arabic comma U+060C) | Persian uses Arabic comma; Latin comma also acceptable |
| . (period) | . (period) | Same |
| : (colon) | : (colon) | Same |
| ; (semicolon) | ؛ (Arabic semicolon U+061B) | Persian uses Arabic semicolon; Latin also acceptable |
| ? (question mark) | ؟ (Arabic question mark U+061F) | Persian uses reversed question mark |
| ! (exclamation) | ! (exclamation) | Same |
| % (percent) | ٪ (Arabic percent U+066A) | Persian places ٪ after the number: ۵۰٪ |
| $100 | ۱۰۰ دلار / ۱۰۰ $ | Currency symbol → written after amount |
| "Chapter 1" | فصل ۱ / بخش ۱ | Persian may use numbered expressions differently |
| "e.g." / "i.e." | مثلاً / یعنی | Use Persian abbreviations |
| "etc." | و غیره (va gheyr-e) | Persian equivalent |
| "c.f." | مقایسه کنید | "Compare" in Persian |
| "N.B." | توجه مهم / دikkat | Use Persian phrase |
| "Inc." | شرکت سهامی / شرکت | Company suffix localization |
| "Ltd." | شرکت با مسئولیت محدود | Limited company |
| Bullet list punctuation | Same as English | Persian convention: items may end without punctuation |
| ( ) parentheses | ( ) | Same; but content inside is LTR within RTL text |
| [ ] square brackets | [ ] | Same usage |

### Bidirectional Text Handling
- Persian is RTL; English text embedded in Persian is LTR.
- Numbers within Persian text are LTR: `قیمت ۱,۰۰۰ تومان`.
- Latin abbreviations within Persian may break RTL flow: `سیستم‌عامل (OS)`.
- Use the Unicode bidirectional algorithm correctly.
- The Persian half-space (نیم‌فاصله, U+200C) joins compound words: `خودرو` (not `خود ر`).

## 7. Sentence Restructuring (SVO → SOV)

English uses Subject-Object-Verb (SVO) word order. Persian uses Subject-Object-Verb (SOV). Translating English sentences into Persian requires restructuring the verb to the end of the clause.

### 7.1 Basic SVO → SOV Conversion

| English (SVO) | Persian (SOV) | Notes |
|---|---|---|
| Ali eats an apple. | علی سیبی می‌خورد. | SVO → SOV |
| The company signed the contract. | شرکت قرارداد را امضا کرد. | Verb moves to end |
| She will send the report tomorrow. | او فردا گزارش را ارسال خواهد کرد. | Future tense at end |
| They have built a new bridge. | آنها پل جدیدی ساخته‌اند. | Perfect tense at end |
| We can solve this problem. | ما می‌توانیم این مشکل را حل کنیم. | Modal + verb at end |

### 7.2 Complex Sentence Restructuring

| English Structure | Persian Equivalent | Notes |
|---|---|---|
| Although it was raining, we went out. | اگرچه باران می‌بارید، ما بیرون رفتیم. | Subordinate clause first, then main clause + verb |
| The report that was submitted yesterday was approved. | گزارشی که دیروز ارائه شد، تأیید شد. | Relative clause before main verb |
| If you need help, please contact us. | اگر به کمک نیاز دارید، لطفاً با ما تماس بگیرید. | Conditional first, then imperative |
| The document must be signed before the deadline. | سند باید قبل از مهلت امضا شود. | Modal + infinitive at end |
| According to the regulation, all employees shall attend. | طبق مقررات، همه کارکنان باید شرکت کنند. | Adverbial phrase first, obligation at end |

### 7.3 Passive Voice Handling

| English Passive | Persian Active (preferred) | Persian Passive (if needed) |
|---|---|---|
| The decision was made by the board. | هیئت مدیره تصمیم گرفت. | تصمیم توسط هیئت مدیره گرفته شد. |
| The law was enacted in 2020. | قانون در سال ۲۰۲۰ تصویب شد. | — (passive natural here) |
| The files were sent to the client. | فایل‌ها به مشتری ارسال شدند. | — (passive natural here) |
| The system was designed by our team. | تیم ما سیستم را طراحی کرد. | سیستم توسط تیم ما طراحی شد. |

**Rule**: Persian uses passive voice more frequently than English in formal/administrative texts. The suffix `-شد` (shod) or `-شده` (shodeh) creates the passive form.

## 8. Ezafe Construction in Translation

The Ezafe (کسره اضافه) is a grammatical particle linking nouns to their modifiers or possessor. It is a critical feature of Persian syntax that has no direct English equivalent.

### 8.1 Ezafe Patterns

| English Pattern | Persian Pattern | Example |
|---|---|---|
| Noun + adjective | Noun + Ezafe + adjective | خانه بزرگ (bozorg khāneh) = big house |
| Noun + possessor | Noun + Ezafe + possessor | کتاب علی (Ali-ye ketāb) = Ali's book |
| Noun + noun (compound) | Noun + Ezafe + noun | قانون مدنی (madanī-ye qānūn) = civil law |
| Adjective + Ezafe + adjective | Stacked Ezafe | ساختمان بزرگ و زیبا (bozorg-o-zibā-ye sākhtemān) = big and beautiful building |

### 8.2 Ezafe in Compound Terms

| English Compound | Persian Ezafe Construction | Notes |
|---|---|---|
| Civil law | حقوق مدنی (hoqūq-e madanī) | Ezafe between noun and adjective |
| Central bank | بانک مرکزی (bānk-e markazi) | Ezafe between noun and adjective |
| Trade agreement | توافق تجاری (tavāghogh-e tejārī) | Ezafe between noun and adjective |
| Company registration | ثبت شرکت (sabt-e sherkat) | Ezafe between noun and noun |
| Tax return | اظهارنامه مالیاتی (ezhār-nāmeh-ye māliyātī) | Ezafe between compound noun and adjective |

### 8.3 Ezafe After Vowels
- After consonants: Ezafe is `-e` (unvocalized): `خانه بزرگ` (bozorg khāneh).
- After vowels: Ezafe is `-ye` (written as `ی`): `کتاب علی` (Ali-ye ketāb).
- In modern unvocalized text, the Ezafe is typically unwritten (implied by context).
