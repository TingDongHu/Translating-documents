# Persian Adaptation Guide: Chinese (Simplified) → Persian

## 1. Formality

Chinese uses two second-person pronouns: 您 (formal) and 你 (informal). Persian uses شما (shomareh) for formal/plural address and تو (tu) for informal. Both languages have a clear T-V distinction, but the usage contexts differ.

| Chinese | Persian | Notes |
|---|---|---|
| 您 (nín) | شما (Shomareh) | Formal/plural second person |
| 你 (nǐ) | تو (Tu) | Informal second person |
| 先生 (xiānsheng) | آقای (Agha) / جناب آقای (Janab-e Agha) | "آقای" for general; "جناب آقای" for very formal |
| 女士 (nǚshì) | خانم (Khanom) / سرکار خانم (Sarkar Khanom) | "خانم" for general; "سرکار خانم" for very formal |
| 小姐 (xiǎojiě) | خانم (Khanom) | "دختر" (dokhtar) is informal; avoid in formal contexts |
| 同志 (tóngzì) | رفیق (Rafīgh) / همکار (Hamkār) | "رفیق" in political contexts; "همکار" for colleague |
| 师傅 (shifu) | استاد (Ostad) / آقا (Agha) | "استاد" for skilled师傅; "آقا" for general service |
| 老师 (lǎoshī) | استاد (Ostad) / آموزگار (Āmuzgār) | "استاد" for university; "آموزگار" for school |
| 尊敬的 (zūnjìng de) | گرامی (Gerāmi) / محترم (Mohteram) | Use in formal correspondence openings |
| 亲爱的 (qīn'ài de) | عزیز (Aziz) / گرامی (Gerāmi) | Use in semi-formal; avoid in official texts |
| 贵姓 (guìxìng) | نام شما چیست؟ / اسم حضرتعالی؟ | "نام شما چیست" is standard; very formal: "اسم حضرتعالی" |
| 请 (qǐng) | لطفاً (Lotfan) | Standard polite marker |
| 麻烦您 (máfan nín) | معذرت می‌خواهم / زحمت می‌دهم | Expresses polite inconvenience |
| 您先请 (nín xiān qǐng) | بفرمایید / شما بفرمایید | Polite offer |
| 大家好 (dàjiā hǎo) | سلام به همه / درود بر هماه | Standard greeting to a group |
| 诸位 (zhūwèi) | حضار محترم / بزرگواران | Literary "everyone"; formal address |

### Formality System Comparison

| Feature | Chinese | Persian | Notes |
|---------|---------|---------|-------|
| Formal pronoun | 您 (nín) | شما (Shomareh) | Persian شما also covers plural |
| Informal pronoun | 你 (nǐ) | تو (Tu) | Both are singular informal |
| Formal address title | 先生/女士 | آقای/خانم | Persian titles precede name |
| Honorific suffix | 同志/老师 | استاد/جناب | Different honorific systems |
| Politeness marker | 请 (qǐng) | لطفاً (Lotfan) | Both are standard polite markers |

## 2. Dates, Times, and Numbers

Chinese uses unique number words (万 10,000; 亿 100,000,000) that have no direct Persian equivalents. Date formats also differ significantly.

| Chinese | Persian | Notes |
|---|---|---|
| 万 (wàn, 10,000) | ۱۰,۰۰۰ (۱۰ هزار) | Never translate as single word; use digit format |
| 亿 (yì, 100,000,000) | ۱۰۰,۰۰۰,۰۰۰ (۱۰۰ میلیون) | Chinese 亿 has no single-word Persian equivalent |
| 百万 (bǎi wàn, 1,000,000) | ۱,۰۰۰,۰۰۰ (۱ میلیون) | Same concept, different grouping |
| 千万 (qiān wàn, 10,000,000) | ۱۰,۰۰۰,۰۰۰ (۱۰ میلیون) | — |
| 万亿/兆 (wàn yì/zhào, 10¹²) | ۱,۰۰۰,۰۰۰,۰۰۰,۰۰۰ (۱ تریلیون) | Verify context; 兆 can also mean 10⁶ |
| 几十万 (jǐ shí wàn) | صدها هزار | Convert to Persian numeric phrasing |
| 百分之... (%) | ... درصد (%) / ... ٪ | E.g., 百分之五十 → ۵۰ درصد |
| 2025年1月15日 | ۲۵ دی ۱۴۰۳ | Convert Gregorian to Solar Hijri |
| 2025-01-15 | ۱۴۰۳/۱۰/۲۵ | ISO → Solar Hijri |
| 第1 (dì 1, ordinal) | ۱اُم / اول (Avval) | Persian ordinal suffixes |
| 第一 (dì yī) | ۱اُم / اول | Can use numeral+suffix or written-out |
| 1点半 (1:30) | ۱۳:۳۰ | Persian uses 24-hour format |
| 上午9点 (shàngwǔ 9 diǎn) | ۰۹:۰۰ | 24-hour format exclusively |
| 下午3点 (xiàwǔ 3 diǎn) | ۱۵:۰۰ | Convert PM to 24-hour clock |
| 电话号码: 12345678 | تلفن: ۱۲۳۴۵۶۷۸ | Same format; area codes may need +98 prefix |
| 邮政编码 (yóuzhèng biānmǎ) | کد پستی | Chinese postal codes: 6 digits; Iranian: 10 digits |
| 平方米 (píngfāng mǐ) | متر مربع (m²) | Same unit symbol |

### Chinese Measure Words (量词) → Persian

Chinese requires specific measure words for counting objects. Persian does not use measure words in the same way.

| Chinese Measure Word | Used For | Persian Strategy | Example |
|---------------------|----------|-----------------|---------|
| 个 (gè) | General objects | No equivalent needed; omit | 一个人 → یک نفر |
| 只 (zhī) | Animals, one of a pair | No equivalent needed; omit | 一只猫 → یک گربه |
| 本 (běn) | Books, magazines | No equivalent needed; omit | 一本书 → یک کتاب |
| 张 (zhāng) | Flat objects (paper, tables) | No equivalent needed; omit | 一张纸 → یک کاغذ |
| 件 (jiàn) | Clothes, matters | No equivalent needed; omit | 一件衣服 → یک لباس |
| 台 (tái) | Machines, vehicles | No equivalent needed; omit | 一台电脑 → یک رایانه |
| 辆 (liàng) | Vehicles | No equivalent needed; omit | 一辆车 → یک خودرو |
| 杯 (bēi) | Cups of liquid | Use لیوان (līvān) as noun | 一杯茶 → یک لیوان چای |
| 瓶 (píng) | Bottles | Use بطری (botri) as noun | 一瓶水 → یک بطری آب |

**Rule**: Chinese measure words are **always removed** in Persian translation. Persian uses direct noun-quantity constructions without classifier words.

## 6. Punctuation

Chinese and Persian punctuation differ in several key areas. Chinese uses full-width punctuation; Persian uses standard-width punctuation with specific Arabic-derived marks.

| Chinese | Persian | Rule |
|---|---|---|
| 。 (period) | . (period) | Chinese full-width period → standard period |
| ， (comma) | ، (Arabic comma) | Chinese full-width comma → Arabic comma (U+060C) |
| 、 (enumeration comma) | ، (Arabic comma) | Chinese dunhao has no Persian equivalent; use Arabic comma |
| "" (double quotation marks) | « » (guillemets) | Chinese double quotes → Persian guillemets |
| '' (single quotation marks) | « » (nested) or ' ' | Same as above |
| 《》(book title marks) | « » or italics | No direct equivalent; use guillemets or italics |
| 【】(bracket) | [ ] | Chinese square brackets → standard brackets |
| —— (em dash, 2-wide) | — (em dash, standard) | Chinese double em dash → single em dash |
| …… (ellipsis, 6 dots) | ... (3 dots) | Chinese six-dot ellipsis → standard three-dot |
| · (middle dot) | - (hyphen) or omit | Used in Chinese for foreign names; adapt to Persian |
| No space after punctuation | No space after Persian punctuation | Both Chinese and Persian typically do not add space after punctuation |
| ： (full-width colon) | : (standard colon) | Full-width → standard-width |
| ； (full-width semicolon) | ؛ (Arabic semicolon) | Full-width → Arabic semicolon (U+061B) |
| ？ (full-width question mark) | ؟ (Arabic question mark) | Full-width → Arabic reversed question mark (U+061F) |
| ！ (full-width exclamation) | ! (standard exclamation) | Full-width → standard-width |
| （） (full-width parentheses) | ( ) (standard parentheses) | Full-width → standard-width |
| — (number dash in ranges) | تا / – (en dash) | Use "تا" or en dash for ranges |
| No hyphenation at line breaks | No hyphenation | Neither language typically hyphenates at line breaks |
| % (percentage) | ٪ (Arabic percent) | Use Arabic percent sign: ۵۰٪ |

## 7. Sentence Restructuring (SVO → SOV)

Chinese uses SVO word order, similar to English. Persian uses SOV word order. Both languages are topic-prominent, but Persian requires the verb to appear at the end of the clause.

### 7.1 Basic SVO → SOV Conversion

| Chinese (SVO) | Persian (SOV) | Notes |
|---|---|---|
| 他吃苹果。 (Tā chī píngguǒ.) | او سیب می‌خورد. | SVO → SOV |
| 公司签了合同。 (Gōngsī qiān le hétong.) | شرکت قرارداد را امضا کرد. | Verb moves to end |
| 明天发送报告。 (Míngtiān fāsòng bàogào.) | او فردا گزارش را ارسال خواهد کرد. | Time expression first in Persian |
| 我们可以解决这个问题。 (Wǒmen kěyǐ jiějué zhège wèntí.) | ما می‌توانیم این مشکل را حل کنیم. | Modal + verb at end |

### 7.2 Complex Sentence Restructuring

| Chinese Structure | Persian Equivalent | Notes |
|---|---|---|
| 虽然下雨，但我们出去了。 (Suīrán xià yǔ, dàn wǒmen chūqù le.) | اگرچه باران می‌بارید، ما بیرون رفتیم. | Concessive clause first, then main + verb |
| 昨天提交的报告已被批准。 (Zuótiān tíjiāo de bàogào yǐ bèi pīzhǔn.) | گزارشی که دیروز ارائه شد، تأیید شد. | Relative clause before main verb |
| 如果需要帮助，请联系我们。 (Rúguǒ xūyào bāngzhù, qǐng liánxì wǒmen.) | اگر به کمک نیاز دارید، لطفاً با ما تماس بگیرید. | Conditional first, then imperative |
| 文件必须在截止日期前签署。 (Wénjiàn bìxū zài jiézhǐ rìqī qián qiānshǔ.) | سند باید قبل از مهلت امضا شود. | Modal + infinitive at end |

### 7.3 Topic-Comment Structure

Both Chinese and Persian are topic-prominent languages. The topic-comment structure is natural in both:

| Chinese Topic-Comment | Persian Topic-Comment | Notes |
|---|---|---|
| 这本书，我已经看过了。 (Zhè běn shū, wǒ yǐjīng kàn guò le.) | این کتاب را، من قبلاً خوانده‌ام. | Topic first, then comment |
| 那个问题，我们已经解决了。 (Nàge wèntí, wǒmen yǐjīng jiějué le.) | آن مشکل را، ما قبلاً حل کرده‌ایم. | Same structure |

## 8. Ezafe Construction in Translation

The Ezafe (کسره اضافه) is a grammatical particle linking nouns to their modifiers or possessor. Chinese does not have an equivalent grammatical feature.

### 8.1 Ezafe Patterns

| Chinese Pattern | Persian Ezafe Pattern | Example |
|---|---|---|
| Adjective + 的 + Noun | Noun + Ezafe + adjective | 大房子 (dà fángzi) → خانه بزرگ (bozorg khāneh) |
| Noun + 的 + Noun | Noun + Ezafe + noun | 阿里的书 (Ālǐ de shū) → کتاب علی (Ali-ye ketāb) |
| Compound noun + 的 + Noun | Noun + Ezafe + compound noun | 公司的政策 (gōngsī de zhèngcè) → سیاست شرکت (sīāsat-e sherkat) |

### 8.2 的 (de) to Ezafe Conversion

| Chinese 的 Usage | Persian Ezafe | Example |
|---|---|---|
| Possessive 的 | Ezafe | 我的书 (wǒ de shū) → کتاب من (man ketāb) |
| Adjective modifier 的 | Ezafe | 漂亮的房子 (piàoliang de fángzi) → خانه زیبا (zibā-ye khāneh) |
| Relative clause 的 | که (ke) relative pronoun | 我买的房子 (wǒ mǎi de fángzi) → خانه‌ای که خریدم (khāneh-i ke kharīdam) |

## 9. Measure Word Removal

Chinese requires specific measure words (量词 liàngcí) for counting objects. Persian does not use measure words. All Chinese measure words must be removed during translation.

### 9.1 Common Chinese Measure Words

| Chinese Measure Word | Chinese Example | Persian Translation | Notes |
|---|---|---|---|
| 个 (gè) - general | 一个人 (yí ge rén) | یک نفر (yek nafar) | Remove 个, use یک |
| 只 (zhī) - animals | 一只猫 (yì zhī māo) | یک گربه (yek gorbeh) | Remove 只 |
| 本 (běn) - books | 一本书 (yì běn shū) | یک کتاب (yek ketāb) | Remove 本 |
| 张 (zhāng) - flat objects | 一张纸 (yì zhāng zhǐ) | یک کاغذ (yek kāghaz) | Remove 张 |
| 件 (jiàn) - clothes/matters | 一件衣服 (yí jiàn yīfu) | یک لباس (yek lebās) | Remove 件 |
| 台 (tái) - machines | 一台电脑 (yì tái diànnǎo) | یک رایانه (yek rāyāneh) | Remove 台 |
| 辆 (liàng) - vehicles | 一辆车 (yí liàng chē) | یک خودرو (yek khodro) | Remove 辆 |
| 杯 (bēi) - cups | 一杯茶 (yì bēi chá) | یک لیوان چای (yek līvān chāy) | Remove 杯, use لیوان as noun |
| 瓶 (píng) - bottles | 一瓶水 (yì píng shuǐ) | یک بطری آب (yek botri āb) | Remove 瓶, use بطری as noun |
| 条 (tiáo) - long objects | 一条鱼 (yì tiáo yú) | یک ماهی (yek māhi) | Remove 条 |
| 把 (bǎ) - graspable objects | 一把椅子 (yì bǎ yǐzi) | یک صندلی (yek sandali) | Remove 把 |
| 位 (wèi) - polite people counter | 一位老师 (yí wèi lǎoshī) | یک استاد (yek ostād) | Remove 位 |

### 9.2 Quantity Expression Conversion

| Chinese Pattern | Persian Pattern | Notes |
|---|---|---|
| 数量 + 量词 + 名词 | 数量 + 名词 | Remove measure word |
| 三个苹果 (sān ge píngguǒ) | سه سیب (se sib) | No measure word needed |
| 五本书 (wǔ běn shū) | پنج کتاب (panj ketāb) | Direct number + noun |
| 两台电脑 (liǎng tái diànnǎo) | دو رایانه (do rāyāneh) | Direct number + noun |
| 十个人 (shí ge rén) | ده نفر (dah nafar) | Use نفر for people |

## 12. Punctuation Details

### 12.1 Full-Width to Half-Width Conversion

| Chinese (Full-Width) | Persian (Half-Width/Arabic) | Unicode |
|---|---|---|
| ， (U+FF0C) | ، (U+060C) | Arabic comma |
| ； (U+FF1B) | ؛ (U+061B) | Arabic semicolon |
| ？ (U+FF1F) | ؟ (U+061F) | Arabic question mark |
| ！ (U+FF01) | ! (U+0021) | Standard exclamation |
| ： (U+FF1A) | : (U+003A) | Standard colon |
| （ (U+FF08) | ( (U+0028) | Standard parenthesis |
| ） (U+FF09) | ) (U+0029) | Standard parenthesis |
| 【 (U+3010) | [ (U+005B) | Standard bracket |
| 】 (U+3011) | ] (U+005D) | Standard bracket |
| 《 (U+300A) | « (U+00AB) | Opening guillemet |
| 》 (U+300B) | » (U+00BB) | Closing guillemet |
| …… (U+2026×2) | ... (U+002E×3) | Standard ellipsis |
| —— (U+2014×2) | — (U+2014) | Single em dash |

### 12.2 Chinese-Specific Punctuation Patterns

| Chinese Pattern | Persian Pattern | Notes |
|---|---|---|
| 顿号列举 (、) | Arabic comma (،) | Use comma for lists |
| 书名号 (《》) | Guillemets («») or italics | Use guillemets for titles |
| 破折号 (——) | Em dash (—) | Single em dash |
| 省略号 (……) | Ellipsis (...) | Three dots |
| 间隔号 (·) | Hyphen (-) or space | For foreign name transliteration |

### 12.3 Spacing Rules Comparison

| Feature | Chinese | Persian | Notes |
|---|---|---|---|
| Space after punctuation | No | No | Both typically omit space after punctuation |
| Space between words | No (no word boundaries) | Yes (word spaces) | Persian uses spaces between words |
| Space before/after parentheses | No | Yes | Persian adds space before ( and after ) |
| Space around em dash | No | No | Neither adds space around em dash |
| Number spacing | No | No | Numbers connect directly to adjacent text |
